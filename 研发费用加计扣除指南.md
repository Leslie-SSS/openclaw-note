## 一、政策概述

### 1.1 政策背景

研发费用加计扣除是我国激励企业加大研发投入、推动科技创新的重要税收优惠政策。该政策允许企业在计算应纳税所得额时，对实际发生的研发费用在据实扣除的基础上，再加计扣除一定比例，从而降低企业税负，增强企业创新动力。

自2008年企业所得税法实施以来，研发费用加计扣除政策经历了多次调整完善：

| 年份 | 政策演进 | 主要变化 |
|------|----------|----------|
| 2008 | 《企业所得税法》实施 | 首次确立研发费用加计扣除制度 |
| 2015 | 财税〔2015〕119号 | 扩大研发活动范围，放宽研发费用归集范围 |
| 2017 | 财税〔2017〕34号 | 科技型中小企业扣除比例提至75% |
| 2018 | 财税〔2018〕99号 | 所有企业扣除比例统一提至75% |
| 2021 | 财政部公告2021年第13号 | 制造业企业扣除比例提至100% |
| 2022 | 财政部公告2022年第16号 | 科技型中小企业恢复100%扣除 |
| 2023 | 财政部公告2023年第7号 | 企业研发费用加计扣除比例统一提高至100% |

### 1.2 政策意义

**（一）降低企业创新成本**

研发费用加计扣除政策直接减少企业应纳税所得额，相当于政府为企业承担部分研发成本，有效降低企业创新活动的边际成本。

**（二）引导资源向研发领域配置**

通过税收杠杆，引导企业将更多资源投入到研发活动中，促进企业由要素驱动向创新驱动转变。

**（三）提升企业核心竞争力**

持续的研发投入有助于企业掌握核心技术，培育自主知识产权，提升市场竞争力和可持续发展能力。

**（四）服务国家创新驱动发展战略**

政策实施有助于提高全社会研发投入强度，推动产业转型升级，服务国家创新驱动发展战略。

### 1.3 适用主体

研发费用加计扣除政策的适用主体为：

**（一）会计核算健全**

实行查账征收的居民企业，且会计核算健全、能够准确归集研发费用。

**（二）研发项目符合规定**

企业从事的研发活动必须符合政策规定的研发活动定义和认定标准。

**（三）费用归集规范**

企业能够按照规定准确归集研发费用，并建立健全研发费用辅助账。

**不适用情形：**

- 核定征收企业
- 企业为会计核算不健全的
- 企业无法准确归集研发费用的

## 二、研发费用加计扣除政策

### 2.1 基本规定

#### 2.1.1 政策依据

研发费用加计扣除政策的主要依据包括：

1. **《中华人民共和国企业所得税法》**第三十条
2. **《中华人民共和国企业所得税法实施条例》**第九十五条
3. **《财政部 国家税务总局 科技部关于完善研究开发费用税前加计扣除政策的通知》**（财税〔2015〕119号）
4. **《国家税务总局关于企业研究开发费用税前加计扣除政策有关问题的公告》**（国家税务总局公告2015年第97号）
5. **《财政部 税务总局关于企业委托境外研究开发费用税前加计扣除有关政策问题的通知》**（财税〔2018〕64号）
6. **《财政部 税务总局关于进一步完善研发费用税前加计扣除政策的公告》**（财政部 税务总局公告2023年第7号）
7. **《国家税务总局关于研发费用税前加计扣除归集范围有关问题的公告》**（国家税务总局公告2017年第40号）

#### 2.1.2 扣除方式

研发费用加计扣除采用"据实扣除+加计扣除"的方式：

**据实扣除**：企业实际发生的研发费用，按照实际发生额在税前扣除。

**加计扣除**：在据实扣除的基础上，再按照研发费用实际发生额的一定比例加计扣除。

**计算公式：**

```
加计扣除金额 = 研发费用实际发生额 × 加计扣除比例
税前扣除总额 = 研发费用实际发生额 + 加计扣除金额
```

**举例说明：**

某企业2024年度研发费用实际发生额为1000万元，加计扣除比例为100%。

```
据实扣除金额 = 1000万元
加计扣除金额 = 1000万元 × 100% = 1000万元
税前扣除总额 = 1000 + 1000 = 2000万元
```

若企业所得税税率为25%，则可减少企业所得税：1000万元 × 25% = 250万元

### 2.2 扣除比例

#### 2.2.1 现行扣除比例（2023年起）

根据财政部 税务总局公告2023年第7号规定，自2023年1月1日起，企业开展研发活动中实际发生的研发费用，未形成无形资产计入当期损益的，在按规定据实扣除的基础上，自2023年1月1日起，再按照实际发生额的**100%**在税前加计扣除；形成无形资产的，自2023年1月1日起，按照无形资产成本的**200%**在税前摊销。

| 情形 | 加计扣除比例 | 政策依据 |
|------|-------------|----------|
| 研发费用计入当期损益 | 100% | 财政部 税务总局公告2023年第7号 |
| 研发费用形成无形资产 | 200%（成本摊销） | 财政部 税务总局公告2023年第7号 |

#### 2.2.2 历史扣除比例对比

| 期间 | 一般企业 | 制造业企业 | 科技型中小企业 |
|------|----------|------------|----------------|
| 2016-2017年 | 50% | 50% | 50% |
| 2018-2020年 | 75% | 75% | 75% |
| 2021年 | 75% | 100% | 75% |
| 2022年 | 75% | 100% | 100% |
| 2023年起 | 100% | 100% | 100% |

#### 2.2.3 政策执行期限

根据财政部 税务总局公告2023年第7号规定，该政策执行期限为2023年1月1日至2027年12月31日。

### 2.3 费用范围

研发费用加计扣除政策适用的费用范围包括：

**（一）自主研发费用**

企业自行开展研发活动发生的费用。

**（二）合作研发费用**

企业与其他企业、科研机构、高等院校合作开展研发活动发生的费用。

**（三）集中研发费用**

企业集团根据生产经营需要，对技术要求高、投资数额大，需要集中研发的项目进行集中开发所发生的费用。

**（四）委托研发费用**

企业委托外部机构或个人开展研发活动支付的费用。

### 2.4 排除项目

下列活动不适用加计扣除政策：

**（一）企业产品（服务）的常规性升级**

仅对现有产品或服务进行小规模、非实质性的改进，不涉及核心技术或功能的变化。

**（二）对某项科研成果的直接应用**

直接使用现有科研成果，未进行后续研究开发活动。

**（三）企业在商品化后为顾客提供的技术支持活动**

产品或服务投入商业化运营后提供的常规技术支持。

**（四）对现存产品、服务、技术、材料或工艺流程进行的重复或简单改变**

虽有一定变化，但这种变化是重复性的或简单的，不具备创新性。

**（五）市场调研、效率调研或管理研究**

属于市场分析、运营效率研究或管理领域的研究活动。

**（六）作为工业（服务）流程环节或常规的质量控制、测试分析、维修维护**

属于日常生产经营活动中的常规性工作。

**（七）社会科学、艺术或人文学方面的研究**

不属于自然科学或工程技术领域的研究活动。

### 2.5 特殊行业规定

#### 2.5.1 负面清单行业

以下行业企业不适用研发费用加计扣除政策：

1. 烟草制造业
2. 住宿和餐饮业
3. 批发和零售业
4. 房地产业
5. 租赁和商务服务业
6. 娱乐业
7. 财政部和国家税务总局规定的其他行业

**判断标准：**

上述行业以《国民经济行业分类与代码（GB/T 4754-2017）》为准，并随《国民经济行业分类》调整而调整。

企业所属行业按照以下原则确定：

- 以企业实际从事的主营业务为基础
- 主营业务收入占收入总额50%以上的业务对应的行业
- 如无单一业务收入占比超过50%，则以收入占比最大的业务确定

#### 2.5.2 经营多业企业

企业同时从事多个行业业务，只要某一行业业务属于负面清单范围，则该企业的全部研发费用均不得加计扣除。

## 三、研发活动认定标准

### 3.1 研发活动定义

根据财税〔2015〕119号文件规定，研发活动是指企业为获得科学与技术新知识，创造性运用科学技术新知识，或实质性改进技术、产品（服务）、工艺而持续进行的具有明确目标的系统性活动。

**定义解析：**

**（一）获得科学与技术新知识**

企业开展的基础研究活动，目的是获取新的科学知识或技术知识。

**（二）创造性运用科学技术新知识**

将已有的科学知识或技术知识创造性地应用于新的领域或场景。

**（三）实质性改进**

对现有技术、产品、服务或工艺进行具有实质意义的改进，而非简单的重复或改变。

**（四）明确目标**

研发活动应有明确的目标和预期成果，而非漫无目的的探索。

**（五）系统性活动**

研发活动应按照一定的方法、程序和步骤进行，具有系统性。

### 3.2 七大认定条件

研发活动的认定应满足以下七大条件：

#### 条件一：创新性

**具体要求：**

研发活动应当具有创新性，即在技术、产品、服务或工艺方面具有实质性的改进或突破。

**判断要点：**

1. 活动是否涉及新的技术领域或应用场景
2. 是否存在技术挑战或技术难点
3. 是否需要突破现有技术瓶颈
4. 是否产生了新的技术成果或知识产权

**案例分析：**

| 情形 | 是否满足创新性 | 说明 |
|------|---------------|------|
| 开发全新技术 | 满足 | 具有原始创新性 |
| 将现有技术应用于新领域 | 满足 | 具有应用创新性 |
| 产品外观颜色变更 | 不满足 | 不涉及技术创新 |
| 软件版本常规更新 | 视情况 | 需判断是否有实质改进 |

#### 条件二：目标性

**具体要求：**

研发活动应当具有明确的目标，包括技术目标和预期成果。

**判断要点：**

1. 是否有明确的研发计划或方案
2. 是否设定了具体的技术指标或参数
3. 是否明确了预期成果形式（专利、技术秘密、新产品等）
4. 目标是否具有可实现性和可验证性

**目标类型：**

- **技术目标**：开发新技术、新工艺、新材料等
- **产品目标**：开发新产品、改进现有产品功能等
- **工艺目标**：改进生产流程、提高生产效率等

#### 条件三：系统性

**具体要求：**

研发活动应当是系统性的活动，而非零散的、偶然的活动。

**判断要点：**

1. 是否按照研发计划或流程进行
2. 是否包含研发活动的完整环节（立项、实施、验收等）
3. 是否有相应的管理制度和流程
4. 活动是否具有连续性和持续性

**研发活动环节：**

```
立项 → 方案设计 → 技术研究 → 试验验证 → 成果总结 → 验收
```

#### 条件四：风险性

**具体要求：**

研发活动应当具有一定的技术风险和不确定性。

**判断要点：**

1. 活动结果是否具有不确定性
2. 是否存在技术失败的可能性
3. 是否需要反复试验和验证
4. 是否涉及技术攻关或难题突破

**注意：**

研发活动的风险性并不意味着必须有失败的可能性，而是指结果存在不确定性，需要通过研发活动来获得确定的结果。

#### 条件五：专业性

**具体要求：**

研发活动应当由具有相应专业知识和技能的人员实施。

**判断要点：**

1. 研发人员是否具有相关专业的学历或技术背景
2. 是否有专业的研发团队或组织
3. 研发人员是否具备开展研发活动的能力
4. 是否有技术负责人或专家指导

#### 条件六：记录性

**具体要求：**

研发活动应当有完整的记录和档案。

**判断要点：**

1. 是否有研发项目立项文件
2. 是否有研发过程记录（试验记录、测试报告等）
3. 是否有研发成果报告或验收报告
4. 是否有相应的财务记录和费用归集

#### 条件七：成果性

**具体要求：**

研发活动应当产生相应的成果，包括技术成果、产品成果或知识产权。

**成果形式：**

| 成果类型 | 具体形式 |
|----------|----------|
| 技术成果 | 新技术、新工艺、新方法、技术秘密 |
| 产品成果 | 新产品、改进产品、原型样机 |
| 知识产权 | 专利、软件著作权、集成电路布图设计 |
| 其他成果 | 技术标准、论文、技术报告 |

**注意：**

研发活动不一定必须产生成功的成果，失败的研发项目只要是真实的研发活动，其发生的费用也可以加计扣除。

### 3.3 判断流程

研发活动的判断可按照以下流程进行：

```
第一步：明确活动目的
    ↓
第二步：判断是否属于排除项目
    ↓
第三步：评估创新性
    ↓
第四步：确认目标和系统性
    ↓
第五步：核实风险性和专业性
    ↓
第六步：检查记录和成果
    ↓
第七步：综合判断
```

### 3.4 研发活动与常规生产活动的区分

| 区分维度 | 研发活动 | 常规生产活动 |
|----------|----------|--------------|
| 目的 | 获取新知识、新技术 | 生产既定产品或服务 |
| 不确定性 | 结果具有不确定性 | 结果可预期 |
| 创新性 | 具有技术创新 | 重复已有流程 |
| 周期 | 相对较长 | 相对固定 |
| 人员 | 专业研发人员 | 生产人员 |
| 管理 | 研发项目管理 | 生产运营管理 |

### 3.5 争议处理

**（一）企业自我判断**

企业应根据政策规定，结合实际情况自行判断其活动是否符合研发活动定义。

**（二）专业机构鉴定**

如企业对研发活动认定存在疑问，可聘请科技部门或专业机构进行鉴定。

**（三）税务机关核查**

税务机关在进行后续管理时，对研发活动有异议的，可以转请科技部门出具鉴定意见。

## 四、研发费用归集范围

### 4.1 人员人工费用

#### 4.1.1 归集范围

人员人工费用是指直接从事研发活动人员的工资薪金、基本养老保险费、基本医疗保险费、失业保险费、工伤保险费、生育保险费和住房公积金，以及外聘研发人员的劳务费用。

**具体包括：**

1. **工资薪金**
   - 基本工资
   - 奖金
   - 津贴
   - 补贴
   - 年终加薪
   - 加班工资
   - 与任职或受雇有关的其他支出

2. **五险一金**
   - 基本养老保险费
   - 基本医疗保险费
   - 失业保险费
   - 工伤保险费
   - 生育保险费
   - 住房公积金

3. **外聘研发人员劳务费**
   - 劳务派遣人员费用
   - 聘请专家咨询费（与研发相关）

#### 4.1.2 人员范围

**直接从事研发活动人员包括：**

1. 研究人员
2. 技术人员
3. 辅助人员

**人员分类说明：**

| 人员类别 | 工作内容 | 归集要求 |
|----------|----------|----------|
| 研究人员 | 从事研发活动的专业人员 | 直接归集 |
| 技术人员 | 参与技术研发的技术人员 | 直接归集 |
| 辅助人员 | 为研发活动提供直接辅助的人员 | 直接归集 |

**注意：**

仅为研发活动提供间接服务的人员（如财务、行政、后勤等），其费用不计入研发费用加计扣除范围。

#### 4.1.3 归集要求

**（一）人员要求**

1. 研发人员必须与企业存在劳动关系或劳务关系
2. 研发人员必须实际参与研发活动
3. 应保留人员参与研发活动的记录

**（二）费用归集原则**

1. 研发人员专门从事研发活动的，其费用全额归集
2. 研发人员同时从事研发和非研发活动的，其费用应按实际工时占比分摊
3. 共用人员的分摊比例应有合理依据

**（三）分摊计算**

```
研发人员费用分摊 = 人员总费用 × （研发工时 ÷ 总工时）
```

#### 4.1.4 留存备查资料

1. 研发人员名单（注明岗位、职责）
2. 工资发放记录
3. 五险一金缴纳记录
4. 研发工时记录或考勤记录
5. 人员分摊比例计算表（如有分摊）

### 4.2 直接投入费用

#### 4.2.1 归集范围

直接投入费用是指研发活动过程中消耗的材料、燃料、动力等费用。

**具体包括：**

1. **材料费用**
   - 原材料
   - 辅助材料
   - 试验用品
   - 其他材料

2. **燃料费用**
   - 燃料油
   - 燃气
   - 其他燃料

3. **动力费用**
   - 电费
   - 水费
   - 蒸汽费
   - 其他动力费

4. **模具、样品、样机及一般测试手段购置费**
   - 专门用于研发活动的模具
   - 研发过程中使用的样品、样机
   - 一般测试手段的购置费

5. **试验、检验、测试等费用**
   - 试验费
   - 检验费
   - 测试费
   - 计量费

#### 4.2.2 归集要求

**（一）材料费用**

1. 材料必须实际用于研发活动
2. 应保留材料领用记录
3. 研发材料与非研发材料应分别核算

**（二）动力费用**

1. 动力费用应与研发活动直接相关
2. 无法直接归集的，应按合理方法分摊
3. 分摊方法应保持一致性

**（三）模具、样品、样机**

1. 专门用于研发活动的模具、样品、样机可全额归集
2. 同时用于研发和生产活动的，应合理分摊
3. 研发活动结束后处置模具、样品、样机取得的收入应冲减研发费用

#### 4.2.3 分摊方法

动力费用的分摊可采用以下方法：

| 分摊方法 | 适用情形 | 计算公式 |
|----------|----------|----------|
| 直接计量 | 有独立计量设备 | 实际计量数 × 单价 |
| 设备功率 | 按设备功率分摊 | 总费用 × （研发设备功率 ÷ 总功率） |
| 面积比例 | 按研发区域面积分摊 | 总费用 × （研发区面积 ÷ 总面积） |
| 工时比例 | 按研发工时分摊 | 总费用 × （研发工时 ÷ 总工时） |

#### 4.2.4 留存备查资料

1. 材料领用单或出库单
2. 材料费用明细账
3. 动力费用分摊计算表
4. 模具、样品、样机购置发票
5. 试验、检验、测试报告及费用凭证

### 4.3 折旧费用

#### 4.3.1 归集范围

折旧费用是指用于研发活动的仪器、设备的折旧费用。

**具体包括：**

1. 专门用于研发活动的仪器、设备折旧
2. 同时用于研发和生产活动的仪器、设备折旧（按比例分摊）
3. 经营租赁方式租入的用于研发活动的仪器、设备租赁费

**注意：**

房屋、建筑物等不动产的折旧费用不属于研发费用加计扣除范围。

#### 4.3.2 归集要求

**（一）仪器设备认定**

1. 仪器设备必须用于研发活动
2. 应明确仪器设备的用途和使用情况
3. 研发设备应单独登记或标记

**（二）折旧计算**

1. 折旧方法应符合会计准则规定
2. 折旧年限和残值率应合理
3. 折旧费用应与研发活动时间相匹配

**（三）分摊原则**

1. 专门用于研发活动的设备折旧全额归集
2. 共用设备折旧按使用时间或工作量分摊
3. 分摊比例应有合理依据

#### 4.3.3 分摊计算

```
研发设备折旧费用 = 设备年折旧额 × （研发使用时间 ÷ 设备总使用时间）
```

或

```
研发设备折旧费用 = 设备年折旧额 × （研发使用工时 ÷ 设备总工时）
```

#### 4.3.4 留存备查资料

1. 仪器设备清单（注明用途）
2. 固定资产折旧计算表
3. 设备使用记录或工时记录
4. 共用设备分摊比例计算表
5. 租赁设备合同及租赁费发票

### 4.4 无形资产摊销费用

#### 4.4.1 归集范围

无形资产摊销费用是指用于研发活动的软件、专利权、非专利技术（包括许可证、专有技术、设计和计算方法等）的摊销费用。

**具体包括：**

1. **软件摊销费用**
   - 专门用于研发的软件
   - 研发活动使用的专业软件

2. **专利权摊销费用**
   - 专门用于研发的专利权
   - 引进的专利技术

3. **非专利技术摊销费用**
   - 专有技术
   - 技术诀窍
   - 设计方案
   - 计算方法

#### 4.4.2 归集要求

1. 无形资产必须用于研发活动
2. 摊销方法应符合会计准则规定
3. 摊销期限应与法定或约定使用期限一致
4. 共用无形资产应合理分摊

#### 4.4.3 留存备查资料

1. 无形资产清单
2. 无形资产购置合同或协议
3. 摊销计算表
4. 无形资产使用记录

### 4.5 新产品设计费等

#### 4.5.1 归集范围

新产品设计费、新工艺规程制定费、新药研制的临床试验费、勘探开发技术的现场试验费等。

**具体包括：**

1. **新产品设计费**
   - 新产品的设计、绘图费用
   - 产品模型制作费用

2. **新工艺规程制定费**
   - 工艺流程设计费用
   - 工艺规程编制费用

3. **新药研制的临床试验费**
   - 临床前研究费用
   - 临床试验费用

4. **勘探开发技术的现场试验费**
   - 现场试验设计费用
   - 现场试验实施费用

#### 4.5.2 归集要求

1. 设计、制定、试验等活动必须与研发项目直接相关
2. 应保留相关合同、协议、报告等资料
3. 费用应与研发项目对应

#### 4.5.3 留存备查资料

1. 设计合同或协议
2. 设计图纸、工艺规程文件
3. 临床试验报告
4. 现场试验报告
5. 费用支付凭证

### 4.6 其他相关费用

#### 4.6.1 归集范围

其他相关费用是指与研发活动直接相关的其他费用，包括技术图书资料费、资料翻译费、专家咨询费、高新科技研发保险费，研发成果的检索、分析、评议、论证、鉴定、评审、评估、验收费用，知识产权的申请费、注册费、代理费，差旅费、会议费等。

**具体包括：**

| 费用类别 | 具体项目 |
|----------|----------|
| 图书资料费 | 技术图书、资料、期刊费用 |
| 翻译费 | 资料翻译费用 |
| 咨询费 | 专家咨询费、技术顾问费 |
| 保险费 | 高新科技研发保险费 |
| 评审费 | 检索、分析、评议、论证、鉴定、评审、评估、验收费用 |
| 知识产权费 | 专利申请费、注册费、代理费、年费 |
| 差旅费 | 研发相关差旅费用 |
| 会议费 | 研发相关会议费用 |

#### 4.6.2 限额规定

其他相关费用总额不得超过可加计扣除研发费用总额的**10%**。

**计算公式：**

```
其他相关费用限额 = （人员人工费用 + 直接投入费用 + 折旧费用 
                    + 无形资产摊销费用 + 新产品设计费等）× 10% ÷ (1 - 10%)
```

**简化公式：**

```
其他相关费用限额 = （除其他费用外的研发费用）÷ 9
```

#### 4.6.3 计算示例

某企业研发费用明细如下：

- 人员人工费用：500万元
- 直接投入费用：300万元
- 折旧费用：100万元
- 无形资产摊销费用：50万元
- 新产品设计费：50万元
- 其他相关费用：80万元

**计算步骤：**

1. 除其他费用外的研发费用 = 500 + 300 + 100 + 50 + 50 = 1000万元

2. 其他相关费用限额 = 1000 ÷ 9 = 111.11万元

3. 实际其他相关费用 = 80万元

4. 由于80万元 < 111.11万元，可全额加计扣除

5. 可加计扣除研发费用总额 = 1000 + 80 = 1080万元

### 4.7 委托外部研发费用

#### 4.7.1 归集范围

委托外部研发费用是指企业委托其他机构或个人开展研发活动所支付的费用。

**委托对象包括：**

1. 境内研发机构
2. 境内高校
3. 境外研发机构
4. 境外高校
5. 个人

#### 4.7.2 加计扣除规定

**（一）委托境内研发**

按照实际发生额的80%计入委托方研发费用并计算加计扣除。

**（二）委托境外研发**

按照实际发生额的80%计入委托方研发费用，且委托境外研发费用不超过境内符合条件的研发费用三分之二的部分，可以加计扣除。

**计算公式：**

```
委托境外研发费用加计扣除限额 = 委托境外研发费用 × 80%
且 ≤ 境内研发费用 × 2/3
```

#### 4.7.3 计算示例

**案例：**

某企业2024年度研发费用如下：
- 自主研发费用：800万元
- 委托境内研发费用：100万元
- 委托境外研发费用：200万元

**计算：**

1. 委托境内研发费用计入额 = 100 × 80% = 80万元

2. 委托境外研发费用计入额 = 200 × 80% = 160万元

3. 委托境外研发费用限额 = 800 × 2/3 = 533.33万元

4. 由于160万元 < 533.33万元，可全额计入

5. 可加计扣除研发费用总额 = 800 + 80 + 160 = 1040万元

### 4.8 合作/集中研发费用

#### 4.8.1 合作研发

**归集原则：**

合作研发由合作各方就各自承担的研发费用分别计算加计扣除。

**要求：**

1. 合作各方应签订合作协议
2. 明确各方承担的研发任务和费用
3. 各方按照实际承担的研发费用计算加计扣除

#### 4.8.2 集中研发

**归集原则：**

企业集团根据生产经营需要，对技术要求高、投资数额大、需要集中研发的项目进行集中开发的，应按照权利和义务相一致、费用支出与收益分享相配比的原则，合理确定研发费用的分摊方法。

**分摊方法：**

1. 按各成员企业的销售收入比例分摊
2. 按各成员企业的资产总额比例分摊
3. 按各成员企业的利润比例分摊
4. 其他合理方法

**要求：**

1. 应保留集中研发项目的立项文件
2. 应保留费用分摊协议
3. 集中研发费用应以集团公司名义归集
4. 分摊方法应保持一致性

## 五、申报流程与材料

### 5.1 申报方式

#### 5.1.1 申报时点

研发费用加计扣除实行"自行判别、申报享受、相关资料留存备查"的办理方式。企业可在以下时点享受优惠：

1. **预缴申报**：可在7月、10月企业所得税预缴申报时享受上半年度或前三季度的研发费用加计扣除优惠
2. **年度汇算清缴**：在年度企业所得税汇算清缴时享受全年度的研发费用加计扣除优惠

#### 5.1.2 申报方式

企业可以通过以下方式申报：

1. 电子税务局网上申报
2. 办税服务厅上门申报
3. 委托税务代理机构申报

### 5.2 申报表填写

#### 5.2.1 年度申报表

年度申报时需填写以下报表：

1. **《中华人民共和国企业所得税年度纳税申报表（A类）》（A100000）**
   - 第17行"免税、减计收入及加计扣除"相关栏次

2. **《免税、减计收入及加计扣除优惠明细表》（A107010）**
   - 第26-28行：研发费用加计扣除相关栏次

3. **《研发费用加计扣除优惠明细表》（A107012）**
   - 详细填报各项研发费用

#### 5.2.2 预缴申报表

预缴申报时需填写：

1. **《中华人民共和国企业所得税月（季）度预缴纳税申报表（A类）》（A200000）**
   - 第7行"免税收入、减计收入、加计扣除"相关栏次

2. **《研发费用加计扣除优惠明细表》（A107012）**
   - 按照实际发生额填报

#### 5.2.3 填写要点

1. **研发项目信息**
   - 准确填写研发项目名称
   - 正确选择研发形式（自主研发、合作研发、集中研发、委托研发）
   - 准确填写研发费用金额

2. **费用明细**
   - 按照费用类别分别填报
   - 其他相关费用不得超过限额
   - 委托研发按照80%计入

3. **加计扣除金额**
   - 系统自动计算加计扣除金额
   - 核对计算结果是否正确

### 5.3 留存备查资料

## 六、常见问题与风险

### 6.1 常见问题

#### 问题1：研发费用加计扣除与高新技术企业认定有什么关系？

**解答：**

研发费用加计扣除与高新技术企业认定是两项相互独立但又相互关联的政策：

| 对比项 | 研发费用加计扣除 | 高新技术企业认定 |
|--------|------------------|------------------|
| 政策性质 | 税收优惠 | 资质认定+税收优惠 |
| 优惠方式 | 费用加计扣除 | 15%优惠税率 |
| 适用范围 | 所有符合条件的企业 | 认定通过的企业 |
| 研发费用口径 | 加计扣除口径 | 高企认定口径 |
| 管理部门 | 税务机关 | 科技/财政/税务 |

**关联性：**

1. 高新技术企业认定需要符合研发费用占比要求
2. 研发费用加计扣除的辅助账可作为高企认定的证明材料
3. 两者研发费用的归集口径有所不同，需要分别建立台账

#### 问题2：失败的研发项目能否享受加计扣除？

**解答：**

可以。根据政策规定，研发活动具有不确定性和风险性，失败的研发项目只要符合研发活动定义，且费用真实发生、归集准确，其研发费用也可以享受加计扣除。

**注意：**

企业需要保留完整的研发过程记录，证明该活动确实是研发活动，而非虚假项目。

#### 问题3：研发人员的范围如何界定？

**解答：**

研发人员包括研究人员、技术人员和辅助人员三类：

- **研究人员**：主要从事研发活动的专业人员
- **技术人员**：参与研发活动的技术人员
- **辅助人员**：为研发活动提供直接辅助的人员

**不包括：**

仅为研发活动提供间接服务的人员，如财务、行政、后勤、销售等部门人员。

#### 问题4：其他相关费用超过限额如何处理？

**解答：**

其他相关费用超过限额的，以限额为准进行加计扣除，超过部分不予加计扣除。

**计算公式：**

```
可加计扣除的其他相关费用 = MIN(实际发生额, 限额)
限额 = （除其他费用外的研发费用）÷ 9
```

#### 问题5：委托研发如何计算加计扣除？

**解答：**

委托研发费用按照实际发生额的80%计入委托方研发费用：

**委托境内研发：**

```
计入金额 = 委托费用 × 80%
```

**委托境外研发：**

```
计入金额 = 委托费用 × 80%
且不超过境内研发费用的2/3
```

#### 问题6：研发形成的下脚料、残次品如何处理？

**解答：**

研发活动过程中产生的下脚料、残次品等，其销售收入应冲减研发费用。

**处理方式：**

1. 研发过程中产生的下脚料、残次品销售收入
2. 研发活动结束后处置研发资产取得的收入
3. 以上收入均应冲减研发费用

#### 问题7：预缴申报享受加计扣除后，年度汇算清缴如何处理？

**解答：**

预缴申报享受的加计扣除是预享受，年度汇算清缴时应按照全年实际发生的研发费用进行清算：

1. 年度汇算清缴时，重新计算全年可加计扣除金额
2. 与预缴时已享受的加计扣除金额进行比较
3. 多退少补

#### 问题8：研发费用如何分摊？

**解答：**

需要分摊的情形包括：

**（一）人员费用分摊**

同时从事研发和非研发活动的人员，按工时分摊：

```
研发人员费用 = 人员总费用 × （研发工时 ÷ 总工时）
```

**（二）共用设备折旧分摊**

同时用于研发和生产活动的设备，按使用时间分摊：

```
研发设备折旧 = 设备折旧总额 × （研发使用时间 ÷ 总使用时间）
```

**（三）动力费用分摊**

按合理方法分摊，如面积比例、设备功率比例、工时比例等。

### 6.2 风险提示

#### 风险一：研发活动认定风险

**风险描述：**

企业将非研发活动认定为研发活动，导致研发费用归集不准确。

**常见情形：**

1. 将常规生产活动作为研发活动
2. 将产品简单改进作为研发活动
3. 将技术支持服务作为研发活动

**防范措施：**

1. 严格按照七大认定条件判断研发活动
2. 保留研发项目立项、实施、验收全过程资料
3. 必要时请科技部门或专业机构鉴定

#### 风险二：费用归集风险

**风险描述：**

研发费用归集不规范，导致加计扣除金额不准确。

**常见情形：**

1. 研发费用与生产费用混淆
2. 共用费用分摊不合理
3. 其他相关费用超限额
4. 委托研发计算错误

**防范措施：**

1. 建立健全研发费用辅助账
2. 制定合理的费用分摊方法并保持一致性
3. 准确计算其他相关费用限额
4. 正确计算委托研发费用

#### 风险三：资料留存风险

**风险描述：**

备查资料不完整，无法证明研发活动真实性和费用归集准确性。

**常见情形：**

1. 研发项目立项文件缺失
2. 研发过程记录不完整
3. 研发成果报告缺失
4. 费用归集依据不充分

**防范措施：**

1. 建立研发活动档案管理制度
2. 完整保存研发活动各环节资料
3. 按照规定保存研发费用辅助账
4. 保留费用归集的计算依据

#### 风险四：行业限制风险

**风险描述：**

属于负面清单行业的企业享受了加计扣除优惠。

**负面清单行业：**

1. 烟草制造业
2. 住宿和餐饮业
3. 批发和零售业
4. 房地产业
5. 租赁和商务服务业
6. 娱乐业

**防范措施：**

1. 准确判断企业所属行业
2. 多业经营企业需特别注意
3. 如有疑问，咨询税务机关

#### 风险五：负面清单项目风险

**风险描述：**

企业将不适用加计扣除的活动纳入研发费用。

**负面清单项目：**

1. 产品（服务）的常规性升级
2. 对某项科研成果的直接应用
3. 商品化后为顾客提供的技术支持活动
4. 对现存产品、服务、技术、材料或工艺流程的重复或简单改变
5. 市场调研、效率调研或管理研究
6. 常规的质量控制、测试分析、维修维护
7. 社会科学、艺术或人文学方面的研究

**防范措施：**

1. 准确理解负面清单项目的含义
2. 对研发活动进行实质性判断
3. 保留判断依据和相关资料

#### 风险六：后续管理风险

**风险描述：**

税务机关后续核查时，企业无法提供充分证据证明研发活动真实性和费用归集准确性。

**常见情形：**

1. 研发活动被认定为非研发活动
2. 费用归集被认定为不准确
3. 备查资料不完整

**防范措施：**

1. 建立健全研发管理制度
2. 完整保存研发活动资料
3. 建立研发费用辅助账
4. 定期自查研发费用归集情况

### 6.3 合规要点

#### 要点一：研发活动真实性

1. 研发活动必须真实发生
2. 研发活动必须符合政策定义
3. 保留研发活动全过程资料

#### 要点二：费用归集准确性

1. 建立健全研发费用辅助账
2. 准确归集各项研发费用
3. 合理分摊共用费用

#### 要点三：备查资料完整性

1. 按照规定准备备查资料
2. 资料之间相互印证
3. 资料保存10年以上

#### 要点四：申报信息准确性

1. 准确填写申报表
2. 如实申报研发费用
3. 准确计算加计扣除金额

#### 要点五：内部管理制度

1. 建立研发项目管理制度
2. 建立研发费用归集制度
3. 建立研发档案管理制度
4. 建立研发人员管理制度

### 8.4 常见问题解答

#### Q1：研发费用加计扣除是否需要税务机关审批？

A：不需要。研发费用加计扣除实行"自行判别、申报享受、相关资料留存备查"的办理方式，企业可自行判断是否符合条件，在申报时直接享受优惠，相关资料留存备查。

#### Q2：研发费用加计扣除是否有金额限制？

A：没有金额限制。只要符合政策规定的研发费用，均可按照规定比例加计扣除。

#### Q3：企业同时享受多项税收优惠政策如何处理？

A：研发费用加计扣除优惠可以与高新技术企业15%税率优惠、小型微利企业优惠等同时享受。但需注意，研发费用本身已在税前扣除，不能重复享受其他扣除优惠。

#### Q4：研发费用加计扣除追溯享受期限是多久？

A：企业符合研发费用加计扣除条件但在2016年1月1日以后未及时享受该项税收优惠的，可以追溯享受，追溯期限最长为3年。

#### Q5：企业集团集中研发如何享受优惠？

A：企业集团集中研发的费用，应按照权利和义务相一致、费用支出与收益分享相配比的原则，合理确定研发费用的分摊方法，由各成员企业分别计算加计扣除。

## 总结

研发费用加计扣除政策是国家激励企业创新的重要税收优惠措施。企业应准确把握政策要点，规范研发活动管理，准确归集研发费用，完整保留备查资料，依法依规享受税收优惠，降低企业创新成本，提升核心竞争力。

**核心要点回顾：**

1. **扣除比例**：2023年起统一提高至100%（计入当期损益）或200%（形成无形资产）

2. **研发活动认定**：需满足创新性、目标性、系统性、风险性、专业性、记录性、成果性七大条件

3. **费用归集范围**：包括人员人工、直接投入、折旧、无形资产摊销、设计费、其他相关费用六大类

4. **其他相关费用限额**：不得超过可加计扣除研发费用总额的10%

5. **委托研发**：按照实际发生额的80%计入

6. **办理方式**：自行判别、申报享受、相关资料留存备查

7. **风险防范**：关注研发活动认定、费用归集、资料留存、行业限制、负面清单项目等风险点
