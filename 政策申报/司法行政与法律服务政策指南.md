## 二、广州市司法行政政策

### 2.1 法律援助政策

#### 2.1.1 《广州市法律援助办法》

**政策依据**：《广州市法律援助办法》（广州市人民政府令第168号）

**主管部门**：广州市司法局法律援助处

**实施时间**：2019年1月1日起施行

**政策要点**：

1. **经济困难标准**
   - 一般标准：家庭人均月收入低于广州市最低工资标准的1.5倍
   - 特殊群体：残疾人、老年人、未成年人、农民工等适当放宽
   - 免审查情形：见义勇为、重大灾难受害者等

2. **援助范围扩大**
   - 民事案件：劳动争议、交通事故、医疗纠纷、家庭暴力等
   - 行政案件：行政复议、行政诉讼
   - 刑事案件：侦查、审查起诉、审判全阶段
   - 非诉讼事项：调解、公证、司法鉴定等

3. **补贴标准**
   - 刑事案件：侦查阶段800-1500元/件，审查起诉阶段1200-2000元/件，审判阶段1500-3000元/件
   - 民事案件：简单案件1500-2500元/件，复杂案件2500-5000元/件
   - 行政案件：2000-4000元/件
   - 值班律师：300-500元/天

4. **申请流程**
   - 窗口申请：市、区法律援助中心
   - 网上申请：广东法律服务网、广州司法行政微信公众号
   - 电话申请：12348法律服务热线
   - 承诺时限：3个工作日内审查决定

**申报条件**：

| 条件类型 | 具体要求 |
|---------|---------|
| 经济困难 | 家庭人均收入低于最低工资1.5倍，提供收入证明 |
| 案件类型 | 属于法律援助事项范围 |
| 管辖规定 | 案件由广州法院管辖或当事人住所地在广州 |
| 材料齐全 | 身份证明、经济困难证明、案件材料 |

**联系方式**：

- 广州市法律援助处：广州市越秀区连新路19号
- 咨询电话：020-83333531
- 12348法律服务热线：24小时服务
- 网上申请：https://gd.12348.gov.cn

#### 2.1.2 刑事法律援助全覆盖政策

**政策依据**：《最高人民法院 司法部关于开展刑事案件律师辩护全覆盖试点工作的办法》

**实施范围**：广州市两级法院

**政策要点**：

1. **全覆盖内容**
   - 一审程序：所有未委托辩护人的被告人
   - 二审程序：上诉案件、检察院抗诉案件
   - 死刑复核：必须提供法律援助
   - 申诉案件：符合条件的重要申诉

2. **值班律师制度**
   - 设置地点：看守所、法院、检察院
   - 服务内容：法律咨询、程序选择建议、见证认罪认罚
   - 补贴标准：300-500元/天

3. **质量保障**
   - 指派时限：法院通知后3日内指派
   - 会见要求：至少会见当事人2次
   - 阅卷要求：完整阅卷并制作阅卷笔录
   - 出庭要求：全程参与庭审并提交辩护意见

**申报流程**：

```
当事人 → 向法院/检察院/看守所提出申请
        ↓
     机构审核资格
        ↓
     通知法律援助中心
        ↓
     指派承办律师
        ↓
     律师介入案件
```

#### 2.1.3 农民工法律援助专项

**政策要点**：

1. **优先服务**
   - 开通绿色通道，优先受理、优先指派
   - 免除经济困难审查
   - 当日申请、当日指派

2. **重点案件类型**
   - 欠薪纠纷
   - 工伤赔偿
   - 劳动合同纠纷
   - 人身损害赔偿

3. **服务保障**
   - 异地协作：与其他省市法律援助机构协作
   - 诉讼费用减免：协调法院减免诉讼费
   - 执行协助：协助申请强制执行

**补贴标准**：
- 欠薪案件：基本补贴+追回金额的5%-10%（上限1万元）
- 工伤案件：基本补贴+赔偿金额的3%（上限5000元）

### 2.2 公证服务政策

#### 2.2.1 公证费用减免政策

**政策依据**：《广东省公证服务收费管理办法》《广州市公证服务收费减免办法》

**减免对象**：

1. **个人减免**
   - 低保户、特困人员：全免

4. 公证处负责人审批

## 三、广东省律师行业发展政策

### 3.1 律师事务所扶持政策

#### 3.1.1 品牌律师事务所认定与扶持

**政策依据**：《广东省关于加快律师行业发展的意见》《广州市关于加快律师行业发展的实施意见》

**主管部门**：广东省司法厅、广州市司法局

**认定标准**：

| 指标类别 | 具体要求 |
|---------|---------|
| 规模要求 | 执业律师30人以上，或分支机构5个以上 |
| 收入要求 | 年业务收入5000万元以上 |
| 专业要求 | 专业化比例不低于60%，设有3个以上专业部门 |
| 管理要求 | 完善的内部管理制度，连续3年无行政处罚 |
| 人才要求 | 党员律师比例不低于20%，硕士以上学历律师不低于30% |
| 社会责任 | 年办理法律援助案件不少于50件，参与公益法律服务 |

**扶持措施**：

1. **资金扶持**
   - 落户奖励：新落户品牌律所最高200万元
   - 办公用房补贴：连续3年，每年最高100万元
   - 税收优惠：按其对地方财政贡献给予一定比例奖励

2. **人才奖励**
   - 高端人才补贴：合伙人每人最高10万元
   - 青年律师培养补贴：每人每年5000元
   - 涉外律师培训补贴：每人3-5万元

3. **优先推荐**
   - 政府法律顾问优先
   - 国企法律服务优先
   - 重大案件代理优先

**申报流程**：

1. 律师事务所自评
2. 提交申报材料
3. 区司法局初审
4. 市司法局审核
5. 专家评审
6. 公示认定
7. 发放扶持资金

**申报材料**：

- 《品牌律师事务所认定申请表》
- 律师事务所执业许可证复印件

### 7.1 法律援助申请流程

#### 7.1.1 一般申请流程

**Step 1：准备材料**
- 身份证明：身份证或户口簿原件及复印件
- 经济困难证明：民政部门出具或承诺书
- 案件材料：起诉状、判决书、合同等相关材料
- 其他证明材料：根据案件类型提供

**Step 2：提交申请**

| 申请方式 | 地址/渠道 | 服务时间 |
|---------|----------|---------|
| 窗口申请 | 市、区法律援助中心 | 工作日9:00-17:00 |
| 网上申请 | 广东法律服务网 | 24小时 |
| 电话申请 | 12348热线 | 24小时 |
| 微信申请 | "广州司法行政"公众号 | 24小时 |

**Step 3：审核审查**
- 收到申请后3个工作日内完成审查
- 需要补充材料的，一次性告知
- 符合条件的，作出给予法律援助决定

**Step 4：指派律师**
- 收到决定后2个工作日内指派
- 优先考虑当事人意愿
- 特殊案件指定专业律师

**Step 5：律师服务**
- 承办律师及时联系当事人
- 了解案情、收集证据
- 代理诉讼或提供法律帮助

**流程图**：

```
准备材料 → 提交申请 → 审核审查 → 作出决定 → 指派律师 → 律师服务
   (1天)     (即时)     (3天)      (即时)     (2天)     (全程)
```

   - 评分排名

   - 拟认定名单公示7天
   - 处理异议

6. **认定阶段**（6月底）
   - 正式发文认定
   - 颁发证书

**申报材料清单**：

- [ ] 《品牌律师事务所认定申请表》
- [ ] 律师事务所执业许可证复印件
- [ ] 律师名册及执业证书复印件
- [ ] 年度财务审计报告
- [ ] 内部管理制度汇编
- [ ] 专业部门设置说明及人员名单
- [ ] 年度业务清单及收入明细
- [ ] 法律援助案件清单及证明
- [ ] 公益法律服务证明材料
- [ ] 获奖证明、荣誉证书
- [ ] 其他证明材料

#### 7.3.2 落户奖励申报

**申报条件**：
- 在番禺区新注册或新迁入
- 取得执业许可证
- 承诺经营5年以上

**申报流程**：

1. 律师事务所落户后6个月内提出申请
2. 提交申请材料
3. 区司法局审核
4. 区财政局复核
5. 区政府审批
6. 公示7天
7. 发放奖励资金

**申报材料**：
- 《落户奖励申请表》
- 营业执照
- 执业许可证
- 承诺书
- 租赁合同或产权证明

### 7.4 涉外律师培养申报流程

#### 7.4.1 境外培训申报

**申报时间**：每年4月、10月

**申报条件**：
- 执业3年以上
- 英语六级或同等水平
- 有涉外法律事务经验
- 律师事务所推荐

**申报流程**：

1. 关注广东省律师协会公告
2. 提交《境外培训申请表》
3. 提供相关证明材料
4. 省律协审核筛选
5. 公示培训名单
6. 参加培训
7. 提交培训报告
8. 发放补贴

**补贴发放**：
- 培训结束提交报告后1个月内
- 汇入律师事务所账户

#### 7.4.2 港澳律师执业考试报名

**报名时间**：每年6月、12月（具体以司法部公告为准）

**报名流程**：

1. 登录司法部网站
2. 填写报名信息
3. 上传证明材料
4. 资格审核
5. 缴纳考试费
6. 打印准考证
7. 参加考试

**考试费用**：
- 报名费：300元
- 考试合格后申请执业：免费

### 7.5 仲裁申请流程

#### 7.5.1 常规仲裁申请

**申请条件**：
- 有有效的仲裁协议
- 有明确的被申请人
- 有具体的仲裁请求和事实理由
- 属于仲裁委员会受理范围

**申请流程**：

1. **提交申请**
   - 仲裁申请书
   - 仲裁协议
   - 证据材料
   - 身份证明

2. **缴费立案**
   - 收到受理通知书后缴费
   - 缴费后正式立案

3. **组成仲裁庭**
   - 简易程序：1名仲裁员
   - 普通程序：3名仲裁员

4. **开庭审理**
   - 举证、质证
   - 辩论
   - 最后陈述

5. **作出裁决**
   - 简易程序：75日内
   - 普通程序：4个月内

#### 7.5.2 网络仲裁申请

**适用条件**：
- 电子商务纠纷
- 当事人约定网络仲裁
- 证据主要为电子数据

**操作流程**：

1. 登录广州仲裁委网络仲裁平台
2. 注册账号并实名认证
3. 在线填写仲裁申请书
4. 上传电子证据材料
5. 在线缴纳仲裁费用
6. 等待立案通知
7. 参加在线庭审
8. 接收电子裁决书

**技术要求**：
- 电脑或智能手机
- 摄像头、麦克风
- 稳定的网络环境
