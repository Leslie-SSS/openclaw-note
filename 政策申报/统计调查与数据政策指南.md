## 第一部分：广州市统计调查政策

### 一、政策框架概述

#### （一）法律依据

广州市统计调查工作主要依据以下法律法规：

1. **《中华人民共和国统计法》**（2009年修订，2024年修正）
   - 规定了统计工作的基本原则、统计机构和统计人员的职责
   - 明确了统计调查对象的义务和权利
   - 强化了统计违法行为的法律责任

2. **《中华人民共和国统计法实施条例》**（2017年）
   - 细化统计调查项目管理
   - 规范统计资料管理和公布
   - 明确统计监督检查程序

3. **《广东省统计管理条例》**（2020年修订）
   - 结合广东省实际情况细化统计管理
   - 规范地方统计调查项目
   - 建立统计信用体系

4. **《广州市统计管理条例》**（2019年修订）
   - 广州市统计工作的地方性法规
   - 规范统计调查项目审批
   - 明确统计资料报送要求

### 二、统计调查项目管理

#### （一）调查项目分类

**1. 国家统计调查项目**

由国家统计局制定，在全国范围内实施的统计调查。广州市必须严格执行，包括：

- **经济普查**：每5年一次（逢3、8年份）
- **人口普查**：每10年一次（逢0年份）
- **农业普查**：每10年一次（逢6年份）
- **常规统计调查**：月报、季报、年报

**2. 地方统计调查项目**

由广东省、广州市统计机构制定，在本行政区域内实施的统计调查：

- **广东省统计调查项目**：由省统计局审批
- **广州市统计调查项目**：由市统计局审批

**3. 部门统计调查项目**

由县级以上人民政府有关部门制定的统计调查：

- 需报同级统计机构审批
- 不得与国家、地方统计调查项目重复矛盾

### 三、统计调查对象义务

#### （一）法定义务

根据《统计法》第七条规定，统计调查对象必须真实、准确、完整、及时地提供统计资料，不得提供不真实或不完整的统计资料，不得迟报、拒报统计资料。

**1. 申报主体分类**

| 主体类型 | 判定标准 | 主要报表 | 报送周期 |
|---------|---------|---------|---------|
| **"四上"企业** | 年主营业务收入达到规模标准 | 工业产销总值、财务状况等 | 月报/季报/年报 |
| **规模以上工业企业** | 年主营收入≥2000万元 | B204-1、B203表 | 月报/年报 |
| **有资质建筑业企业** | 具有总承包或专业承包资质 | C204-1、C203表 | 季报/年报 |
| **限额以上批发零售企业** | 批发≥2000万、零售≥500万 | E204-1、E203表 | 月报/年报 |
| **限额以上住宿餐饮企业** | 年营收≥200万元 | S204-1、S203表 | 月报/年报 |
| **房地产开发经营企业** | 有开发经营行为 | X204-1、X203表 | 月报/年报 |
| **规模以上服务业企业** | 年营收≥1000万或200万 | F204、F203表 | 月报/年报 |
| **固定资产投资项目** | 总投资≥500万元 | 204-1、206表 | 月报 |

**2. 统计人员要求**

- 每个统计调查单位应指定专（兼）职统计人员
- 统计人员应持有《统计从业资格证书》或具备相应能力
- 参加统计业务培训，每年不少于24学时
- 建立健全统计台账和原始记录制度

#### （二）统计台账制度

**1. 台账设置要求**

统计台账是统计调查对象按照统计制度要求，为便于填报统计报表而设置的一种系统积累统计资料的表册。

| 台账类型 | 设置范围 | 主要内容 | 保存期限 |
|---------|---------|---------|---------|
| 工业台账 | 工业企业 | 产品产量、销售、库存 | ≥3年 |
| 能源台账 | 用能单位 | 能源购进、消费、库存 | ≥3年 |
| 投资台账 | 项目单位 | 投资、资金、工程进度 | ≥5年 |
| 劳资台账 | 用人单位 | 人员、工资、福利 | ≥3年 |
| 财务台账 | 所有单位 | 收入、成本、利润 | ≥5年 |

**2. 原始记录要求**

- 原始记录必须真实、准确、完整
- 及时登记，不得事后补记
- 按时间顺序连续记录
- 与统计报表数据保持一致
- 妥善保管，以备核查

### 四、统计数据报送

#### （一）报送渠道

**1. 联网直报**

国家统计局联网直报系统是"四上"企业和投资项目的法定报送渠道。

| 网址 | 适用范围 | 登录方式 |
|-----|---------|---------|
| http://lwzb.stats.gov.cn | 全国"四上"企业 | 用户名+密码+CA证书 |
| http://219.235.129.78 | 固定资产投资项目 | 项目代码+密码 |

**2. 线下报送**

部分调查项目采用线下报送方式：

- **纸质报表**：填写纸质统计报表，加盖公章后报送
- **电子报表**：将电子版报表发送至指定邮箱
- **现场填报**：由统计人员上门采集数据

#### （二）报送时间要求

| 报表类型 | 报送时间 | 迟报后果 |
|---------|---------|---------|
| 月报 | 次月1-10日 | 警告，责令改正 |
| 季报 | 季末次月1-15日 | 警告，纳入信用记录 |
| 年报 | 次年1-3月 | 警告，影响信用评级 |
| 普查 | 按普查方案规定 | 罚款，纳入失信名单 |
| 一次性调查 | 按调查通知规定 | 警告，责令限期报送 |

#### （三）数据质量要求

**1. 数据真实性**

- 按照实际发生数填报，不得虚报、瞒报
- 与会计报表、业务记录保持一致
- 发现差错应及时申请更正

**2. 数据准确性**

- 严格按照指标解释填列
- 计量单位使用正确
- 数据逻辑关系合理

**3. 数据完整性**

- 按报表规定内容完整填报
- 不得遗漏应报指标
- 暂无数据的应注明原因

**4. 数据时效性**

- 在规定时限内报送
- 遇节假日不顺延（另有规定除外）
- 数据发生重大变化应及时报告

### 五、统计执法检查

#### （一）检查类型

| 检查类型 | 检查主体 | 检查频率 | 检查内容 |
|---------|---------|---------|---------|
| 日常检查 | 统计机构 | 不定期 | 数据真实性、台账建设 |
| 专项检查 | 统计机构 | 每年1-2次 | 重点领域、重点指标 |
| 联合检查 | 多部门联合 | 每年1次 | 综合性、系统性问题 |
| 投诉检查 | 统计机构 | 接投诉后 | 违法举报线索 |

#### （二）检查程序

```
1. 检查准备
   ├─ 制定检查方案
   ├─ 确定检查对象和内容
   ├─ 通知检查对象（突击检查除外）
   └─ 准备检查文书

2. 现场检查
   ├─ 出示执法证件
   ├─ 说明检查事由
   ├─ 查验相关资料
   └─ 制作检查笔录

3. 检查结论
   ├─ 未发现违法：制作检查结论书
   ├─ 发现违法：立案处理
   └─ 限期整改：发出整改通知书

4. 整改复查
   ├─ 跟踪整改情况
   ├─ 验收整改效果
   └─ 结案归档
```

| 迟报统计资料 | 《统计法》第42条 | 企业：1万-5万元 |

- 限制参与政府采购

## 第二部分：广东省数据要素市场化政策

### 一、政策背景与发展历程

#### （一）国家政策背景

1. **《中共中央 国务院关于构建数据基础制度更好发挥数据要素作用的意见》**（2022年12月）

即"数据二十条"，首次明确提出数据作为新型生产要素，建立数据资源持有权、数据加工使用权、数据产品经营权"三权分置"的产权运行机制。

2. **《数字中国建设整体布局规划》**（2023年2月）

提出加快建立数据产权制度，开展数据资产计价研究，建立数据要素按价值贡献参与分配机制。

3. **《数据安全法》《个人信息保护法》**（2021年）

为数据要素市场化配置提供法律保障，规范数据处理活动。

#### （二）广东省政策演进

### 二、广东省数据条例核心内容

#### （一）数据分类分级

### 二、跨境数据流动规则

#### （一）数据出境路径

**1. 数据出境方式**

| 出境方式 | 适用情形 | 审批/备案部门 | 办理周期 |
|---------|---------|--------------|---------|
| 安全评估 | 重要数据或大个人信息 | 国家网信办 | 45个工作日 |
| 标准合同 | 较小规模个人信息 | 省网信办备案 | 15个工作日 |
| 认证 | 特定行业或场景 | 国家网信办 | 60个工作日 |
| 自由流动 | 粤港澳大湾区试点 | 备案管理 | 7个工作日 |

**2. 安全评估条件**

| 评估情形 | 具体标准 | 评估要点 |
|---------|---------|---------|
| 重要数据出境 | 任何数量 | 数据重要性、安全保障 |
| 大个人信息出境 | 100万人以上或累计100万 | 个人权益影响、安全措施 |
| 关键信息基础设施运营者 | 累计10万人 | 基础设施安全、数据保护 |

**3. 标准合同适用条件**

| 适用情形 | 条件要求 | 合同要点 |
|---------|---------|---------|
| 非关键信息基础设施运营者 | - | 不适用 |
| 处理个人信息人数 | <100万人 | 不适用安全评估 |
| 累计出境个人信息 | <10万人 | 可用标准合同 |
| 累计出境敏感个人信息 | <1万人 | 可用标准合同 |

#### （二）粤港澳大湾区跨境数据流动试点

**1. 试点范围**

| 试点区域 | 试点内容 | 试点行业 |
|---------|---------|---------|
| 横琴粤澳深度合作区 | 澳门数据入境 | 金融、医疗、科研 |
| 前海深港现代服务业合作区 | 港澳数据双向流动 | 金融、物流、商贸 |
| 广州南沙粤港澳全面合作区 | 港澳数据入境 | 科技创新、专业服务 |
| 深圳河套深港科技创新合作区 | 科研数据流动 | 生物医药、人工智能 |

**2. 简化流程**

```
粤港澳大湾区跨境数据流动简化流程：

传统流程：
企业申请 → 网信办评估 → 45个工作日 → 批准 → 数据出境

简化流程（试点区域）：
企业备案 → 合作区管理机构审核 → 7个工作日 → 备案完成 → 数据出境

适用条件：
├─ 注册在试点区域的企业
├─ 符合试点行业目录
├─ 数据类型属于试点范围
├─ 通过数据安全合规审查
└─ 签署跨境数据流动承诺书
```

**3. 试点行业数据类型**

| 行业 | 可跨境数据类型 | 用途限制 |
|-----|---------------|---------|
| 金融 | 客户基本信息、交易记录 | 金融监管、风险管理 |
| 医疗 | 电子病历、检验报告 | 跨境医疗、远程诊疗 |
| 科研 | 研究数据、实验结果 | 联合研究、学术交流 |
| 物流 | 货物信息、运输轨迹 | 供应链管理、贸易便利化 |
| 商贸 | 商品信息、订单数据 | 跨境电商、贸易服务 |

**2. 服务流程**

```
跨境数据服务申请流程：

1. 需求确认
   ├─ 明确数据出境需求
   ├─ 确定数据类型和数量
   └─ 了解合规要求

2. 方案设计
   ├─ 选择合适的出境路径
   ├─ 制定数据安全保护措施
   └─ 准备申请材料

3. 合规申请
   ├─ 提交安全评估/标准合同/认证申请
   ├─ 配合审查
   └─ 获得批准/备案

4. 数据出境
   ├─ 通过安全通道传输
   ├─ 全程监控记录
   └─ 合规存证

5. 持续合规
   ├─ 定期合规评估
   ├─ 变更报告
   └─ 应急响应
```

### 五、跨境数据流动合规要点

#### （一）合规义务清单

| 合规事项 | 具体要求 | 违规后果 |
|---------|---------|---------|
| 数据分类分级 | 识别重要数据和个人信息 | 责令整改、警告 |
| 安全评估 | 符合条件的必须进行评估 | 罚款5000万元或营业额5% |
| 标准合同 | 签订标准合同并备案 | 责令整改、警告 |
| 安全措施 | 采取必要技术和管理措施 | 责令整改、罚款 |
| 主体告知 | 告知个人信息主体 | 责令整改、警告 |
| 单独同意 | 获得个人信息主体单独同意 | 责令整改、罚款 |
| 影响评估 | 进行个人信息保护影响评估 | 责令整改 |
| 记录保存 | 保存个人信息出境活动记录 | 责令整改 |
| 定期评估 | 定期评估数据出境合规性 | 责令整改 |

#### （二）合规自检清单

```
跨境数据流动合规自检清单：

一、数据分类
□ 是否完成数据分类分级
□ 是否识别出重要数据
□ 是否识别出个人信息（含敏感个人信息）
□ 是否建立数据目录清单

二、出境必要性
□ 是否确有必要向境外提供数据
□ 是否已评估境内替代方案
□ 是否明确数据出境目的和范围

三、合规路径选择
□ 是否符合安全评估条件
□ 是否可采用标准合同
□ 是否需要进行认证
□ 是否属于粤港澳大湾区试点范围

四、安全保护措施
□ 是否建立数据安全管理制度
□ 是否采取加密、脱敏等技术措施
□ 是否与境外接收方签订安全协议
□ 是否建立数据安全事件应急机制

五、主体权益保护
□ 是否告知个人信息主体
□ 是否获得个人信息主体单独同意
□ 是否建立主体权利响应机制
□ 是否建立投诉举报渠道

六、合规评估
□ 是否进行个人信息保护影响评估
□ 是否定期评估数据出境合规性
□ 是否保存评估记录

七、监管配合
□ 是否配合监管检查
□ 是否及时报告变更情况
□ 是否及时报告安全事件
```

### 附录一：术语解释

| 术语 | 定义 |
|-----|------|
| **统计数据** | 统计活动过程中获得的、反映经济社会发展情况的数据 |
| **数据要素** | 能够参与社会生产经营活动、带来经济利益的数据资源 |
| **数据资产** | 企业拥有或控制的、能够为企业带来经济利益的数据资源 |
| **数据交易** | 数据产品或数据服务在不同主体之间的有偿转让行为 |
| **数据出境** | 数据处理者向境外提供数据的行为 |
| **重要数据** | 一旦遭到篡改、破坏、泄露或者非法获取、非法利用，可能危害国家安全、经济运行、社会公共利益等的数据 |
| **个人信息** | 以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息 |
| **敏感个人信息** | 一旦泄露或者非法使用，容易导致自然人的人格尊严受到侵害或者人身、财产安全受到危害的个人信息 |
| **PUE** | Power Usage Effectiveness，数据中心能源效率指标，为数据中心总能耗与IT设备能耗之比 |
| **数据商** | 为数据交易双方提供数据产品开发、发布、承销等服务的中介服务机构 |
