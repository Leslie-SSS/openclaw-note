## 第一章 广州市企业数字化转型政策

### 一、政策概述

广州市作为国家中心城市和粤港澳大湾区核心引擎，高度重视企业数字化转型工作。近年来，广州市相继出台《广州市数字经济发展规划》《广州市工业互联网创新发展行动计划》《广州市制造业数字化转型实施方案》等一系列政策文件，形成了覆盖全产业链、全生命周期的数字化转型政策体系。

广州市数字化转型政策的总体目标是：到2025年，推动超过5000家规模以上工业企业实现数字化转型，培育100个以上工业互联网平台，打造50个智能制造示范工厂，全市数字经济核心产业增加值占GDP比重超过15%。

### 二、核心政策解读

#### （一）工业互联网创新发展项目

**1. 政策依据**
- 《广州市工业互联网创新发展行动计划（2022-2025年）》
- 《广州市促进工业互联网发展若干措施》

**2. 支持方向**
- 工业互联网平台建设与运营
- 工业设备联网与数据采集
- 工业APP开发与应用
- 工业网络安全体系建设
- 5G+工业互联网融合应用

**3. 申报条件**
```
基本条件：
（1）在广州市行政区域内依法注册的企业法人；
（2）具有独立法人资格，经营状况良好；
（3）上年度主营业务收入不低于1000万元；
（4）近两年无重大安全、环保、质量事故；
（5）无严重失信记录。

专项条件：
（1）工业互联网相关投入不低于100万元；
（2）拥有自主知识产权的工业互联网产品或服务；
（3）具备工业互联网平台运营能力；
（4）已实现一定规模的企业用户接入。
```

**4. 奖励标准**
- 平台建设类：按项目总投资的30%给予补助，最高500万元
- 应用推广类：按实际服务收入的20%给予奖励，最高200万元
- 示范项目类：一次性奖励100-300万元

**5. 申报材料**
- 企业营业执照复印件
- 上年度财务审计报告
- 项目可行性研究报告
- 工业互联网投入明细及发票
- 知识产权证明材料
- 用户服务合同及证明
- 其他相关证明材料

**6. 申报流程**
```
第一步：网上注册 → 广州市工业和信息化局项目管理系统
第二步：材料填报 → 在线填写申报表，上传附件材料
第三步：区级初审 → 所在区工信部门审核推荐
第四步：市级评审 → 市工信局组织专家评审
第五步：现场核查 → 实地考察项目建设情况
第六步：公示公告 → 官网公示7个工作日
第七步：资金拨付 → 公示无异议后拨付资金
```

**7. 申报时间**
每年3-4月集中申报，具体时间以市工信局发布的申报指南为准。

#### （二）企业上云上平台补贴

**1. 政策背景**
为降低中小企业数字化转型门槛，广州市设立企业上云上平台专项补贴，鼓励企业使用云计算、大数据等数字化服务，加快业务系统云化迁移。

**2. 支持范围**
- 基础云服务：云服务器、云存储、云数据库等
- 平台服务：云化ERP、云化CRM、云化OA等
- 软件服务：云化设计软件、云化生产管理软件等
- 安全服务：云安全、数据备份等

**3. 申报条件**
```
（1）在广州市注册的中小企业（符合《中小企业划型标准规定》）；
（2）连续使用云服务满12个月以上；
（3）年度云服务费用不低于10万元；
（4）与具有资质的云服务商签订服务合同；
（5）业务系统已实现云化部署。
```

**4. 奖励标准**
- 一般企业：按实际支付云服务费用的30%给予补贴
- 专精特新企业：按实际支付云服务费用的40%给予补贴
- 高新技术企业：按实际支付云服务费用的35%给予补贴
- 单个企业年度补贴上限100万元

**5. 云服务商资质要求**
- 获得工信部云计算服务牌照
- 通过信息安全等级保护三级以上认证
- 在广州市设有服务团队
- 列入市工信局推荐目录

**6. 申报材料清单**
- [ ] 企业上云补贴申请表
- [ ] 营业执照复印件
- [ ] 中小企业证明材料
- [ ] 云服务合同复印件
- [ ] 云服务费发票及付款凭证
- [ ] 云服务使用情况报告
- [ ] 云服务商资质证明
- [ ] 业务系统云化情况说明

**7. 注意事项**
- 云服务费用不包含网络带宽费用
- 一次性预付费用需分年度申报
- 同一云服务合同只能申报一次补贴
- 已享受其他财政补贴的不得重复申报

#### （三）智能制造试点示范项目

**1. 政策目标**
通过支持企业实施智能化改造，建设智能车间、智能工厂，提升制造业数字化、网络化、智能化水平，打造一批具有示范引领作用的智能制造标杆企业。

**2. 支持方向**
```
智能车间建设：
• 自动化生产线改造
• 工业机器人应用
• 智能检测与质量控制
• 数字化生产管理系统

智能工厂建设：
• 智能设计与仿真
• 智能生产调度
• 智能仓储物流
• 智能运维服务

新模式应用：
• 大规模个性化定制
• 网络协同制造
• 远程运维服务
• 共享制造
```

**3. 申报条件**
```
基本条件：
（1）广州市规模以上工业企业；
（2）企业智能化改造投入不低于500万元；
（3）拥有完善的信息化基础设施；
（4）建立了智能制造相关团队；
（5）近三年平均营收不低于3000万元。

关键指标：
• 关键工序数控化率≥70%
• 生产设备联网率≥60%
• 数字化研发设计工具普及率≥80%
• 智能制造就绪度≥二级
```

**4. 奖励标准**
- 市级智能车间：一次性奖励100万元
- 市级智能工厂：一次性奖励300万元
- 省级智能制造示范：配套奖励200万元
- 国家级智能制造示范：配套奖励500万元
- 项目建设补助：按设备投资的10%给予补助，最高1000万元

**5. 评价指标体系**

| 评价指标 | 权重 | 一级指标要求 |
|---------|-----|-------------|
| 数字化设计 | 15% | CAD/CAM/CAE应用率≥90% |
| 智能生产 | 30% | 关键工序数控化率≥70% |
| 智能装备 | 20% | 工业机器人密度≥150台/万人 |
| 智能管理 | 15% | MES/ERP系统覆盖率≥80% |
| 智能服务 | 10% | 远程运维服务比例≥30% |
| 数据应用 | 10% | 数据采集与分析应用≥3个场景 |

**6. 申报流程图**
```
项目储备 → 项目立项 → 项目建设 → 中期评估 → 项目验收 → 奖励申请
   ↓          ↓          ↓          ↓          ↓          ↓
  入库      备案       实施       监督       评审       拨付
```

#### （四）工业软件推广应用项目

**1. 政策背景**
工业软件是制造业数字化转型的核心支撑。为解决工业软件"卡脖子"问题，广州市设立专项支持国产工业软件推广应用，促进工业软件产业发展和应用普及。

**2. 支持领域**
- CAD/CAE/CAM等研发设计软件
- MES/APS等生产管控软件
- ERP/PLM等经营管理软件
- MRO等运维服务软件
- 工业控制软件

**3. 申报条件**
```
软件采购企业：
（1）广州市制造业企业；
（2）采购国产工业软件授权费用不低于50万元；
（3）软件应用于生产环节；
（4）承诺软件使用期限不低于3年。

软件开发企业：
（1）广州市软件企业；
（2）拥有软件著作权；
（3）软件产品通过第三方测试；
（4）用户数量不低于10家。
```

**4. 奖励标准**
- 采购补贴：按软件采购金额的20%给予补贴，最高200万元
- 开发奖励：按软件销售收入的10%给予奖励，最高300万元
- 示范应用：每个示范案例奖励50万元

**5. 国产软件认定标准**
- 软件著作权由中国公民或法人拥有
- 核心技术具有自主知识产权
- 国产化率不低于60%
- 源代码自主可控

#### （五）5G+工业互联网应用项目

**1. 政策意义**
5G技术具有高带宽、低时延、大连接的特点，与工业互联网深度融合将带来生产方式的革命性变革。广州市鼓励企业探索5G在工业领域的创新应用。

**2. 应用场景**
```
智能安防：5G+高清视频监控、AI安全识别
远程操控：5G+AGV远程驾驶、远程设备操控
智能质检：5G+机器视觉、在线质量检测
协同设计：5G+AR/VR远程协作、虚拟仿真
预测维护：5G+传感器网络、设备健康监测
柔性制造：5G+无线工控、柔性生产线
```

**3. 申报条件**
（1）已在广州市建设5G应用场景不少于3个；
（2）5G网络覆盖核心生产区域；
（3）与基础电信运营商签署5G服务协议；
（4）5G相关投入不低于100万元；
（5）形成可复制推广的解决方案。

**4. 奖励标准**
- 应用建设：按5G相关投入的30%补助，最高300万元
- 示范推广：每个推广案例奖励20万元
- 标杆项目：一次性奖励100万元

**5. 技术要求**
- 5G网络时延≤10ms
- 单基站并发连接数≥1000
- 视频传输分辨率≥4K
- 数据安全加密传输

## 第二章 广东省制造业数字化转型政策

### 一、政策体系概述

广东省作为制造业大省，率先在全国推进制造业数字化转型工作。省级政策体系以《广东省制造业数字化转型实施方案》为统领，配套出台了一系列专项政策措施，构建了覆盖战略规划、资金支持、平台建设、人才培养的完整政策框架。

**核心政策文件**：
- 《广东省制造业数字化转型实施方案（2021-2025年）》
- 《广东省工业互联网示范区建设方案》
- 《广东省智能制造发展规划（2021-2025年）》
- 《广东省培育智能制造战略性新兴产业集群行动计划》

**发展目标**：

```

（1）在广东省内依法注册的制造业企业；
（2）具有独立法人资格，持续经营三年以上；

（2）已建成省级及以上企业技术中心；
（3）数字化转型项目已完工并验收合格；
（4）形成可复制推广的解决方案；
（5）带动产业链上下游企业数字化转型数量≥10家。
```

**3. 支持方向**
```
数字化研发设计：
• 三维建模与仿真分析
• 产品数据管理（PDM）
• 协同设计与知识管理

数字化生产制造：
• 智能排产与调度

• 质量在线检测与追溯

数字化经营管理：
• 企业资源计划（ERP）
```

#### （三）常见问题解答

**Q1：省级项目与市级项目可以同时申报吗？**
A：可以同时申报，但同一项目内容不得重复获得财政资金支持。建议错开申报，先申报市级项目，获得认定后再申报省级配套奖励。

**Q2：子公司可以使用母公司的资质吗？**
A：不可以。申报主体必须独立拥有相关资质，母子公司之间的资质不能混用。

**Q3：项目验收后多久可以申报奖励？**
A：项目验收合格后，原则上在下一批次申报时即可申请。具体以申报指南要求为准。

### 四、常见问题解答

**Q1：项目已经实施完成，能否申报？**
A：部分项目类型支持"先建后补"，可以申报。但示范类项目通常要求在建或拟建，需仔细阅读申报指南。

**Q2：同一项目可以同时申报多个级别的资金吗？**
A：可以同时申报，但同一项目内容不得重复获得财政资金支持。建议按照"就高不重复"原则统筹安排。

**Q3：企业有轻微失信记录，能否申报？**
A：列入严重失信名单的企业不得申报。轻微失信记录影响评分，但不构成否决条件。建议先进行信用修复。

**Q4：申报材料可以修改吗？**
A：提交截止日前可修改。截止后如需补充材料，需联系主管部门，看是否允许补正。

**Q5：项目验收不通过，需要退还资金吗？**
A：需要。项目验收不通过，企业须退还已拨付资金。建议严格按照项目计划实施，确保验收通过。

## 结语

企业数字化转型是推动制造业高质量发展的必由之路。本指南系统梳理了广州市、广东省及粤港澳大湾区的数字化转型相关政策，旨在帮助企业了解政策、用好政策，实现数字化转型目标。

企业在申报过程中应注意：
1. **提前规划**：结合企业实际制定数字化转型规划
2. **精准匹配**：选择最适合企业情况的政策项目
3. **规范准备**：严格按照要求准备申报材料
4. **跟踪落实**：及时跟进项目进度，确保验收通过

本指南内容仅供参考，具体以各政策文件和申报指南为准。如有疑问，请咨询相关主管部门或专业机构。
