## 二、广州市产业投资基金体系

### 2.1 体系概述

广州市产业投资基金体系以"1+N"模式构建，形成覆盖企业全生命周期的基金支持网络。"1"指广州市新兴产业发展引导基金作为母基金，"N"指多个细分领域子基金，包括天使基金、创投基金、产业基金、并购基金等。

#### 2.1.1 基金架构设计

**母基金层**

广州市新兴产业发展引导基金作为核心母基金，由广州市发展和改革委员会主管，广州市新兴产业发展基金管理有限公司负责运营。母基金主要通过参股方式与优秀股权投资机构合作设立子基金，引导社会资本投向广州市战略性新兴产业。

**子基金层**

子基金按照产业领域分类设立，每个子基金聚焦特定产业方向。目前重点支持的产业领域包括：
- 新一代信息技术（集成电路、人工智能、大数据、云计算）
- 生物医药与健康（创新药、医疗器械、生物技术）
- 新能源与节能环保（新能源汽车、储能技术、环保装备）
- 高端装备制造（智能制造、机器人、航空航天）
- 新材料（先进材料、纳米材料、复合材料）

**直投基金层**

对于重大项目和重点企业，引导基金可通过直投方式进行股权投资。直投主要针对以下情形：
- 投资额超过5000万元的重点项目
- 产业链关键环节的龙头企业
- 具有重大带动效应的平台型项目

#### 2.1.2 运作机制

**出资比例**

引导基金对子基金出资比例不超过子基金规模的20%，且不作为第一大出资人或股东。这一设计确保了市场化运作，避免政府资金主导投资决策。

**返投要求**

子基金投资期内投资于广州市行政区域内企业的比例原则上不低于引导基金实缴出资额的2.5倍。例如，引导基金出资2000万元参股1亿元子基金，则子基金需投资广州企业不少于5000万元。

**产业聚焦**

子基金投资须聚焦申报时确定的产业领域，该领域投资比例原则上不低于子基金总规模的60%。这一要求确保子基金形成专业化投资能力。

**收益分配**

子基金收益分配遵循"先回本后分利"原则。引导基金可适当让利，对超额收益部分，引导基金可按照一定比例让渡给其他出资人和基金管理人。

### 2.2 广州市新兴产业发展引导基金

#### 2.2.1 政策依据

《广州市新兴产业发展引导基金管理暂行办法》（穗发改规字〔2023〕7号）是该基金的核心管理文件，明确了基金的设立目的、运作原则、申报条件和监管要求。

#### 2.2.2 申报条件

**基本条件**

1. **机构资质**
   - 在中国境内注册，具有独立法人资格
   - 注册资本不低于1000万元人民币
   - 实缴资本不低于注册资本的50%
   - 成立时间满1年或核心团队具有5年以上投资经验

2. **管理资质**
   - 已完成私募基金管理人登记（中国证券投资基金业协会）
   - 管理机构及其高管无重大违法违规记录
   - 具有完善的内部控制制度和风险管理制度

3. **投资能力**
   - 累计管理基金规模不低于2亿元人民币
   - 至少管理过2支创业投资基金或股权投资基金
   - 至少成功退出过3个以上投资项目
   - 拥有不少于3名具有5年以上股权投资经验的全职高级管理人员

**优先支持条件**

1. 获得国家级、省级引导基金参股的管理机构
2. 管理团队具有世界500强企业投资部门、知名投资机构工作经验
3. 在广州市设有固定办公场所和专职团队
4. 承诺子基金管理人在广州缴纳税收

#### 2.2.3 基金规模与出资

**子基金规模**

单支子基金规模原则上不低于5000万元人民币，不高于5亿元人民币。对于特别优质的GP，经审批可适当放宽上限。

**引导基金出资**

- 出资比例：不超过子基金规模的20%
- 出资上限：单支子基金不超过1亿元
- 出资方式：与其他出资人同比例分期到位

**社会资本要求**

- 社会资本（除政府资金外）占比不低于60%
- 单一社会资本出资人出资比例不超过50%
- 鼓励引入险资、银行理财资金等长期资本

#### 2.2.4 投资限制

**禁止投资领域**

1. 房地产、基础设施等传统行业
2. 金融服务、小额贷款等类金融业务
3. 产能过剩行业（钢铁、水泥、电解铝等）
4. 高污染、高耗能行业
5. 从事二级市场股票、期货、房地产等投机交易

**单项目投资限制**

- 单个项目投资额不超过子基金总规模的20%
- 对同一企业累计投资不超过子基金总规模的30%
- 不得成为被投企业第一大股东

#### 2.2.5 管理费与收益分配

**管理费**

- 投资期：实缴出资的2%/年
- 退出期：已投资未退出金额的1.5%/年
- 延长期：不收取管理费

**收益分配顺序**

1. 向各出资人返还实缴出资
2. 向出资人支付门槛收益（年化6%单利）
3. 超额收益部分：
   - 80%按出资比例分配
   - 20%作为绩效奖励分配给基金管理人

**让利机制**

当子基金实现全部退出且整体收益率超过年化10%时，引导基金可将超出部分的一定比例让渡给其他出资人和基金管理人，最高让利比例可达引导基金收益的50%。

### 2.3 区级产业基金

#### 2.3.1 荔湾区产业投资基金

**基金规模**

荔湾区产业投资基金规模5亿元，主要通过参股子基金方式运作，也可根据需要进行直投。

**申报条件**

- 子基金管理人注册地或承诺迁入荔湾区
- 管理团队具有3年以上投资经验
- 累计管理基金规模不低于1亿元
- 子基金规模不低于3000万元

**投资方向**

重点支持荔湾区"一带三区"产业发展：
- 白鹅潭商务区：现代商贸、金融服务
- 海龙围科创区：科技创新、数字经济
- 大坦沙健康生态岛：生物医药、健康产业
- 荔湾文商旅活力区：文化创意、旅游消费

**特殊优惠**

- 投资荔湾区企业比例达到引导基金出资额3倍的，给予额外让利
- 协助引入企业落地荔湾区的，给予招商奖励

#### 2.3.2 天河区产业投资基金

**基金规模**

天河区产业投资基金规模10亿元，是广州市各区中规模较大的产业基金之一。

**重点支持**

- 天河智慧城：人工智能、云计算、大数据
- 广州国际金融城：金融科技、财富管理
- 天河中央商务区：总部经济、专业服务

## 三、广东省产业投资基金

### 3.1 省级基金体系架构

广东省产业投资基金体系采用"1+N+X"架构模式：

- **"1"**：广东省产业发展基金（总规模500亿元）
- **"N"**：若干支行业性、区域性子基金
- **"X"**：市场化运作的专业基金

### 3.2 广东省产业发展基金

#### 3.2.1 基金定位

广东省产业发展基金是省政府设立的省级综合性产业基金，由广东省财政厅出资，广东省粤财投资控股有限公司受托管理。基金定位为：
- 战略性产业的引导者和推动者
- 重大项目的投资人和支持者
- 区域协调发展的促进者

#### 3.2.2 投资方向

**十大战略性支柱产业**

1. 半导体与集成电路
2. 智能家电
3. 汽车产业
4. 先进材料
5. 现代轻工
6. 软件与信息服务
7. 超高清视频显示
8. 生物医药与健康
9. 现代农业与食品
10. 新能源

**十大战略性新兴产业**

1. 高端装备制造
2. 智能机器人
3. 激光与增材制造
4. 数字创意
5. 新材料
6. 新一代电子信息
7. 绿色低碳
8. 海洋经济
9. 航空航天
10. 前沿新材料

#### 3.2.3 支持方式

**参股子基金**

- 出资比例：不超过子基金规模的25%
- 返投要求：投资广东省内企业比例不低于引导基金出资额的1.5倍
- 决策参与：不参与日常投资决策，但享有一票否决权

**直接投资**

- 单项目投资：5000万-5亿元
- 投资方式：股权投资、可转债
- 持股比例：原则上不超过20%

**投贷联动**

与银行合作，为投资企业提供配套贷款支持：
- 贷款额度：最高可达投资金额的2倍
- 贷款利率：享受优惠利率
- 风险分担：基金与银行按比例分担风险

### 3.3 广东省科技创新基金

#### 3.3.1 基金规模

广东省科技创新基金规模100亿元，专注支持科技创新领域。

#### 3.3.2 投资范围

- 省实验室、重大科技基础设施建设
- 高新技术企业、科技型中小企业
- 技术转移转化项目
- 创新创业孵化载体

#### 3.3.3 特色支持

**科技成果转化专项**

- 重点支持高校、科研院所科技成果转化
- 可采用"先投后股"模式
- 给予技术团队股权激励支持

**科技企业培育专项**

- 支持高企认定、专精特新企业培育
- 给予研发费用补贴
- 提供科技金融服务

### 3.4 市场化基金合作

#### 3.4.1 广东省与知名GP合作

广东省积极与国内外知名投资机构合作，引入优质GP资源：

**已合作机构**

- 深圳市创新投资集团（深创投）
- 红杉资本中国基金
- 高瓴资本
- IDG资本
- 经纬中国
- 君联资本

**合作模式**

1. 省级基金参股GP设立的专项基金
2. 联合设立产业并购基金
3. 项目跟投合作
4. 产业研究合作

#### 3.4.2 地方国企参与

广东省属国企通过多种方式参与产业基金：

- **粤财控股**：省级基金受托管理机构
- **粤科金融**：科技创投领域主力军
- **恒健控股**：重大产业投资平台
- **广业集团**：环保与绿色产业基金
- **广晟集团**：电子信息与矿业基金

## 四、广州市创业投资引导基金

### 4.1 基金概述

广州市创业投资引导基金是由广州市政府设立的政策性基金，旨在引导社会资本投向初创期、早中期科技型企业，促进创新创业和科技成果转化。

### 4.2 政策依据

- 《广州市创业投资引导基金管理办法》
- 《广州市促进创业投资持续健康发展的若干措施》
- 《广州市科技创新条例》

### 4.3 申报条件

#### 4.3.1 子基金管理人条件

**基本资格**

1. 注册在中国境内，已完成私募基金管理人登记
2. 注册资本不低于1000万元，实缴资本不低于500万元
3. 管理团队稳定，核心成员共事时间超过2年
4. 累计管理创业投资基金规模不低于1亿元

**专业能力**

1. 至少有3名具备5年以上创业投资经验的全职高级管理人员
2. 至少完成过5个以上创业投资项目的投资
3. 至少有2个以上成功退出案例
4. 团队成员具有科技、金融复合背景

**诚信记录**

1. 管理机构及高管无重大违法违规记录
2. 管理的基金无重大风险事件
3. 无不良信用记录

#### 4.3.2 子基金设立条件

**规模要求**

- 子基金规模不低于3000万元
- 存续期不超过10年
- 投资期不超过5年

**出资结构**

- 引导基金出资不超过25%
- 单一社会出资人出资不超过50%
- 管理团队跟投不低于1%

**投资方向**

- 投资于广州市企业的资金不低于引导基金出资额的2倍
- 投资于初创期、早中期企业的比例不低于60%
- 重点投资于高新技术企业、科技型中小企业

### 4.4 投资标准

#### 4.4.1 投资阶段定义

**初创期企业**

同时满足以下条件：
1. 成立不满5年
2. 职工人数不超过300人
3. 上年度销售收入不超过5000万元
4. 具有核心技术或创新商业模式

**早中期企业**

同时满足以下条件：
1. 成立不满10年
2. 职工人数不超过500人
3. 上年度销售收入不超过2亿元
4. 研发投入占比不低于3%

#### 4.4.2 重点支持领域

1. **新一代信息技术**
   - 集成电路设计与制造
   - 人工智能算法与应用
   - 工业软件与工业互联网
   - 网络与信息安全

2. **生物医药**
   - 创新药物研发
   - 高端医疗器械
   - 生物技术与服务
   - 医疗大数据与AI

3. **新材料**
   - 电子信息材料
   - 新能源材料
   - 生物医用材料
   - 高端装备材料

4. **新能源与节能环保**
   - 新能源汽车关键零部件
   - 储能技术与设备
   - 智能电网
   - 环境治理技术

5. **高端装备制造**
   - 智能制造装备
   - 工业机器人
   - 检测检验设备
   - 航空航天装备

### 4.5 奖励与让利

#### 4.5.1 投资奖励

**高新技术企业投资奖励**

投资于广州市高新技术企业，按实际投资金额的1%给予奖励，单个项目奖励不超过50万元。

**科技型中小企业投资奖励**

投资于广州市入库科技型中小企业，按实际投资金额的0.5%给予奖励，单个项目奖励不超过30万元。

**天使投资奖励**

对投资广州市种子期、初创期科技型企业的天使投资人，按实际投资金额的10%给予风险补贴，单年度补贴不超过100万元。

#### 4.5.2 税收优惠

**创业投资税收抵扣**

公司制创业投资企业采取股权投资方式直接投资于初创科技型企业满2年的，可按投资额的70%在股权持有满2年的当年抵扣该公司制创业投资企业的应纳税所得额。

**合伙制创投税收优惠**

有限合伙制创业投资企业法人合伙人可享受上述同等待遇，按投资额的70%抵扣法人合伙人的应纳税所得额。

#### 4.5.3 引导基金让利

**基本让利**

子基金整体收益率超过年化8%时，引导基金可将超额收益部分的20%让渡给其他出资人。

**返投奖励让利**

子基金投资广州企业比例达到引导基金出资额3倍以上的，引导基金让利比例提高到30%。

**产业培育让利**

子基金投资项目中成功上市、被并购的比例超过50%的，引导基金让利比例提高到40%。

| 风控能力 | 15% | 制度建设、执行情况、应急能力 |
| 本地化程度 | 10% | 办公场所、团队配置、返投能力 |

#### 7.4.2 一票否决项

存在以下情形之一的，实行一票否决：

1. 提供虚假申报材料
2. 存在重大违法违规行为
3. 严重违反诚信承诺
4. 过往管理基金发生重大风险事件
5. 团队核心成员存在利益冲突

### 7.5 常见问题解答

**Q1：外资机构能否申报？**

A：可以。符合条件的境外股权投资管理机构均可申报。鼓励境外知名GP通过QFLP等方式参与。

**Q2：能否同时申报多个基金？**

A：可以。但需注意不同基金的返投要求，确保具备足够的投资能力。

**Q3：申报失败后能否再次申报？**

A：可以。补充完善材料后可在下一年度再次申报。

**Q4：引导基金出资何时到位？**

A：与社会资本同比例到位。通常分期出资，首期出资在基金设立完成后到位。

**Q5：子基金能否投资房地产？**

A：不能。引导基金参股的子基金禁止投资房地产、基础设施等领域。

## 九、附录：实用工具

### 9.1 投资收益计算工具

**IRR计算公式**

```
NPV = Σ(CFt / (1+IRR)^t) = 0
```

其中：
- CFt：第t期现金流
- IRR：内部收益率
- t：期数

**DPI（投入资本分红率）**

```
DPI = 累计已分配金额 / 实缴出资
```

**TVPI（投入资本总倍数）**

```
TVPI = (累计已分配金额 + 剩余估值) / 实缴出资
```

### 9.2 返投比例计算示例

**案例**：
- 引导基金出资：2000万元
- 子基金规模：1亿元
- 返投要求：2.5倍

**计算**：
- 应投资广州企业金额：2000万 × 2.5 = 5000万元
- 子基金可投资外地金额：10000万 - 5000万 = 5000万元

## 十、政策动态与展望

### 10.1 2026年政策趋势

**一、基金规模持续扩大**

- 广州市计划新增引导基金规模20亿元
- 广东省战略性产业基金规模有望突破1000亿元
- 粤港澳大湾区基金将进一步扩容

**二、投资领域不断拓展**

- 人工智能大模型应用成为重点支持方向
- 新能源汽车产业链投资持续加码
- 生物医药创新药、医疗器械重点关注
- 低空经济、商业航天等新兴领域纳入支持范围

**三、申报流程持续优化**

- 推行"不见面审批"，全程网上办理
- 简化材料要求，减少证明事项
- 压缩审批时限，实现快速落地
- 建立容错机制，鼓励大胆创新

**四、跨境合作更加便利**

- QFLP试点进一步扩大
- 外汇管理持续优化
- 跨境资金流动更加便利
- 港澳专业人士从业限制放宽

### 10.2 政策建议

**对投资机构的建议**

1. **提前布局**：密切关注政策动态，提前准备申报材料
2. **专业化发展**：深耕细分领域，形成专业投资能力
3. **本地化运营**：在广州设立办公室，配备专职团队
4. **投后赋能**：提供增值服务，助力企业发展壮大
5. **诚信经营**：恪守职业道德，维护行业声誉

**对企业的建议**

1. **关注基金动态**：了解各类产业基金投资方向
2. **主动对接**：积极与基金管理人建立联系
3. **规范运营**：完善公司治理，规范财务管理
4. **提升价值**：专注核心业务，增强核心竞争力
5. **善用政策**：充分利用各类扶持政策，降低融资成本

## 结语

产业投资基金作为政府支持实体经济发展的重要工具，在促进产业转型升级、培育新兴产业、推动科技创新等方面发挥着越来越重要的作用。广州市、广东省及粤港澳大湾区各级政府高度重视产业投资基金发展，不断完善政策体系，优化发展环境，为投资机构和企业提供了广阔的发展空间。

本指南系统梳理了广州市产业投资基金体系、广东省产业投资基金、广州市创业投资引导基金、番禺区产业基金专项、粤港澳大湾区基金合作等方面的政策要点，详细介绍了申报条件和流程，并提供了实用的联系方式和资源汇总，希望能为广大投资机构和创业者提供有价值的参考。

在实际申报和运作过程中，建议：

1. **认真研读政策文件**：准确把握政策精神和具体要求
2. **提前做好准备工作**：充分准备申报材料，确保符合条件
3. **加强与主管部门沟通**：及时了解最新动态，解决疑难问题
4. **规范基金运作管理**：建立健全内控制度，依法合规经营
5. **发挥基金引导作用**：聚焦主业投资，助力产业发展

产业投资基金的发展离不开各方的共同努力。我们期待与广大投资机构、创业企业携手合作，共同推动广州市、广东省及粤港澳大湾区产业高质量发展，为实现中国式现代化贡献力量。
