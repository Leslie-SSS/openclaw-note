## 一、政策概述

### 1.1 政策背景

标准化是国家治理体系和治理能力现代化的基础性制度，是高质量发展的重要支撑。近年来，国家、省、市各级政府高度重视标准化工作，出台了一系列支持企业参与标准化建设的政策措施。

**政策体系框架**：

```

└─────────────────────────────────────────────────────────────┘
                            ↓
```

### 1.2 企业参与标准化的意义

#### 1.2.1 战略价值

| 维度 | 具体价值 |
|------|----------|
| **市场竞争** | 掌握行业话语权，塑造竞争优势 |
| **技术创新** | 推动研发成果标准化，保护知识产权 |
| **品牌建设** | 提升企业公信力和行业影响力 |
| **供应链整合** | 建立产业链协作标准，优化资源配置 |
| **国际拓展** | 通过国际标准制定，突破技术性贸易壁垒 |

#### 1.2.2 经济效益

1. **直接收益**：政府资金资助（最高可达100万元）
2. **间接收益**：提升产品溢价能力、降低采购成本、增强客户信任
3. **融资支持**：标准化成果可作为无形资产，提升企业估值

### 1.3 标准层级与分类

#### 1.3.1 标准层级

```
国家标准（GB/GB/T）
    ↑ 行业标准（如：YD通信、JB机械、HG化工）
        ↑ 地方标准（DB44广东省、DB4401广州市）
            ↑ 团体标准（T/协会名）
                ↑ 企业标准（Q/企业代号）
```

#### 1.3.2 标准类型

| 类型 | 代号 | 说明 | 适用范围 |
|------|------|------|----------|
| 强制性国家标准 | GB | 必须执行 | 全国 |
| 推荐性国家标准 | GB/T | 推荐采用 | 全国 |
| 行业标准 | 各行业代号 | 行业通用 | 特定行业 |
| 地方标准 | DB+行政区代码 | 区域适用 | 特定地区 |
| 团体标准 | T/团体代号 | 市场驱动 | 团体成员及自愿采用 |
| 企业标准 | Q/企业代号 | 企业自主 | 企业内部 |

## 二、广州市企业标准化政策

### 2.1 政策依据

#### 2.1.1 主要文件

1. 《广州市标准化战略专项资金管理办法》
2. 《广州市质量强市专项资金管理办法》
3. 《广州市市场监督管理局关于印发广州市标准化资助实施办法的通知》
4. 《广州市人民政府关于实施质量强市战略的决定》

#### 2.1.2 管理部门

- **主管部门**：广州市市场监督管理局（知识产权局）
- **具体处室**：标准化处
- **配合部门**：广州市工业和信息化局、广州市科技局

### 2.2 标准制定/修订资助

#### 2.2.1 资助范围

**可资助的标准类型**：

| 标准类型 | 资助额度 | 条件要求 |
|----------|----------|----------|
| 国际标准 | 50-100万元 | 主导制定单位 |
| 国家标准 | 20-50万元 | 主导制定单位 |
| 行业标准 | 10-30万元 | 主导制定单位 |
| 地方标准 | 5-15万元 | 主导制定单位 |
| 团体标准 | 3-10万元 | 主导制定单位 |

#### 2.2.2 "主导制定"的认定标准

**国际标准主导制定**：
1. 担任国际标准组织（ISO、IEC、ITU）技术委员会（TC）或分技术委员会（SC）主席、秘书处单位
2. 担任国际标准工作组（WG）召集人或项目牵头人
3. 提出的标准提案被正式采纳并发布

**国家标准主导制定**：
1. 在国家标准前言中列明为第一起草单位
2. 担任全国专业标准化技术委员会（TC）或分技术委员会（SC）主任委员/秘书长单位
3. 担任国家标准起草工作组组长单位

**行业标准主导制定**：
1. 在行业标准前言中列明为第一起草单位
2. 担任行业标准技术归口单位
3. 担任行业标准起草工作组组长单位

#### 2.2.3 资助申请条件

**基本条件**：
1. 在广州市行政区域内依法注册的企业、事业单位、社会团体
2. 标准已正式发布并实施
3. 申请时距标准发布日期不超过2年
4. 同一标准未获得过同级政府资助
5. 企业近3年无严重违法失信记录

**优先支持**：
1. 战略性新兴产业领域标准
2. 涉及关键技术、核心专利的标准
3. 具有显著经济社会效益的标准
4. 粤港澳大湾区合作制定的标准

### 2.3 标准化技术组织建设

#### 2.3.1 技术委员会资助

| 类型 | 资助额度 | 条件 |
|------|----------|------|
| 国际TC/SC秘书处 | 50万元/年 | 连续资助3年 |
| 国家TC秘书处 | 30万元/年 | 连续资助3年 |
| 国家SC秘书处 | 20万元/年 | 连续资助3年 |
| 省级TC秘书处 | 10万元/年 | 连续资助3年 |

#### 2.3.2 申报条件

1. 秘书处承担单位为广州市注册企业
2. 按照相关规定正常开展工作
3. 年度考核合格以上
4. 积极组织标准制修订工作

### 2.4 企业标准化良好行为

#### 2.4.1 认定等级

| 等级 | 要求 | 资助 |
|------|------|------|
| AAAA级 | 标准体系完善、实施有效、成效显著 | 10万元 |
| AAA级 | 标准体系较完善、实施有效 | 5万元 |
| AA级 | 标准体系基本建立、实施有效 | 2万元 |

#### 2.4.2 认定标准

1. **标准体系建立**：建立健全以技术标准为主体，包括管理标准、工作标准的企业标准体系
2. **标准实施**：企业标准体系有效运行
3. **自我声明公开**：执行的标准在国家标准信息公共服务平台公开
4. **成效评价**：标准化工作对企业发展的贡献度

### 2.5 标准化示范试点

#### 2.5.1 项目类型

| 项目类型 | 申报层级 | 资助额度 |
|----------|----------|----------|
| 国家级标准化示范项目 | 国家标准委 | 50万元 |
| 省级标准化示范项目 | 省市场监管局 | 30万元 |
| 市级标准化示范项目 | 市市场监管局 | 20万元 |

#### 2.5.2 申报条件

1. 企业注册经营3年以上
2. 具有较完善的标准化工作机制
3. 近2年内主导或参与制定标准1项以上
4. 标准化工作经验具有示范推广价值
5. 行业领域具有代表性

### 2.7 申报材料清单

#### 2.7.1 基本材料

## 三、广东省标准化示范企业认定

### 3.1 政策依据

#### 3.1.1 主要文件

1. 《广东省标准化条例》
2. 《广东省标准化示范企业管理办法》
3. 《广东省人民政府关于深化标准化工作改革推进广东标准化工作的意见》
4. 《广东省市场监督管理局关于印发广东省标准化示范企业认定工作方案的通知》

#### 3.1.2 管理部门

- **主管部门**：广东省市场监督管理局
- **具体处室**：标准化处
- **配合部门**：广东省工业和信息化厅

### 3.2 认定标准

#### 3.2.1 基本条件

| 条件 | 具体要求 |
|------|----------|
| **注册年限** | 在广东省依法注册登记3年以上 |
| **经营规模** | 年主营业务收入达到一定规模（工业企业5000万元以上，服务业企业2000万元以上） |
| **信用状况** | 近3年无严重违法失信记录，无重大质量安全事故 |
| **行业地位** | 在行业内具有较高的知名度和影响力 |

#### 3.2.2 标准化工作条件

**标准制修订要求**（近3年）：

| 标准类型 | 要求 | 备注 |
|----------|------|------|
| 国际标准 | 主导1项或参与2项 | 担任主要起草单位 |
| 国家标准 | 主导2项或参与5项 | 前三起草单位 |
| 行业标准 | 主导3项或参与8项 | 前三起草单位 |
| 地方标准 | 主导5项以上 | 第一起草单位 |
| 团体标准 | 主导5项以上 | 第一起草单位 |

**技术组织参与要求**（满足以下之一）：

1. 承担国际TC/SC秘书处工作
2. 担任国家TC/SC主任委员或秘书长单位
3. 担任省TC主任委员或秘书长单位
4. 担任国家或省TC/SC委员单位

**企业标准化体系要求**：

1. 建立完善的企业标准体系
2. 通过标准化良好行为企业确认（AAA级以上）
3. 执行标准在国家标准信息公共服务平台自我声明公开
4. 企业标准体系有效运行2年以上

### 3.5 申报材料

#### 3.5.1 申报书内容

1. **企业基本情况**：
   - 企业注册信息
   - 主营业务及市场地位
   - 组织架构及人员构成
   - 财务状况（近3年）

2. **标准化工作情况**：
   - 标准化组织机构
   - 标准化制度建设
   - 标准制修订情况
   - 技术委员会参与情况

3. **标准体系运行情况**：
   - 企业标准体系框架
   - 标准实施情况
   - 标准化工作成效

4. **效益分析**：
   - 经济效益（可量化指标）
   - 社会效益
   - 行业影响力

5. **发展规划**：
   - 未来3年标准化工作计划
   - 预期目标

#### 3.5.2 附件材料

1. 营业执照（复印件）
2. 近3年财务审计报告
3. 主导/参与制定的标准清单及证明材料
4. 技术委员会相关证明文件
5. 标准化良好行为确认证书
6. 标准自我声明公开截图
7. 获奖证书、荣誉证明等
8. 其他证明材料

### 3.6 管理与监督

#### 3.6.1 动态管理

1. **信息报送**：每年12月底前报送年度标准化工作总结
2. **变更报告**：企业名称、地址、法人等变更需在30日内报告
3. **重大事项报告**：发生重大质量安全事故等需及时报告

#### 3.6.2 复审制度

1. **复审周期**：每3年复审一次
2. **复审内容**：
   - 标准化工作持续开展情况
   - 标准制修订成果
   - 标准体系运行情况
   - 经济社会效益
3. **复审结果**：
   - 合格：继续保留称号
   - 基本合格：限期整改
   - 不合格：取消称号

#### 3.6.3 退出机制

**取消称号的情形**：

1. 企业申请注销
2. 复审不合格且整改不到位
3. 发生重大质量安全事故
4. 严重违法失信
5. 提供虚假申报材料

| 交通运输 | 交通设施、物流标准 | 积极推进 |

**湾区标准定义**：

由粤港澳大湾区相关方共同制定，在三地共同实施，引领产业高质量发展的先进标准。

2. 协商一致
3. 高于现行标准
4. 三地互认共用

**湾区标准标识**：

- 统一使用"湾区标准"标识
- 标注"GBA Standard"

## 十、附录：常见问题解答

### 10.1 资格条件类

**Q1：企业必须成立多长时间才能申报标准化资助？**

A：市级标准化资助对企业成立年限没有特殊要求，只要在广州市注册登记、具有独立法人资格即可。但省级标准化示范企业认定要求企业注册经营3年以上。

**Q2：外资企业可以申请标准化资助吗？**

A：可以。只要在广州市依法注册登记的企业，不论内资外资，均可申请。

**Q3：个体工商户可以申请吗？**

A：不可以。申请主体必须是具有独立法人资格的企业、事业单位或社会团体。

**Q4：同一标准可以同时申请市、区两级资助吗？**

A：可以。市、区资助是相互独立的，同一标准可以同时申请市级和区级资助，但同一级别政府不能重复资助。

### 10.2 标准认定类

**Q5：如何证明企业是"主导制定"标准？**

A：主要证明材料包括：
1. 标准前言中列明为第一起草单位
2. 担任技术委员会相关职务的证明文件
3. 工作组组长聘任文件
4. 标准组织出具的证明文件

**Q6：参与标准制定可以获得资助吗？**

A：可以，但资助额度低于主导制定。参与制定标准的企业需在前三位起草单位之列，才能申请资助。

**Q7：企业标准可以获得资助吗？**

A：一般不单独资助企业标准。但如果企业标准转化为团体标准、地方标准或以上层级，可按相应标准申请资助。

**Q8：团体标准如何认定"优秀"？**

A：优秀团体标准通常需满足：
1. 填补标准空白
2. 技术指标先进
3. 实施效果显著
4. 行业认可度高
5. 获得省级以上标准化主管部门认定

### 10.3 申报流程类

**Q9：标准化资助每年什么时候申报？**

A：广州市一般在每年3-4月发布申报通知，申报期约30天。番禺区一般在4-5月。具体时间以当年通知为准。

**Q10：申报材料需要准备纸质版和电子版吗？**

A：是的。需要通过网上申报系统提交电子版，同时向所在区市场监管局提交纸质版材料。

**Q11：申报后多久能获得资助？**

A：从申报到资金拨付一般需要3-4个月，包括初审、审核、专家评审、公示等环节。

**Q12：错过申报时间怎么办？**

A：标准化资助实行年度集中申报制，错过当年申报时间需等待下一年度。建议提前准备材料，关注申报通知。

### 10.4 资金使用类

**Q13：资助资金有使用限制吗？**

A：资助资金应专款专用，主要用于：
1. 标准化研究开发
2. 标准制修订活动
3. 标准化人才培养
4. 标准化设备购置
5. 标准化国际合作

不得用于与标准化无关的支出。

**Q14：资助资金需要归还吗？**

A：不需要归还，属于政府奖励性资助。但如果存在虚报冒领、挪用资金等违规行为，将被追回资金。

**Q15：获得资助后需要报送使用情况吗？**

A：需要。获得资助的单位应按规定报送资金使用情况和绩效自评报告，接受监督检查。

### 10.5 示范企业类

**Q16：省级标准化示范企业认定标准是什么？**

A：主要标准包括：
1. 企业注册3年以上
2. 近3年主导制定标准数量达标
3. 参与标准化技术组织
4. 标准化体系有效运行
5. 近3年无严重违法失信

**Q17：省级示范企业认定后有什么好处？**

A：主要好处包括：
1. 省级奖励50万元
2. 市级配套20万元
3. 番禺区配套30万元
4. 采购、融资等方面优先支持
5. 品牌形象提升

**Q18：示范企业认定有效期多长？**

A：有效期为3年。期满后需要复审，复审合格可继续保留称号并获得10万元奖励。

### 10.6 湾区标准类

**Q19：企业如何参与粤港澳大湾区标准制定？**

A：参与路径：
1. 关注粤港澳大湾区标准化研究中心公告
2. 提交参与申请
3. 加入标准起草工作组
4. 参与标准制定工作

**Q20：湾区标准与国家标准有什么区别？**

A：湾区标准是由粤港澳三地共同制定、在三地共同实施的标准，主要特点：
1. 三地协商一致
2. 适应湾区市场需求
3. 有助于三地互联互通
4. 可作为国家标准的补充

**Q21：湾区标准的资助额度是多少？**

A：主导制定湾区标准可获得30-50万元资助，参与制定可获得10-20万元资助。番禺区还有额外5万元奖励。

### 10.7 技术委员会类

**Q22：企业如何申请承担技术委员会秘书处？**

A：申请流程：
1. 关注国家标准委或省市场监管局公告
2. 提交申请材料
3. 通过竞争性遴选
4. 获批后承担秘书处工作

**Q23：承担秘书处有什么好处？**

A：主要好处：
1. 获得年度资助（国际50万、国家30万、省级10万）
2. 掌握标准制定主导权
3. 提升行业影响力
4. 获取行业信息资源
