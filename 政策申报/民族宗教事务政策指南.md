## 二、广州市民族事务政策

### 2.1 民族团结进步创建活动

#### 2.1.1 政策依据

- 《广州市民族团结进步创建活动管理办法》（穗民宗规〔2020〕1号）
- 《广东省民族团结进步创建活动指导意见》（粤民宗发〔2019〕15号）
- 《关于全面深入持久开展民族团结进步创建工作铸牢中华民族共同体意识的意见》（中办发〔2018〕65号）

#### 2.1.2 创建类型与标准

**（一）创建类型**

1. **民族团结进步示范区（单位）**
   - 区级示范区
   - 市级示范区
   - 省级示范区
   - 国家级示范区

2. **民族团结进步教育基地**
   - 主题教育类
   - 实践体验类
   - 文化展示类

3. **民族团结进步示范社区**
   - 城市社区
   - 农村社区
   - 混合型社区

**（二）创建标准**

**示范区（单位）标准：**
- 组织领导健全，成立民族团结进步创建工作领导小组
- 工作机制完善，有专职工作人员和专项工作经费
- 宣传教育到位，民族团结理念深入人心
- 服务管理规范，各族群众合法权益得到保障
- 示范作用明显，创建工作经验可复制推广

**教育基地标准：**
- 有固定的教育场所和展示区域
- 有完整的教育内容和讲解体系
- 有专业的讲解员队伍
- 年接待参观人数不少于5000人次
- 开展主题教育活动不少于10次/年

**示范社区标准：**
- 少数民族常住人口占比5%以上或绝对数500人以上
- 建立少数民族流动人口服务管理站
- 配备专兼职民族工作联络员
- 少数民族居民满意率85%以上
- 无涉及民族因素的群体性事件

#### 2.1.3 申报条件

**基本条件：**
1. 在广州市行政区域内登记注册的机关、企事业单位、社会团体、社区等
2. 开展民族团结进步创建工作满2年以上
3. 近3年内未发生涉及民族因素的违法违规事件
4. 已获得下一级示范单位称号

**加分条件：**
1. 创新民族工作方法，形成可推广经验（加10分）
2. 少数民族员工占比达30%以上（加5分）
3. 获得省级以上民族工作相关荣誉（加10分）
4. 承担市级以上民族工作试点任务（加8分）
5. 开展跨区域民族交流合作项目（加5分）

#### 2.1.4 奖励标准

| 认定级别 | 奖励金额 | 后续支持 |
|---------|---------|---------|
| 区级示范单位 | 5万元 | 优先推荐市级评选 |
| 市级示范单位 | 15万元 | 给予工作经费补助 |
| 省级示范单位 | 30万元 | 纳入省级重点项目库 |
| 国家级示范单位 | 50万元 | 给予创建工作专项奖励 |

#### 2.1.5 申报材料

1. 广州市民族团结进步创建示范单位申报表
2. 创建工作总结报告（3000字以上）
3. 创建工作规划和实施方案
4. 相关证明材料：
   - 组织机构文件
   - 工作制度汇编
   - 活动开展记录
   - 媒体宣传报道
   - 荣誉证书复印件
5. 申报单位营业执照或组织机构代码证复印件
6. 法定代表人身份证复印件

### 2.2 民族团结宣传教育专项

#### 2.2.1 支持方向

1. **民族团结主题宣传**
   - 民族团结进步宣传月活动
   - 民族政策法规宣传活动
   - 民族知识普及教育

2. **民族文化展示交流**
   - 民族文化艺术展演
   - 民族传统体育赛事
   - 民族美食文化节

3. **民族团结教育培训**
   - 民族工作干部培训
   - 民族政策法规培训
   - 民族团结教育进校园

#### 2.2.2 申报条件

**宣传项目：**
- 活动规模500人以上
- 有详细的活动方案和预算
- 有主流媒体宣传报道计划
- 预期覆盖人群1万人以上

**文化项目：**
- 有3个以上民族的文化元素展示
- 有专业团队参与策划执行
- 活动持续3天以上
- 有文化传承和教育意义

**培训项目：**
- 培训人数100人以上
- 有专业师资团队
- 有完整的培训课程体系
- 有培训效果评估机制

#### 2.2.3 资助标准

| 项目类型 | 资助比例 | 最高限额 | 特别说明 |
|---------|---------|---------|---------|
| 宣传活动 | 实际支出50% | 20万元 | 市级以上媒体宣传 |
| 文化活动 | 实际支出40% | 30万元 | 纳入文化惠民工程 |
| 培训项目 | 实际支出60% | 10万元 | 需提供培训合格证 |
| 综合性项目 | 实际支出45% | 50万元 | 多类活动组合 |

### 2.3 民族工作进社区专项

#### 2.3.1 建设内容

**"六个一"工程：**

1. **一个服务窗口**
   - 设立少数民族服务窗口
   - 提供政策咨询、证件办理等服务
   - 配备双语服务人员

2. **一个联络站**
   - 建立少数民族流动人口联络站
   - 提供就业、住房、子女入学等信息
   - 协调解决实际困难

3. **一支联络员队伍**
   - 每个社区配备1-2名民族工作联络员
   - 负责信息收集、矛盾调解
   - 建立微信工作群

4. **一个活动室**
   - 设置民族文化活动室
   - 配备民族文化展示设施
   - 定期开展联谊活动

5. **一套工作制度**
   - 制定社区民族工作制度
   - 建立信息报送机制
   - 完善应急处置预案

6. **一本服务手册**
   - 编印少数民族服务指南
   - 提供多语言版本
   - 定期更新内容

#### 2.3.2 建设标准与资金

| 建设内容 | 建设标准 | 补助金额 | 验收要求 |
|---------|---------|---------|---------|
| 服务窗口 | 不少于10平方米 | 3万元 | 正常运营1年以上 |
| 联络站 | 有固定场所和人员 | 5万元 | 建立工作台账 |
| 联络员队伍 | 每万名少数民族配1名 | 2000元/人/年 | 完成培训考核 |
| 活动室 | 不少于50平方米 | 8万元 | 开展活动12次/年 |
| 制度建设 | 制度上墙公示 | 1万元 | 制度完备有效 |
| 服务手册 | 印制500份以上 | 5000元 | 发放到户 |

## 三、广东省宗教事务管理

### 3.1 宗教活动场所管理

#### 3.1.1 法律法规依据

- 《宗教事务条例》（国务院令第686号）
- 《广东省宗教事务条例》（2020年修订）
- 《宗教活动场所管理办法》（国家宗教事务局令第2号）
- 《广东省宗教活动场所管理规定》（粤府令第254号）

#### 3.1.2 宗教活动场所分类

**按宗教类别分类：**

| 宗教类别 | 场所类型 | 主要分布区域 | 登记数量（广东） |
|---------|---------|-------------|----------------|
| 佛教 | 寺院、庵堂 | 全省各地 | 约1500处 |
| 道教 | 宫观 | 广州、惠州、罗浮山 | 约300处 |
| 伊斯兰教 | 清真寺 | 广州、深圳、东莞 | 约20处 |
| 天主教 | 教堂、会所 | 广州、汕头、江门 | 约100处 |
| 基督教 | 教堂、聚会点 | 全省各地 | 约2000处 |

**按规模等级分类：**

1. **大型宗教活动场所**
   - 占地面积10000平方米以上
   - 可容纳信众2000人以上
   - 年接待信众10万人次以上

2. **中型宗教活动场所**
   - 占地面积3000-10000平方米
   - 可容纳信众500-2000人
   - 年接待信众3-10万人次

3. **小型宗教活动场所**
   - 占地面积3000平方米以下
   - 可容纳信众500人以下
   - 年接待信众3万人次以下

#### 3.1.3 场所登记管理

**登记条件：**
1. 有固定的场所名称、地点和符合规定的布局
2. 有经常参加宗教活动的信教公民
3. 有民主协商成立的管理组织
4. 有主持宗教活动的宗教教职人员或符合本宗教规定的其他人员
5. 有必要的资金
6. 有布局合理、符合安全要求的建筑设施

**登记程序：**

```
申请 → 乡镇街道初审 → 区民族宗教局审核 → 市民族宗教局复核 → 省民族宗教委登记 → 发证
```

**办理时限：**
- 乡镇街道初审：10个工作日
- 区民族宗教局审核：20个工作日
- 市民族宗教局复核：15个工作日
- 省民族宗教委登记：30个工作日
- 总计：75个工作日

**年度检验：**

| 检验内容 | 检验标准 | 结果认定 |
|---------|---------|---------|
| 组织管理 | 管理组织健全，制度完善 | 合格/基本合格/不合格 |
| 财务管理 | 账目清楚，收支合规 | 合格/基本合格/不合格 |
| 活动管理 | 活动规范，无违规行为 | 合格/基本合格/不合格 |
| 安全管理 | 安全制度健全，设施完好 | 合格/基本合格/不合格 |
| 资产管理 | 资产清晰，使用规范 | 合格/基本合格/不合格 |

**年度检验时间：** 每年1月1日至3月31日

### 3.2 宗教团体管理

#### 3.2.1 团体设立条件

**市级宗教团体：**
1. 有可考证的、信教群众公认的历史传承
2. 有广泛的信教群众基础（不少于3000人）
3. 有代表性和影响力的教职人员（不少于10人）
4. 有固定的办公场所和必要的经费
5. 有合法的资产和经费来源
6. 有独立承担民事责任的能力

**区级宗教团体：**
1. 有一定的信教群众基础（不少于500人）
2. 有相对固定的教职人员（不少于3人）
3. 有固定的办公场所
4. 有合法的经费来源

#### 3.2.2 管理制度要求

**"四项制度"建设：**

1. **民主管理制度**
   - 建立民主管理组织
   - 实行民主决策
   - 定期召开会议
   - 公开重大事项

2. **财务管理制度**
   - 执行国家财务会计制度
   - 建立健全财务账目
   - 实行财务公开
   - 接受审计监督

3. **人员管理制度**
   - 建立教职人员档案
   - 实行考核评价
   - 加强教育培训
   - 规范流动管理

4. **活动管理制度**
   - 活动申报审批
   - 重大活动报告
   - 活动记录存档
   - 安全应急预案

#### 3.2.3 团体职能定位

| 职能类别 | 具体内容 | 责任主体 |
|---------|---------|---------|
| 政府联系 | 反映信教群众诉求，传达政策精神 | 各级宗教团体 |
| 教务指导 | 制定教规制度，指导教务活动 | 各级宗教团体 |
| 人才培养 | 举办培训班，培养教职人员 | 省市级团体 |
| 文化建设 | 开展宗教文化研究交流 | 各级宗教团体 |
| 公益慈善 | 组织开展公益慈善活动 | 各级宗教团体 |
| 对外交流 | 开展港澳台及国际交流 | 省市级团体 |

### 3.3 宗教教职人员管理

#### 3.3.1 备案管理

**备案条件：**
1. 拥护中国共产党的领导，拥护社会主义制度
2. 遵守宪法、法律、法规和规章
3. 信教公民，品德良好
4. 具有本宗教认定的宗教教职人员身份
5. 身体健康，能履行宗教教职人员职责

**备案程序：**

```
个人申请 → 宗教团体推荐 → 场所管理组织同意 → 区民族宗教局初审 → 市民族宗教局复核 → 省民族宗教委备案
```

**备案材料：**
1. 宗教教职人员备案申请表
2. 身份证复印件
3. 宗教教职人员资格证书复印件
4. 宗教团体推荐意见
5. 宗教活动场所管理组织同意意见
6. 户籍所在地公安派出所出具的无犯罪记录证明
7. 健康证明

#### 3.3.2 权利与义务

**权利：**
1. 在宗教活动场所内主持宗教活动
2. 从事宗教经典研究和宗教文化学术交流
3. 接受宗教教育培训
4. 参与宗教团体活动
5. 获得合法收入

**义务：**
1. 遵守宪法、法律、法规和规章
2. 维护国家统一、民族团结和社会稳定
3. 抵制非法宗教活动和宗教极端思想
4. 尊重其他宗教和信仰
5. 接受政府宗教事务部门的依法管理

### 3.4 宗教活动管理

#### 3.4.1 大型宗教活动审批

**定义：**
跨省、跨市举行的超过宗教活动场所容纳规模的大型宗教活动，或者在宗教活动场所外举行的大型宗教活动。

**审批条件：**
1. 活动内容符合国家法律、法规、规章和政策
2. 活动方案合理可行
3. 安全措施到位
4. 有明确的组织者和责任人
5. 有足够的工作人员和志愿者
6. 不影响社会秩序和公共利益

**申请材料：**
1. 大型宗教活动申请书
2. 活动方案（包括时间、地点、内容、规模等）
3. 安全工作方案
4. 组织者基本情况
5. 参加人员情况说明
6. 场地使用证明或场地业主同意文件
7. 应急处置预案

**审批流程：**

| 活动规模 | 审批机关 | 办理时限 |
|---------|---------|---------|
| 跨省活动 | 国家宗教事务局 | 30个工作日 |
| 跨市活动 | 省民族宗教委 | 20个工作日 |
| 跨区活动 | 市民族宗教局 | 15个工作日 |
| 市内大型活动 | 市民族宗教局 | 15个工作日 |

**申请时间：** 活动举行前30日提出申请

## 四、广州市少数民族扶持政策

### 4.1 少数民族企业发展扶持

#### 4.1.1 政策依据

- 《广州市少数民族特需商品定点企业认定管理办法》
- 《广东省民族贸易和民族特需商品生产贷款贴息管理暂行办法》
- 《广州市支持民营经济发展若干措施》

#### 4.1.2 民族特需商品定点企业认定

**认定条件：**

1. **基本条件**
   - 在广州市注册登记的企业法人
   - 从事民族特需商品生产或销售满2年
   - 年销售额达到500万元以上
   - 资产负债率不超过70%
   - 近2年无违法违规记录

2. **产品条件**
   - 产品列入《民族特需商品目录》
   - 产品质量达到国家标准或行业标准
   - 有稳定的原材料供应渠道
   - 有完善的质量管理体系

3. **服务条件**
   - 主要服务于少数民族群众
   - 有健全的销售网络
   - 产品价格合理公道

**民族特需商品目录（部分）：**

| 商品类别 | 主要商品 | 认证要求 |
|---------|---------|---------|
| 清真食品 | 清真牛羊肉、清真乳制品、清真糕点 | 清真认证 |
| 民族服装 | 藏袍、蒙古袍、维吾尔族服装等 | 传统工艺 |
| 民族用品 | 酥油茶具、蒙古包材料、藏香等 | 原产地证明 |
| 民族医药 | 藏药、蒙药、维吾尔药等 | 药品批文 |
| 民族文体用品 | 冬不拉、马头琴、唐卡等 | 非遗认定 |

#### 4.1.3 扶持措施

**（一）财政补贴**

| 补贴类型 | 补贴标准 | 申请条件 | 最高限额 |
|---------|---------|---------|---------|
| 技改补贴 | 设备投资20% | 技改投资100万以上 | 100万元 |
| 贷款贴息 | 贷款利息50% | 定点企业 | 50万元/年 |
| 市场开拓 | 展位费70% | 参加指定展会 | 20万元/年 |
| 品牌建设 | 认证费用80% | 获得省级以上品牌 | 10万元 |

**（二）税收优惠**

- 民族贸易企业：增值税先征后返50%
- 民族特需商品生产企业：所得税减免15%
- 西部民族地区来穗投资企业：享受西部大开发税收优惠延续

**（三）金融支持**

1. **贷款授信**
   - 定点企业可申请最高1000万元专项贷款
   - 贷款利率下浮10%
   - 信用贷款额度可达200万元

2. **担保支持**
   - 纳入政府性融资担保体系
   - 担保费率优惠至1%以下
   - 免除反担保要求

3. **保险服务**
   - 企业财产保险费率优惠20%
   - 产品质量保险保费补贴50%
   - 出口信用保险保费补贴80%

#### 4.1.4 申报流程

```
企业申报 → 区民族宗教局初审 → 市民族宗教局复核 → 市商务局认定 → 公示 → 发证
```

**申报时间：** 每年5月1日至6月30日

**申报材料：**
1. 广州市民族特需商品定点企业认定申请表
2. 企业营业执照复印件
3. 近2年财务审计报告
4. 产品生产销售情况报告
5. 产品质量检测报告
6. 服务少数民族群众情况说明
7. 其他证明材料

### 4.2 少数民族文化传承扶持

#### 4.2.1 非物质文化遗产保护

**保护类别：**

1. **少数民族语言文字**
   - 保护濒危少数民族语言
   - 开发少数民族语言学习资料
   - 培养少数民族语言传承人

2. **少数民族传统技艺**
   - 传统手工艺品制作技艺
   - 传统食品加工技艺
   - 传统医药制作技艺

3. **少数民族传统表演艺术**
   - 民族歌舞
   - 民族戏剧
   - 民族曲艺

4. **少数民族民俗活动**
   - 传统节庆
   - 传统礼仪
   - 传统体育竞技

**扶持措施：**

| 项目类型 | 扶持方式 | 扶持金额 | 周期 |
|---------|---------|---------|------|
| 国家级非遗 | 专项资金 | 100万元/年 | 5年 |
| 省级非遗 | 专项资金 | 30万元/年 | 3年 |

#### 4.2.2 民族文化传承基地建设

**基地类型：**

1. **综合型传承基地**
   - 集展示、传习、研究于一体
   - 建筑面积不少于1000平方米
   - 有3个以上传承项目

2. **专业型传承基地**
   - 专注于某一民族或某一技艺
   - 有专业传承人常驻
   - 年培训不少于100人次

3. **展示型传承基地**
   - 以展示、体验为主
   - 有完整的展示体系
   - 年接待参观不少于1万人次

**建设标准：**

| 基地类型 | 场地要求 | 人员要求 | 活动要求 | 补助标准 |
|---------|---------|---------|---------|---------|
| 综合型 | 1000㎡以上 | 专职5人以上 | 年活动50次以上 | 50万元 |
| 专业型 | 300㎡以上 | 专职2人以上 | 年活动30次以上 | 20万元 |
| 展示型 | 500㎡以上 | 专职3人以上 | 年活动40次以上 | 30万元 |

**认定程序：**

1. 申报单位提交申请材料
2. 区民族宗教局推荐
3. 专家评审
4. 市民族宗教局审定
5. 公示7天
6. 授予牌匾

### 7.2 宗教事务审批流程

#### 7.2.1 宗教活动场所登记

**登记流程图：**

```
申请人 → 乡镇街道初审 → 区民族宗教局审核 → 市民族宗教局复核 → 省民族宗教委登记 → 发证
   │           │              │               │                │
   └─提交材料   └─10个工作日    └─20个工作日     └─15个工作日      └─30个工作日
```

**详细步骤：**

**第一步：准备申请材料**
- 宗教活动场所登记申请表
- 场所基本情况说明
- 所在地信教公民情况说明
- 管理组织成员名单及身份证明
- 主持宗教活动人员情况说明
- 资金证明
- 场所建筑物所有权或使用权证明
- 布局平面图

**第二步：乡镇街道初审**
- 受理申请
- 实地核查
- 征求群众意见
- 出具初审意见

**第三步：区民族宗教局审核**
- 审核材料完整性
- 实地考察
- 征求相关部门意见
- 组织专家论证（大型场所）
- 出具审核意见

**第四步：市民族宗教局复核**
- 复核材料合规性
- 必要时实地核查
- 出具复核意见

**第五步：省民族宗教委登记**
- 审查材料
- 必要时实地核查
- 作出登记决定
- 颁发登记证书

**办理时限：** 总计75个工作日

#### 7.2.2 大型宗教活动审批

**审批条件：**
- 活动内容符合法律法规
- 有明确的活动方案
- 安全措施到位
- 有责任单位和责任人

**申请材料：**
1. 大型宗教活动申请书
2. 活动方案（含时间、地点、内容、规模）
3. 安全保卫工作方案
4. 应急处置预案
5. 责任人身份证明
6. 场地使用证明
7. 参加人员情况说明

**审批时限：**
- 申请时间：活动前30日
- 审批时限：15个工作日

**审批机关：**
- 跨省活动：国家宗教事务局
- 跨市活动：省民族宗教委
- 市内大型活动：市民族宗教局

### 7.3 资金申报流程

#### 7.3.1 专项资金申请

**申请流程：**

```
项目入库 → 发布申报通知 → 单位申报 → 区级审核 → 市级评审 → 公示 → 资金下达 → 项目实施 → 绩效评价
```

**各阶段要求：**

| 阶段 | 时间 | 主要工作 | 责任单位 |
|------|------|---------|---------|
| 项目入库 | 常年 | 项目储备，可行性论证 | 申报单位 |
| 发布通知 | 按年度 | 发布申报指南 | 业务主管部门 |
| 单位申报 | 按通知 | 提交申报材料 | 申报单位 |
| 区级审核 | 10个工作日 | 审核推荐 | 区主管部门 |
| 市级评审 | 30个工作日 | 专家评审 | 市主管部门 |
| 公示 | 7天 | 社会公示 | 市主管部门 |
| 资金下达 | 公示后30日 | 下达资金 | 财政部门 |
| 项目实施 | 按计划 | 项目执行 | 项目单位 |
| 绩效评价 | 项目完成后 | 评价考核 | 市主管部门 |

#### 7.3.2 资金使用管理

**使用原则：**
1. 专款专用，不得挪用
2. 按计划执行，不得擅自调整
3. 规范核算，账目清楚
4. 绩效导向，注重实效

**报销流程：**

1. 项目实施产生费用
2. 收集整理票据
3. 填写报销单
4. 项目负责人签字
5. 财务部门审核
6. 领导审批
7. 财务报销

**注意事项：**
- 发票必须是正规发票
- 费用与项目相关
- 按预算执行
- 留存相关凭证

### 7.4 常见问题解答

#### 7.4.1 申报资格类

**问：非广州市户籍的少数民族可以申请创业补贴吗？**
答：可以。在广州市创业的少数民族人员，不限户籍，符合条件均可申请。

**问：宗教活动场所必须登记吗？**
答：是的。根据《宗教事务条例》，宗教活动场所应当办理登记。

**问：个人可以申报民族文化传承项目吗？**
答：可以。传承人个人或团体均可申报，需提供相关资质证明。

#### 7.4.2 材料准备类

**问：申报材料需要盖章吗？**
答：是的。所有申报材料需加盖申报单位公章。

**问：财务审计报告有什么要求？**
答：需由具有资质的会计师事务所出具，报告期为近2年。

**问：传承人证明如何获取？**
答：由区级以上文化主管部门或民族宗教部门认定出具。

#### 7.4.3 审批流程类

**问：申报后多久能知道结果？**
答：一般项目30-60个工作日，大型项目90个工作日。

**问：评审未通过可以再次申报吗？**
答：可以。根据评审意见完善后，下一批次可再次申报。

**问：资金什么时候能到位？**
答：公示结束后30个工作日内下达。
