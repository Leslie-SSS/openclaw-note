## 一、知识产权金融服务概述

### 1.1 知识产权金融定义

知识产权金融是指以知识产权（专利、商标、著作权、地理标志等）为核心资产，通过质押、证券化、保险等方式实现知识产权价值变现的金融服务模式。

### 1.2 知识产权金融服务类型

| 服务类型 | 定义 | 适用主体 | 典型案例 |
|---------|------|---------|---------|
| **质押融资** | 以专利、商标等知识产权作为质押物获取银行贷款 | 中小企业、高新技术企业 | 专利权质押贷款500万元 |
| **专利保险** | 为专利申请、维权、侵权诉讼等提供保险保障 | 创新型企业 | 专利执行险保费补贴50% |
| **商标质押** | 以注册商标专用权作为质押物获取贷款 | 品牌企业 | 商标质押贷款1000万元 |
| **证券化** | 将知识产权未来收益权打包发行资产支持证券 | 优质企业集群 | 知识产权ABS发行5亿元 |
| **融资租赁** | 以知识产权作为融资租赁标的物 | 轻资产企业 | 专利售后回租融资 |
| **信托服务** | 设立知识产权信托实现资产隔离与管理 | 家族企业、科研院所 | 知识产权家族信托 |

#### 2.1.2 政策核心要点

**（一）风险补偿机制**

| 项目 | 内容 |
|-----|------|
| **补偿对象** | 开展知识产权质押贷款的银行机构 |
| **补偿比例** | 单笔贷款本金损失的50%-70% |
| **补偿上限** | 单笔贷款最高补偿500万元 |
| **触发条件** | 借款企业违约，处置质押物后仍有损失 |
| **申报周期** | 每季度申报一次 |

**（二）贴息支持政策**

| 项目 | 内容 |
|-----|------|
| **贴息对象** | 通过知识产权质押获得贷款的企业 |
| **贴息比例** | 实际支付利息的30%-50% |
| **贴息上限** | 单家企业每年最高100万元 |
| **贴息期限** | 贷款期限内，最长3年 |
| **申报时间** | 每年4月、10月集中申报 |

**（三）评估费补贴**

| 评估机构类型 | 补贴比例 | 补贴上限 |
|------------|---------|---------|
| 资产评估机构 | 评估费的50% | 单笔最高5万元 |
| 律师事务所 | 法律意见书费用50% | 单笔最高3万元 |
| 担保机构 | 担保费30% | 单笔最高10万元 |

### 2.2 适用条件与申请要求

#### 2.2.1 企业准入条件

**基本条件（必须全部满足）**：

| 序号 | 条件 | 具体要求 |
|-----|------|---------|
| 1 | 企业注册地 | 在广州市行政区域内注册登记 |
| 2 | 企业性质 | 具有独立法人资格的企业 |
| 3 | 经营年限 | 连续正常经营1年以上 |
| 4 | 信用状况 | 企业及实际控制人无严重不良信用记录 |
| 5 | 知识产权权属 | 拥有自主有效的专利权、商标权等 |
| 6 | 财务状况 | 具有还款能力，资产负债率不超过70% |

**优先支持条件（满足任一即可获得优先审批）**：

| 序号 | 优先条件 | 支持力度 |
|-----|---------|---------|
| 1 | 高新技术企业 | 贴息比例提高10% |
| 2 | 专精特新企业 | 贷款额度上限提高20% |
| 3 | 科技型中小企业 | 评估费补贴提高至60% |
| 4 | 知识产权示范企业 | 风险补偿比例提高至70% |
| 5 | 战略性新兴产业 | 审批绿色通道，7个工作日内完成 |

#### 2.2.2 知识产权准入条件

**专利权质押条件**：

| 项目 | 要求 |
|-----|------|
| **专利类型** | 发明专利、实用新型专利、外观设计专利 |
| **专利状态** | 已获得授权，处于有效期内 |
| **剩余年限** | 发明专利剩余保护期不少于5年，实用新型不少于3年 |
| **法律状态** | 无权属纠纷，无质押，无许可（或已取得许可方同意） |
| **技术价值** | 具有市场应用价值或产业化前景 |
| **评估价值** | 单件专利评估价值不低于50万元 |

**商标权质押条件**：

| 项目 | 要求 |
|-----|------|
| **商标类型** | 商品商标、服务商标、集体商标、证明商标 |
| **商标状态** | 已注册，处于有效期内 |
| **剩余年限** | 注册商标剩余有效期不少于3年 |
| **使用情况** | 商标已实际使用满2年，具有市场知名度 |
| **法律状态** | 无权属纠纷，无质押，无侵权纠纷 |
| **评估价值** | 单件商标评估价值不低于100万元 |

#### 2.4.2 典型银行产品详解

```

• 纯知识产权质押，无需固定资产抵押

• 还款方式：按月付息，到期还本/分期还本

├── 银行进行初步审核

│   └── 确定质押率与贷款额度
└── 风险评审
    ├── 银行内部审批
    ├── 上报授信审批部门
    └── 获得贷款批复

第4步：签订合同（3-5个工作日）
├── 签订借款合同
├── 签订质押合同
├── 办理质押登记（国家知识产权局）
│   ├── 专利权质押登记：5个工作日
│   ├── 商标权质押登记：3个工作日
│   └── 著作权质押登记：5个工作日
└── 开立监管账户

第5步：贷款发放（1-3个工作日）
├── 银行放款至监管账户
├── 资金用途监管
└── 开始计息

第6步：贷后管理（持续）
├── 定期提交经营报告
├── 知识产权状态监控
├── 按时还本付息
└── 贷款结清后解除质押
```

## 三、广东省专利保险政策

### 3.1 政策背景与意义

#### 3.1.1 专利保险概念

专利保险是指以专利及相关利益为保险标的的保险产品，保障专利权人在专利申请、维权、侵权诉讼等环节可能面临的经济损失风险。

#### 3.1.2 广东省专利保险发展历程

| 时间 | 里程碑事件 |
|-----|----------|
| 2012年 | 广东省启动专利保险试点（佛山、东莞） |
| 2015年 | 《广东省专利保险试点工作方案》出台 |
| 2017年 | 试点扩大至全省21个地市 |
| 2019年 | 《广东省专利保险补贴办法》发布 |
| 2021年 | 专利保险纳入广东省知识产权保护体系 |
| 2023年 | 《广东省知识产权保险扶持政策》全面实施 |
| 2024年 | 专利保险覆盖企业突破5000家 |

### 3.2 专利保险产品类型

#### 3.2.1 专利保险产品矩阵

| 产品类型 | 保障内容 | 保费范围 | 保险金额 |
|---------|---------|---------|---------|
| **专利申请险** | 专利申请被驳回的申请费用损失 | 500-2000元/件 | 最高5万元/件 |
| **专利执行险** | 专利维权调查费、诉讼费、律师费 | 3000-10000元/年 | 最高50万元/年 |
| **专利侵权责任险** | 被诉侵权赔偿金、诉讼费 | 5000-20000元/年 | 最高100万元/年 |
| **专利许可信用险** | 被许可方违约损失 | 保费的2-5% | 最高500万元 |
| **专利质押贷款保证险** | 借款人违约时银行损失补偿 | 贷款金额的1-2% | 与贷款金额匹配 |
| **海外专利维权险** | 海外专利诉讼费用 | 10000-50000元/年 | 最高200万元/年 |

#### 3.2.2 主要保险产品详解

**（一）专利执行险**

```
【产品定义】
保障专利权人为维护自身专利权，
对侵权方进行调查、取证、诉讼所产生的费用。

【保障范围】
• 调查费用：侵权调查、取证、公证费用
• 诉讼费用：法院诉讼费、财产保全费
• 律师费用：委托律师代理费用
• 鉴定费用：技术鉴定、专利评估费用
• 差旅费用：异地维权差旅费

【保险条款】
• 保险期限：1年（可续保）
• 保费标准：3000-10000元/年
• 保险金额：20万元-50万元
• 免赔额：5000元或损失金额的10%
• 理赔比例：损失金额的80%-90%

【典型案例】
某电子企业拥有发明专利15件，投保专利执行险，
年保费8000元，保险金额40万元。
发现某公司侵犯其专利权，产生维权费用35万元，
保险公司理赔：35万×90%-5000元=31万元
```

**（二）专利侵权责任险**

```
【产品定义】
保障被保险人因非故意侵犯他人专利权，
被判决赔偿损失及承担相关诉讼费用。

【保障范围】
• 侵权赔偿金：法院判决的损害赔偿
• 诉讼费用：原告/被告的合理诉讼费用
• 律师费用：委托律师辩护费用
• 和解费用：经保险公司同意的和解金额

【保险条款】
• 保险期限：1年（可续保）
• 保费标准：5000-20000元/年
• 保险金额：50万元-100万元
• 免赔额：10000元或损失金额的10%
• 理赔比例：损失金额的80%-90%

【典型案例】
某制造企业生产新产品，投保专利侵权责任险，
年保费15000元，保险金额80万元。
被诉侵犯他人专利权，法院判决赔偿60万元，
保险公司理赔：60万×90%-10000元=53万元
```

**（三）海外专利维权险**

```
【产品定义】
保障企业在海外遭遇专利诉讼、337调查等
知识产权纠纷时的维权费用。

【保障范围】
• 海外律师费：美国、欧盟等地区律师费用
• 诉讼费用：海外法院诉讼费用
• 337调查费用：美国ITC调查应对费用
• 和解费用：经同意的和解金额
• 翻译公证费：法律文书翻译、公证费用

【保险条款】
• 保险期限：1年
• 保费标准：10000-50000元/年
• 保险金额：100万元-200万元
• 保障区域：全球（指定国家/地区）
• 特别约定：需事先经保险公司同意方可启动程序

【典型案例】
某电子企业产品出口美国，投保海外专利维权险，
年保费30000元，保险金额150万元。
产品遭遇美国337调查，应对费用120万元，
保险公司理赔：120万×85%=102万元
```

## 五、番禺区知识产权证券化政策

### 5.1 知识产权证券化概述

#### 5.1.1 基本概念

知识产权证券化（IP Securitization）是指将缺乏流动性但能够产生可预见现金流的知识产权资产，通过重新组合打包，以其产生的现金流为支撑在金融市场发行可流通证券的融资方式。

#### 5.1.2 知识产权证券化运作模式

```
知识产权证券化交易结构

知识产权证券化参与主体关系图

┌─────────────┐
│  原始权益人  │ (拥有知识产权的企业)
│ (借款企业)   │
└──────┬──────┘
       │ 转让知识产权收益权
       ↓
┌─────────────┐
│  发起机构    │ (券商/融资租赁公司)
│ (计划管理人) │
└──────┬──────┘
       │ 设立专项计划
       ↓
┌─────────────┐         ┌─────────────┐
│  SPV专项计划  │←─担保───│  增信机构    │
│ (资产支持专项)│         │ (担保公司)   │
└──────┬──────┘         └─────────────┘
       │ 发行证券
       ↓
┌─────────────┐         ┌─────────────┐
│  资本市场    │←─投资───│  投资者      │
│ (交易所)     │         │ (机构投资者) │
└─────────────┘         └─────────────┘

现金流：
投资者购买证券 → 资金流入SPV → 
支付给原始权益人 → 知识产权产生收益 → 
支付本息给投资者
```

## 八、常见问题解答

### 8.1 知识产权质押融资常见问题

**Q1：哪些知识产权可以用于质押融资？**

| 知识产权类型 | 是否可用于质押 | 备注 |
|------------|--------------|------|
| 发明专利 | ✅可以 | 最受银行认可 |
| 实用新型专利 | ✅可以 | 需评估技术价值 |
| 外观设计专利 | ✅可以 | 价值相对较低 |
| 注册商标 | ✅可以 | 需具有品牌价值 |
| 软件著作权 | ✅可以 | 部分银行接受 |

**Q2：贷款额度如何确定？**

贷款额度 = 知识产权评估价值 × 质押率

影响质押率的因素：知识产权类型、质量、剩余保护期限、市场应用情况、企业综合实力。

**Q3：质押期间知识产权还可以使用吗？**

| 使用方式 | 是否允许 | 说明 |
|---------|---------|------|
| 自己实施 | ✅允许 | 不影响质押 |
| 许可他人使用 | ⚠️需征得质权人同意 | 可能影响质押权 |
| 转让 | ❌不允许 | 质押期间不得转让 |

### 8.2 专利保险常见问题

**Q4：专利保险保费是多少？政府有补贴吗？**

| 保险类型 | 保费范围 | 政府补贴比例 |
