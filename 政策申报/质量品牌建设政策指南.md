# 第一章 广州市质量奖申报指南

## 一、广州市市长质量奖

### 1.1 政策概述

广州市市长质量奖是广州市人民政府设立的最高质量荣誉奖，旨在表彰在质量管理、经营绩效、创新发展等方面取得卓越成就的组织和个人。该奖项每两年评选一届，是广州市质量领域的最高荣誉。

**设立依据：**
- 《广州市市长质量奖管理办法》（穗府〔2018〕14号）
- 《广州市质量强市工作领导小组办公室关于印发广州市市长质量奖评审实施细则的通知》

**奖项设置：**
- 组织奖：不超过5个，每家奖励100万元
- 个人奖：不超过3人，每人奖励10万元

### 1.2 申报条件

#### 1.2.1 基本条件（必须同时满足）

**组织申报条件：**

1. **注册与经营要求**
   - 在广州市行政区域内登记注册5年以上
   - 具有独立法人资格
   - 从事合法生产经营活动

2. **质量管理体系要求**
   - 建立并实施卓越绩效管理模式
   - 通过质量管理体系认证（ISO 9001或等同标准）
   - 质量管理制度健全，运行有效

3. **经营业绩要求**
   - 主要经济、技术和质量指标在全市同行业中处于领先水平
   - 近3年主营业务收入、利润总额、纳税额等主要经济指标持续增长
   - 具有良好的发展前景

4. **创新能力要求**
   - 具有较强的自主创新能力
   - 拥有自主知识产权或核心技术
   - 研发投入占主营业务收入比例不低于行业平均水平

5. **社会信誉要求**
   - 诚信经营，具有良好的社会信誉
   - 近3年无重大质量、安全、环保等事故
   - 近3年无重大违法违纪行为
   - 产品或服务符合国家产业、环保、质量安全等政策

**个人申报条件：**

1. 在本市从事质量相关工作8年以上
2. 在质量管理领域有突出贡献和创新成果
3. 积极推广应用先进质量管理方法
4. 具有良好的职业道德和社会声誉

#### 1.2.2 优先条件（满足以下条件可优先推荐）

1. 获得过区级质量奖或省级以上质量荣誉
2. 参与国家标准、行业标准或国际标准制定
3. 拥有国家级、省级企业技术中心或工程研究中心
4. 获得国家级、省级科技进步奖
5. 产品或服务质量指标达到国际先进水平
6. 在抗击重大灾害、疫情防控等方面作出突出贡献

#### 1.2.3 否决条件（存在以下情况不得申报）

1. 不符合国家产业、环保、质量、安全等政策
2. 近3年内有重大质量、安全、环保等事故
3. 近3年内有重大违法违纪行为
4. 近3年内产品或服务在国家、省、市质量监督抽查中出现不合格
5. 存在拖欠劳动者报酬、社会保险费等情况
6. 其他违反法律法规的行为

### 1.3 评审标准

广州市市长质量奖评审采用《卓越绩效评价准则》（GB/T 19580）国家标准，总分为1000分。

#### 1.3.1 评审内容与分值

| 评审项目 | 分值 | 主要内容 |
|---------|------|---------|
| 领导 | 120分 | 高层领导的作用、组织治理、社会责任 |
| 战略 | 90分 | 战略制定、战略部署 |
| 顾客与市场 | 90分 | 顾客和市场的了解、顾客关系与顾客满意 |
| 资源 | 130分 | 人力资源、财务资源、信息和知识资源、技术资源、基础设施、相关方关系 |
| 过程管理 | 110分 | 价值创造过程、支持过程 |
| 测量分析与改进 | 100分 | 测量与分析、改进 |
| 经营结果 | 360分 | 顾客与市场的结果、财务结果、资源结果、过程有效性结果、组织治理和社会责任结果 |

#### 1.3.2 评分规则

1. **方法（Approach）- 50分**
   - 方法的适宜性、有效性
   - 方法的系统性、可重复性
   - 方法的创新性

2. **展开（Deployment）- 25分**
   - 方法在组织内部的实施程度
   - 方法在各个部门、层级的贯彻情况

3. **学习（Learning）- 15分**
   - 方法是否经过评价和改进
   - 是否有持续改进的机制

4. **整合（Integration）- 10分**
   - 方法是否与其他过程协调一致
   - 是否形成有机整体

### 1.4 申报流程

#### 第一阶段：申报准备（1-2月）

1. **自我评估**
   - 对照《卓越绩效评价准则》进行自我评价
   - 识别组织优势和改进机会
   - 形成自评报告

2. **材料准备**
   - 收集近3年的经营数据
   - 整理质量管理体系文件
   - 准备相关证明材料

#### 第二阶段：提交申报（3-4月）

1. **网上申报**
   - 登录广州市市场监督管理局官网
   - 进入市长质量奖申报系统
   - 填写申报表格

2. **材料提交**
   - 申报表（加盖公章）
   - 组织概述（3000字以内）
   - 自评报告（5万字以内）
   - 证实性材料

3. **材料审核**
   - 区市场监督管理局初审
   - 市市场监督管理局形式审查

#### 第三阶段：材料评审（5-6月）

1. **专家组评审**
   - 由7-9名专家组成评审组
   - 独立评审申报材料
   - 形成书面评审意见

2. **进入现场评审名单确定**
   - 综合材料评审结果
   - 确定现场评审组织名单（原则上不超过15家）

#### 第四阶段：现场评审（7-8月）

1. **现场评审实施**
   - 专家组进驻企业（2-3天）
   - 高层领导座谈
   - 各部门访谈
   - 现场查看

2. **评审内容**
   - 核实申报材料真实性
   - 深入了解经营实践
   - 评估卓越绩效模式实施情况

#### 第五阶段：审议公示（9-10月）

1. **评审委员会审议**
   - 听取评审组汇报
   - 审议拟获奖名单
   - 投票表决

2. **社会公示**
   - 公示期7天
   - 接受社会监督
   - 处理异议投诉

#### 第六阶段：批准表彰（11-12月）

1. **报批程序**
   - 报市政府批准
   - 发文公布获奖名单

2. **表彰奖励**
   - 召开表彰大会
   - 颁发奖牌、证书
   - 发放奖励资金

### 1.5 奖励标准与资金使用

#### 1.5.1 奖励金额

- **组织奖**：每家奖励100万元
- **个人奖**：每人奖励10万元

#### 1.5.2 资金使用范围

奖励资金应用于以下方面：

1. **质量提升**
   - 质量管理体系建设与改进
   - 质量人才培养
   - 质量检测设备购置

2. **技术创新**
   - 技术研发投入
   - 创新平台建设
   - 标准制修订

3. **品牌建设**
   - 品牌推广宣传
   - 品牌保护
   - 市场开拓

#### 1.5.3 资金监管

- 专款专用，独立核算
- 接受财政、审计部门监督
- 使用情况纳入绩效评价

## 三、广东省质量奖

### 3.1 政策概述

广东省质量奖是广东省人民政府设立的最高质量荣誉，旨在表彰在质量管理和经营绩效方面取得卓越成就的组织和个人。

**奖项设置：**
- 组织奖：不超过10个，每家奖励200万元
- 个人奖：不超过5人，每人奖励20万元

### 3.2 申报条件

**组织申报条件：**

1. 在广东省登记注册5年以上
2. 具有独立法人资格
3. 建立并实施卓越绩效管理模式3年以上

# 第二章 广东省名牌产品认定指南

## 一、广东省名牌产品认定

### 1.1 政策概述

广东省名牌产品认定是广东省品牌建设的重要组成部分，旨在培育和扶持一批具有较强市场竞争力和影响力的产品品牌。

**政策依据：**
- 《广东省名牌产品管理办法》
- 《广东省质量强省工作领导小组办公室关于开展广东省名牌产品认定工作的通知》

**认定特点：**
- 有效期3年，到期需复评
- 分为工业产品和农业产品两大类
- 采用自愿申报、科学评价、动态管理

### 1.2 申报条件

#### 1.2.1 基本条件

1. **企业资质要求**
   - 在广东省登记注册
   - 具有独立法人资格
   - 生产经营3年以上
   - 年销售收入达到规定标准（工业产品一般不低于5000万元）

2. **产品质量要求**
   - 产品符合国家标准、行业标准或国际标准
   - 近3年省级以上质量监督抽查全部合格
   - 产品质量水平达到国内先进水平

3. **品牌要求**
   - 具有自主注册商标
   - 商标注册3年以上
   - 商标与产品名称一致

4. **市场地位要求**
   - 产品市场占有率居省内同行业前列
   - 具有较高的知名度和美誉度
   - 顾客满意度较高

5. **创新能力要求**
   - 具有一定的研发能力
   - 拥有自主知识产权
   - 积极采用先进技术标准

#### 1.2.2 否决条件

存在以下情况之一不得申报：

1. 不符合国家产业政策
2. 近3年内有重大质量事故
3. 近3年内产品在国家、省级监督抽查不合格
4. 近3年内有重大违法违规行为
5. 存在知识产权纠纷
6. 环保不达标

### 1.3 评价标准

广东省名牌产品评价采用综合评分法，总分1000分。

#### 1.3.1 评价指标体系

| 评价指标 | 分值 | 主要内容 |
|---------|------|---------|
| 市场评价 | 400分 | 市场占有率、销售额、出口额、品牌知名度 |
| 质量评价 | 300分 | 产品标准水平、质量认证、质量水平、质量保证能力 |
| 效益评价 | 150分 | 利润总额、税收贡献、劳动生产率 |
| 发展评价 | 150分 | 技术创新能力、研发投入、专利数量、发展前景 |

#### 1.3.2 市场评价指标（400分）

1. **市场占有率（150分）**
   - 省内市场占有率
   - 国内市场占有率
   - 国际市场占有率

2. **销售额（100分）**
   - 年销售收入规模
   - 销售收入增长率
   - 人均销售额

3. **出口额（50分）**
   - 出口创汇额
   - 出口增长率
   - 出口市场分布

4. **品牌知名度（100分）**
   - 品牌认知度调查
   - 媒体报道情况
   - 广告宣传投入

#### 1.3.3 质量评价指标（300分）

1. **产品标准水平（80分）**
   - 执行标准级别
   - 是否参与标准制定
   - 标准技术水平

## 二、广东优质认证

### 2.1 政策概述

广东优质认证是对达到国际先进质量水平的广东产品和服务的高端品质认证，是广东省质量品牌建设的重要标志。

**认证特点：**
- 采用国际先进标准
- 严格的认证程序
- 持续的监督管理
- 高端品牌形象

### 2.2 认证范围

广东优质认证范围包括：

1. **消费品**
   - 电子信息产品
   - 家用电器
   - 纺织服装
   - 食品饮料
   - 日用百货

2. **工业品**
   - 机械装备
   - 建筑材料
   - 化工产品
   - 金属材料

3. **服务**
   - 物流服务
   - 金融服务
   - 信息技术服务
   - 旅游服务

### 2.3 认证条件

#### 基本条件

1. 在广东省登记注册的企业生产的产品或提供的服务
2. 建立健全的质量管理体系
3. 产品或服务质量达到国际先进标准
4. 近3年无重大质量事故
5. 近3年无违法违规行为

#### 质量要求

1. **标准要求**
   - 产品执行标准达到国际先进水平
   - 关键质量指标优于国家标准
   - 符合国际市场准入要求

2. **认证要求**
   - 通过质量管理体系认证
   - 主要产品通过强制性认证
   - 鼓励通过国际认证

3. **检测要求**
   - 产品经国家认可实验室检测
   - 检测项目覆盖关键质量指标
   - 检测结果达到认证要求

### 2.4 认证程序

1. **申请受理**
   - 提交认证申请书
   - 提供相关证明材料
   - 认证机构受理审查

2. **文件审查**
   - 审查质量管理体系文件
   - 审查产品标准文件
   - 审查检测报告

3. **现场审核**
   - 审核组进驻企业
   - 核实生产条件
   - 验证质量保证能力

4. **产品检测**
   - 抽样送检
   - 全项目检测
   - 出具检测报告

5. **认证决定**
   - 认证委员会审议
   - 做出认证决定
   - 颁发认证证书

6. **监督审核**
   - 年度监督审核
   - 证书有效期3年
   - 到期复评

### 2.5 认证标志使用

**标志规格：**
- 标准版：直径30mm
- 小型版：直径15mm
- 大型版：直径50mm

**使用范围：**
- 获证产品包装
- 产品说明书
- 宣传推广材料
- 获证企业招牌

**使用要求：**
- 按照认证机构规定的式样使用
- 不得擅自改变标志颜色、比例
- 认证撤销后立即停止使用

## 三、广东老字号认定

### 3.1 政策概述

广东老字号认定旨在保护和发展具有广东传统文化特色的老字号品牌，传承优秀传统文化，促进老字号振兴发展。

**认定主体：**

**认定周期：** 原则上每年认定一次

### 3.2 认定条件

1. **品牌历史要求**
   - 品牌创立时间达到30年（含）以上
   - 有历史传承证明材料
   - 具有广东地域文化特色

2. **企业资质要求**
   - 在广东省登记注册
   - 具有独立法人资格

3. **专家评审**
   - 省商务厅组织专家评审
   - 现场核查
   - 形成评审意见

4. **社会公示**
   - 公示拟认定名单
   - 公示期7天
   - 处理异议

5. **认定公布**
   - 省商务厅审定
   - 发布认定公告
   - 颁发牌匾证书

### 3.4 认定材料

4. 品牌历史传承证明材料
   - 历史文献记载
   - 老照片、老物件
   - 口述历史资料

6. 企业发展历史资料
7. 获得的荣誉证书
8. 媒体报道资料
9. 其他证明材料

### 3.5 老字号保护与扶持

   - 域名保护

# 第五章 粤港澳大湾区品牌合作政策

## 一、粤港澳大湾区品牌发展概况

   - 湾区品牌国际竞争力全球领先

3. **企业层面**

## 二、粤港澳大湾区品牌认定

### 2.1 湾区品牌认定

**认定主体：**
- 粤港澳大湾区标准化研究中心
- 粤港澳三地质量品牌组织

**认定范围：**
- 产品品牌
- 服务品牌
- 企业品牌
- 区域品牌

**认定条件：**

1. **基本条件**
   - 企业注册地在粤港澳大湾区
   - 具有独立法人资格
   - 经营状况良好
   - 品牌在湾区市场具有较高知名度

2. **质量条件**
   - 产品或服务质量达到国际先进水平
   - 通过质量管理体系认证
   - 近3年无重大质量事故

3. **市场条件**
   - 品牌在湾区市场占有率较高
   - 消费者认可度高
   - 具有良好的口碑

4. **创新条件**
   - 具有较强的创新能力
   - 拥有自主知识产权
   - 技术水平领先

### 2.2 湾区认证

**认证特点：**
- 一次认证，三地认可
- 采用国际先进标准
- 统一认证标志

**认证范围：**
- 消费品
- 食品农产品
- 工业品
- 服务

1. **申请受理**
   - 向湾区认证机构提交申请
   - 提供相关证明材料

2. **文件审查**
   - 审查质量管理体系文件
   - 审查产品标准文件

3. **现场审核**
   - 审核组进驻企业
   - 核实生产条件和质量保证能力

4. **产品检测**
   - 抽样送检
   - 按湾区标准检测

5. **认证决定**
   - 认证委员会审议
   - 做出认证决定

6. **颁发证书**
   - 颁发湾区认证证书
   - 准许使用湾区认证标志

# 总结

本指南系统梳理了质量品牌建设相关政策，涵盖广州市质量奖、广东省名牌产品、品牌建设扶持、番禺区品牌专项和粤港澳大湾区品牌合作等五大板块。企业可根据自身情况选择适合的政策进行申报，建议重点关注以下几个方面：

## 申报要点总结

1. **提前准备**：质量奖、名牌产品等认定需要提前1-2年准备，建立完善的质量管理体系和品牌管理体系。

2. **条件匹配**：仔细研究申报条件，确保符合基本条件，争取满足优先条件。

3. **材料完整**：申报材料要完整、真实、规范，突出亮点和特色。

4. **持续改进**：品牌建设是长期工作，需要持续投入和改进。

5. **政策跟踪**：及时关注政策变化，把握申报时机。

## 发展建议

1. **中小企业**：建议先申报区级质量奖、区级品牌扶持，逐步提升到市级、省级。

2. **龙头企业**：可以同时申报多个项目，包括质量奖、名牌产品、品牌扶持等。

3. **出口企业**：重点关注国际品牌打造、湾区品牌合作等政策。

4. **老字号企业**：积极申报广东老字号、中华老字号认定，用好扶持政策。

5. **创新型企业**：关注标准制定、专利资助、知识产权示范等政策。
