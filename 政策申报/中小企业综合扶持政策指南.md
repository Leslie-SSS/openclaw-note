# 第一部分：广州市中小企业发展政策

## 一、专精特新企业扶持政策

### （一）政策背景与目标

广州市深入贯彻习近平总书记关于"培育一批'专精特新'中小企业"的重要指示精神，全面落实国家和广东省关于促进中小企业发展的决策部署，以提升中小企业创新能力和专业化水平为核心，构建完善的梯度培育体系，推动中小企业走"专精特新"发展道路。

专精特新企业是指具有"专业化、精细化、特色化、新颖化"特征的中小企业，是推动产业基础高级化、产业链现代化的生力军。广州市通过财政资金引导、政策协同支持、服务资源汇聚等方式，培育一批在细分领域掌握关键核心技术、市场占有率高的优质企业。

### （二）培育梯度体系

广州市建立"创新型中小企业—专精特新中小企业—专精特新'小巨人'企业"三级梯度培育体系：

**1. 创新型中小企业（基础层）**

申报条件：
- 在广州市登记注册满两年，具有独立法人资格
- 符合《中小企业划型标准规定》的中小型企业标准
- 企业主导产品或服务符合广州市产业发展方向
- 上年度研发费用占营业收入比重不低于3%
- 拥有有效知识产权（发明专利、实用新型专利、软件著作权等）不少于1项
- 上年度资产负债率不高于70%

**2. 专精特新中小企业（骨干层）**

申报条件：
- 原则上已认定为创新型中小企业满6个月
- 从事特定细分市场时间达到2年以上
- 上年度研发费用总额不低于100万元，且占营业收入总额比重不低于5%
- 上年度营业收入总额在1000万元以上，或近两年新增股权融资总额（合格机构投资者的实缴额）达到2000万元以上
- 拥有与主导产品相关的有效知识产权（发明专利1项以上或实用新型专利/软件著作权等5项以上）
- 主导产品在细分市场占有率达到10%以上或近两年平均增长率不低于5%

**3. 专精特新"小巨人"企业（领军层）**

申报条件：
- 原则上已认定为专精特新中小企业满12个月
- 从事特定细分市场时间达到3年以上
- 上年度研发费用总额不低于300万元，且占营业收入总额比重不低于5%
- 上年度营业收入总额在2000万元以上，或近两年新增股权融资总额（合格机构投资者的实缴额）达到4000万元以上
- 拥有与主导产品相关的有效知识产权（发明专利2项以上）并实际应用
- 主导产品在细分市场占有率达到15%以上或近两年平均增长率不低于8%
- 自建或与高校、科研机构联合建立研发机构

### （三）奖励标准

**1. 认定奖励**

| 认定级别 | 广州市奖励 | 区级配套奖励 | 合计最高 |
|---------|----------|------------|---------|
| 创新型中小企业 | 5万元 | 5-10万元 | 15万元 |
| 专精特新中小企业 | 20万元 | 10-30万元 | 50万元 |
| 专精特新"小巨人"企业 | 100万元 | 50-100万元 | 200万元 |

**2. 研发投入补助**

- 对专精特新企业上年度研发费用，按实际支出的20%给予补助
- 单家企业年度补助上限：专精特新中小企业50万元，专精特新"小巨人"企业100万元

**3. 技术改造奖励**

- 对专精特新企业实施的技术改造项目，按设备投资额的10%给予奖励
- 单个项目最高奖励500万元

**4. 贷款贴息**

- 对专精特新企业获得银行贷款，按同期LPR的50%给予贴息
- 单家企业年度贴息上限100万元，期限最长2年

### （四）申报流程

**第一步：企业自评**
- 登录工信部优质中小企业梯度培育平台（zjtx.miit.gov.cn）
- 完成企业信息填报和自评打分
- 对照评分标准确认是否符合申报条件

**第二步：材料准备**
- 企业营业执照复印件
- 近三年财务审计报告
- 研发费用专项审计报告
- 知识产权证书复印件
- 产品市场占有率证明材料
- 其他佐证材料（荣誉证书、合同等）

**第三步：线上申报**
- 通过梯度培育平台提交申报材料
- 同时登录广州市政务服务网进行备案
- 上传相关证明文件

**第四步：区级审核**
- 所属区工信部门进行形式审查
- 实地核查企业生产经营情况
- 推荐至市工信局

**第五步：市级评审**
- 市工信局组织专家评审
- 公示拟认定名单（5个工作日）
- 发布正式认定通知

**第六步：资金拨付**
- 认定后3个月内拨付奖励资金
- 资金直接划入企业账户

## 二、小巨人企业专项扶持

### （一）政策概述

专精特新"小巨人"企业是专精特新企业中的佼佼者，是专注于细分市场、创新能力强、市场占有率高、掌握关键核心技术、质量效益优的排头兵企业。广州市对"小巨人"企业实行"一企一策"精准扶持，在资金、土地、人才、金融等方面给予全方位支持。

### （二）申报条件详解

**基本条件：**
1. 在广州市注册登记、连续经营3年以上
2. 具有独立法人资格，符合《中小企业划型标准规定》
3. 未被列入经营异常名录或严重失信主体名单
4. 提供的产品（服务）不属于国家禁止、限制或淘汰类
5. 近三年未发生重大安全、质量、环境污染事故

**专项条件：**

| 指标类别 | 具体要求 |
|---------|---------|
| 市场化指标 | 上年度营业收入≥1亿元；或近两年新增股权融资≥4000万元 |
| 细分市场 | 从事特定细分市场≥3年，主导产品收入占比≥70% |
| 创新能力 | 研发费用占比≥5%，研发人员占比≥10% |
| 知识产权 | 发明专利≥2项或Ⅰ类知识产权≥3项 |
| 数字化水平 | 达到两化融合管理体系贯标或数字化转型成熟度2级以上 |
| 产品质量 | 主导产品通过国际标准认证或获得省级以上质量荣誉 |

**分类条件（满足其一即可）：**

1. 专业化：主导产品在全国细分市场占有率达到15%以上
2. 精细化：取得相关管理体系认证，或产品生产执行标准达到国际先进水平
3. 特色化：产品或服务具有独特性、独有性、独家生产的特点
4. 新颖化：拥有自主知识产权，科技成果转化能力强

### （三）奖励政策

**1. 认定奖励**
- 国家级"小巨人"：一次性奖励100万元
- 省级"专精特新"中小企业：一次性奖励50万元

**2. 成长奖励**
- "小巨人"企业年营业收入首次突破5000万元、1亿元、3亿元、5亿元、10亿元的，分别给予10万元、20万元、30万元、50万元、100万元奖励

**3. 上市奖励**
- 在境内证券交易所首发上市的，给予最高300万元奖励
- 在境外主要证券交易所上市的，给予最高200万元奖励
- 在新三板挂牌的，给予50万元奖励

**4. 质量品牌奖励**
- 获得中国质量奖的，给予200万元奖励
- 获得广东省政府质量奖的，给予100万元奖励
- 获得广州市市长质量奖的，给予50万元奖励

**5. 标准制定奖励**
- 主导制定国际标准的，每项奖励50万元
- 主导制定国家标准的，每项奖励30万元
- 主导制定行业标准的，每项奖励15万元

### （四）配套支持措施

**1. 用地保障**
- 纳入市重点建设项目清单的，优先安排用地指标
- 支持符合条件的"小巨人"企业联合申请产业用地
- 工业用地出让底价可按所在土地等别对应工业用地最低价标准的70%执行

**2. 人才引进**
- 核心技术人才可申请广州市人才公寓
- 符合条件的高层次人才，可享受最高100万元住房补贴
- 企业引进的急需紧缺人才，可申请人才绿卡

**3. 金融支持**
- 纳入广州市中小微企业信贷风险补偿资金池支持范围
- 优先推荐进入"金种子"企业库，对接多层次资本市场
- 支持发行中小企业集合债、集合票据

**4. 市场拓展**
- 优先推荐参与政府采购和政府投资项目
- 支持参加国内外知名展会，给予展位费50%补助
- 支持建立海外营销网络，给予最高50万元补助

### （五）申报注意事项

1. **时间节点**：每年3-4月集中申报，具体时间以工信部公告为准
2. **材料真实性**：所有申报材料须真实有效，弄虚作假将被取消认定资格并追回奖励资金
3. **持续合规**：认定后需每年定期报送企业发展情况，存在重大违法违规行为将被撤销认定
4. **动态管理**：专精特新"小巨人"企业有效期3年，到期需申请复核

# 第二部分：广东省中小企业扶持政策

## 一、省级专项资金政策

### （一）广东省中小企业发展专项资金

**1. 支持方向**

广东省中小企业发展专项资金重点支持以下领域：

- **技术创新**：支持中小企业开展关键核心技术攻关、科技成果转化
- **数字化转型**：支持中小企业上云用云、智能化改造
- **融资促进**：支持中小企业获得银行贷款、股权融资
- **市场开拓**：支持中小企业参加展会、拓展市场
- **公共服务**：支持中小企业公共服务平台建设
- **人才培训**：支持中小企业经营管理和技术人才培训

**2. 申报条件**

| 支持类别 | 基本条件 | 专项条件 |
|---------|---------|---------|
| 技术创新 | 在广东省注册的中小企业 | 研发费用占比≥3%，拥有自主知识产权 |
| 数字化转型 | 上年度营业收入≥500万元 | 有明确的数字化转型方案和投资计划 |
| 融资促进 | 正常经营满2年 | 已获得银行贷款或股权投资 |
| 市场开拓 | 有出口实绩或参展计划 | 产品具有竞争力，市场前景良好 |
| 公共服务 | 服务中小企业的专业机构 | 服务企业数量≥100家/年 |

**3. 资助标准**

- **技术创新项目**：按项目总投资的30%给予补助，单个项目最高500万元
- **数字化转型项目**：按软硬件投资额的20%给予补助，最高200万元
- **贷款贴息**：按同期LPR的30%给予贴息，年度最高50万元
- **参展补助**：按展位费的50%给予补助，单次最高10万元
- **培训补贴**：按培训费用的50%给予补贴，年度最高20万元

### （二）广东省专精特新中小企业培育资金

**1. 政策内容**

省财政每年安排专项资金，支持培育国家级专精特新"小巨人"企业和省级专精特新中小企业。

**2. 奖励标准**

| 级别 | 省级奖励 | 备注 |
|-----|---------|------|
| 国家级专精特新"小巨人" | 100万元 | 首次认定给予奖励 |
| 省级专精特新中小企业 | 20万元 | 首次认定给予奖励 |
| 复核通过 | 10万元 | 有效期满复核通过 |

**3. 申报流程**

（1）企业申报：登录广东省政务服务网，在线填写申报表并上传相关材料
（2）地市推荐：各地市工信部门审核推荐，于每年5月底前报送省工信厅
（3）省级评审：省工信厅组织专家评审和现场核查
（4）公示发布：在省工信厅网站公示5个工作日后发布认定名单

**4. 申报材料清单**

- 广东省专精特新中小企业认定申请表
- 企业营业执照复印件
- 近三年财务审计报告
- 研发费用明细及专项审计报告
- 知识产权证书复印件
- 主导产品市场占有率证明
- 企业信用报告
- 其他相关证明材料

### （三）广东省民营经济及中小企业发展专项资金

**1. 支持对象**

- 在广东省内注册的民营企业、中小企业
- 为中小企业提供公共服务的机构
- 中小企业产业园、创业基地运营单位

**2. 支持方式**

- **无偿资助**：对符合条件的特定项目给予直接资金支持
- **贷款贴息**：对企业获得的银行贷款给予利息补贴
- **以奖代补**：对已完成的优秀项目给予奖励性补助
- **政府购买服务**：通过购买服务方式支持中小企业

**3. 重点项目**

**（1）小微企业创业创新示范基地**

- 支持标准：按基地建设投资的30%给予补助，最高500万元
- 申报条件：入驻小微企业≥50家，入驻率≥80%，服务设施完善

**（2）中小企业公共服务示范平台**

- 支持标准：按服务收入的20%给予奖励，最高100万元
- 申报条件：年服务中小企业≥200家，服务满意度≥85%

**（3）中小企业数字化转型**

- 支持标准：按转型投入的25%给予补助，最高100万元
- 申报条件：有完整的数字化转型方案并付诸实施

## 二、广东省科技创新普惠性政策

### （一）企业研发费用加计扣除

**1. 政策内容**

企业开展研发活动中实际发生的研发费用，未形成无形资产计入当期损益的，在按规定据实扣除的基础上，自2023年1月1日起，再按照实际发生额的100%在税前加计扣除；形成无形资产的，自2023年1月1日起，按照无形资产成本的200%在税前摊销。

**2. 适用条件**

- 在广东省内注册的居民企业
- 会计核算健全、实行查账征收
- 能够准确归集研发费用
- 研发活动符合《科技部、财政部、国家税务总局关于完善研究开发费用税前加计扣除政策的通知》规定

**3. 研发费用范围**

| 费用类别 | 具体内容 |
|---------|---------|
| 人员费用 | 直接从事研发活动人员的工资薪金、五险一金 |
| 直接投入 | 研发活动直接消耗的材料、燃料、动力费用 |
| 折旧费用 | 用于研发的仪器、设备折旧费 |
| 无形资产摊销 | 用于研发的软件、专利权摊销费 |
| 新产品设计 | 新产品设计费、新工艺规程制定费 |
| 其他费用 | 专家咨询费、研发成果检索分析费等（不超过总费用的20%） |

**4. 办理流程**

（1）企业自主判别：判断是否符合加计扣除条件
（2）归集核算：按照研发项目归集研发费用，建立辅助账
（3）年度申报：在年度企业所得税申报时填报优惠明细表
（4）留存备查：准备相关资料留存备查，包括研发项目立项书、费用归集表等

### （二）高新技术企业认定

**1. 奖励标准**

# 第三部分：广州市民营经济发展政策

## 一、广州市促进民营经济发展壮大若干措施

### （一）政策背景

民营经济是广州市经济社会发展的重要支撑。2023年，广州市出台《关于促进民营经济发展壮大的若干措施》，从优化营商环境、强化要素保障、支持创新发展、拓展市场空间等方面，推出一系列含金量高的政策措施，全力推动民营经济高质量发展。

### （二）核心政策内容

**1. 降低市场准入门槛**

- 全面实施市场准入负面清单制度，确保各类市场主体依法平等进入
- 在民营企业投资领域，除国家法律法规明确禁止外，一律开放
- 鼓励民间资本参与基础设施、公共服务等领域建设

**2. 缓解融资难融资贵**

| 措施 | 具体内容 |
|-----|---------|
| 贷款风险补偿 | 设立10亿元信贷风险补偿资金池，对银行向民营企业发放贷款产生的损失给予补偿 |
| 融资担保支持 | 政府性融资担保机构担保费率降至1%以下，单户担保额度提高至1000万元 |
| 贷款贴息 | 对符合条件的民营企业贷款，按LPR的30%给予贴息，年度最高50万元 |
| 股权投资 | 设立100亿元民营经济发展基金，以股权投资方式支持优质民营企业 |

**3. 降低企业成本**

- 落实国家减税降费政策，确保惠企政策直达快享
- 降低用能成本，参与电力市场化交易
- 降低物流成本，规范物流收费行为
- 降低用工成本，延续实施阶段性降低失业保险、工伤保险费率政策

**4. 支持创新发展**

- 对民营企业研发投入，按实际支出的15%给予补助
- 对民营企业建设技术创新中心、重点实验室，给予最高500万元支持
- 支持民营企业参与国家、省、市重大科技项目
- 鼓励民营企业引进高层次人才，享受人才政策待遇

**5. 拓展市场空间**

- 政府采购项目预留30%以上份额给中小企业
- 支持民营企业参与政府采购和政府投资项目
- 鼓励民营企业开拓国际市场，给予参展补助和出口信用保险补贴
- 支持民营企业参与"一带一路"建设和粤港澳大湾区建设

## 二、广州市民营经济发展条例解读

### （一）条例要点

2023年，《广州市民营经济发展促进条例》正式实施，这是广州市首部专门促进民营经济发展的地方性法规，从法律层面保障民营企业权益。

**1. 平等保护**

- 保障民营企业依法平等使用生产要素
- 公平参与市场竞争
- 同等受到法律保护

**2. 权益保障**

- 禁止干预民营企业依法自主经营
- 规范涉企检查行为，减少对企业正常经营的影响
- 保障民营企业家人身和财产安全

**3. 政策支持**

- 建立民营经济发展专项资金
- 完善政府与民营企业沟通机制
- 建立民营企业诉求快速响应机制

### （二）企业权益保护

**1. 涉企收费管理**

- 实行涉企收费清单制度
- 任何单位不得擅自设立收费项目
- 对违规收费行为，企业可向12345热线投诉

- 实行"双随机、一公开"监管
- 同一部门对同一企业检查每年不超过2次
- 推行跨部门联合检查，减少检查频次

**3. 企业家保护**

- 依法保护民营企业家人身自由
- 慎用查封、扣押、冻结等强制措施
- 建立涉企冤错案件依法甄别纠正常态化机制

- 按技术合同金额的30%给予补助

**3. 港澳青年创新创业基地**

支持建设面向港澳青年的创新创业平台。

资助标准：

# 第八部分：常见问题解答

## 一、专精特新申报常见问题

**Q1：专精特新中小企业和"小巨人"有什么区别？**

A：专精特新中小企业是省级或市级认定，"小巨人"是国家级认定。"小巨人"要求更高，研发费用门槛为300万元（专精特新为100万元），营收门槛为2000万元（专精特新为1000万元），奖励金额也更高。

**Q2：可以跳过专精特新中小企业直接申报"小巨人"吗？**

A：原则上需要先认定为专精特新中小企业满12个月，才能申报"小巨人"。但特别优秀的企业（如获得国家级科技奖项、主导制定国际标准等）可以申请"直通车"。

**Q3：研发费用如何归集？**

A：研发费用包括人员工资、直接投入、折旧费用、无形资产摊销、设计费、委托外部研发费用等。需要建立研发费用辅助账，保留相关凭证。

**Q4：市场占有率如何证明？**

A：可通过行业协会出具证明、第三方咨询机构报告、招投标文件、客户证明等方式证明。建议准备2种以上证明材料。

## 二、融资政策常见问题

**Q5：贷款贴息如何申请？**

A：先获得银行贷款，然后向所在区工信部门提交贴息申请，需提供贷款合同、借据、还款凭证等材料。一般每季度集中受理一次。

**Q6：政府性融资担保如何办理？**

A：联系广州金控、广州担保等政府性融资担保机构，提供企业基本资料、财务报表、贷款用途说明等，担保机构审核通过后为企业提供担保。

**Q7：上市奖励何时申请？**

A：企业成功上市后6个月内，向市地方金融监管局提交申请材料。需提供上市证明、募集资金验资报告等。

## 三、创新政策常见问题

**Q8：研发费用加计扣除如何操作？**

A：在年度企业所得税汇算清缴时，填报研发费用加计扣除优惠明细表。需提前准备研发项目立项书、费用归集表、人员考勤记录等备查资料。

**Q9：高新技术企业认定需要多长时间？**

A：从准备材料到获得认定一般需要6-12个月。建议提前1年规划，完善知识产权布局，规范研发费用归集。

**Q10：科技成果转化补助如何申请？**

A：企业与高校、科研院所签订技术转让合同后，在技术合同登记机构进行登记，然后向科技部门提交补助申请。
