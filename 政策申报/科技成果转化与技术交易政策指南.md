## 前言与编制说明

### 一、编制背景

科技成果转化是科技创新活动的关键环节，是将科学研究与技术开发所产生的具有实用价值的科技成果，通过技术转让、技术许可、技术开发、技术咨询、技术服务等方式，实现科技成果商业化、产业化和市场化的过程。

近年来，国家高度重视科技成果转化工作，相继出台了一系列政策法规，为科技成果转化提供了制度保障和政策支持。广州市作为粤港澳大湾区核心引擎城市，在科技成果转化领域出台了大量创新政策，为科研机构、高等院校和企业提供了丰富的政策红利。

### 二、政策体系概述

当前，科技成果转化与技术交易政策体系主要包括以下几个层面：

| 政策层级 | 主要政策文件 | 发布部门 | 适用范围 |
|---------|------------|---------|---------|
| 国家层面 | 《中华人民共和国促进科技成果转化法》（2015年修订）| 全国人大常委会 | 全国 |
| 国家层面 | 《技术合同认定登记管理办法》（国科发政字〔2000〕63号）| 科技部 | 全国 |
| 省级层面 | 《广东省促进科技成果转化条例》| 广东省人大常委会 | 全省 |
| 市级层面 | 《广州市进一步促进科技成果转移转化若干措施》（穗府办〔2024〕7号）| 广州市政府 | 广州市 |
| 市级层面 | 《广州市促进科技成果转化实施办法》（穗科规字〔2022〕）| 广州市科技局 | 广州市 |
| 区级层面 | 各区配套政策 | 各区政府 | 各区 |

### 三、政策支持方向

科技成果转化政策支持主要包括以下几个方向：

1. **资金支持**：财政补贴、税收优惠、金融支持
2. **服务支持**：技术合同登记、成果评估、对接服务
3. **平台支持**：孵化器、众创空间、中试平台
4. **人才支持**：技术经理人培养、职称评审、奖励激励
5. **机制创新**：职务科技成果单列管理、赋权改革、"先用后转"

## 第一章 广州市科技成果转化政策

### 一、政策框架

#### （一）核心政策文件

**1. 《广州市进一步促进科技成果转移转化若干措施》**

- **文号**：穗府办〔2024〕7号
- **发布时间**：2024年4月22日
- **有效期**：3年（2024年4月22日至2027年4月21日）
- **发布部门**：广州市人民政府办公厅

**政策目标**：
- 促进科技成果高质量转移转化
- 构建以企业为主体、市场为导向、需求为牵引的科技成果转化体系
- 推动科技成果需求端、供给端、服务端同向发力
- 完善从技术到样品到产品再到产业的科技成果转化链条

#### （二）政策主要内容

**1. 强化市场需求牵引**

| 政策措施 | 具体内容 | 支持力度 |
|---------|---------|---------|
| 创新联合体协同攻关 | 实施"揭榜挂帅""赛马制"机制 | 每个项目最高200万元 |
| 重大科技成果落地产业化 | 承接国家重大科技项目、颠覆性技术项目落地转化 | 最高1000万元"补改投"支持 |
| 应用场景牵引 | 开放重点产业领域应用场景、政务服务应用场景 | 政策支持 |
| 首台套/首批次/首版次 | 创新产品优先采购、推广应用 | 单一来源采购、相关补助 |

**2. 提升成果供给水平**

| 政策措施 | 具体内容 | 实施部门 |
|---------|---------|---------|
| 优化科技成果源头质量 | 设置与战略性新兴产业相匹配的学科专业 | 市教育局、市人社局 |
| 成果转化绩效纳入考核 | 将成果转化绩效纳入市属高校院所领导班子考核 | 市科技局、市委组织部 |
| 激发转化动力活力 | 赋予科研人员职务科技成果所有权或长期使用权 | 市科技局、市教育局 |
| 职务科技成果单列管理 | 符合条件的职务科技成果不纳入国有资产保值增值管理 | 市科技局、市财政局 |
| 现金奖励税收优惠 | 职务科技成果转化现金奖励可减按50%计入工资薪金 | 市税务局、市财政局 |

**3. 增强服务体系效能**

| 支持类型 | 支持内容 | 支持方式 |
|---------|---------|---------|
| 概念验证中心 | 为实验阶段科技成果提供技术概念验证、商业化开发服务 | 先创建后认定 |
| 中试小试平台 | 加速成果产品化和产业化进程 | 先创建后认定 |
| 科技成果转化"特区" | 在环五山、环大学城、环中大、南沙、越秀等区域建设孵化载体 | 政策支持 |
| 天使母基金 | 破解科技成果转化最初一公里融资难题 | 资金支持 |
| 技术经理人队伍建设 | 构建技术经纪工程技术人才职称评价体系 | 最高50万元奖励 |

**4. 强化组织保障**

- 建立市促进科技成果转化联席会议制度
- 定期发布广州科技成果转化白皮书
- 制定高校院所、医疗机构等科技成果转化"排行榜"
- 打造区域性的成果转化数字平台

### 二、"补改投"试点政策详解

#### （一）政策背景

**文件名称**：《广州市科技成果转化"补改投"试点工作办法（试行）》

**发布时间**：2026年1月

**政策意义**：全国首个以市场化遴选加速科技成果转化落地的"补改投"政策

#### （二）核心内容

**1. 运作机制**

"补改投"是指将原本用于补贴的财政资金，改为股权投资方式支持科技成果转化项目，通过市场化运作实现财政资金的循环使用和保值增值。

| 环节 | 传统补贴方式 | "补改投"方式 |
|------|------------|-------------|
| 资金性质 | 无偿资助 | 股权投资 |
| 退出机制 | 无 | 通过股权转让、并购、上市等方式退出 |
| 资金循环 | 一次性消耗 | 循环使用 |
| 风险分担 | 政府承担 | 政府与社会资本共担 |
| 激励机制 | 相对较弱 | 市场化激励，收益共享 |

**2. 支持对象**

- 聚焦社会资本"不愿投"的早期科技成果转化项目
- 具有产业化潜力的核心技术成果
- 符合广州市战略性新兴产业发展方向

**3. 支持额度**

- 单个项目最高支持：1000万元
- 支持方式：股权投资
- 投资期限：根据项目情况确定

#### （三）申报条件

| 条件类别 | 具体要求 |
|---------|---------|
| 主体资格 | 在广州市注册的企业或科研机构 |
| 技术要求 | 具有自主知识产权的核心技术 |
| 市场前景 | 技术成果具有明确的产业化方向和市场前景 |
| 团队要求 | 具有稳定的技术研发和管理团队 |
| 财务要求 | 财务管理规范，具备接受股权投资的条件 |

### 三、职务科技成果"先用后转"改革

#### （一）政策依据

**文件名称**：《广东省推进职务科技成果"先用后转"改革实施方案》

**发布部门**：广东省科学技术厅等五部门

**发布时间**：2026年1月14日

#### （二）改革要点

**1. 核心机制**

"先用后转"是指在职务科技成果转化中，允许企业先免费或低成本使用科技成果，待产生经济效益后再进行转让或许可的机制。

| 环节 | 传统模式 | "先用后转"模式 |
|------|---------|---------------|
| 前期投入 | 企业需支付高额转让费 | 低成本或免费试用 |
| 风险评估 | 企业承担全部风险 | 成果单位与企业共担风险 |
| 转化效率 | 相对较低 | 提高转化成功率 |
| 激励机制 | 相对单一 | 多元化激励 |

**2. 配套措施**

- 建立规范合同文本
- 构建覆盖全省的系统性服务网络
- 组织开展科技成果"先用后转"激励支持
- 尽职免责认定机制

**3. 责任部门**

| 部门 | 职责 |
|------|------|
| 省科技厅 | 牵头组织实施 |
| 省教育厅 | 指导高校成果单位 |
| 省卫健委 | 指导医疗机构成果单位 |
| 省市场监管局 | 知识产权相关服务 |
| 技术转移服务机构 | 专业服务支持 |

## 第二章 广东省技术合同认定登记管理

### 一、政策依据

#### （一）国家层面

**1. 《技术合同认定登记管理办法》**

- **文号**：国科发政字〔2000〕63号
- **发布部门**：科学技术部
- **适用范围**：全国

**核心内容**：
- 第四条：省、自治区、直辖市和计划单列市科学技术行政部门及技术合同登记机构，应当通过技术合同的认定登记，加强对技术市场和科技成果转化工作的指导、管理和服务，并进行相关的技术市场统计和分析工作。

**2. 《技术合同认定规则》**

- 规范技术合同认定登记的具体操作
- 明确各类技术合同的认定标准

#### （二）省级层面

**《广东省技术合同认定登记管理办法》**

- 规范广东省技术合同认定登记工作
- 加强技术市场管理
- 保障科技成果转化政策落实

### 二、技术合同分类

#### （一）合同类型

| 合同类型 | 定义 | 特点 |
|---------|------|------|
| 技术开发合同 | 当事人之间就新技术、新产品、新工艺或者新材料及其系统的研究开发所订立的合同 | 创新性强、风险共担 |
| 技术转让合同 | 当事人之间就专利权转让、专利申请权转让、专利实施许可、技术秘密转让所订立的合同 | 权利转移明确 |
| 技术咨询合同 | 当事人一方为另一方就特定技术项目提供可行性论证、技术预测、专题技术调查、分析评价报告所订立的合同 | 知识服务 |
| 技术服务合同 | 当事人一方以技术知识为另一方解决特定技术问题所订立的合同 | 解决实际问题 |

#### （二）合同认定标准

**1. 技术开发合同认定要点**

| 要素 | 要求 |
|------|------|
| 标的 | 新技术、新产品、新工艺或新材料 |
| 内容 | 具有创新性的研究开发活动 |
| 成果 | 形成新的技术成果或技术方案 |
| 风险 | 开发风险由当事人约定分担 |

**2. 技术转让合同认定要点**

| 要素 | 要求 |
|------|------|
| 标的 | 现有的、特定的、权利化的技术 |
| 内容 | 专利权、专利申请权、专利实施权、技术秘密权的转移 |
| 权利 | 技术权利归属明确 |
| 对价 | 有明确的技术交易价格 |

**3. 技术咨询合同认定要点**

| 要素 | 要求 |
|------|------|
| 标的 | 特定技术项目 |
| 内容 | 可行性论证、技术预测、专题技术调查、分析评价 |
| 形式 | 提供咨询报告或意见 |
| 性质 | 知识密集型服务 |

**4. 技术服务合同认定要点**

| 要素 | 要求 |
|------|------|
| 标的 | 特定技术问题 |
| 内容 | 解决专业技术问题 |
| 方式 | 提供技术服务 |
| 成果 | 解决技术难题、提高技术效果 |

### 三、认定登记流程

#### （一）办理流程

```
┌─────────────────────────────────────────────────────────────┐
│                   技术合同认定登记流程                          │
├─────────────────────────────────────────────────────────────┤
│  第一步：签订技术合同                                          │
│     ↓                                                        │
│  第二步：准备申请材料                                          │
│     ├─ 技术合同文本                                           │
│     ├─ 技术方案说明                                           │
│     ├─ 相关证明材料                                           │
│     └─ 营业执照/身份证                                        │
│     ↓                                                        │
│  第三步：提交申请                                              │
│     └─ 向所在地技术合同登记机构提交                             │
│     ↓                                                        │
│  第四步：审查认定                                              │
│     ├─ 形式审查                                               │
│     ├─ 技术内容审查                                           │
│     └─ 实质审查                                               │
│     ↓                                                        │
│  第五步：登记备案                                              │
│     └─ 核发技术合同认定登记证明                                │
│     ↓                                                        │
│  第六步：享受优惠政策                                          │
│     ├─ 税收优惠                                               │
│     ├─ 财政补贴                                               │
│     └─ 其他政策支持                                           │
└─────────────────────────────────────────────────────────────┘
```

#### （二）办理时限

| 环节 | 时限要求 |
|------|---------|
| 材料提交 | 合同签订后30日内 |
| 形式审查 | 5个工作日 |
| 技术内容审查 | 10个工作日 |
| 整体认定 | 一般不超过30个工作日 |

#### （三）注意事项

**1. 合同必备条款**

| 条款类别 | 具体内容 |
|---------|---------|
| 标的条款 | 明确技术内容、范围、技术指标 |
| 价款条款 | 技术交易金额及支付方式 |
| 履行条款 | 履行期限、地点、方式 |
| 保密条款 | 技术秘密保护措施 |
| 风险条款 | 技术开发风险的承担 |
| 成果条款 | 技术成果的归属和分享 |
| 验收条款 | 技术验收标准和方法 |

**2. 常见问题及解决方案**

| 问题 | 原因分析 | 解决方案 |
|------|---------|---------|
| 合同性质认定不清 | 技术内容与商务内容混合 | 明确技术内容范围，合理划分价款 |
| 技术含量不足 | 缺乏技术创新性 | 补充技术方案说明，突出创新点 |
| 材料不齐全 | 申请材料准备不充分 | 按清单逐项准备材料 |
| 金额核定争议 | 技术价款与非技术价款未区分 | 提供详细的价格构成说明 |

### 四、登记优惠政策

#### （一）税收优惠

| 优惠政策 | 适用范围 | 优惠内容 |
|---------|---------|---------|
| 增值税优惠 | 技术开发、技术转让 | 免征增值税 |
| 企业所得税优惠 | 技术转让所得 | 500万元以内免征，超过部分减半征收 |
| 个人所得税优惠 | 职务科技成果转化现金奖励 | 可减按50%计入工资薪金 |

#### （二）财政补贴

| 补贴类型 | 补贴标准 | 申请条件 |
|---------|---------|---------|
| 技术合同登记补贴 | 按合同成交额一定比例 | 完成技术合同认定登记 |
| 技术交易奖励 | 根据交易额确定 | 促成重大技术交易 |
| 技术输出补贴 | 按技术输出额确定 | 技术输出到市外 |

## 第三章 广州市技术交易补贴政策

### 一、技术交易奖励补贴

#### （一）技术输出奖励

**1. 政策内容**

对向广州市以外单位输出技术的企业、科研机构和高等院校，按照技术合同成交额的一定比例给予奖励。

| 技术输出额 | 奖励比例 | 最高奖励 |
|-----------|---------|---------|
| 100万元以下 | 3% | 3万元 |
| 100-500万元 | 2.5% | 12.5万元 |
| 500-1000万元 | 2% | 20万元 |
| 1000万元以上 | 1.5% | 50万元 |

**2. 申报条件**

| 条件 | 要求 |
|------|------|
| 主体资格 | 在广州市注册的企业、科研机构、高等院校 |
| 合同要求 | 已完成技术合同认定登记 |
| 技术方向 | 向广州市以外单位输出技术 |

### 三、常见问题解答

#### （一）技术合同认定登记类

**Q1：技术合同认定登记需要多长时间？**

A：一般情况下，技术合同认定登记需要在合同签订后30日内提出申请。登记机构在收到申请材料后，一般在30个工作日内完成认定登记。

**Q2：技术开发合同和技术转让合同有什么区别？**

A：技术开发合同是针对尚未存在的新技术进行研究开发；技术转让合同是针对已有的、特定的技术进行权利转移。两者的核心区别在于技术是否已经存在。

**Q3：技术合同认定登记后可以享受哪些优惠？**

A：主要优惠包括：增值税免征（技术开发、技术转让）、企业所得税优惠（技术转让所得500万元以内免征，超过部分减半征收）、财政补贴等。

#### （二）科技成果转化类

**Q4：职务科技成果转化奖励如何分配？**

A：根据《促进科技成果转化法》，对完成、转化职务科技成果做出重要贡献的人员给予奖励和报酬的比例不低于科技成果转化净收入或者许可净收入的50%。

**Q5：科技成果"先用后转"有哪些风险？**

A：主要风险包括：技术泄密风险、合作方违约风险、收益分配争议等。建议签订规范的合同，明确各方权利义务和违约责任。

**Q6：如何认定科技成果转化的经济效益？**

A：主要通过技术合同成交额、产品销售收入、技术许可费收入、股权收益等方式认定。需要提供相关的财务凭证和效益证明材料。

#### （三）产学研合作类

**Q7：产学研合作项目如何选择合作高校？**

A：建议考虑以下因素：高校在该领域的技术优势、科研团队的实力、过往合作基础、知识产权归属约定、合作费用预算等。

**Q8：产学研合作中知识产权如何归属？**

A：按照《专利法》和《合同法》规定，合作开发完成的发明创造，除当事人另有约定外，申请专利的权利属于合作开发的当事人共有。

## 政策解读与实操要点

### 技术合同认定登记实操要点

#### 1. 合同文本规范

**建议使用技术合同示范文本：**
- 技术开发（委托）合同示范文本
- 技术开发（合作）合同示范文本
- 技术转让合同示范文本
- 技术咨询合同示范文本
- 技术服务合同示范文本

**合同必备条款：**
- 项目名称：准确反映技术内容
- 标的内容：明确技术目标、技术内容、技术方法和技术成果
- 履行期限：明确起止时间
- 价款报酬：明确金额和支付方式
- 技术资料：明确技术资料的提交时间、地点和方式
- 技术秘密：明确保密范围和保密期限
- 风险责任：明确技术风险的责任承担
- 技术成果归属：明确知识产权归属
- 验收标准：明确技术成果的验收标准和方式
- 违约责任：明确违约责任和争议解决方式

#### 2. 常见问题及解决方案

| 问题 | 解决方案 |
|------|---------|
| 合同名称与技术内容不符 | 使用准确的技术合同名称，避免使用"服务合同"、"咨询合同"等模糊表述 |
| 技术内容表述不清晰 | 详细描述技术目标、技术路线、关键技术、预期成果 |
| 知识产权归属不明确 | 明确约定技术成果的知识产权归属，包括专利申请权、专利权、技术秘密等 |
| 价款构成不合理 | 合理划分技术开发费、技术服务费、设备费等 |
| 缺少技术附件 | 提供技术方案、可行性报告等技术附件 |

#### 3. 税收优惠备案要点

**增值税免税备案：**
- 备案时间：享受免税政策前或首次申报纳税时
- 备案材料：技术合同认定登记证明、合同复印件、免税备案表
- 备案渠道：主管税务机关办税服务厅或电子税务局

**企业所得税优惠备案：**
- 备案时间：年度企业所得税汇算清缴时
- 备案材料：技术转让所得计算说明、技术合同认定登记证明、技术转让收入证明
- 优惠计算：技术转让所得 = 技术转让收入 - 技术转让成本 - 相关税费

### 科技成果转化收益分配实操

#### 1. 收益分配比例

| 分配对象 | 最低比例 | 说明 |
|---------|---------|------|
| 成果完成人 | 50% | 科技成果转化净收入或净收益的50%以上 |
| 转化贡献者 | 10-20% | 对成果转化做出重要贡献的人员 |
| 单位 | 20-30% | 单位留存部分，用于科技研发和成果转化 |

#### 2. 收益计算方法

**技术转让净收入 = 技术转让收入 - 技术转让成本 - 相关税费**

其中：
- 技术转让收入：技术转让合同约定的价款
- 技术转让成本：技术成果的研发成本、评估费、中介费等
- 相关税费：与技术成果转让相关的税费

#### 3. 分配程序

1. 科技成果完成转化后，财务部门核算净收入
2. 成果完成人提出收益分配申请
3. 单位审核分配方案
4. 内部公示（不少于5个工作日）
5. 单位审批并实施分配

| 团队能力 | 20% | 核心团队背景、研发能力、管理经验 |
| 财务状况 | 15% | 营业收入、利润率、资产负债率 |
| 社会效益 | 15% | 就业带动、产业链影响、环保贡献 |

#### 2. 尽职调查内容

**业务尽调：**
- 公司基本情况
- 业务模式和盈利模式
- 核心技术和知识产权
- 主要客户和供应商
- 市场竞争分析

**财务尽调：**
- 财务报表分析
- 盈利能力评估
- 现金流分析
- 资产负债分析
- 财务风险识别

**法务尽调：**
- 公司治理结构
- 合规经营情况
- 重大合同审查
- 知识产权权属
- 诉讼仲裁情况

#### 3. 投后管理要点

- 定期跟踪企业经营情况（季度/半年度报告）
- 参与企业重大决策（股东会/董事会）
- 提供增值服务（资源对接、战略咨询）
- 监控风险并及时预警
- 适时启动退出程序

### 产学研合作实操要点

#### 1. 合作协议关键条款

**合作范围条款：**
- 明确合作的技术领域和研发方向
- 约定合作的起止时间和阶段目标
- 界定各方的权利和义务

**知识产权条款：**
- 明确合作前各方自有知识产权的归属
- 约定合作期间产生知识产权的归属
- 明确知识产权的使用许可方式
- 约定知识产权侵权的责任承担

**利益分配条款：**
- 明确合作成果的收益分配比例
- 约定收益分配的计算方法和支付方式
- 明确收益分配的税务处理

**保密条款：**
- 明确保密信息的范围
- 约定保密期限和保密义务
- 明确保密违约的责任

**退出机制条款：**
- 约定合作终止的条件
- 明确终止后的知识产权处理
- 约定终止后的利益分配

#### 2. 合作模式选择

| 合作模式 | 适用场景 | 优势 | 劣势 |
|---------|---------|------|------|
| 委托开发 | 企业需要特定技术开发 | 目标明确、风险可控 | 成本较高、缺乏参与感 |
| 合作开发 | 双方共同攻关技术难题 | 资源共享、优势互补 | 利益分配复杂、管理难度大 |
| 联合实验室 | 长期深度合作 | 合作稳定、持续产出 | 投入较大、见效较慢 |
| 人才联合培养 | 需要专业人才支持 | 人才定制、产学研融合 | 周期较长、培养成本高 |
| 科技成果转化 | 高校/科研院所有成熟成果 | 见效快、风险低 | 成果适用性、价格谈判 |

## 案例分析

### 案例一：技术合同认定登记享受税收优惠

**企业背景：**
某软件企业承接了一项大型软件开发项目，合同金额500万元。

**操作步骤：**
1. 使用技术开发合同示范文本签订合同
2. 在广东省科技业务管理阳光政务平台申请认定登记
3. 取得技术合同认定登记证明
4. 向主管税务机关办理增值税免税备案

**优惠效果：**
- 增值税免征：500万 ÷ 1.06 × 6% = 28.30万元
- 企业所得税：如符合技术转让条件，500万元以内部分免征

**经验总结：**
- 合同签订时使用规范的技术合同示范文本
- 明确技术内容、技术目标、技术成果
- 及时办理认定登记和税收备案

### 案例二：产学研合作项目申报

**企业背景：**
某制造企业需要攻克一项关键技术，与华南理工大学开展产学研合作。

**操作步骤：**
1. 与华南理工大学签订产学研合作协议
2. 联合申报广州市重点领域研发计划产学研合作项目
3. 项目通过评审，获得300万元财政资金支持
4. 企业自筹300万元配套资金
5. 项目实施2年，取得3项发明专利，实现产业化

**经验总结：**
- 合作协议要明确知识产权归属和利益分配
- 项目申报前充分沟通，明确分工
- 项目实施中保持密切合作，定期沟通
- 项目成果要及时申报专利，做好知识产权保护

### 案例三："补改投"试点项目

**企业背景：**
某科技型初创企业在广州科技创新创业大赛中获得一等奖，有融资需求。

**操作步骤：**
1. 提交"补改投"融资需求申请
2. 配合完成尽职调查
3. 通过投资决策委员会审议
4. 签订投资协议，获得200万元股权投资
5. 企业快速发展，估值提升3倍

**经验总结：**
- 积极参加科技创新创业大赛
- 保持财务规范和信息透明
- 配合尽职调查，提供真实完整资料
- 善用政府背书，撬动更多社会资本

## 常见问题解答（FAQ）

### Q1：技术合同认定登记需要多长时间？

**A：** 一般情况下7个工作日内完成审核，复杂情况15个工作日内完成。建议在合同签订后30日内办理，以免影响税收优惠的享受。

### Q2：高新技术企业认定需要哪些核心条件？

**A：** 核心条件包括：
- 企业注册成立一年以上
- 拥有核心自主知识产权
- 主要产品（服务）属于国家重点支持的高新技术领域
- 科技人员占比不低于10%
- 研发费用占销售收入比例达标（3%-5%）
- 高新技术产品（服务）收入占比不低于60%

### Q3：职务科技成果所有权改革对科研人员有什么好处？

**A：** 主要好处包括：
- 可以获得职务科技成果的所有权或长期使用权
- 科技成果转化收益的50%以上奖励给成果完成人
- 降低成果转化的审批程序和时间成本
- 提高科研人员的创新创业积极性

### Q4：番禺区的高新技术企业奖励什么时候可以申请？

**A：** 企业通过高新技术企业认定后，可凭认定证书向番禺区科工商信局申请区级奖励。一般每年集中申报一次，具体时间以区科技部门通知为准。

### Q5：粤港澳大湾区的科技合作有哪些特殊政策？

**A：** 特殊政策包括：
- 境外研发机构奖励（100-120万元）
- 科技要素跨境流动便利化
- 粤港澳联合实验室建设支持
- 高校科技成果转化中心平台支持
- 知识产权跨境交易服务
