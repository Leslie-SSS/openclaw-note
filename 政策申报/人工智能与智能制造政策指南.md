## 二、广州市人工智能产业发展政策

### 2.1 政策背景与目标

广州市于2024年发布《广州市人工智能产业发展行动计划（2024-2026年）》，明确提出将广州建设成为全国领先的人工智能产业集聚区和创新应用高地。主要目标包括：

- **产业规模**：到2026年，全市人工智能产业规模突破2000亿元
- **企业培育**：培育100家人工智能领军企业、500家高新技术企业
- **应用场景**：打造100个人工智能示范应用场景
- **人才引进**：引进培育100名人工智能领域高层次人才

### 2.2 主要扶持政策

#### 2.2.1 企业落户奖励

**政策依据**：《广州市人工智能产业发展行动计划》《广州市促进总部经济发展暂行办法》

**申报条件**：
1. 企业在广州市行政区域内注册、纳税，具有独立法人资格
2. 主营业务为人工智能技术研发、产品制造或应用服务
3. 实缴注册资本不低于1000万元
4. 拥有不少于10名全职人工智能领域研发人员
5. 持有不少于5项人工智能相关知识产权（专利或软件著作权）

**奖励标准**：
| 企业类型 | 实缴资本 | 年营收要求 | 奖励金额 |
|---------|---------|-----------|---------|
| 总部企业 | ≥5000万元 | ≥10亿元 | 5000万元 |
| 总部企业 | ≥3000万元 | ≥5亿元 | 3000万元 |
| 领军企业 | ≥1000万元 | ≥1亿元 | 1000万元 |
| 骨干企业 | ≥500万元 | ≥3000万元 | 500万元 |
| 创新型企业 | ≥300万元 | ≥1000万元 | 200万元 |

**发放方式**：
- 首期发放40%（完成注册后）
- 第二期发放30%（运营满1年且达标后）
- 第三期发放30%（运营满2年且达标后）

**申报材料**：
1. 广州市人工智能产业扶持资金申请表
2. 企业营业执照复印件
3. 验资报告或实缴资本证明
4. 上年度财务审计报告
5. 知识产权证明材料
6. 研发人员名册及社保证明
7. 企业发展规划书

#### 2.2.2 研发投入补贴

**政策依据**：《广州市人工智能产业发展行动计划》《广州市企业研发经费投入后补助实施方案》

**申报条件**：
1. 在广州市注册的人工智能企业
2. 上年度研发经费支出不低于100万元
3. 研发经费支出占营业收入比例不低于5%
4. 已完成研发费用税前加计扣除备案

**奖励标准**：
| 研发投入金额 | 补贴比例 | 最高补贴 |
|-------------|---------|---------|
| 100万-500万元 | 10% | 50万元 |
| 500万-2000万元 | 15% | 300万元 |
| 2000万元以上 | 20% | 500万元 |

**特别说明**：
- 连续3年符合条件的企业，第4年起补贴比例提高5个百分点
- 国家级、省级企业技术中心额外奖励100万元、50万元
- 获得国家级科技计划项目立项的，给予1:1配套支持

#### 2.2.3 场景应用示范项目

**政策依据**：《广州市人工智能场景应用示范项目建设实施方案》

**申报条件**：
1. 项目实施主体为在广州市注册的企业或事业单位
2. 项目总投资不低于300万元
3. 项目具有明确的人工智能技术应用场景
4. 项目实施周期不超过2年
5. 项目已完成可行性研究，具备实施条件

**重点支持领域**：
1. **智能制造**：智能工厂、数字化车间、质量检测
2. **智慧交通**：智能驾驶、交通优化、智慧停车
3. **智慧医疗**：辅助诊断、影像分析、健康管理
4. **智慧教育**：个性化学习、智能评测、虚拟课堂
5. **智慧金融**：风险控制、智能投顾、反欺诈
6. **智慧城市**：城市治理、环境监测、公共安全
7. **智慧商贸**：智能推荐、无人零售、供应链优化

**奖励标准**：
| 项目类型 | 投资规模 | 补贴比例 | 最高补贴 |
|---------|---------|---------|---------|
| 示范项目 | ≥1000万元 | 30% | 1000万元 |
| 重点项目 | ≥500万元 | 25% | 500万元 |
| 一般项目 | ≥300万元 | 20% | 200万元 |

**项目验收标准**：
1. 完成项目可行性研究报告确定的全部建设内容
2. 实现预定的技术指标和应用效果
3. 形成可复制、可推广的解决方案
4. 完成项目财务审计报告

### 2.3 产业园区扶持政策

**政策依据**：《广州市人工智能产业园区认定管理办法》

**园区认定条件**：
1. 园区规划面积不少于50亩或建筑面积不少于5万平方米
2. 园区内人工智能企业占比不低于50%
3. 园区内企业上年度总产值不低于5亿元
4. 具有完善的公共服务平台和配套设施
5. 建立专业的园区运营管理团队

**扶持措施**：
| 扶持项目 | 扶持标准 |
|---------|---------|
| 园区认定奖励 | 国家级1000万元，省级500万元，市级200万元 |
| 公共服务平台建设 | 投资额30%，最高500万元 |
| 企业租金补贴 | 租金30%，每家企业每年最高50万元 |
| 招商引资奖励 | 每引进1家规模以上企业奖励10万元 |

### 2.4 人才引进与培养

**政策依据**：《广州市人工智能人才引进培养行动计划》

**人才认定标准**：
| 人才类别 | 认定条件 | 奖励标准 |
|---------|---------|---------|
| 领军人才 | 国家级人才计划入选者、院士 | 500万元安家补贴 |
| 高端人才 | 省部级人才计划入选者、博士+5年经验 | 200万元安家补贴 |
| 骨干人才 | 硕士+3年AI行业经验、年薪≥50万元 | 50万元安家补贴 |
| 紧缺人才 | 本科+2年AI行业经验、年薪≥30万元 | 20万元安家补贴 |

**人才培养支持**：
1. 企业开展人工智能培训，按培训费用50%给予补贴，每年最高50万元
2. 与高校共建人工智能学院，给予500万元建设经费支持
3. 承办国家级人工智能赛事，给予100万元补贴

## 三、广东省智能制造发展政策

### 3.1 政策框架

广东省智能制造政策体系以《广东省智能制造发展规划（2021-2025年）》为核心，配套多项专项资金和扶持政策，形成覆盖智能制造全产业链的政策支持体系。

**发展目标**：
- 到2025年，全省制造业关键工序数控化率达到70%
- 创建100个省级智能制造示范工厂
- 培育1000家智能制造系统解决方案供应商
- 智能装备产业规模突破1万亿元

### 3.2 智能制造试点示范项目

**政策依据**：《广东省智能制造试点示范项目管理办法》

**申报条件**：
1. 企业在广东省内注册，具有独立法人资格
2. 企业运营和财务状况良好，上年度营业收入不低于5000万元
3. 智能制造项目已完成或进入实施阶段，投资规模不低于1000万元
4. 项目技术水平处于国内领先地位，具有示范带动作用
5. 企业已通过两化融合管理体系评定

**示范类型**：
1. **离散型智能制造**：机械、汽车、电子、家电等行业
2. **流程型智能制造**：石化、钢铁、有色、建材等行业
3. **网络协同制造**：产业链协同、产业集群
4. **大规模个性化定制**：服装、家具、消费品行业
5. **远程运维服务**：装备制造、工业产品

**奖励标准**：
| 项目级别 | 投资规模要求 | 奖励比例 | 最高奖励 |
|---------|-------------|---------|---------|
| 国家级示范 | ≥5000万元 | 投资额25% | 2000万元 |
| 省级示范 | ≥2000万元 | 投资额20% | 1000万元 |
| 市级示范 | ≥1000万元 | 投资额15% | 500万元 |

**申报材料**：
1. 广东省智能制造试点示范项目申报书
2. 企业营业执照、组织机构代码证
3. 上年度财务审计报告
4. 项目可行性研究报告
5. 项目投资明细及凭证
6. 两化融合评定证书
7. 知识产权证明材料
8. 项目成效证明材料

### 3.3 工业互联网发展专项资金

**政策依据**：《广东省工业互联网发展专项资金管理办法》

**支持方向**：

#### 3.3.1 工业互联网平台建设

**申报条件**：
1. 平台已上线运营满1年
2. 平台注册企业用户不少于100家
3. 平台连接工业设备不少于1000台套
4. 平台年服务收入不低于500万元

**奖励标准**：
| 平台级别 | 认定条件 | 奖励金额 |
|---------|---------|---------|
| 国家级双跨平台 | 工信部认定 | 3000万元 |
| 省级跨行业平台 | 省工信厅认定 | 1000万元 |
| 行业特色平台 | 省工信厅认定 | 500万元 |

#### 3.3.2 工业企业上云上平台

**申报条件**：
1. 在广东省内注册的工业企业
2. 与省级以上工业互联网平台签订服务合同
3. 实现核心业务系统上云

**补贴标准**：
| 企业规模 | 补贴比例 | 最高补贴 |
|---------|---------|---------|
| 规上企业 | 服务费50% | 100万元/年 |
| 中小企业 | 服务费70% | 50万元/年 |
| 小微企业 | 服务费80% | 20万元/年 |

**支持云服务类别**：
1. 基础云服务：云服务器、云存储、云数据库
2. 核心业务系统：ERP、MES、PLM、SCM
3. 工业应用：CAD、CAE、CAM、EDA
4. 工业大数据：数据采集、存储、分析
5. 工业AI应用：质量检测、预测性维护、工艺优化

#### 3.3.3 工业互联网标识解析体系

**申报条件**：
1. 建设国家顶级节点或二级节点
2. 通过工信部验收认定
3. 标识注册量不低于100万个

**奖励标准**：
| 节点类型 | 奖励金额 |
|---------|---------|
| 国家顶级节点 | 5000万元 |
| 二级节点 | 500万元 |
| 企业节点 | 50万元 |

### 3.4 智能制造装备推广

**政策依据**：《广东省智能制造装备推广应用的实施意见》

**支持范围**：

4. **智能检测与装配装备**：视觉检测、在线测量设备
5. **智能物流装备**：AGV、智能仓储、分拣系统

**购置补贴标准**：
| 设备类型 | 国产设备补贴 | 进口设备补贴 |
|---------|-------------|-------------|

| 工业机器人 | 购置价20% | 购置价15% |
| 智能检测装备 | 购置价20% | 购置价15% |
| 智能物流装备 | 购置价15% | 购置价10% |

**单家企业年度补贴上限**：500万元

### 3.5 数字化转型服务商培育

**政策依据**：《广东省智能制造系统解决方案供应商培育工作方案》

| 五星级 | ≥5亿元 | ≥50个案例 | ≥500人 |
| 四星级 | ≥2亿元 | ≥30个案例 | ≥200人 |
| 三星级 | ≥5000万元 | ≥15个案例 | ≥50人 |
| 二星级 | ≥1000万元 | ≥5个案例 | ≥20人 |

**扶持措施**：

2. 获评省级供应商，奖励100万元

## 四、广州市机器人产业政策

### 4.1 政策背景

广州市将机器人产业作为战略性新兴产业重点发展，出台《广州市机器人产业发展专项规划（2023-2027年）》，致力打造全国重要的机器人产业基地。

**发展目标**：
- 到2027年，全市机器人产业规模突破800亿元
- 培育10家年营收超10亿元的机器人龙头企业
- 建成5个机器人产业特色园区
- 工业机器人密度达到200台/万名工人

### 4.2 工业机器人扶持政策

**政策依据**：《广州市工业机器人产业发展专项资金管理办法》

#### 4.2.1 机器人整机生产企业扶持

**申报条件**：
1. 企业在广州市注册，主营业务为工业机器人整机生产
2. 具有自主研发能力和自主品牌
3. 年销售额不低于1000万元
4. 拥有不少于3项机器人相关核心专利

**奖励标准**：
| 奖励类型 | 奖励标准 |

## 七、申报流程与实操指南

### 7.1 申报前准备工作

#### 7.1.1 企业资质自查清单

在申报任何政府扶持政策前，企业应先完成以下资质自查：

| 检查项目 | 检查内容 | 要求标准 |
|---------|---------|---------|
| 企业注册 | 是否在政策适用区域内注册 | 广州市/番禺区 |
| 经营年限 | 企业成立时间是否符合要求 | 一般≥1年 |
| 法人资格 | 是否具有独立法人资格 | 是 |
| 纳税关系 | 税务征管关系是否在本地 | 是 |
| 财务状况 | 财务是否健康、有无重大违法违规 | 良好、无违规 |
| 知识产权 | 是否拥有相关知识产权 | ≥2项专利/软著 |
| 信用记录 | 是否被列入失信名单 | 无失信记录 |

#### 7.1.2 常见申报材料清单

**基础材料**（适用于所有申报）：
1. 政策扶持资金申请表（官方模板）
2. 企业营业执照复印件（加盖公章）
3. 法人代表身份证复印件
4. 企业章程
5. 上年度财务审计报告
6. 近三年纳税证明
7. 社保缴纳证明
8. 信用中国查询报告

**专项材料**（根据申报类型）：

| 申报类型 | 专项材料清单 |
|---------|------------|
| 落户奖励 | 验资报告、实缴资本证明、发展规划书 |
| 研发补贴 | 研发费用明细、加计扣除备案表、研发项目立项书 |
| 项目补贴 | 可行性研究报告、项目投资明细、设备购置发票 |
| 人才补贴 | 人才认定材料、劳动合同、学历/职称证明 |
| 平台认定 | 平台建设方案、设备清单、团队人员名单 |
| 知识产权 | 专利证书、软件著作权登记证书 |

#### 7.1.3 财务准备工作

**审计报告要求**：
1. 由具有资质的会计师事务所出具
2. 包含资产负债表、利润表、现金流量表
3. 附注完整，披露关联交易
4. 审计意见为无保留意见

**研发费用归集要点**：
1. 建立独立的研发费用辅助账
2. 研发人员工资单独核算
3. 研发设备折旧单独计算
4. 委托外部研发费用需有合同和发票
5. 研发费用需与研发项目一一对应

### 7.2 申报流程详解

#### 7.2.1 省级项目申报流程

**广东省工信厅项目（智能制造、工业互联网）**：

```
第一步：关注通知（省工信厅官网）
    ↓
第二步：网上注册（广东省政务服务网）
    ↓
第三步：在线填报（广东省企业服务平台）
    ↓
第四步：上传材料（PDF格式，单个文件≤10MB）
    ↓
第五步：提交审核（确认提交后不可修改）
    ↓
第六步：区级初审（10个工作日）
    ↓
第七步：市级复审（15个工作日）
    ↓
第八步：省级终审（20个工作日）
    ↓
第九步：公示（7天）
    ↓
第十步：下达资金（公示无异议后30日内）
```

**关键时间节点**：
| 项目类型 | 通知发布 | 申报截止 | 结果公示 |
|---------|---------|---------|---------|
| 智能制造示范 | 每年3月 | 每年5月 | 每年8月 |
| 工业互联网 | 每年4月 | 每年6月 | 每年9月 |
| 技术改造 | 每年1月 | 每年4月 | 每年7月 |

#### 7.2.2 市级项目申报流程

**广州市工信局项目（AI产业、机器人）**：

```
第一步：政策解读会（关注市工信局通知）
    ↓
第二步：网上申报（广州市工业和信息化局项目库）
    ↓
第三步：材料准备（按清单准备纸质材料）
    ↓
第四步：提交材料（线上提交+线下递交）
    ↓
第五步：形式审查（5个工作日）
    ↓
第六步：专家评审（15个工作日）
    ↓
第七步：现场核查（必要时）
    ↓
第八步：部门联审（10个工作日）
    ↓
第九步：公示（7天）
    ↓
第十步：资金拨付（公示后15个工作日）
```

**常见问题及解决方案**：
| 问题 | 原因 | 解决方案 |
|-----|-----|---------|
| 系统无法登录 | 浏览器兼容性 | 使用Chrome或IE11 |
| 材料上传失败 | 文件格式/大小超限 | 转PDF，压缩至10MB内 |
| 提交后需修改 | 联系主管部门退回 | 电话联系经办人 |
| 审核进度查询 | 系统显示滞后 | 直接电话咨询 |

#### 7.2.3 区级项目申报流程

**番禺区科工商信局项目**：

```
第一步：政策咨询（区科工商信局企业服务科）
    ↓
第二步：材料准备（按指南准备全套材料）
    ↓
第三步：窗口递交（番禺区政务服务中心）
    ↓
第四步：受理回执（当场出具）
    ↓
第五步：初审（5个工作日）
    ↓
第六步：实地考察（必要时）
    ↓
第七步：审定（区政策联席会议）
    ↓
第八步：公示（5天）
    ↓
第九步：资金拨付（公示后10个工作日）
```

**区级项目优势**：
1. 审批流程更短（整体周期约30天）
2. 可与市级政策叠加享受
3. 专人对接服务，沟通便利
4. 验收标准相对灵活

### 7.3 项目验收管理

#### 7.3.1 验收准备

**验收材料清单**：
1. 项目验收申请表
2. 项目执行情况报告
3. 项目财务决算报告
4. 项目审计报告（专项审计）
5. 成果证明材料（专利、论文、产品）
6. 应用效果证明材料（用户报告、检测报告）

**验收关键指标**：
| 项目类型 | 核心验收指标 |
|---------|------------|
| 智能制造示范 | 设备到位率≥95%、产能提升≥20%、良品率提升≥5% |
| AI应用示范 | 系统上线运行≥6个月、用户满意度≥85% |
| 研发平台 | 设备到位率≥90%、团队到位率≥80%、研发项目≥3个 |
| 技术改造 | 投资完成率≥90%、产能提升≥15% |

#### 7.3.2 验收流程

```
企业自查 → 提交验收申请 → 资料审核 → 现场验收 → 专家评议 → 验收结论
```

**验收结论类型**：
1. **通过**：全额拨付剩余资金
2. **基本通过**：按比例拨付剩余资金
3. **不通过**：追回已拨付资金
4. **延期验收**：延期6个月，仅限一次

#### 7.3.3 验收注意事项

1. **时间节点**：项目完工后6个月内必须申请验收
2. **资金使用**：财政资金使用率≥90%
3. **变更管理**：重大变更需提前报批
4. **档案管理**：项目档案保存≥10年

### 7.4 常见申报问题与解决方案

#### 7.4.1 申报失败常见原因

| 原因类型 | 具体表现 | 占比 |
|---------|---------|-----|
| 资质不符 | 企业规模、年限不达标 | 30% |
| 材料不全 | 缺少关键证明材料 | 25% |
| 申报超期 | 错过申报截止日期 | 15% |
| 项目质量不高 | 技术水平、创新性不足 | 15% |
| 财务问题 | 审计报告问题、资金使用不规范 | 10% |
| 其他 | 信用记录、重复申报等 | 5% |

#### 7.4.2 提高申报成功率的建议

**申报前**：
1. 建立政策跟踪机制，及时获取申报信息
2. 提前准备基础材料，确保材料齐全
3. 聘请专业机构进行申报辅导
4. 与主管部门保持良好沟通

**申报中**：
1. 严格按照指南要求填写和准备材料
2. 突出项目亮点和竞争优势
3. 数据真实准确，避免夸大
4. 按时提交，预留修改时间

**申报后**：
1. 关注审核进度，及时补充材料
2. 配合现场核查，展示项目成效
3. 做好项目过程管理和档案管理
4. 按期完成项目验收
