## 二、广州市社会信用体系建设

### 2.1 政策背景与发展历程

广州市作为全国首批社会信用体系建设示范城市，近年来持续推进社会信用体系建设高质量发展。根据《广州市推进社会信用体系建设高质量发展 更好服务和融入新发展格局行动方案》，广州着力打造"信用广州"品牌，构建以信用为基础的新型监管机制。

#### 2.1.1 核心政策文件

1. **《广州市社会信用条例》**  
   - 施行时间：2022年6月1日
   - 主要内容：明确信用信息归集、披露、使用、权益保护等方面的规范要求
   - 适用对象：广州市行政区域内的信用信息管理及相关活动

2. **《广州市推进社会信用体系建设高质量发展 更好服务和融入新发展格局行动方案》**  
   - 发布单位：广州市发展和改革委员会
   - 主要目标：到2025年，基本建成与广州城市发展水平相适应的社会信用体系
   - 重点任务：完善信用制度体系、加强信用信息归集共享、创新信用监管机制

3. **《信用修复管理办法》2025年第36号令**  
   - 发布时间：2025年11月27日
   - 主要内容：规范信用信息修复工作，保护信用主体合法权益
   - 适用范围：失信行为纠正后的信用信息修复

### 2.2 信用评价体系

#### 2.2.1 行业信用评价

广州市建立了覆盖多领域的行业信用评价体系，主要包括：

**（一）纳税缴费信用评价**

政策依据：《纳税缴费信用管理办法》

评价对象：已纳入纳税缴费信用管理的经营主体

评价周期：每年一次年度评价

评价等级：A级（优秀）、B级（良好）、M级（新设）、C级（一般）、D级（较差）

**申报条件：**
- 完成税务登记并正常经营满一年
- 按规定报送纳税申报资料
- 无重大税收违法行为

**优惠措施：**
- A级纳税人可享受发票按需领用
- 税收优惠政策优先享受
- 办税绿色通道服务
- 联合激励措施

**（二）企业环境信用评价**

政策依据：《广州市企业环境信用评价修复指引（试行）》《企业环境信用评价办法（试行）》（环发〔2013〕150号）

评价对象：污染物排放总量大、环境风险高、生态环境影响大的企业

评价等级：环保诚信企业（绿牌）、环保良好企业（蓝牌）、环保警示企业（黄牌）、环保不良企业（红牌）

**评价内容：**
1. 污染防治设施运行情况
2. 污染物排放达标情况
3. 环境管理制度落实情况
4. 环境风险防控措施
5. 社会责任履行情况

**申报流程：**
1. 企业登录"广州市生态环境污染源企业管理系统"
2. 在线填写自评信息并上传证明材料
3. 生态环境部门组织核查评审
4. 公示评价结果
5. 发布年度评价等级

**（三）消防安全信用等级评定**

政策依据：《消防安全信用管理办法》

评价对象：机关、团体、企业、事业等单位

评价等级：A级（信用优秀）、B级（信用良好）、C级（信用一般）、D级（信用较差）

**评定标准：**
1. 消防安全责任制落实情况
2. 消防设施设备完好率
3. 火灾隐患整改情况
4. 消防安全培训开展情况
5. 火灾事故发生情况

**激励与惩戒：**
- A级单位：减少检查频次、优先评先评优
- D级单位：列为重点监管对象、增加检查频次

#### 2.2.2 公共信用综合评价

广州市依托广州市公共信用信息管理系统，开展公共信用综合评价，评价结果应用于：

1. **行政审批**：信用良好主体享受容缺受理、告知承诺等便利措施
2. **政府采购**：将信用状况作为供应商资格审查重要依据
3. **招投标**：在招标文件中明确信用评价结果应用规则
4. **财政补贴**：优先支持信用良好的企业和项目
5. **日常监管**：根据信用等级实施分级分类监管

### 2.3 信用应用场景

#### 2.3.1 "信易+"系列产品

**（一）信易贷**

平台：全国中小企业融资综合信用服务平台（广州站）

功能：为信用良好的中小微企业提供便捷融资服务

特点：
- 纯信用贷款，无需抵押担保
- 贷款利率优惠
- 审批速度快
- 贷款额度灵活

**申报条件：**
- 在广州市注册登记的企业
- 信用记录良好，无严重失信记录
- 经营状况正常，有还款能力
- 符合银行其他贷款条件

**申报流程：**
1. 登录全国中小企业融资综合信用服务平台
2. 完成企业信用信息授权
3. 选择合作银行和贷款产品
4. 在线提交贷款申请
5. 银行审批放款

**（二）信易租**

适用对象：信用良好的企业

功能：在场地租赁方面享受优惠待遇

优惠措施：
- 租金折扣
- 押金减免
- 租期灵活
- 服务优先

**（三）信易批**

适用对象：信用状况良好的企业和个人

功能：在行政审批中享受便利服务

便利措施：
- 容缺受理
- 告知承诺
- 绿色通道
- 并联审批

#### 2.3.2 信用惠民便企

**（一）信用+政务服务**

广州市将信用评价结果深度融入政务服务：

1. **信用承诺制审批**  
   对信用良好的申请人，在办理行政审批时可采用告知承诺方式，减少证明材料提交。

2. **容缺受理机制**  
   对信用良好的申请人，主要申报材料齐全、次要材料缺失的，可先行受理，在规定时间内补齐材料。

3. **绿色通道服务**  
   对信用等级高的企业和个人，提供优先办理、专人服务、延时服务等便利。

**（二）信用+市场监管**

1. **分级分类监管**  
   根据企业信用状况，采取差异化监管措施：
   - 信用优秀：减少检查频次，以自查为主
   - 信用良好：正常监管，定期抽查
   - 信用一般：加强监管，增加检查频次
   - 信用较差：重点监管，实施联合惩戒

2. **联合奖惩机制**  
   建立跨部门联合奖惩机制，对守信主体实施联合激励，对失信主体实施联合惩戒。

## 三、广东省信用服务产业

### 3.1 征信机构管理

#### 3.1.1 征信机构分类

根据《征信业管理条例》和相关监管规定，广东省征信机构主要分为以下类别：

**（一）企业征信机构**

业务范围：
- 企业信用信息采集、整理、保存、加工
- 企业信用报告出具
- 企业信用评级
- 企业信用评估咨询

设立条件：
- 注册资本不低于500万元人民币
- 有符合要求的信用信息数据库
- 有健全的组织机构和管理制度
- 有符合任职条件的高级管理人员
- 符合国家信息安全等级保护要求

**备案管理：**
广东省企业征信机构需向中国人民银行广州分行备案，备案材料包括：
1. 企业征信机构备案表
2. 营业执照复印件
3. 股东名册及出资比例
4. 组织架构和业务流程说明
5. 信用信息数据库建设情况
6. 信息安全管理制度
7. 高级管理人员任职资格证明

**（二）个人征信机构**

业务范围：
- 个人信用信息采集、整理、保存、加工
- 个人信用报告出具
- 个人信用评分

设立条件：
- 注册资本不低于5000万元人民币
- 主要股东信誉良好，最近3年无重大违法违规记录
- 有符合要求的信用信息数据库和安全保障措施
- 有健全的组织机构、内部控制和风险管理制度
- 符合国务院征信业监督管理部门规定的其他条件

**审批管理：**
个人征信机构须经中国人民银行审批，取得个人征信业务经营许可证后方可开展业务。

#### 3.1.2 征信机构监管要求

**信息采集规范：**

1. **合法原则**  
   征信机构采集信用信息应当遵循合法、正当、必要的原则，不得采集法律法规禁止采集的信息。

2. **告知原则**  
   征信机构采集个人信息应当告知信息主体本人，并取得其同意（法律法规另有规定的除外）。

3. **禁止采集信息**  
   征信机构不得采集下列个人信息：
   - 宗教信仰、基因、指纹、血型、疾病和病史
   - 法律、行政法规规定禁止采集的其他个人信息

**信息安全要求：**

1. 征信机构应当建立健全信息安全管理制度
2. 采取必要的技术措施保障信息安全
3. 对信用信息进行分类管理
4. 建立信息泄露应急处置机制
5. 定期开展信息安全风险评估

**信息主体权益保护：**

1. 知情权：信息主体有权查询自身信用报告
2. 异议权：信息主体有权对错误信息提出异议
3. 更正权：征信机构应当及时更正错误信息
4. 删除权：符合条件的信息主体有权要求删除相关信息

### 3.2 信用产品与服务

#### 3.2.1 信用报告产品

**（一）企业信用报告**

内容构成：
1. **基本信息**：企业名称、注册号、法定代表人、注册资本、经营范围等
2. **经营信息**：经营状态、股东信息、主要管理人员、分支机构等
3. **信用信息**：信贷记录、担保信息、欠税信息、行政处罚信息等
4. **公共信息**：资质认证、荣誉奖励、司法信息等
5. **声明信息**：企业自主申报的说明信息

获取渠道：
- 信用中国网站（免费查询基础信息）
- 信用广东网站
- 信用广州网站
- 商业征信机构（详细报告需付费）

**（二）个人信用报告**

内容构成：
1. **个人基本信息**：姓名、证件号码、婚姻状况、工作单位等
2. **信贷信息**：贷款、信用卡、担保等信贷记录
3. **公共信息**：社保、公积金、欠税、行政处罚等
4. **查询记录**：信用报告被查询的记录

获取渠道：
- 中国人民银行征信中心官网
- 人民银行柜台查询（每年2次免费）
- 商业银行网点自助查询机
- "粤省事"小程序

#### 3.2.2 信用评级产品

**（一）企业主体信用评级**

评级对象：企业法人

评级等级：AAA、AA、A、BBB、BB、B、CCC、CC、C（AAA为最高等级）

评级用途：
- 债券发行
- 银行贷款
- 招投标
- 商业合作

**（二）债券信用评级**

评级对象：企业债券、公司债券、债务融资工具等

评级内容：
- 发行人主体信用状况
- 债券条款分析
- 偿债能力评估
- 风险因素分析

**（三）绿色债券评估认证**

评估内容：
- 绿色项目符合性
- 环境效益评估
- 资金用途合规性
- 持续跟踪评价

#### 3.2.3 信用增值服务

**（一）信用风险预警**

服务内容：
- 重点客户信用监控
- 信用状况变化提醒
- 行业风险分析报告
- 定制化风险预警方案

**（二）信用管理咨询**

服务内容：
- 企业信用制度建设
- 应收账款管理优化
- 客户信用评估体系
- 信用风险防控方案

**（三）信用修复指导**

服务内容：
- 失信信息分析
- 修复方案制定
- 修复材料指导
- 修复进度跟踪

**重要提醒：**
信用广州网站和信用广东网站均为公众提供**免费公益性**公共信用服务，从未组织或委托任何机构开展有偿公共信用服务辅导或培训。企业应谨防上当受骗。

### 3.3 信用服务产业发展政策

#### 3.3.1 广东省促进信用服务产业发展措施

**（一）培育信用服务市场**

1. 扩大信用信息开放共享
2. 鼓励信用产品创新应用
3. 培育引进信用服务机构
4. 建立信用服务行业标准

1. 加强征信机构监管
2. 建立信用服务评价体系
3. 严厉打击违法违规行为
4. 保护信息主体合法权益

**（三）支持信用服务机构发展**

1. 税收优惠政策
2. 场地租金补贴
3. 人才引进支持
4. 融资便利措施

## 四、广州市企业信用管理

### 4.1 信用修复机制

#### 4.1.1 行政处罚信息信用修复

**政策依据：**
《失信行为纠正后的信用信息修复管理办法（试行）》（国家发展改革委令2023年第58号）

**修复对象：**
在"信用中国"网站、地方信用门户网站和国家企业信用信息公示系统公示的行政处罚信息

**修复前提条件：**

1. **公示期限要求**
   - 以简易程序作出的对法人和非法人组织的行政处罚信息，"信用中国"网站不进行归集和公示
   - 以普通程序作出的行政处罚信息，最短公示期为三个月，最长公示期为三年
   - 涉及食品、药品、特种设备、安全生产、消防领域的行政处罚信息，最短公示期为一年
   - 仅被处以警告、通报批评的行政处罚信息，不予公示

2. **修复条件**
   - 完全履行行政处罚决定规定的义务，纠正违法行为
   - 达到最短公示期限
   - 公开作出信用承诺

**特别规定：**
- 法律、法规对相关违法违规行为规定了附带期限的惩戒措施的，在相关期限届满前，行政处罚信息不得提前终止公示
- 同一行政处罚决定涉及多种处罚类型的，其公示期限以期限最长的类型为准

**办理形式：**
- 企业、事业单位、社会组织：在"信用中国"网站提出修复申请
- 个体工商户：在"信用广东"网站提出修复申请

**材料清单：**

材料一：《失信行为纠正后的信用信息修复业务办理授权委托书》或者《法定代表人身份证明书》

材料二：行政处罚决定书明确的责任义务已履行完毕的证明材料（如缴交罚款的收据或《失信行为纠正后的信用信息修复表》、行政处罚机关出具的其他相关整改证明材料）

材料三：《行政处罚信息信用信息修复承诺书》

**注：** 行政处罚机关出具的整改证明材料需清晰写明联系人及联系方式，联系人需为本单位在编业务人员，联系电话为单位座机号码。

**修复流程：**

1. **网站在线申请**  
   进入相应网站首页，在搜索栏输入主体名称或者统一社会信用代码，在行政处罚一栏找到需要修复的行政处罚决定书文号，点击在线申请修复，按照申请页面提示的信息，上传相关材料。根据行政处罚机关所在地选择信用修复受理地点。

2. **受理核查**  
   法人和非法人组织提交的修复申请材料，符合《失信行为纠正后的信用信息修复管理办法》规定的，由市信用主管部门受理和初审，对不符合条件的予以退回，经初审合格的，提交省进行复核，复核通过的提交国家进行终审。

3. **修复**  
   经三级审核，对未履行相关法律责任、未按要求提供相关材料的，不予修复；对已经纠正违法行为、履行法律责任、完成有效整改的，"信用中国"集中处理下线。

4. **数据同步**  
   信用平台网站与国家企业信用信息公示系统、有关行业部门信用信息系统建立信用信息修复共享机制，自"信用中国"做出信用修复决定之日起3个工作日内，将修复信息共享至相关系统。

**办理时限：**
市级受理初审1个工作日（自企业提交修复申请的工作日24小时内），省级复审+国家终审5个工作日，共6个工作日。

#### 4.1.2 严重失信主体名单移出

**（一）失信被执行人名单**

**政策依据：**
《最高人民法院关于公布失信被执行人名单信息的若干规定》

**移出条件：**
具有下列情形之一的，人民法院应当在三个工作日内删除失信信息：
1. 被执行人已履行生效法律文书确定的义务或人民法院已执行完毕的
2. 当事人达成执行和解协议且已履行完毕的
3. 申请执行人书面申请删除失信信息，人民法院审查同意的
4. 终结本次执行程序后，通过网络执行查控系统查询被执行人财产两次以上，未发现有可供执行财产，且申请执行人或者其他人未提供有效财产线索的
5. 因审判监督或破产程序，人民法院依法裁定对失信被执行人中止执行的
6. 人民法院依法裁定不予执行的
7. 人民法院依法裁定终结执行的

**申请材料：**

法人：
1. 信用承诺书
2. 法定代表人身份证
3. 加盖法人公章的营业执照复印件
4. 法定代表人身份证明书
5. 加盖法人公章的授权委托书及委托代理人身份证明材料（如有）

自然人：
1. 信用承诺书
2. 自然人身份证明（签名加指模）
3. 授权委托书及委托代理人身份证明材料（如有）

**办理流程：**
1. 失信被执行人向执行法院提出书面纠正申请，由执行法院进行审查是否予以纠正
2. 如执行法院予以纠正的，申请人可向执行法院申请出具信用修复证明书。如执行法院决定驳回纠正申请的，申请人可向上级法院申请复议

**办理时限：**
公民、法人或其他组织对被纳入失信被执行人名单申请纠正的，执行法院应当自收到书面纠正申请之日起十五日内审查，理由成立的，应当在三个工作日内纠正；理由不成立的，决定驳回。公民、法人或其他组织对驳回决定不服的，可以自决定书送达之日起十日内向上一级人民法院申请复议。上一级人民法院应当自收到复议申请之日起十五日内作出决定。

#### 4.1.4 信用评价等级提升

**修复条件：**
经营主体发生纳税缴费失信行为，符合相应纳税缴费信用修复条件的，可按以下方式进行修复：
1. 失信行为已记入纳税缴费信用评价结果的，经营主体可在次年年度评价前填写《纳税缴费信用修复申请表》向主管税务机关申请信用修复
2. 失信行为尚未记入纳税缴费信用评价结果的，经营主体无需提出申请，税务机关在开展年度评价时根据失信行为纠正情况统一更新结果

**注意事项：**
经营主体应对信用修复申请内容的真实性作出承诺。税务机关发现经营主体虚假承诺的，撤销相应的纳税缴费信用修复结果，并按照纳税缴费信用评价指标，对虚假承诺行为予以扣分。

**办理形式：**
- 线下办理：办税服务厅（场所）
- 网上办理："广东省电子税务局"官网-"我要办税"模块

**办理时限：**
自受理申请之日起15个工作日内完成修复

**（二）企业环境信用评价修复**

**政策依据：** 《广州市企业环境信用评价修复指引（试行）》

**修复对象：**
上一年度环境信用评价等级为"环保警示企业"（黄牌企业）或"环保不良企业"（红牌企业）的企业

**修复条件：**
企业环境信用评价结果公布之日起6个月内，"环保警示企业"（黄牌企业）或"环保不良企业"（红牌企业）主动改善环境行为、实施有效整改的，可向市生态环境部门提交修复申请并提供相应辅证材料。

**不予修复情形：**
1. 本评价周期内已经进行过环境信用修复（包括"即时修复"）的
2. 提供的信用修复申请材料弄虚作假或未按要求提供完整修复材料的
3. 市生态环境部门依法依规认定存在不能实施信用修复的其他情形的

**材料清单：**
- 企业环境信用修复辅证材料一览表
- 企业环境信用修复承诺书

**修复流程：**

1. **在线申请**  
   企业登录"广州市生态环境污染源企业管理系统"，在线提交修复申请、作出信用修复承诺，并在线上传相关证明材料。

2. **受理核查**  
   市生态环境部门受理申请后于15个工作日内按照企业环境信用评价指标和评分方法，对企业从评价年度年初至提出信用修复申请期间环境行为进行核查。需现场核查的，由生态环境部门开展现场核查，提出初步处理意见。

   生态环境部门经审查后，对未履行相关法律责任、未按要求完成整改的信用修复申请，不予修复；对已经改善环境行为、履行法律责任、完成有效整改的信用修复申请，予以信用修复公示。

3. **公示**  
   在广州市生态环境部门网站进行信用修复公示，公示期为7天。公示期间若有异议，市生态环境部门于5个工作日内进行核实处理，作出处理决定并通知申请单位。

4. **修复**  
   对通过信用修复公示的修复申请，市生态环境部门及时调整该申请单位上一年度的环境信用等级，并推送至广州市公共信用信息管理系统及广州市政府信息共享平台。

### 4.2 信用承诺制度

#### 4.2.1 信用承诺类型

**（一）主动型信用承诺**

**适用场景：**
企业主动向社会作出信用承诺，接受社会监督

**承诺内容：**
- 依法诚信经营
- 遵守行业规范
- 履行社会责任
- 接受社会监督

**办理方式：**
登录信用广州网站在线提交

**（二）审批替代型信用承诺**

**适用场景：**
在行政审批中，申请人以书面形式承诺符合审批条件，审批部门依据承诺直接作出审批决定

**适用事项：**
已在各行业主管部门公布的事项清单中列明

**承诺内容：**
- 符合审批条件
- 如实提交材料
- 承担不实承诺的法律责任

**（三）信用修复型信用承诺**

**适用场景：**
信用主体在申请信用修复时，承诺已纠正失信行为并维持良好信用

**承诺内容：**
- 已完全履行法定义务
- 已纠正失信行为
- 承诺今后诚信守法经营
- 愿意承担违反承诺的责任

**（四）行业自律型信用承诺**

**适用场景：**
行业协会会员向协会作出的信用承诺

**承诺内容：**
- 遵守协会章程和行业规范
- 维护行业良好秩序
- 诚信经营，公平竞争

#### 4.2.2 信用承诺申报流程

1. **选择承诺类型**  
   根据实际需要选择相应的信用承诺类型

2. **准备承诺材料**  
   根据承诺类型准备相应材料，包括：
   - 信用承诺书
   - 企业营业执照复印件
   - 法定代表人身份证明
   - 其他相关证明材料

3. **在线提交申请**  
   登录信用广州网站，进入信用承诺申报页面，填写信息并上传材料

4. **审核公示**  
   主管部门审核通过后，在信用广州网站公示

5. **承诺履行**  
   企业应严格履行承诺事项，接受社会监督

### 4.3 企业信用管理制度建议

#### 4.3.1 建立健全内部信用管理制度

**（一）信用信息共享**

合作内容：
1. 建立跨境信用信息交换机制

## 七、申报流程详解

### 7.1 信用修复申报流程

#### 7.1.1 行政处罚信息信用修复流程

```
第一步：确认修复条件
├── 检查是否达到最短公示期限
├── 确认已完全履行行政处罚决定
└── 准备信用承诺

第二步：准备申请材料
├── 授权委托书或法定代表人身份证明
├── 义务履行证明材料（罚款收据、整改证明等）
└── 信用修复承诺书

第三步：在线提交申请
├── 登录信用中国网站（企业/事业单位/社会组织）
├── 或登录信用广东网站（个体工商户）
├── 搜索企业名称或统一社会信用代码
├── 找到需修复的行政处罚记录
├── 点击"在线申请修复"
└── 上传申请材料

第四步：等待审核
├── 市级初审（1个工作日）
├── 省级复核（约2个工作日）
└── 国家终审（约3个工作日）

第五步：结果反馈
├── 审核通过：信息停止公示
├── 审核不通过：退回并说明原因
└── 数据同步至相关系统（3个工作日内）
```

#### 7.1.2 纳税信用修复流程

```
第一步：确认修复资格
├── 查看纳税信用评价结果
├── 确认失信行为已纠正
└── 判断是否在年度评价前

第二步：准备申请材料
├── 纳税缴费信用修复申请表
├── 失信行为纠正证明
└── 其他相关证明材料

第三步：提交申请
├── 方式一：办税服务厅窗口
└── 方式二：广东省电子税务局

第四步：等待审核
├── 税务机关核实材料（15个工作日内）
├── 作出是否调整等级决定
└── 反馈审核结果
```

#### 7.1.3 企业环境信用修复流程

```
第一步：确认修复条件
├── 上一年度评价为黄牌或红牌企业
├── 评价结果公布后6个月内
└── 已主动改善环境行为并有效整改

第二步：准备申请材料
├── 企业环境信用修复承诺书
├── 环境行为改善证明材料
└── 其他辅证材料（参照辅证材料一览表）

第三步：在线提交申请
├── 登录广州市生态环境污染源企业管理系统
├── 在线填写修复申请
├── 上传承诺书和证明材料
└── 提交申请

第四步：受理核查
├── 生态环境部门受理（15个工作日内核查）
├── 必要时开展现场核查
└── 提出初步处理意见

第五步：公示（7天）
├── 在生态环境部门网站公示
├── 接受社会监督
└── 处理异议（5个工作日）

第六步：结果反馈
├── 通过：调整信用等级并推送信息
└── 不通过：说明原因
```

### 7.2 信用承诺申报流程

#### 7.2.1 主动型信用承诺流程

```
第一步：登录平台
└── 访问信用广州网站

第二步：选择承诺类型
├── 进入"信用承诺"栏目
└── 选择"主动型信用承诺"

第三步：填写信息
├── 填写企业基本信息
├── 选择承诺事项
└── 上传相关材料

第四步：提交审核
├── 核对信息无误
└── 提交申请

第五步：公示
├── 审核通过后在网站公示
└── 接受社会监督
```

### 7.3 信用报告查询流程

#### 7.3.1 企业信用报告查询

**方式一：信用中国网站**
1. 访问信用中国网站（www.creditchina.gov.cn）
2. 在搜索框输入企业名称或统一社会信用代码
3. 点击查询，查看企业信用信息
4. 基础信息免费查询

**方式二：信用广州网站**
1. 访问信用广州网站（credit.gz.gov.cn）
2. 在搜索框输入企业名称或统一社会信用代码
3. 查询企业信用信息
4. 可查看信用承诺、行政处罚等信息

**方式三：商业征信机构**
1. 选择正规征信机构
2. 提交查询申请
3. 支付服务费用
4. 获取详细信用报告

#### 7.3.2 个人信用报告查询

**方式一：人民银行柜台**
1. 携带身份证到人民银行分支机构
2. 填写查询申请表
3. 每年可免费查询2次
4. 现场获取信用报告

**方式二：人民银行征信中心网站**
1. 访问中国人民银行征信中心官网
2. 注册并实名认证
3. 提交查询申请
4. 24小时后获取报告

**方式三：商业银行网点**
1. 前往合作银行网点
2. 使用自助查询机
3. 刷身份证进行人脸识别
4. 打印信用报告
