## 二、广州市退役军人就业创业政策

### 2.1 就业扶持政策

#### 2.1.1 自主就业退役士兵一次性经济补助

**【政策依据】**  
《广州市自主就业退役士兵一次性经济补助发放办法》（穗退役军人规字〔2020〕1号）

**【补助对象】**  
- 广州市户籍退役士兵
- 退役时选择自主就业方式安置
- 不符合由政府安排工作条件的义务兵和士官

**【补助标准】**  
1. **义务兵**: 按照广州市上年度城镇居民人均可支配收入的2倍发放，2024年标准约为35,000元/人
2. **士官**: 在义务兵补助基础上，每多服役1年增发补助标准的10%
   - 下士（服役5年）：约42,000元
   - 中士（服役8年）：约52,500元
   - 上士（服役12年）：约66,500元

**【申报条件】**  
- 已完成退役报到手续
- 选择自主就业安置方式
- 未领取国家安排工作相关待遇

**【申报材料】**  
1. 《广州市自主就业退役士兵一次性经济补助申请表》
2. 退出现役证件原件及复印件
3. 身份证原件及复印件
4. 户口簿原件及复印件
5. 本人银行账户信息
6. 退役报到证明

**【申报流程】**  
1. 退役后30日内到户籍地街道（镇）退役军人服务站报到
2. 填写《自主就业退役士兵登记表》
3. 提交补助申请材料
4. 街道（镇）初审，区退役军人事务局复核
5. 公示7个工作日
6. 补助金发放至个人账户

**【办理时限】**  
- 材料齐全后30个工作日内完成审核
- 补助金于审核通过后次月发放

**【注意事项】**  
- 补助金应在退役后1年内申请，逾期视为自动放弃
- 已领取国家安排工作相关待遇的不再享受此项补助
- 补助金不计入家庭收入，不影响低保等社会救助待遇

---

#### 2.1.2 就业服务与岗位推荐

**【政策依据】**  
《广州市退役军人就业创业服务管理办法》

**【服务内容】**  

**1. 专场招聘会**
- 每季度至少举办1次退役军人专场招聘会
- 每年举办2次大型退役军人就业专场活动
- 参会企业需提供适合退役军人的岗位

**2. 岗位定向招聘**
- 市属国有企业按年度招聘计划5%比例定向招聘退役军人
- 机关事业单位工勤岗位优先招用退役军人
- 社区工作者、辅警等岗位适当放宽条件招用退役军人

**3. 就业信息推送**
- 建立退役军人就业信息服务平台
- 每周推送不少于5条就业信息
- 提供一对一就业指导服务

**4. 就业见习**
- 认定退役军人就业见习基地
- 见习期间给予见习补贴（当地最低工资标准的80%）
- 见习期限3-12个月

**【申请条件】**  
- 已完成退役报到并登记就业需求
- 有就业意愿的退役军人

**【申请方式】**  
- 线上：广州市退役军人服务管理系统
- 线下：户籍地街道（镇）退役军人服务站

---

#### 2.1.3 公益性岗位安置

**【政策依据】**  
《广州市就业困难人员公益性岗位管理办法》

**【岗位类型】**  
- 社区服务岗位
- 城市管理协管岗位
- 治安巡逻岗位
- 环境卫生岗位
- 公共设施维护岗位

**【安置条件】**  
- 符合就业困难人员条件的退役军人
- 连续失业6个月以上
- 有就业意愿且能够正常工作

**【待遇标准】**  
- 工资不低于当地最低工资标准
- 按规定缴纳社会保险
- 岗位补贴：按当地最低工资标准的100%给予补贴
- 社保补贴：按实际缴纳社保费用的100%给予补贴

**【安置期限】**  
- 一般不超过3年
- 距退休不足5年的可延长至退休

### 2.2 创业补贴政策

#### 2.2.1 创业担保贷款

**【政策依据】**  
《广州市创业担保贷款实施办法》

**【贷款对象】**  
- 退役5年内的自主就业退役士兵
- 在广州市创办企业、个体工商户、农民专业合作社等

**【贷款额度】**  
- 个人创业：最高50万元
- 合伙创业：最高220万元
- 小微企业：最高500万元

**【贷款期限】**  
- 个人贷款：最长3年
- 小微企业贷款：最长2年
- 可申请展期1次，期限不超过1年

**【贷款利率】**  
- 贷款利率上限为LPR+150BP
- 个人贷款财政全额贴息
- 小微企业贷款财政贴息50%

**【申报条件】**  
1. 退役5年内的自主就业退役士兵
2. 在广州市行政区域内创办经营实体
3. 经营实体正常经营6个月以上
4. 无不良信用记录
5. 除助学贷款、扶贫贷款、住房贷款、购车贷款、5万元以下小额消费贷款外，无其他贷款

**【申报材料】**  
1. 《创业担保贷款申请表》
2. 退出现役证件原件及复印件
3. 营业执照副本原件及复印件
4. 经营场所证明（租赁合同或产权证）
5. 经营流水或财务报表
6. 身份证、户口簿原件及复印件
7. 个人征信报告
8. 担保材料（如需）

**【办理流程】**  
1. 向户籍地街道（镇）人社服务中心提交申请
2. 人社部门资格审核（5个工作日）
3. 经办银行贷前调查（10个工作日）
4. 银行审批并签订贷款合同
5. 贷款发放

**【注意事项】**  
- 贷款资金仅用于生产经营，不得用于消费、投资等用途
- 按时还款可再次申请
- 逾期还款将影响个人征信及后续贷款资格

#### 2.2.2 一次性创业资助

**【政策依据】**  
《广州市创业补贴申领管理办法》

**【资助标准】**  
- 一次性发放10000元

**【申报条件】**  
1. 退役5年内的自主就业退役士兵
2. 首次在广州市创办企业、个体工商户
3. 正常经营6个月以上
4. 未领取过同类创业补贴

**【申报材料】**  
1. 《一次性创业资助申请表》
2. 营业执照副本原件及复印件
3. 退出现役证件原件及复印件
4. 身份证原件及复印件
5. 经营证明材料（纳税证明或银行流水）
6. 开户许可证或基本存款账户信息

**【办理流程】**  
1. 登录"广东政务服务网"在线申请
2. 或到户籍地街道（镇）人社服务中心窗口办理
3. 区人社部门审核（20个工作日）
4. 公示5个工作日
5. 补贴发放至企业银行账户

#### 2.2.3 创业场地租金补贴

**【政策依据】**  
《广州市创业孵化基地管理办法》

**【补贴标准】**  
- 每年最高6000元
- 最长补贴3年
- 实际租金低于6000元的按实际租金补贴

**【申报条件】**  
1. 退役5年内的自主就业退役士兵
2. 在广州市级以上创业孵化基地内租用场地
3. 正常经营并缴纳租金

**【申报材料】**  
1. 《创业场地租金补贴申请表》
2. 营业执照副本复印件
3. 场地租赁合同及租金发票
4. 创业孵化基地入驻证明
5. 退出现役证件复印件
6. 身份证复印件

**【办理流程】**  
1. 每年1月、7月集中受理
2. 向区人社部门提交申请
3. 审核通过后次月发放

#### 2.2.4 创业带动就业补贴

**【政策依据】**  
《广州市就业创业补贴申领管理办法》

**【补贴标准】**  
- 招用3人（含）以下：每人2000元
- 招用3人以上：每增加1人增发3000元
- 补贴总额最高不超过3万元

**【申报条件】**  
1. 退役5年内的创业者创办的企业
2. 招用法定劳动年龄内的劳动者
3. 与招用人员签订1年以上劳动合同
4. 连续缴纳社会保险费6个月以上

**【申报材料】**  
1. 《创业带动就业补贴申请表》
2. 营业执照副本复印件
3. 招用人员身份证复印件
4. 劳动合同复印件
5. 社会保险缴费证明
6. 退出现役证件复印件

**【办理流程】**  
- 每季度首月1-15日集中受理
- 审核通过后30个工作日内发放

### 2.3 税收优惠政策

#### 2.3.1 退役士兵创业税收扣减

**【政策依据】**  
《关于进一步支持和促进重点群体创业就业有关税收政策的通知》（财税〔2023〕15号）

**【优惠内容】**  
- 自主就业退役士兵从事个体经营
- 3年内按每户每年24000元为限额
- 依次扣减其当年实际应缴纳的增值税、城市维护建设税、教育费附加、地方教育附加和个人所得税

**【享受条件】**  
1. 自主就业退役士兵
2. 从事个体经营（不含个人独资企业）
3. 办理个体工商户登记
4. 申报纳税时持退出现役证件

**【办理方式】**  
- 税务申报时直接享受
- 留存退出现役证件备查

#### 2.3.2 企业招用退役士兵税收优惠

**【优惠内容】**  
- 企业招用自主就业退役士兵
- 3年内按每人每年7800元定额标准
- 依次扣减增值税、城市维护建设税、教育费附加、地方教育附加和企业所得税

**【享受条件】**  
1. 与退役士兵签订1年以上期限劳动合同
2. 依法缴纳社会保险费

**【办理方式】**  
- 持退出现役证件、劳动合同、社保缴纳证明
- 向主管税务机关备案后享受

## 三、广东省退役军人教育培训政策

### 3.1 免费职业技能培训

**【政策依据】**  
《广东省退役军人职业技能培训实施办法》（粤退役军人规〔2021〕2号）

**【培训对象】**  
- 自主就业退役士兵
- 退役后1年内可免费参加一次职业技能培训

**【培训项目】**  

**1. 热门技能培训专业**
- 汽车驾驶员、汽车维修工
- 电工、焊工、钳工
- 物业管理员、保安员
- 电子商务师、网络营销师
- 中式烹调师、西式面点师
- 消防设施操作员
- 计算机程序设计员
- 物流师、仓储管理员

**2. 新兴产业培训专业**
- 无人机驾驶员
- 工业机器人操作员
- 新能源汽车维修
- 智能家居安装与维护
- 大数据分析师
- 云计算工程师

**【培训费用】**  
- 培训费：全额补贴，最高8000元/人
- 鉴定费：初次鉴定费用全额补贴
- 教材费：按实际费用报销，最高500元

**【生活补助】**  
- 培训期间按当地城镇最低生活保障标准发放生活补助
- 广州市标准：每月约1200元
- 按实际培训天数计算

**【申报条件】**  
1. 自主就业退役士兵
2. 退役后1年内
3. 未参加过同类免费培训

**【申报材料】**  
1. 《退役军人职业技能培训申请表》
2. 退出现役证件原件及复印件
3. 身份证原件及复印件
4. 户口簿原件及复印件
5. 1寸免冠照片2张

**【申报流程】**  
1. 退役报到时填写培训意愿
2. 选择定点培训机构和专业
3. 区退役军人事务部门审核
4. 到培训机构参加培训
5. 培训结束后参加技能鉴定
6. 生活补助按月发放

**【定点培训机构】**（广州市部分）
- 广州市技师学院
- 广州市工贸技师学院
- 广州市轻工技师学院

### 7.3 常见问题解答

#### Q1: 退役后多久内需要报到？
**A:** 退役后30日内到户籍地街道（镇）退役军人服务站报到，逾期可能影响相关待遇享受。

#### Q2: 优待证丢失怎么办？
**A:** 携带身份证到户籍地街道退役军人服务站办理挂失和补办手续，补办周期约15个工作日。

#### Q3: 创业补贴可以重复申请吗？
**A:** 不同类型的创业补贴可以叠加申领（如一次性创业资助和场地租金补贴），但同类补贴每人只能享受一次。

#### Q4: 技能培训期间的生活补助如何发放？
**A:** 生活补助按月发放，通常于每月15日前打入个人银行账户，需在培训机构签到打卡满规定天数。

#### Q5: 非广州户籍退役军人能否享受广州的政策？
**A:** 部分政策（如免费技能培训、就业服务）对非户籍退役军人开放，但补贴类政策通常要求具有广州市户籍。建议咨询户籍地退役军人事务部门。

#### Q6: 创业贷款逾期还款有什么后果？
**A:** 逾期还款将影响个人征信记录，可能导致财政停息，并影响后续贷款资格。建议按时还款或提前申请展期。

#### Q7: 港澳台退役军人在广州能否享受优待？
**A:** 原籍广州的港澳台退役军人可申请广州市退役军人优待证，享受相应优待服务。

#### Q8: 退役大学生士兵考研加分政策如何申请？
**A:** 在研究生报名时勾选"退役大学生士兵"选项，现场确认时提交退出现役证件，初试成绩自动加分。
