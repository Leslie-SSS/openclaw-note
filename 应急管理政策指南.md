## 二、广州市应急管理政策

### 2.1 政策框架

#### 2.1.1 核心政策文件

**《广州市安全生产条例》**
- 发布机关：广州市人民代表大会常务委员会
- 施行日期：2021年修订版
- 适用范围：广州市行政区域内从事生产经营活动的单位

**核心要点：**
1. 生产经营单位是安全生产的责任主体
2. 主要负责人对本单位安全生产工作全面负责
3. 建立健全安全生产责任制和规章制度
4. 保证安全生产投入的有效实施
5. 督促、检查安全生产工作，消除事故隐患

**《广州市突发事件应对条例》**
- 发布机关：广州市人民代表大会常务委员会
- 施行日期：2020年
- 调整范围：突发事件的预防与应急准备、监测与预警、应急处置与救援、事后恢复与重建

**主要制度：**
1. 突发事件应急预案制度
2. 突发事件风险评估制度
3. 应急资源储备制度
4. 应急救援队伍管理制度
5. 应急演练制度

#### 2.1.2 政策目标

**总体目标：**
到2025年，建成与国家中心城市地位相适应的应急管理体制和机制，形成统一指挥、专常兼备、反应灵敏、上下联动的应急管理体制，应急管理能力和水平显著提升。

**具体指标：**
1. 生产安全事故死亡人数下降15%以上
2. 亿元GDP生产安全事故死亡率下降30%以上
3. 工矿商贸就业人员十万人生产安全事故死亡率下降20%以上
4. 特大事故有效遏制，重大事故起数下降15%以上
5. 自然灾害造成的直接经济损失占GDP比重控制在1%以内

### 2.2 安全生产政策

#### 2.2.1 安全生产责任制

**三级责任体系：**

**市级层面：**
- 成立市安全生产委员会，市长担任主任
- 设立安全生产专项资金，每年不少于5000万元
- 建立安全生产目标责任考核制度
- 实施安全生产"一票否决"制度

**区级层面：**
- 各区设立安全生产委员会
- 配备专职安全生产监管人员
- 建立安全生产网格化管理体系
- 落实属地监管责任

**企业层面：**
- 建立安全生产责任制度
- 设置安全生产管理机构或配备专职人员
- 落实安全生产投入
- 开展安全生产教育培训

**企业负责人责任清单：**
1. 建立、健全本单位安全生产责任制
2. 组织制定本单位安全生产规章制度和操作规程
3. 保证本单位安全生产投入的有效实施
4. 督促、检查本单位的安全生产工作，及时消除生产安全事故隐患
5. 组织制定并实施本单位的生产安全事故应急救援预案
6. 及时、如实报告生产安全事故

#### 2.2.2 安全生产专项资金

**《广州市安全生产专项资金管理办法》**

**支持方向：**
1. 安全生产标准化建设
2. 安全隐患排查治理
3. 安全生产技术改造
4. 应急救援能力建设
5. 安全生产宣传教育培训
6. 安全生产科技研发与推广

**申报条件：**
1. 在广州市依法登记注册的企业或社会组织
2. 具有独立法人资格
3. 财务管理规范，制度健全
4. 近三年内无重大安全生产责任事故
5. 符合年度资金支持重点方向

**奖励标准：**

| 项目类别 | 奖励标准 | 最高限额 |
|---------|---------|---------|
| 一级安全生产标准化达标 | 一次性奖励50万元 | 50万元 |
| 二级安全生产标准化达标 | 一次性奖励20万元 | 20万元 |
| 重大隐患治理项目 | 按投资额30%补助 | 100万元 |
| 安全生产技术改造 | 按设备投资额20%补助 | 50万元 |
| 应急救援装备购置 | 按购置费用30%补助 | 30万元 |
| 安全生产信息化建设 | 按投入费用40%补助 | 30万元 |
| 安全生产培训基地建设 | 按建设费用30%补助 | 50万元 |

**申报材料：**
1. 广州市安全生产专项资金申请表
2. 企业营业执照副本复印件
3. 组织机构代码证复印件
4. 税务登记证复印件
5. 近三年安全生产情况说明
6. 项目可行性研究报告
7. 项目投资预算及资金来源证明
8. 其他相关证明材料

**申报流程：**
1. 项目单位向所在区应急管理局提交申请
2. 区应急管理局初审并推荐
3. 市应急管理局组织专家评审
4. 公示拟支持项目
5. 下达资金计划
6. 资金拨付

**申报时间：**
每年3月1日至4月30日（以当年公告为准）

#### 2.2.3 安全生产许可制度

**《安全生产许可证条例》实施办法**

**许可范围：**
1. 矿山企业
2. 建筑施工企业
3. 危险化学品生产企业
4. 烟花爆竹生产企业
5. 民用爆破器材生产企业

**许可条件：**
1. 建立、健全安全生产责任制，制定完备的安全生产规章制度和操作规程
2. 安全投入符合安全生产要求
3. 设置安全生产管理机构，配备专职安全生产管理人员
4. 主要负责人和安全生产管理人员经考核合格
5. 特种作业人员经有关业务主管部门考核合格，取得特种作业操作资格证书
6. 从业人员经安全生产教育和培训合格
7. 依法参加工伤保险，为从业人员缴纳保险费
8. 厂房、作业场所和安全设施、设备、工艺符合有关安全生产法律、法规、标准和规程的要求
9. 有职业危害防治措施，并为从业人员配备符合国家标准或者行业标准的劳动防护用品
10. 依法进行安全评价
11. 有重大危险源检测、评估、监控措施和应急预案
12. 有生产安全事故应急救援预案、应急救援组织或者应急救援人员，配备必要的应急救援器材、设备
13. 法律、法规规定的其他条件

**许可程序：**
1. 申请：向应急管理部门提交申请材料
2. 受理：材料齐全、符合要求的予以受理
3. 审查：对申请材料进行审查，必要时进行现场核查
4. 决定：自受理之日起45日内作出准予或不予许可的决定
5. 发证：准予许可的，自决定之日起10日内颁发安全生产许可证

**许可证管理：**
- 有效期：3年
- 延期：有效期届满前3个月申请延期
- 变更：企业名称、地址、法定代表人等变更后10日内申请变更
- 注销：企业终止经营或不再从事许可范围内的活动

### 2.3 应急响应政策

#### 2.3.1 应急预案管理

**《广州市突发事件应急预案管理办法》**

**预案体系：**
1. 总体应急预案
2. 专项应急预案
3. 部门应急预案
4. 企事业单位应急预案
5. 重大活动应急预案

**编制要求：**

**总体应急预案：**
- 由市、区人民政府组织编制
- 明确应急组织体系
- 规定应急响应级别和程序
- 建立应急资源保障体系

**专项应急预案：**
- 由相关职能部门组织编制
- 针对特定类型突发事件
- 明确预警级别和响应措施
- 规定应急资源调配程序

**企事业单位应急预案：**
- 结合本单位实际情况编制
- 明确应急组织机构和职责
- 规定应急处置程序
- 建立应急资源清单

**预案评审：**
1. 形式评审：审查预案格式、内容完整性
2. 实质评审：评估预案科学性、可操作性
3. 专家评审：组织相关领域专家进行评审
4. 实践检验：通过演练检验预案效果

**预案备案：**
- 市级总体应急预案报省政府备案
- 区级总体应急预案报市政府备案
- 专项应急预案报上级主管部门备案
- 企业应急预案报所在地应急管理部门备案

**预案演练：**
- 综合应急预案演练每年至少1次
- 专项应急预案演练每半年至少1次
- 现场处置方案演练每季度至少1次

#### 2.3.2 应急响应机制

**《广州市突发事件应急响应规程》**

**响应级别划分：**

**Ⅰ级响应（特别重大）：**
- 启动条件：死亡30人以上，或重伤100人以上，或直接经济损失1亿元以上
- 响应主体：市政府统一指挥
- 响应措施：全市总动员，请求省级支援

**Ⅱ级响应（重大）：**
- 启动条件：死亡10-29人，或重伤50-99人，或直接经济损失5000万元-1亿元
- 响应主体：市政府统一指挥
- 响应措施：市级资源统筹，必要时请求省级支援

**Ⅲ级响应（较大）：**
- 启动条件：死亡3-9人，或重伤10-49人，或直接经济损失1000万-5000万元
- 响应主体：区政府统一指挥
- 响应措施：区级资源统筹，市级应急力量待命

**Ⅳ级响应（一般）：**
- 启动条件：死亡1-2人，或重伤10人以下，或直接经济损失1000万元以下
- 响应主体：区级部门或街道（镇）指挥
- 响应措施：区级或以下力量处置

**响应程序：**
1. 信息接报：24小时值班制度，接到报告后立即核实
2. 先期处置：事发单位立即启动应急响应，采取必要措施
3. 级别研判：根据初步信息研判事件级别
4. 响应启动：按规定启动相应级别响应
5. 指挥协调：成立现场指挥部，统一指挥救援
6. 信息发布：按规定发布事件信息
7. 响应终止：事件得到控制后终止响应
8. 总结评估：对应急处置工作进行全面总结评估

**应急资源保障：**

**应急队伍：**
1. 综合应急救援队伍：以消防救援队伍为主
2. 专业应急救援队伍：按行业领域组建
3. 专兼职应急救援队伍：企事业单位组建
4. 志愿者应急救援队伍：社会组织和志愿者

**应急物资：**
1. 生活救助类：食品、饮用水、衣被、帐篷等
2. 医疗救护类：医疗设备、药品、器械等
3. 抢险救援类：救援装备、工具、车辆等
4. 通信保障类：通信设备、电力设备等

**应急资金：**
1. 财政预算安排应急专项资金
2. 建立应急资金快速拨付机制
3. 鼓励社会力量参与应急救援

#### 2.3.3 应急救援体系建设

**《广州市应急救援队伍建设管理办法》**

**队伍分类：**

**综合应急救援队伍：**
- 依托消防救援队伍组建
- 承担综合性应急救援任务
- 实行24小时执勤备战
- 配备专业救援装备

**专业应急救援队伍：**
1. 危险化学品应急救援队
2. 矿山应急救援队
3. 建筑工程施工应急救援队
4. 水上搜救应急救援队
5. 道路交通事故应急救援队
6. 特种设备应急救援队
7. 电力应急救援队
8. 通信应急救援队
9. 医疗卫生应急救援队
10. 环境污染应急处置队

**队伍建设标准：**

| 队伍类别 | 人数要求 | 装备配置 | 培训要求 | 演练频次 |
|---------|---------|---------|---------|---------|
| 综合救援队 | ≥50人 | 全套装备 | 年培训≥48学时 | 月演练≥1次 |
| 专业救援队 | ≥30人 | 专业装备 | 年培训≥40学时 | 季演练≥1次 |
| 企业救援队 | ≥15人 | 基础装备 | 年培训≥24学时 | 半年演练≥1次 |

**队伍保障：**
1. 经费保障：纳入财政预算，建立稳定的经费保障机制
2. 装备保障：按标准配备救援装备，定期更新维护
3. 培训保障：建立培训基地，定期组织培训演练
4. 待遇保障：按规定购买人身意外伤害保险，发放岗位津贴

**社会动员：**
1. 建立应急救援志愿者队伍
2. 开展应急救援知识技能培训
3. 组织志愿者参与应急救援演练
4. 对表现突出的志愿者给予表彰奖励

## 三、广东省安全生产标准化

### 3.1 政策概述

#### 3.1.1 政策背景

安全生产标准化建设是落实企业安全生产主体责任的重要手段，是预防事故发生的基础性工作。广东省高度重视安全生产标准化建设，将其作为提升企业本质安全水平的重要抓手。

**政策目标：**
- 到2025年，全省规模以上工矿商贸企业安全生产标准化达标率达到95%以上
- 高危行业企业安全生产标准化一级达标率达到30%以上
- 中小企业安全生产标准化达标率达到80%以上

#### 3.1.2 核心政策文件

**《广东省安全生产标准化建设管理办法》**

**主要内容包括：**
1. 标准化建设的原则和目标
2. 标准化建设的内容和要求
3. 标准化评审的程序和标准
4. 标准化证书的管理
5. 激励措施和监督管理

**《广东省安全生产标准化奖励办法》**

**奖励范围：**
1. 首次取得安全生产标准化一级证书的企业
2. 首次取得安全生产标准化二级证书的企业
3. 标准化建设成效显著的地区
4. 标准化建设先进单位和个人

**《广东省企业安全生产标准化考评实施细则》**

4. 抽查考核

**证书管理：**
- 有效期：3年
- 年度自评：每年开展自评并上报
- 期满复评：有效期满前3个月申请复评

1. 升级：符合更高等级标准可申请升级评审

1. 建立标准化企业数据库
2. 实行动态更新
3. 信息共享公开
4. 接受社会监督

**撤销条件：**
1. 发生生产安全责任事故的

3. 拒不接受监督检查的

5. 其他严重违规情形

**撤销程序：**
1. 调查核实
2. 告知当事人
3. 听取陈述申辩
4. 作出撤销决定
5. 公告撤销
6. 追回奖励资金

### 附录二：名词解释

**安全生产标准化：** 指企业通过建立健全安全生产责任制、制定安全管理制度和操作规程，排查治理隐患和监控重大危险源，建立预防机制，规范生产行为，使各生产环节符合安全生产法律法规和标准规范的要求，人员、机械、材料、工法、环境、测量处于良好的生产状态，不断加强企业安全生产规范化建设。

**突发事件：** 指突然发生，造成或者可能造成严重社会危害，需要采取应急处置措施予以应对的自然灾害、事故灾难、公共卫生事件和社会安全事件。

**应急预案：** 指各级人民政府及其部门、基层组织、企事业单位和社会组织为依法、迅速、科学、有序应对突发事件，最大程度减少突发事件及其造成的损害而预先制定的工作方案。

**应急响应：** 指突发事件发生后，有关人民政府及其部门、单位和个人采取的应急处置措施。

**重大危险源：** 指长期地或者临时地生产、搬运、使用或者储存危险物品，且危险物品的数量等于或者超过临界量的单元（包括场所和设施）。

## 结语

本指南系统梳理了广州市及粤港澳大湾区应急管理政策体系，涵盖安全生产标准化、应急产业发展、区级专项政策及跨境应急合作等内容。企业在申报过程中，应重点关注以下要点：

**核心要点：**

1. **申报条件**：仔细核对企业是否符合申报条件，包括注册年限、安全记录、营收规模等基本要求。

2. **奖励标准**：安全生产标准化一级达标最高可获省市级奖励100万元，应急产业项目扶持最高500万元。

3. **申报时间**：市级标准化奖励每年9-10月申报，应急产业资金常年受理，区级专项每年两批次。

4. **材料准备**：按照申报指南要求准备完整材料，确保真实性和规范性。

5. **流程规范**：逐级申报、配合审核、及时反馈、做好资金使用管理。

**申报建议：**

1. 提前规划：结合企业发展实际，选择适合的政策项目
2. 规范管理：持续做好安全生产标准化建设，保持达标状态
3. 主动对接：加强与主管部门沟通，及时了解政策动态
4. 资料积累：日常做好资料整理，为申报做好准备
5. 专业支持：必要时可寻求专业机构指导

希望本指南能为企业申报应急管理相关政策提供有益参考，助力企业提升安全管理水平，实现高质量发展。
