## 二、专精特新"小巨人"企业认定政策

### 2.1 政策背景与目标

#### 2.1.1 政策背景

"专精特新"是指具有"专业化、精细化、特色化、新颖化"特征的中小企业。2011年，工信部首次提出"专精特新"概念。2021年，习近平总书记强调要"培育一批'专精特新'中小企业"。2022年，工信部发布《优质中小企业梯度培育管理暂行办法》，构建了从创新型中小企业、专精特新中小企业到专精特新"小巨人"企业的梯度培育体系。

专精特新"小巨人"企业是中小企业中的佼佼者，是专注于细分市场、创新能力强、市场占有率高、掌握关键核心技术、质量效益优的排头兵企业。

#### 2.1.2 政策目标

**（一）总体目标**

到2025年，全国培育：
- **专精特新"小巨人"企业**：1万家
- **专精特新中小企业**：10万家
- **创新型中小企业**：100万家

**（二）广东省目标**

- 专精特新"小巨人"企业：1000家
- 省级专精特新中小企业：5000家
- 市级专精特新中小企业：10000家

**（三）广州市目标**

- 专精特新"小巨人"企业：200家
- 省级专精特新中小企业：1000家
- 市级专精特新中小企业：2000家

### 2.2 申报条件详解

#### 2.2.1 基本条件

**（一）企业主体条件**

| 条件类型 | 具体要求 |
|---------|---------|
| 注册登记 | 在中华人民共和国境内工商注册登记、连续经营3年以上、具有独立法人资格 |
| 企业规模 | 符合《中小企业划型标准规定》的中型企业标准 |
| 主营业务 | 专注于产业链某一环节或某一产品 |
| 市场地位 | 主导产品（服务）在国内细分行业占有率≥10% |
| 合规经营 | 近三年未发生重大安全、质量、环境污染事故，无偷漏税等违法违规行为 |

**（二）财务指标条件**

| 指标名称 | 一般企业要求 | 优先考虑 |
|---------|------------|---------|
| 上年度营业收入 | ≥1亿元 | ≥2亿元 |
| 近2年主营业务收入平均增长率 | ≥10% | ≥15% |
| 资产负债率 | ≤70% | ≤60% |
| 净利润率 | ≥5% | ≥8% |
| 主营业务收入占比 | ≥70% | ≥80% |

**（三）创新能力条件**

| 指标名称 | 基本要求 | 优选标准 |
|---------|---------|---------|
| 研发费用占营业收入比重 | ≥3% | ≥5% |
| 研发人员占比 | ≥15% | ≥20% |
| 有效发明专利 | ≥2项 | ≥5项 |
| 有效Ⅰ类知识产权 | ≥2项 | ≥5项 |
| 省级以上研发机构 | 具备1个 | 具备2个以上 |

#### 2.2.2 专项条件（六选一）

企业需同时满足基本条件和以下至少一项专项条件：

**（一）专业化程度高**

| 评估维度 | 具体要求 |
|---------|---------|
| 主营业务收入占比 | ≥80% |
| 细分市场占有率 | 全国前三位或≥10% |
| 细分市场年限 | 经营≥3年 |
| 产业链地位 | 关键环节核心供应商 |

**（二）精细化程度高**

| 评估维度 | 具体要求 |
|---------|---------|
| 质量管理体系 | 通过ISO9001等认证 |
| 精益生产水平 | 实施精益管理体系 |
| 数字化程度 | 关键工序数控化率≥50% |
| 产品质量水平 | 获得省级以上质量奖 |

**（三）特色化程度高**

| 评估维度 | 具体要求 |
|---------|---------|
| 产品独特性 | 拥有独特工艺、配方或技术 |
| 品牌影响力 | 省级以上著名商标或名牌产品 |
| 地域特色 | 地方特色产品或传统技艺传承 |
| 标准制定 | 参与国家标准或行业标准制定 |

**（四）新颖化程度高**

| 评估维度 | 具体要求 |
|---------|---------|
| 技术创新 | 拥有核心自主知识产权 |
| 产品创新 | 近三年推出创新产品≥3个 |
| 模式创新 | 创新商业模式，形成可复制经验 |
| 新兴产业 | 属于战略性新兴产业领域 |

**（五）产业链配套能力强**

| 评估维度 | 具体要求 |
|---------|---------|
| 供应链地位 | 为龙头企业核心供应商 |
| 配套产品 | 关键零部件、基础材料 |
| 替代进口 | 实现进口替代或填补国内空白 |
| 产业链安全 | 对产业链安全具有重要作用 |

**（六）主导产品竞争力强**

| 评估维度 | 具体要求 |
|---------|---------|
| 市场份额 | 细分市场占有率≥20% |
| 技术水平 | 国际领先或国内领先 |
| 客户群体 | 服务行业龙头企业≥5家 |
| 出口能力 | 主导产品出口占比≥30% |

### 3.2 申报流程

#### 3.2.1 流程图

```
企业自评 → 在线申报 → 区级审核 → 市级推荐 → 省级评审 → 公示公告 → 认定授牌
   ↓          ↓          ↓          ↓          ↓          ↓
 自查符合   递交材料   形式审查   初审推荐   专家评审   5个工作日
 条件情况   完整性     合规性     真实性     综合评分   无异议
```

#### 3.2.2 时间节点

| 环节 | 时间安排 | 注意事项 |
|------|---------|---------|
| 申报通知发布 | 每年5月 | 关注省工信厅官网 |
| 企业在线申报 | 5-6月 | 提前准备材料 |
| 区级审核推荐 | 6月 | 配合补充材料 |
| 市级初审推荐 | 6-7月 | 等待审核结果 |
| 省级专家评审 | 7-8月 | 可能需要答辩 |
| 公示公告 | 9月 | 关注公示信息 |
| 认定授牌 | 10月 | 参加授牌仪式 |

### 3.3 奖励政策

#### 4.1.1 主要政策

|---------|------|---------|

| 广州市专精特新中小企业认定管理办法 | 穗工信规字〔2022〕5号 | 规范市级认定管理 |
| 广州市中小企业发展专项资金管理办法 | 穗财规〔2023〕8号 | 资金支持方式 |

#### 4.1.2 培育目标

| 级别 | 2023年目标 | 2025年目标 | 增量 |
|------|:----------:|:----------:|:----:|
| 国家"小巨人" | 120家 | 200家 | +80家 |
| 省级专精特新 | 600家 | 1000家 | +400家 |
| 市级专精特新 | 1200家 | 2000家 | +800家 |

### 4.2 市级认定标准

#### 4.2.1 基本条件

| 条件名称 | 具体要求 |
|---------|---------|
| 注册登记 | 在广州市登记注册、连续经营2年以上 |
| 企业规模 | 符合中小企业划型标准 |
| 主营业务 | 主营业务收入占比≥60% |
| 合规经营 | 近两年无重大违法违规行为 |

#### 4.2.2 专项条件（四选一）

**专业化指标：**
- 主营业务收入占比≥70%
- 细分市场占有率≥5%
- 服务龙头企业≥3家

**精细化指标：**
- 通过ISO9001等质量管理体系认证
- 关键工序数控化率≥40%
- 获得省级以上质量奖

**特色化指标：**
- 拥有独特工艺或技术
- 获得地理标志保护或非遗认定
- 参与行业标准制定

**新颖化指标：**
- 研发费用占比≥2%
- 有效知识产权≥2项
- 近两年推出新产品≥2个

### 4.3 奖励措施

#### 4.3.1 认定奖励

| 认定级别 | 奖励金额 | 发放方式 |
|---------|---------|---------|
| 市级专精特新（首次） | 10万元 | 一次性发放 |
| 晋升省级 | 追加20万元 | 差额奖励 |
| 晋升国家"小巨人" | 追加50万元 | 差额奖励 |

#### 4.3.2 融资支持

| 支持类型 | 具体标准 |

### 7.3 常见问题解答

#### 7.3.1 申报资格问题

| 问题 | 解答 |
|------|------|
| 分公司可以申报吗？ | 不可以，必须是独立法人企业 |
| 新成立企业可以申报吗？ | 国家"小巨人"需3年以上，省级需2年以上 |

| 外资企业可以申报吗？ | 可以，但需在国内注册登记 |

#### 7.3.2 评分标准问题

| 问题 | 解答 |

| 主营业务收入如何界定？ | 占比最大的业务类别收入 |

| 奖励资金有使用限制吗？ | 一般用于研发、设备购置等 |

### 附录二：术语解释

1. **专精特新**：专业化、精细化、特色化、新颖化的简称，指具有这些特征的中小企业。

2. **小巨人**：专精特新中小企业中的佼佼者，是专注于细分市场、创新能力强、市场占有率高、掌握关键核心技术的企业。

3. **细分市场**：指在整个市场中具有特殊需求的较小市场部分，通常具有独特的客户群体或产品特性。

4. **Ⅰ类知识产权**：发明专利（含国防专利）、植物新品种、国家级农作物品种、国家新药、国家一级中药保护品种、集成电路布图设计专有权。

5. **Ⅱ类知识产权**：实用新型专利、外观设计专利、软件著作权等（不含商标）。

6. **研发费用占比**：企业研发费用总额占营业收入总额的比例。

## 结语

专精特新企业培育是国家支持中小企业发展的重要战略举措。本指南系统梳理了从国家到区级的专精特新企业政策体系，希望为企业申报认定和享受政策提供全面指导。

企业在申报过程中应注意：

1. **提前规划**：知识产权布局、研发费用归集、财务规范需要提前准备
2. **逐级申报**：按梯度培育体系，先申报市级再晋升省级和国家级
3. **材料完整**：确保申报材料齐全、真实、有效
4. **动态关注**：关注政策变化，及时调整申报策略
5. **持续创新**：认定后继续保持创新能力，提升核心竞争力
