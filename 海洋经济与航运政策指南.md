## 第一章 广州市海洋经济发展政策

### 一、政策背景与目标

广州市作为粤港澳大湾区核心引擎城市，正全力推进海洋经济高质量发展。根据《广州市海洋经济发展"十四五"规划》及相关配套政策，广州将打造成为国家海洋中心城市和全球海洋经济枢纽。

**发展目标：**
- 到2025年，全市海洋生产总值突破3000亿元
- 海洋战略性新兴产业增加值占海洋生产总值比重达40%
- 培育10家以上海洋领域上市企业
- 建成5个国家级海洋科技创新平台

### 二、支持方向与重点领域

#### （一）海洋高端装备制造业

**支持范围：**
- 海洋工程装备设计与制造
- 海洋油气开采设备
- 海上风电装备
- 海洋新能源装备
- 深海探测与作业装备

**申报条件：**
1. 在广州市注册并实际经营满2年的独立法人企业
2. 上年度海洋装备制造主营业务收入不低于2000万元
3. 拥有自主知识产权或核心技术
4. 财务管理规范，无不良信用记录
5. 项目总投资不低于1000万元

**奖励标准：**
- 新建项目：按固定资产投资额的10%给予补贴，最高不超过1000万元
- 技改项目：按设备投资额的15%给予补贴，最高不超过500万元
- 研发项目：按研发费用的30%给予补贴，最高不超过300万元
- 首台套装备：按销售价格的20%给予补贴，单台最高200万元

#### （二）海洋生物医药产业

**支持范围：**
- 海洋创新药物研发
- 海洋生物制品生产
- 海洋功能性食品开发
- 海洋化妆品原料提取

**申报条件：**
1. 具有药品生产许可证或相关资质
2. 海洋生物医药研发投入占比不低于5%
3. 拥有有效的发明专利或新药证书
4. 项目符合环保要求

**奖励标准：**
- 新药研发：临床前研究补贴100万元，I期临床200万元，II期临床500万元，III期临床1000万元
- 产业化项目：按投资额的15%给予补贴，最高800万元
- 平台建设：国家级平台补贴500万元，省级平台补贴200万元

#### （三）港口物流与航运服务

**支持范围：**
- 港口基础设施建设
- 智慧港口系统建设
- 冷链物流项目
- 多式联运中心建设
- 港航物流信息化项目

**申报条件：**
1. 具有港口经营许可证或相关资质
2. 年货物吞吐量或集装箱吞吐量达到规定规模
3. 项目符合港口总体规划
4. 投资主体信用良好

**奖励标准：**
- 基础设施项目：按投资额的8%给予补贴，最高1500万元
- 智慧港口项目：按软件及设备投资额的20%给予补贴，最高500万元
- 冷链物流项目：按冷库容积给予100元/立方米补贴，最高300万元
- 多式联运项目：每个标准箱补贴50元，年度最高200万元

#### （四）海洋新能源产业

**支持范围：**
- 海上风电项目
- 潮汐能发电项目
- 波浪能利用项目
- 海洋能制氢项目
- 海岛微电网项目

**申报条件：**
1. 项目取得相关主管部门核准或备案
2. 装机容量达到规定规模（风电≥50MW，其他≥5MW）
3. 技术方案经过专业机构评审
4. 具有并网接入条件

**奖励标准：**
- 海上风电：按装机容量给予800元/千瓦补贴，单个项目最高3亿元
- 海洋能发电：按投资额的20%给予补贴，最高5000万元
- 制氢项目：按产能给予500元/千瓦补贴，最高2000万元
- 海岛微电网：按投资额的30%给予补贴，最高1000万元

#### （五）海洋科技研发与创新

**支持范围：**
- 海洋科技重点实验室建设
- 海洋工程技术中心建设
- 海洋产业技术创新联盟
- 海洋科技成果转化项目
- 海洋科技企业孵化器

**申报条件：**
1. 申报单位为高校、科研院所或高新技术企业
2. 研发团队具有高级职称人员不少于5人
3. 近三年承担过省部级以上科研项目
4. 实验场地面积不少于1000平方米

**奖励标准：**
- 国家级重点实验室：建设补贴2000万元，运行经费每年300万元
- 省级重点实验室：建设补贴500万元，运行经费每年100万元
- 工程技术中心：建设补贴300万元
- 孵化器：按在孵企业数量给予补贴，每家企业5万元，最高500万元

### 三、申报流程

**第一步：项目准备（1-2月）**
- 确定项目类别和支持方向

- 填写项目申报表
- 上传相关附件材料

**第三步：材料审核（4-5月）**
- 区级初审
- 市级复审
- 第三方机构评估
- 现场核查

**第四步：项目评审（5-6月）**
- 专家评审会
- 部门联审
- 评审结果公示

**第五步：资金拨付（7-8月）**
- 下达资金计划
- 签订项目协议
- 资金分批拨付

## 第二章 广东省航运产业发展政策

### 一、政策背景与目标

广东省作为全国航运大省，拥有得天独厚的海岸线和港口资源。根据《广东省航运产业发展规划（2021-2030年）》和《广东省关于促进航运业高质量发展的若干措施》，省财政每年安排专项资金支持航运产业发展。

**发展目标：**
- 到2025年，全省港口货物吞吐量达22亿吨
- 集装箱吞吐量达8000万标箱
- 远洋运输船队规模达2000万载重吨
- 航运服务业增加值达500亿元
- 培育5家以上具有国际竞争力的航运企业

### 二、支持方向与重点领域

#### （一）航运运力发展

**支持范围：**
- 新建远洋运输船舶
- 购置二手船舶（船龄<10年）
- 船舶融资租赁
- 船舶营运补贴

**申报条件：**
1. 在广东省注册的航运企业
2. 持有有效的国际航运经营资格
3. 船舶在广东省登记注册
4. 运力规模达到规定要求（总运力≥5万载重吨）
5. 船舶平均船龄不超过15年

**奖励标准：**
- 新建船舶：按载重吨给予补贴，散货船100元/载重吨，集装箱船150元/载重吨，油轮120元/载重吨，单船最高2000万元
- 购置船舶：按购置价格的3%给予补贴，单船最高1000万元
- 融资租赁：按租赁金额的2%给予贴息，年度最高500万元
- 营运补贴：根据航线类型，中远洋航线每载重吨年度补贴20元，近洋航线15元，港澳航线10元

#### （二）航运服务业发展

**支持范围：**
- 船舶代理业务
- 货运代理业务
- 航运保险服务
- 航运法律服务
- 航运经纪服务
- 航运金融服务

**申报条件：**
1. 在广东省注册满3年的独立法人
2. 具有相关行业资质证书
3. 上年度主营业务收入不低于500万元
4. 从业人员不少于20人
5. 服务客户不少于50家

**奖励标准：**
- 新设立的国际航运服务机构：一次性补贴200万元
- 业务规模奖励：按业务量增长给予奖励，最高300万元
- 国际化发展奖励：在境外设立分支机构补贴100万元
- 专业人才奖励：高级人才每人补贴20万元，年度最高200万元

#### （三）智慧航运建设

**支持范围：**
- 船岸一体化信息系统
- 智能船舶系统
- 港航物流信息平台
- 电子航运交易平台
- 航运大数据中心

**申报条件：**
1. 项目总投资不低于500万元
2. 采用自主可控技术
3. 具有创新性和示范性
4. 取得软件著作权或专利

**奖励标准：**
- 平台建设项目：按投资额的25%给予补贴，最高1000万元
- 系统集成项目：按投资额的20%给予补贴，最高500万元
- 数据服务项目：按服务收入的10%给予补贴，年度最高200万元

#### （四）绿色航运发展

**支持范围：**
- 船舶节能改造
- LNG动力船舶
- 岸电设施建设
- 船舶压载水处理系统
- 船舶尾气处理装置

**申报条件：**
1. 船舶在广东省登记注册
2. 改造或设备符合国家技术标准
3. 经权威机构检测合格
4. 正常运营满1年以上

**奖励标准：**
- 节能改造：按改造费用的30%给予补贴，单船最高200万元
- LNG动力船舶：按建造成本的20%给予补贴，单船最高1500万元
- 岸电设施：按建设费用的40%给予补贴，单项目最高500万元
- 压载水处理系统：按设备购置费用的50%给予补贴，单船最高100万元
- 尾气处理装置：按设备购置费用的40%给予补贴，单船最高80万元

#### （五）航运人才培养

**支持范围：**
- 航运人才培训基地建设
- 高级船员培训
- 航运管理人才培养
- 航运专业学历教育
- 国际航运人才引进

**申报条件：**
1. 培训机构具有海事部门认可资质
2. 培训项目符合行业需求
3. 培训质量达标
4. 就业率不低于80%

**奖励标准：**
- 培训基地建设：补贴300万元
- 高级船员培训：每人补贴1万元
- 管理人才培训：每人补贴2万元
- 国际人才引进：每人补贴30万元

**第一步：计划发布（每年1月）**

## 第三章 广州市海事服务政策

### 一、政策背景与目标

广州作为中国南方重要的海事服务中心，拥有完备的海事服务体系和丰富的海事服务资源。根据《广州市海事服务产业发展规划》和《广州市关于加快海事服务业发展的实施意见》，市财政每年安排专项资金支持海事服务产业发展。

**发展目标：**
- 到2025年，海事服务业总产值达200亿元
- 培育10家以上具有国际竞争力的海事服务机构
- 引进50家以上海事服务企业
- 建成3个国家级海事服务示范平台
- 海事服务人才队伍达10000人

### 二、支持方向与重点领域

#### （一）船舶检验与认证服务

**支持范围：**
- 船舶入级检验
- 法定检验
- 公证检验
- 船用产品检验
- 海上设施检验

**申报条件：**
1. 具有海事部门认可的检验资质
2. 在广州市设有固定办公场所
3. 持证验船师不少于10人
4. 年检验船舶不少于200艘次
5. 检验质量优良，无重大责任事故

**奖励标准：**
- 新落户机构：一次性奖励100万元
- 资质升级：从乙级升甲级奖励50万元
- 业务量奖励：按检验艘次，每艘次补贴200元，年度最高100万元
- 设备购置：按购置费用的30%给予补贴，最高200万元

#### （二）船舶代理服务

**支持范围：**
- 国际航行船舶代理
- 国内航行船舶代理
- 港口船舶代理
- 船员服务
- 供应服务

**申报条件：**
1. 取得国际船舶代理经营资格
2. 在广州市实际经营满3年
3. 年代理船舶不少于300艘次
4. 注册资金不低于500万元
5. 专业人员不少于15人

**奖励标准：**
- 业务量奖励：按代理艘次，国际航线每艘次补贴500元，国内航线每艘次补贴200元，年度最高150万元
- 优质服务奖：评选优秀代理企业，奖励30万元
- 国际化发展：在境外设立机构奖励80万元

#### （三）海商法律服务

**支持范围：**
- 海事海商诉讼代理
- 海事仲裁代理
- 海上保险理赔
- 海事调解服务

## 附录四：常见问题解答

### Q1：同一项目可以同时申请多个政策吗？

**A：** 一般情况下，同一项目不能重复享受同级财政资金支持。但可以同时申请不同层级的政策，如区级、市级、省级、国家级政策可叠加。建议在申报前咨询相关部门确认。

### Q2：企业注册时间不足可以申请吗？

**A：** 不同政策对企业注册年限要求不同：
- 广州市海洋经济政策要求注册满2年
- 广东省航运政策要求注册满3年
- 番禺区海洋专项要求注册满1年
- 部分人才引进、创新项目可放宽要求

### Q3：申报材料需要准备多长时间？

**A：** 建议提前2-3个月准备：
- 项目可行性研究报告：1个月
- 财务审计报告：2周
- 相关资质证明：1-2周
- 网上申报填写：1周

### Q4：资金拨付需要多长时间？

**A：** 从申报到资金拨付一般需要4-6个月：
- 材料审核：1个月
- 项目评审：1-2个月
- 公示期：7-15天
- 资金拨付：1-2个月

### Q5：未通过评审可以再次申请吗？

**A：** 可以。未通过评审的项目可根据评审意见进行修改完善，在下一年度重新申报。部分政策设有补充申报批次，可关注补充申报通知。

### Q6：项目实施过程中可以变更吗？

**A：** 项目实施过程中如需重大变更，须提前向主管部门提交变更申请，经批准后方可实施。擅自变更可能导致资金追回或取消资助资格。

### Q7：如何查询申报进度？

**A：** 可通过以下方式查询：
- 登录各地政务服务网在线查询
- 致电相关部门咨询
- 现场咨询窗口查询

## 附录八：申报注意事项

### 申报前准备

1. **仔细研读申报指南**：每年度政策可能有调整，务必以最新指南为准
2. **确认申报资格**：对照申报条件逐一核实，避免无效申报
3. **提前准备材料**：财务审计、环评等材料需要时间，提前准备
4. **确定申报类别**：根据项目特点选择最匹配的支持方向
5. **计算资助金额**：根据奖励标准预估，合理安排项目预算

### 材料准备要点

1. **真实性**：所有材料必须真实有效，不得弄虚作假
2. **完整性**：按要求提供全部材料，缺项可能导致申报失败
3. **规范性**：格式规范、签章齐全、装订有序
4. **时效性**：注意材料的有效期，过期材料无效
5. **一致性**：不同材料中的数据信息应保持一致

### 申报过程中

1. **关注系统状态**：及时查看审核进度和反馈意见
2. **保持通讯畅通**：确保联系电话畅通，及时回应补充材料要求
3. **配合现场核查**：如实介绍项目情况，提供必要配合
4. **留存申报记录**：保存申报材料和凭证，以备后续核查

### 资金使用管理

1. **专款专用**：资助资金必须用于项目规定的用途
2. **单独核算**：建立专项资金台账，独立核算
3. **定期报告**：按要求报送项目进展和资金使用情况
4. **接受监督**：配合审计和绩效评价工作
5. **完成验收**：项目完成后及时申请验收
