## 🎯 平台选择决策树

```
你是哪种卖家？

1. 有品牌、有产品力
   ├── 主攻：Amazon + 独立站 + TikTok Shop
   ├── 辅助：Shopee/Lazada（东南亚）
   └── 策略：品牌溢价+多渠道

2. 工厂/供应链型
   ├── 主攻：Temu + Amazon + Shein（供应商）
   ├── 辅助：1688跨境专供
   └── 策略：薄利多销+规模化

3. 个人/小团队起步
   ├── 主攻：TikTok Shop + Shopee
   ├── 辅助：eBay（特定品类）
   └── 策略：低成本试错+快速迭代

4. 有设计能力/快时尚
   ├── 主攻：独立站 + TikTok Shop
   ├── 辅助：Shein（供应商合作）
   └── 策略：品牌打造+社交电商

5. 特定品类专家
   ├── 汽配：eBay + Amazon
   ├── 时尚：TikTok Shop + 独立站
   ├── 3C：Amazon + 独立站
   └── 根据品类选择优势平台
```
