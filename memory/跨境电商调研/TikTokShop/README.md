## 二、2026年政策重大变化

### 1. 美国履约新规（2026年1月起）

```
发货时效变化：
├── 原要求：3个工作日
├── 新要求：2个工作日
└── 影响：库存管理要求更高

运单号要求：
├── 必须真实有效
├── 必须可追踪
├── 平台审核收紧
└── 影响：打击虚假发货

违规处罚：
├── 订单取消率>2.5%：警告
├── 订单取消率>5%：限制
├── 延迟发货率>4%：警告
└── 严重违规：封店
```

### 2. 美国取消自发货（2026年2-3月）

```
政策时间线：
├── 2026年2月9日：宣布政策
├── 2026年2月25日：本土卖家开始切换
├── 2026年3月31日：全面使用FBT

FBT（Fulfilled by TikTok）要求：
├── 所有美国订单必须使用FBT
├── 货物入TikTok指定仓库
├── 平台统一配送
├── 统一售后处理
└── 费用结构：类似FBA

影响分析：
├── 好处：时效快、体验好、减少售后
├── 挑战：库存管理、资金占用、灵活性降低
└── 应对：精准预测库存、多渠道布局
```

## 三、店铺类型与入驻

### 店铺类型

| 类型 | 要求 | 特点 | 适合 |
|------|------|------|------|
| **本土店** | 当地公司/身份 | 流量支持、政策宽松 | 有海外主体 |
| **跨境店** | 中国公司 | 入驻门槛低 | 大多数卖家 |
| **品牌店** | 品牌+授权 | 品牌标识、流量支持 | 品牌卖家 |
| **达人店** | 达人身份 | 内容带货 | 内容创作者 |

### 入驻要求（美国跨境店）

```
必备材料：
├── 中国大陆或香港营业执照
├── 法人身份证
├── 手机号/邮箱
├── 店铺Logo
├── 品牌授权书（如有）
└── 仓库地址（FBT入仓）

审核周期：
├── 提交资料：1-3天
├── 资质审核：3-7天
├── 缴纳保证金：1-2天
└── 总计：5-14天

常见拒因：
├── 营业执照问题
├── 资料不完整
├── 侵权风险
└── 关联账号问题
```

```
├── 专属链接/折扣码
├── 销售归因

```

```
   ├── 直播引流
   ├── 实时转化
   └── 新兴形式
```

## 七、物流与履约

### 履约方式（美国站）

```
2026年变化：

FBT（Fulfilled by TikTok）- 必选
├── 入TikTok仓库
├── 平台统一配送
├── 2-5天送达
├── 费用：$3-8/件
└── 优点：体验好、合规

FBA（跨境卖家）
├── 部分卖家可用
├── 需提前申请
└── 作为过渡方案

成本结构：
├── FBT配送费：$3-8
├── 仓储费：$0.5-2/件/月
├── 头程：视物流方式
└── 总履约成本：15-25%售价
```

### 库存管理

```
库存策略：

安全库存：
├── 热销品：30-45天销量
├── 常规品：15-30天销量
└── 新品：测试量

补货周期：
├── 头程时间：7-15天
├── 入仓时间：3-5天
├── 总周期：10-20天
└── 提前规划

库存预警：
├── 可售天数<15天：紧急补货
├── 可售天数<7天：可能断货
└── 可售天数>60天：可能积压
```
