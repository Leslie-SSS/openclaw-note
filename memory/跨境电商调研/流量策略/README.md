### 关键词策略

```
关键词分类：

1. 核心词（Head Terms）
   ├── 搜索量大
   ├── 竞争激烈
   ├── 例："running shoes"
   └── 长期目标

2. 长尾词（Long-tail）
   ├── 搜索量小
   ├── 竞争较低
   ├── 转化率高
   ├── 例："best running shoes for flat feet"
   └── 优先优化

3. 商业词（Commercial）
   ├── 购买意图
   ├── 例："buy running shoes online"
   └── 高转化

4. 信息词（Informational）
   ├── 学习意图

关键词工具：
├── Ahrefs（推荐）
├── SEMrush
├── Google Keyword Planner
├── Ubersuggest
└── AnswerThePublic
```

## 三、Meta广告（Facebook/Instagram）

### 广告账户结构

```
账户层级：

Campaign（广告系列）
├── 目标：转化/流量/品牌
├── 预算：CBO
└── 优化目标

Ad Set（广告组）
├── 受众定位
├── 版位
├── 预算（ABO）
└── 出价策略

Ad（广告）
├── 素材
├── 文案
├── CTA
└── 落地页
```

### 受众策略

```
受众类型：

1. 广泛受众（Broad）
   ├── 仅基础定位
   ├── 算法优化
   ├── 适合：大预算、学习期
   └── 例：25-45岁、美国、兴趣广泛

2. 兴趣受众（Interest）
   ├── 兴趣标签
   ├── 行为定位
   ├── 适合：冷启动
   └── 例：跑步爱好者、健身

3. 类似受众（Lookalike）
   ├── 基于种子用户
   ├── 1-10%比例
   ├── 适合：有数据基础
   └── 例：购买者1% LAL

4. 再营销（Retargeting）
   ├── 网站访客
   ├── 视频观看
   ├── 互动用户
   └── 例：加购未购买

受众漏斗：
广泛 → 兴趣 → LAL → 再营销
```

### 素材策略

```
素材类型：

1. 单图广告
   ├── 简单直接
   ├── 制作成本低
   └── 适合：产品展示

2. 轮播广告
   ├── 多产品展示
   ├── 讲故事
   └── 适合：多SKU

3. 视频广告
   ├── 动态展示
   ├── 转化率高
   └── 适合：功能展示

4. Collection广告
   ├── 沉浸式体验
   ├── 移动端
   └── 适合：电商

素材优化：
├── 前3秒Hook
├── 产品卖点清晰
├── 社会证明
├── 明确CTA
├── 多版本测试
└── 7-14天迭代
```

### 成本与优化

```
成本参考（美国市场）：

CPM：
├── Facebook：$10-25
├── Instagram：$12-30
└── Audience Network：$5-15

CPC：
├── 平均：$0.50-1.50
├── 优秀：$0.30-0.80
└── 较高：$1.50+

CAC（获客成本）：
├── 低客单价（$30-50）：$15-30
├── 中客单价（$50-100）：$25-50
└── 高客单价（$100+）：$40-80

ROAS目标：
├── 盈利线：2.5-3x
├── 健康线：3-4x
└── 优秀线：4x+

优化策略：
├── 关停低效广告组
├── 扩展表现好的
├── 迭代素材
├── 调整受众
└── 优化落地页
```

## 四、Google广告

### 广告类型

```
Google广告类型：

1. Google Shopping（50-60%预算）
   ├── 产品列表广告
   ├── 转化率最高
   ├── 需要Merchant Center
   └── 适合：电商

2. 搜索广告（20-30%预算）
   ├── 关键词触发
   ├── 意图明确
   ├── 文字广告
   └── 适合：精准词

3. 展示广告（10-15%预算）
   ├── 再营销为主
   ├── 品牌曝光
   ├── 图片/响应式
   └── 适合：品牌+再营销

4. YouTube广告（10-15%预算）
   ├── 视频广告
   ├── 品牌建设
   ├── 跳过/不可跳过
   └── 适合：视频素材

5. Performance Max（10-20%）
   ├── 全渠道自动
   ├── AI优化
   ├── 新兴形式
   └── 适合：有一定数据
```

### Shopping广告优化

```
Shopping广告优化：

1. Feed优化
   ├── 标题：关键词+卖点
   ├── 描述：详细
   ├── 图片：高质量
   ├── 价格：有竞争力
   └── 属性：完整

2. 出价策略
   ├── 新品：Target ROAS
   ├── 成熟：Maximize Conversion Value
   └── 预算控制

3. 分割策略
   ├── 按品类
   ├── 按利润率
   ├── 按表现
   └── 按价格段

4. 负面关键词
   ├── 排除低质流量
   ├── 定期更新
   └── 成本控制

成本参考：
├── Shopping CPC：$0.50-2.00
├── 搜索CPC：$1.00-5.00
├── CAC：$25-80
└── ROAS：3-5x
```

### 搜索广告策略

```
搜索广告结构：

Campaign
├── 按品类/目标分
└── 预算控制

Ad Group
├── 主题相关关键词
├── 10-20个关键词
└── 相关广告

关键词匹配：
├── 精确匹配（Exact）：[keyword]
├── 词组匹配（Phrase）："keyword"
├── 广泛匹配修饰（Broad Mod）：+keyword
└── 广泛匹配（Broad）：keyword

建议：
├── 品牌词：精确匹配
├── 产品词：词组匹配
├── 长尾词：广泛修饰
└── 避免纯广泛
```

```
   ├── 互动率高
```
