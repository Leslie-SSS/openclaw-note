## 一、PPC广告策略

### 1.1 广告类型对比

| 广告类型 | 适用场景 | 平均CPC | 平均CTR | 平均ACOS | 推荐度 |
|---------|---------|---------|---------|----------|--------|
| 商品推广 (SP) | 新品推广、日常销售 | $0.85-1.35 | 0.35-0.45% | 25-35% | ⭐⭐⭐⭐⭐ |
| 品牌推广 (SB) | 品牌建设、防守竞品 | $1.20-1.85 | 0.25-0.38% | 18-28% | ⭐⭐⭐⭐ |
| 展示型推广 (SD) | 再营销、受众拓展 | $0.65-1.10 | 0.15-0.25% | 15-25% | ⭐⭐⭐⭐ |
| DSP广告 | 高级受众、程序化 | $0.80-1.50 | 0.20-0.35% | 20-30% | ⭐⭐⭐ |

### 1.2 商品推广 (Sponsored Products) 策略

**广告活动结构:**

```
新品推广活动 (自动)
├── 紧密匹配 (Close Match) - 竞价 $1.20
├── 宽松匹配 (Loose Match) - 竞价 $0.65
├── 同类商品 (Substitutes) - 竞价 $0.95
└── 关联商品 (Complements) - 竞价 $0.55

关键词精准活动 (手动)
├── 核心词组 (5-10词) - 竞价 $1.35-1.65
├── 长尾词组 (15-30词) - 竞价 $0.75-1.10
└── 竞品词组 (10-20词) - 竞价 $0.85-1.20

商品定位活动
├── 竞品ASIN (20-50个) - 竞价 $0.65-0.95
└── 互补ASIN (10-30个) - 竞价 $0.45-0.75
```

**新品冷启动策略:**

**第1-2周: 数据收集期**
- 预算: $30-50/天
- 策略: 自动广告为主
- 目标: 收集搜索词数据
- 优化: 否定无关词

**第3-4周: 优化调整期**
- 预算: $50-80/天
- 策略: 手动广告加入
- 目标: 降低ACOS
- 优化: 调整竞价,移除低效词

**第5-8周: 规模扩展期**
- 预算: $80-150/天
- 策略: 多活动并行
- 目标: 提升销量排名
- 优化: 扩展关键词,增加预算

### 1.3 关键词策略

**关键词分类:**

1. **核心关键词 (Head Terms)**
   - 特点: 流量大、竞争激烈
   - 例子: "blender", "yoga mat"
   - 策略: 适度投入,注意转化率
   - 建议: 竞价$1.00-1.50

2. **长尾关键词 (Long-tail)**
   - 特点: 流量小、转化高
   - 例子: "portable blender for smoothies"
   - 策略: 重点投入,ROI最高
   - 建议: 竞价$0.65-1.10

3. **竞品关键词 (Competitor)**
   - 特点: 精准定位竞品流量
   - 例子: "ninja blender alternative"
   - 策略: 选择性投放
   - 建议: 竞价$0.80-1.25

4. **场景关键词 (Scenario)**
   - 特点: 需求明确、转化好
   - 例子: "blender for making baby food"
   - 策略: 深挖场景
   - 建议: 竞价$0.55-0.95

**关键词挖掘方法:**

1. **Amazon搜索建议**
   - 输入核心词看下拉建议
   - 记录所有相关词
   - 整理成关键词表

2. **竞品分析**
   - 查看竞品Listing标题
   - 分析竞品广告关键词
   - 反向ASIN查询

3. **工具挖掘**
   - Helium 10 Cerebro
   - Jungle Scout Keyword Scout
   - Merchant Words
   - Google Keyword Planner

**否定关键词策略:**

**必须否定的词:**
- 品牌词 (除非卖该品牌)
- "free", "cheap", "used"
- 不相关品类词
- 低转化高消费词

**否定匹配类型:**
- 词组否定 (Phrase)
- 精准否定 (Exact)

### 1.4 竞价优化策略

**动态竞价策略:**

| 策略 | 适用场景 | 说明 |
|------|---------|------|
| 动态-仅降低 | 所有情况 | 不提高竞价,仅降低 |
| 动态-提高或降低 | 强推新品 | 可能提高50-100%竞价 |
| 固定竞价 | 精准控制 | 保持设定竞价 |

**竞价调整频率:**

```
每日检查:
- 消耗异常词 (超预算150%)
- 零展现词 (暂停或提价)

每周优化:
- 低ACOS词 (提价10-20%)
- 高ACOS词 (降价10-20%)
- 零转化词 (暂停)

每月复盘:
- 整体ACOS分析
- 关键词词库更新
- 预算重新分配
```

**ACOS目标设定:**

| 产品阶段 | 目标ACOS | 说明 |
|---------|---------|------|
| 新品期 (0-2个月) | 40-60% | 以出单为主 |
| 成长期 (2-6个月) | 25-35% | 平衡增长和利润 |
| 成熟期 (6个月+) | 15-25% | 优化利润 |
| 清仓期 | <15% | 清理库存 |

## 二、自然排名优化 (SEO)

### 2.1 Listing优化

**标题优化公式:**

```
品牌名 + 核心关键词 + 主要特性 + 适用场景 + 颜色/尺寸/数量

例子:
"XYZPro Portable Blender for Smoothies - 16oz USB Rechargeable 
Personal Blender, 6 Blades, BPA Free, for Office, Gym, Travel - Blue"
```

**标题优化要点:**

1. **关键词权重 (从高到低)**
   - 标题 (Title)
   - 五点描述 (Bullet Points)
   - 产品描述 (Description)
   - 搜索词 (Search Terms)
   - A+内容

2. **字符限制**
   - 200字符以内 (大部分类目)
   - 关键词前80字符最重要
   - 移动端只显示前60字符

3. **避免问题**
   - 重复关键词
   - 全大写/全小写
   - 促销用语
   - 特殊符号

**五点描述 (Bullet Points) 优化:**

**结构模板:**

```
• 【核心卖点1】- 详细说明 + 数据支撑
• 【核心卖点2】- 解决痛点 + 使用场景
• 【核心卖点3】- 材质/品质 + 认证信息
• 【核心卖点4】- 规格/参数 + 适用对象
• 【核心卖点5】- 包装内容 + 售后保障
```

**优化技巧:**
- 每点150-200字符
- 开头大写吸引注意
- 埋入长尾关键词
- 强调差异化优势

**产品描述优化:**

**A+内容 (品牌备案卖家):**

1. **标准A+ (免费)**
   - 模块化设计
   - 图文结合
   - 转化率提升5-10%

2. **高级A+ (Premium)**
   - 互动模块
   - 视频展示
   - 转化率提升10-15%

**描述内容结构:**

```
1. 品牌故事/产品理念
2. 核心功能详解
3. 使用场景描述
4. 规格参数表
5. 包装清单
6. 常见问题解答
7. 联系方式
```

### 2.2 图片优化

**主图要求:**

1. **技术要求**
   - 纯白背景 (RGB 255,255,255)
   - 最小1000x1000像素
   - 占图片面积85%以上
   - 无水印、文字、边框

2. **最佳实践**
   - 清晰展示产品全貌
   - 角度标准化
   - 光线充足
   - 无反光/阴影

**辅图策略:**

**必备辅图 (6-8张):**

1. **功能展示图** - 展示核心功能
2. **使用场景图** - 实际使用场景
3. **细节特写图** - 材质/工艺细节
4. **尺寸对比图** - 与常见物品对比
5. **功能图解** - 图解功能原理
6. **包装内容图** - 展示包装内物品
7. **生活方式图** - 使用者+产品
8. **信息图** - 规格参数/对比

**图片优化技巧:**

- 统一风格和色调
- 高质量3D渲染
- 真实场景拍摄
- 符合品牌调性

### 2.3 搜索词优化

**后台搜索词 (Search Terms):**

1. **长度限制**
   - 最多250字节 (249字符)
   - 不要重复标题词
   - 用空格分隔

2. **优化技巧**
   - 同义词/拼写变体
   - 单复数形式
   - 缩写/全称
   - 相关场景词

3. **避免问题**
   - 重复词汇
   - 品牌名 (除非授权)
   - ASIN号码
   - 促销用语

### 2.4 Review策略

**合规获取Review:**

1. **Amazon Vine计划**
   - 费用: $200/ASIN
   - 获取30个专业评价
   - 适合新品

2. **Request a Review按钮**
   - 卖家中心内置功能
   - 订单完成后5-30天可请求
   - 每日限额

3. **产品插页**
   - 合规的售后服务卡
   - 不诱导评价
   - 提供客服联系方式

**Review优化策略:**

1. **提升产品质量**
   - 减少差评根源
   - 产品迭代改进

2. **客服跟进**
   - 主动解决问题
   - 降低负面评价

3. **评价分析**
   - 收集用户反馈
   - 优化产品/Listing

## 四、流量转化优化

### 4.1 Listing转化率优化

**影响转化率的因素:**

1. **价格竞争力** (权重30%)
   - 对比竞品定价
   - 促销策略
   - 价值感知

2. **图片质量** (权重25%)
   - 主图吸引力
   - 辅图完整性
   - 视频展示

3. **评价数量/评分** (权重20%)
   - 至少15个评价
   - 4.3星以上
   - 有图文评价

4. **标题/描述** (权重15%)
   - 关键词相关性
   - 卖点清晰
   - 信息完整

5. **A+内容** (权重10%)
   - 品牌故事
   - 视觉呈现
   - 差异化展示

### 4.2 定价策略

**定价模型:**

1. **成本加成法**
   ```
   价格 = 成本 / (1 - 目标利润率)
   例: $20成本，目标30%利润
   价格 = 20 / (1 - 0.30) = $28.57
   ```

2. **竞争导向法**
   - 对标前5名竞品
   - 略低或持平
   - 突出差异化

3. **价值定价法**
   - 基于感知价值
   - 高品质高价格
   - 品牌溢价

**动态定价:**

```
新品期: 略低于市场价 (-5-10%)
成长期: 持平或略高 (+0-5%)
成熟期: 根据竞争调整
清仓期: 降价清库存 (-20-30%)
```

### 4.3 促销策略

**促销类型:**

1. **Coupon (优惠券)**
   - 5-20%折扣
   - 费用: $0.60/次
   - 提升点击率

2. **Prime专享折扣**
   - 至少10% OFF
   - 会员专属
   - 免费使用

3. **秒杀 (Lightning Deal)**
   - 申请制
   - 费用$150-500
   - 大流量曝光

4. **7天促销**
   - 申请制
   - 费用$300-1000
   - 持续曝光

**促销节奏:**

- 新品: Coupon 10-15%
- 日常: Prime专享 5-10%
- 节日: 秒杀 20-30%
- 清仓: 大折扣 30-50%
