## 二、费用结构

### 主要费用

```
eBay费用构成：

1. 刊登费（Insertion Fee）
   ├── 免费刊登：50-250件/月（按店铺等级）
   ├── 超出后：$0.35/件
   └── 拍卖：每次刊登

2. 成交费（Final Value Fee）
   ├── 按类目不同
   ├── 一般品类：10-12%
   ├── 汽配：12.55%
   ├── 电子产品：10%
   ├── 服装：12.35%
   └── 计算基础：售价+运费

3. 支付费（Payment Processing）
   ├── PayPal：2.9% + $0.30
   ├── eBay Managed Payments：2.9% + $0.30
   └── 已整合到成交费

4. 店铺订阅费（Store Subscription）
   ├── Starter：$4.95/月
   ├── Basic：$21.95/月
   ├── Premium：$59.95/月
   ├── Anchor：$299.95/月
   └── 年付优惠

综合费率：15-25%
```

### 店铺等级对比

| 等级 | 月费 | 免费刊登 | 成交费优惠 | 适合 |
|------|------|----------|------------|------|
| 无店铺 | $0 | 50件/月 | 无 | 偶尔卖 |
| Starter | $4.95 | 100件/月 | 无 | 小规模 |
| Basic | $21.95 | 250件/月 | 无 | 正常运营 |
| Premium | $59.95 | 1000件/月 | 无 | 规模化 |
| Anchor | $299.95 | 10000件/月 | 有 | 大卖家 |
