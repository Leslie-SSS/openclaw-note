# 项目申报指南 - 2026年

**企业基本信息**
- 注册地点：广州市番禺区
- 注册时间：2026年2月
- 主营业务：财务软件开发
- 当前经营时长：约1个月（截至2026年3月）

## 二、需提前准备的政策（需2年以上经营）

### 1. 广州市跨境电商"麒麟"企业及"品牌出海"企业

| 项目 | 内容 |
|------|------|
| **政策名称** | 跨境电商"麒麟"企业及"品牌出海"企业申报 |
| **发布单位** | 广州市商务局 |
| **最高支持** | 最高3500万元 |
| **申报条件** | 1. 依法登记注册并在广州市内实际从事跨境电商经营活动**2年以上**；2. 有税务登记的独立法人；3. 未被列入严重失信主体名单；4. 品牌2023年海外销售额或出口额在500万元以上或境外粉丝量在100万以上 |
| **适用性** | 财务软件如开展跨境业务，可从2028年开始准备申报 |
| **准备要点** | 提前规划跨境电商业务，积累海外销售数据 |

### 2. 广东省跨境电商独立站企业扶持

| 项目 | 内容 |
|------|------|
| **政策名称** | 广东省跨境电商独立站企业扶持政策 |
| **支持内容** | 年度交易额20亿元以上独立站企业在用地、高管落户、子女入学等方面给予支持 |
| **申报条件** | 年度交易额达到20亿元及以上 |
| **适用性** | 规模要求较高，需长期发展 |

## 三、税收优惠政策

### 1. 软件企业税收优惠

| 项目 | 内容 |
|------|------|
| **政策名称** | 国家软件企业税收优惠政策 |
| **优惠内容** | 1. 增值税即征即退（软件产品销售）；2. 企业所得税优惠（两免三减半）；3. 研发费用加计扣除 |
| **申报条件** | 符合软件企业认定条件 |
| **申报时间** | 随税务申报办理 |
| **所需材料** | 软件企业认定证书、研发费用明细等 |
| **优先级** | ⭐⭐⭐⭐⭐ 高度相关，优先申请 |

### 2. 高新技术企业认定

| 项目 | 内容 |
|------|------|
| **政策名称** | 高新技术企业认定 |
| **优惠内容** | 1. 企业所得税减按15%征收；2. 品牌提升；3. 政府项目申报优势 |
| **申报条件** | 1. 注册满一年以上；2. 拥有核心自主知识产权；3. 研发费用占比达标；4. 高新技术产品收入占比60%以上 |
| **申报时间** | 通常每年分批次，3-4月、7-8月 |
| **适用性** | 2027年可开始准备，需积累知识产权和研发费用 |
| **准备要点** | 申请软件著作权、专利，规范研发费用核算 |

### 3. 小微企业税收优惠

| 项目 | 内容 |
|------|------|
| **政策名称** | 小微企业所得税优惠 |
| **优惠内容** | 年应纳税所得额300万元以下按优惠税率征收 |
| **适用性** | 新注册企业通常符合小微条件，自动享受 |

## 七、2026年行动计划

### 立即可做：
1. ✅ **软件企业认定** - 申请软件企业资质，享受税收优惠
2. ✅ **软件著作权登记** - 为财务软件申请软著，积累知识产权
3. ✅ **创业带动就业补贴** - 招聘员工后申请
4. ✅ **小微企业税收优惠** - 自动享受

### 1年内可做：
1. 🔄 **高新技术培育入库** - 进入高新技术企业培育库
2. 🔄 **研发费用加计扣除** - 规范研发费用核算
3. 🔄 **科技创新券** - 购买检测、认证等服务

### 2年内规划：
1. 📋 **高新技术企业认定** - 2027年开始准备
2. 📋 **服务贸易专项资金** - 积累服务出口数据

### 3年以上规划：
1. 📅 **跨境电商"麒麟"企业** - 2028年后可申报
2. 📅 **专精特新认定** - 2028年后可申报

## 八、重要提示

1. **经营年限限制**：大部分补贴政策要求企业成立2年以上，需提前规划
2. **财务规范**：保持财务核算规范，特别是研发费用独立核算
3. **知识产权**：及时申请软件著作权、专利，为后续申报打基础
4. **政策跟踪**：定期关注番禺区政府、广州市商务局、科技局网站
5. **信息渠道**：
   - 番禺区政府网：www.panyu.gov.cn
   - 广州市商务局：sw.gz.gov.cn
   - 穗@i企平台：qyfw.gzonline.gov.cn
   - 广东省跨境电商平台：gdcbec.singlewindow.gd.cn
