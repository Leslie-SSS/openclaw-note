### 4.3 ROAS计算误区与正确方法

#### ⚠️ 常见误区

**误区1：只看广告平台ROAS**
```
错误：FB报告ROAS = 广告收入 / 广告支出
正确：真实ROAS = （毛利 - 退款 - 运费） / 广告支出
```

**误区2：忽略LTV**
```
首购ROAS = 2.0x（看似亏损）
但 LTV = $150，CAC = $30
→ LTV/CAC = 5x（健康模型）
```

**误区3：未考虑归因周期**
| 品类 | 平均决策周期 | 归因设置建议 |
|------|--------------|--------------|
| 快消品 | 1-3天 | 7天点击 |
| 电子产品 | 7-14天 | 14-30天点击 |
| 高端商品 | 14-30天 | 30天点击+浏览 |

### 4.4 盈亏平衡ROAS计算

#### 🧮 计算公式

```
盈亏平衡ROAS = 1 / 毛利率

示例：
├── 产品售价：$100
├── 产品成本：$40
├── 运费：$10
├── 退货率：5%
└── 毛利润：$45
    → 毛利率 = 45%
    → 盈亏平衡ROAS = 1 / 0.45 = 2.22x

如果目标利润率20%：
目标ROAS = 1 / (毛利率 - 目标利润率)
         = 1 / (0.45 - 0.20)
         = 4.0x
```
