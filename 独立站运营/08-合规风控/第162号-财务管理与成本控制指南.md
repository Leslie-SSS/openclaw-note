## 四、利润分析体系

### 4.1 利润计算模型

```
利润计算层级:

┌────────────────────────────────────────────────┐
│                  营业收入                       │
│                    ↓ -                         │
│                  营业成本    →  毛利润          │
│                    ↓ -                         │
│                  运营费用    →  营业利润        │
│                    ↓ -/+                       │
│              营业外收支     →  利润总额          │
│                    ↓ -                         │
│                  所得税      →  净利润          │
└────────────────────────────────────────────────┘

单品利润模型:

收入:
  商品售价
  + 运费收入
  + 包装费收入
  = 总收入

成本:
  商品成本
  + 物流成本
  + 包装成本
  + 平台佣金
  + 支付手续费
  = 总成本

毛利润 = 总收入 - 总成本

费用分摊:
  广告费分摊
  + 人力分摊
  + 其他费用分摊
  = 总费用分摊

净利润 = 毛利润 - 总费用分摊
```

### 4.3 盈亏平衡分析

```yaml
盈亏平衡点计算:

公式:
盈亏平衡销售额 = 固定成本 ÷ 毛利率

示例:
  固定成本: ¥100,000/月
  毛利率: 40%
  
  盈亏平衡点 = 100,000 ÷ 0.4 = ¥250,000/月

安全边际:
  安全边际 = 实际销售额 - 盈亏平衡销售额
  安全边际率 = 安全边际 ÷ 实际销售额
  
  示例:
    实际销售额: ¥400,000
    安全边际: ¥150,000
    安全边际率: 37.5%

各产品盈亏分析:
  单品盈亏平衡销量 = (固定成本 × 产品占比) ÷ 单品毛利
  
  示例:
    产品A固定成本分摊: ¥20,000
    单品毛利: ¥40
    盈亏平衡销量: 500件
```

## 六、定价策略

### 6.1 定价方法

```yaml
定价方法:

成本加成法:
  价格 = 成本 × (1 + 加成率)
  
  优点: 简单, 保证利润
  缺点: 忽略市场和竞争

市场导向法:
  价格 = 市场基准价 × 调整系数
  
  优点: 市场竞争性强
  缺点: 利润可能受影响

价值定价法:
  基于客户感知价值定价
  
  优点: 利润空间大
  缺点: 需要品牌支撑

竞争定价法:
  - 低于竞争者
  - 等于竞争者
  - 高于竞争者
```

### 6.2 动态定价策略

动态定价策略:

时间维度:
