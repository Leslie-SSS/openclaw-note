# 第156号：再营销策略深度指南

## 一、再营销概述

### 1.1 再营销定义

再营销（Remarketing/Retargeting）是指针对已经与品牌有过接触但未完成转化的用户，通过精准广告投放再次触达，引导其完成购买或其他目标行为的营销策略。

### 1.2 再营销的核心价值

| 价值维度 | 具体表现 | 数据支持 |
|---------|---------|---------|
| 转化提升 | 提高弃单用户转化率 | 再营销用户转化率比新用户高3-5倍 |
| 成本降低 | 降低获客成本 | CPA平均降低30-50% |
| 品牌强化 | 增强品牌记忆度 | 品牌认知度提升104% |
| 精准触达 | 高意向用户覆盖 | 点击率提升2-3倍 |
| ROI提升 | 提高广告投资回报 | ROI平均提升150% |

### 1.3 再营销适用场景

```
┌─────────────────────────────────────────────────────────────┐
│                    再营销触发场景                            │
├─────────────────────────────────────────────────────────────┤
│ ▶ 网站访客（浏览未购买）                                     │
│ ▶ 购物车放弃者（加购未结账）                                 │
│ ▶ 商品页浏览者（意向用户）                                   │
│ ▶ 搜索用户（特定关键词搜索）                                 │
│ ▶ 视频观看者（观看未互动）                                   │
│ ▶ 邮件打开者（邮件未点击）                                   │
│ ▶ App用户（安装未活跃）                                      │
│ ▶ 老客户（复购提醒）                                         │
└─────────────────────────────────────────────────────────────┘
```

## 五、再营销投放策略

### 5.1 频次控制

```
频次控制最佳实践:

第一周:
├── 第1-2天: 1-2次/天
├── 第3-4天: 2-3次/天
└── 第5-7天: 2次/天

第二周:
├── 第8-10天: 1-2次/天
└── 第11-14天: 1次/天

第三周及以后:
├── 活跃用户: 1次/2天
└── 非活跃用户: 停止投放或降频

关键指标:
├── 最优频次: 7-10次/受众生命周期
├── 疲劳阈值: 超过15次效果骤降
└── 负面影响: 超过20次品牌好感度下降
```

### 5.2 出价策略

|---------|---------|---------|---------|

| 目标CPA | 转化目标明确 | 设定目标CPA | 初始+30%，逐步降低 |
| 目标ROAS | ROI导向 | 设定目标ROAS | 根据产品毛利设定 |
| 最大化转化 | 预算充足 | 自动出价 | 监控CPA波动 |
| 增强型CPC | 平衡控制与自动 | 基础出价+自动调整 | 适合中级阶段 |

### 8.2 预测性再营销

```python
# 用户转化概率预测模型

def calculate_conversion_score(user):
    """
    计算用户转化概率得分
    """
    score = 0
    
    # 行为权重
    behaviors = {
        'page_views': user.page_views * 2,
        'product_views': user.product_views * 5,
        'add_to_cart': 20 if user.add_to_cart else 0,
        'begin_checkout': 30 if user.begin_checkout else 0,
        'time_on_site': min(user.time_on_site / 60, 10),
        'return_visits': user.return_visits * 3,
    }
    
    # 时间衰减
    recency_weight = calculate_recency_weight(user.last_visit)
    
    # 设备权重
    device_weight = 1.2 if user.device == 'desktop' else 0.8
    
    # 计算总分
    score = sum(behaviors.values()) * recency_weight * device_weight
    
    return score

# 分数解读:
# 0-20: 低意向 - 品牌曝光策略
# 21-50: 中等意向 - 产品展示策略
# 51-80: 高意向 - 优惠转化策略
# 81+: 极高意向 - 紧急跟进策略
```

### 10.1 问题诊断表
