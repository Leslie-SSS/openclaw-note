## 二、广州市电子商务发展政策

### 2.1 政策背景

2025年2月，广州市商务局正式印发《广州市关于推动电子商务高质量发展的若干措施》，从支持电商企业做大做强、培育电子商务标杆企业等方面提出19条政策措施，加速推进传统商贸业数字化转型。

### 2.2 重点政策内容

#### 2.2.1 支持电商企业做大做强

**（一）实物商品网上零售额分级奖励**

对2025-2027年年度实物商品网上零售额达到以下门槛，且连续两年平均增长率满足对应条件的企业，给予一次性奖励：

| 门槛 | 增长率要求 | 奖励金额 |
|:---:|:---:|:---:|
| 30亿元 | ≥10% | 不超过150万元 |
| 100亿元 | ≥6% | 不超过250万元 |
| 200亿元 | ≥4% | 不超过400万元 |
| 500亿元 | ≥3% | 不超过600万元 |
| 1000亿元 | ≥2% | 不超过1000万元 |

**特别说明**：若企业后续进档（如从500亿升至1000亿），可补发差额部分。

**（二）总部企业认定支持**

符合广州总部企业标准的电商企业，可享：
- 人才落户便利
- 人才绿卡
- 子女入学优先
- 人才公寓等服务

#### 2.2.2 培育电子商务标杆企业

**"鸿鹄"企业评选**

- **评选周期**：每两年一次（2025年、2027年）
- **奖励标准**：首次入选企业给予一次性奖励，最高不超过100万元
- **评选方式**：通过综合评价方式择优遴选

#### 2.2.3 推动电商多领域融合创新

鼓励电商企业在以下方面创新发展：
- 新技术应用
- 经营模式创新
- 线上线下融合发展
- 带动传统产业转型升级
- 扩大消费

### 2.3 申报条件

**基本条件**：
1. 在广州市依法注册、具有独立法人资格
2. 从事电子商务相关业务
3. 财务管理规范，无严重失信记录
4. 符合具体项目的专项条件

**优先支持**：
- 国家高新技术企业
- 市级以上电子商务示范企业
- 数字商务企业

## 三、广东省数字贸易发展规划

### 3.1 发展目标

根据《广东省人民政府办公厅关于以高水平开放推动数字贸易和服务贸易高质量发展的实施意见》（2026年1月发布），核心发展目标包括：

#### 3.1.1 近期目标（到2029年）

- 可数字化交付的服务贸易规模稳中有增
- 占服务贸易总额比重提高到**45%以上**
- 数字贸易基础设施布局进一步完善
- 适应数字贸易发展的体制机制基本建立

#### 3.1.2 中期目标（到2035年）

- 可数字化交付的服务贸易占比提高到**50%以上**
- 与国际高标准经贸规则对接全面加强

### 3.2 重点领域

#### 3.2.1 数字产品贸易

| 细分领域 | 发展重点 |
|:---|:---|
| 影视动漫 | 中国国际影视动漫版权保护和贸易博览会 |
| 文化娱乐 | 优质生活性服务进口，涉外电影展映 |
| 医疗健康 | 数字化医疗服务输出 |
| 游戏电竞 | 文化数字化国际协作 |

#### 3.2.2 数字服务贸易

| 细分领域 | 发展重点 |
|:---|:---|
| 数字制造外包 | 提升品牌和标准影响力 |
| 数据服务 | 数据安全、数据资产、数字信用第三方服务国际化 |
| 云计算服务 | 对外贸易拓展 |
| 人工智能服务 | AI大模型示范应用 |

#### 3.2.3 数字技术贸易

**重点发展领域**：
- 物联网
- 云计算
- 人工智能
- 区块链
- 卫星导航

**支持措施**：
- 加强数字技术研发支持
- 拓展国际技术合作网络
- 支持技术成果转化
- 研发中心技术跨境转移便利化

### 3.3 重点项目布局

| 平台/基地 | 功能定位 | 所在地 |
|:---|:---|:---|
| 广州天河中央商务区 | 数据要素集聚发展区 | 广州天河 |
| 广州开发区数字服务出口基地 | 数字贸易公共服务平台 | 广州黄埔 |
| 广州琶洲人工智能与数字经济试验区 | 人工智能大模型示范应用 | 广州海珠 |
| 东莞松山湖数字贸易港 | 信息技术外包、知识流程外包 | 东莞 |
| 横琴数字贸易国际枢纽港 | 创作者经济、数据元宇宙 | 珠海横琴 |

### 3.4 粤港澳数字贸易合作机制

#### 3.4.1 制度层面

- **规则衔接**：数百项规则衔接和改革创新措施落地
- **机制对接**：推动粤澳深度合作的关键
- **数据跨境**：探索数据跨境互联互通、匿名化处理

#### 3.4.2 平台建设

- **澳琴数字贸易国际枢纽港**：入选国家数据基础设施试点
- **穗通云贸平台**：粤港澳大湾区数字贸易新标杆
- **粤港澳大湾区服务贸易大会**：11个重点项目签约，总金额11.05亿元

## 四、广州市跨境电商政策

### 4.1 政策依据

**主要文件**：
- 《广州市进一步推动跨境电子商务高质量发展若干政策措施》（穗商务函〔2024〕24号）
- 《广州市促进商务高质量发展专项资金管理办法》（穗商务规字〔2022〕3号）

### 4.2 主要扶持措施

#### 4.2.1 跨境电商仓储设施资金扶持

| 项目 | 具体要求 |
|:---|:---|
| **支持对象** | 广州市依法注册、从事跨境电商业务的企业 |
| **支持条件** | 租赁海关特殊监管区域内仓库；单个企业租赁面积≥500㎡；租赁时间2024年3月6日至2024年12月31日；未重复享受国家或省级同类奖补 |
| **补助标准** | 按合同面积给予不超过租金30%的补助，单个企业最高150万元 |
| **项目预算** | 1200万元 |

#### 4.2.2 跨境电商"鲲鹏"产业园区认定

| 项目 | 具体要求 |
|:---|:---|
| **奖励标准** | 首次入选园区给予运营主体一次性最高500万元奖励 |
| **园区运营** | ≥1年，2024年1月1日前已开园 |
| **占地面积** | ≥1万㎡ |
| **建筑面积** | ≥2万㎡ |
| **入驻企业** | ≥30家，其中跨境电商及相关生态企业≥10家 |
| **配套设施** | 公共服务中心、直播间、共享展示区等 |
| **产权/租赁** | 有效期至2026年6月30日之后 |

#### 4.2.3 "品牌出海"企业资金扶持

**支持对象**：
- 广州注册
- 实际从事跨境电商出口
- 拥有自主品牌

**申报条件**（二选一）：
1. 品牌2024年销售额≥500万元
2. 境外线上媒体粉丝量≥100万

**扶持标准**：
- 推广费用补贴比例：不超过50%
- 单个企业年度上限：100万元
- 覆盖范围：社交媒体、搜索引擎、跨境直播、第三方平台等境外推广支出

#### 4.2.4 跨境电商"麒麟"企业认定

**政策依据**：《广州市进一步推动跨境电子商务高质量发展若干政策措施》

**奖励标准**：
- 首次进入名单的跨境电商企业给予一次性奖励
- 最高不超过200万元

**企业类型**：
- B2B平台企业
- B2C平台企业
- 第三方服务企业

## 五、番禺区电商产业扶持

### 5.1 扶持政策体系

番禺区电商产业扶持政策主要依据《番禺区扶持电子商务发展暂行办法》，涵盖企业落户、营收奖励、人才补贴、租金支持、跨境电商业务等多个方面。

### 5.2 主要扶持措施

#### 5.2.1 企业落户与成长奖励

| 奖励类型 | 奖励金额 | 条件要求 |
|:---|:---|:---|
| 知名企业落户奖 | ≥100万元 | 新迁入的市级或以上示范企业 |

| 电商十优奖 | 5万元 | 获评番禺区"电商十优"企业 |

| 项目 | 标准 |
|:---|:---|

| 补贴期限 | 3年 |

| 级别 | 奖励金额 |

## 六、粤港澳数字贸易合作

### 6.1 合作机制

#### 6.1.1 制度层面

**规则衔接与机制对接**：
- 数百项规则衔接和改革创新措施已落地
- 横琴以规则衔接、机制对接作为推动粤澳深度合作的关键
- 贯彻落实粤港澳大湾区"一点两地"全新定位

**数据跨境合作**：
- 澳琴数字贸易国际枢纽港入选国家数据基础设施试点
- 探索数据跨境互联互通
- 匿名化处理等个性化任务

#### 6.1.2 平台建设

**澳琴数字贸易国际枢纽港**：
- 建设内容：数据运河实验室、监管和服务创新机制、国际数据中心智算中心
- 功能定位：创作者经济、数据元宇宙产品确权出海平台
- 入选：国家数据基础设施建设第二批先行先试名单

**穗通云贸平台**：
- 功能：粤港澳大湾区数字贸易综合服务平台
- 战略布局：
  - 深化全球多国渠道网络
  - 拓展"一带一路"及新兴市场
  - 产业链金融带动
  - "跨境电商+产业带"标杆模式

### 6.2 重点项目

#### 6.2.1 粤港澳大湾区服务贸易大会

| 届次 | 时间 | 地点 | 成果 |
|:---|:---|:---|:---|
| 第四届 | 2025年10月 | 珠海 | 11个重点项目签约，签约总金额11.05亿元 |

**主题**：服务外包新引擎 数字贸易加速度

**聚焦领域**：
- 服务外包
- 数字贸易
- 跨境电商
- 高端咨询
- 法律服务

#### 6.2.2 重点合作项目

| 项目 | 功能定位 |
|:---|:---|

## 附录：常见问题解答

### Q1：市级和区级政策可以叠加享受吗？

**A**：通常可以叠加，但需注意：
- 同一项目不得重复享受市级和区级同类奖补
- 不同类型奖励（如落户奖+租金补贴）可叠加
- 建议申报前咨询主管部门确认

### Q2：跨境电商企业需要在哪里完成数据报备？

**A**：广州跨境电商公共服务平台（http://www.gzcba.com/）完成阳光化试点注册登记。

### Q3：申报材料需要找会计师事务所审计吗？

**A**：部分项目需要，具体要求：
- 年度财务审计报告：需有资质会计师事务所出具
- 专项审计报告：涉及具体业务数据的项目通常需要
- 建议提前2-3个月准备

### Q4：外资企业可以申报吗？

**A**：可以，只要满足以下条件：
- 在广州市依法注册
- 具有独立法人资格
- 符合具体项目条件

### Q5：申报失败后可以重新申报吗？

**A**：可以。常见原因及建议：
- 材料不完整：补充完善后下批次申报
- 条件不达标：完善业务数据后申报
- 评审未通过：根据反馈意见改进
