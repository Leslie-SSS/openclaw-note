# 311 - 营销自动化系统

## 核心要点

### 一、什么是营销自动化

营销自动化是使用软件自动执行营销任务的过程。核心价值：
- 提升效率：减少重复性人工操作
- 精准触达：基于用户行为触发个性化沟通
- 规模化运营：同时管理大规模用户

### 二、核心功能模块

**1. 触发器（Trigger）**

定义何时触发营销动作：
| 触发类型 | 示例 |
|----------|------|
| 行为触发 | 加入购物车、浏览商品、完成购买 |
| 时间触发 | 生日、注册N天、购买N天 |
| 数据变化 | 标签变更、等级升级 |
| 人工触发 | 导入名单、手动发送 |

**2. 条件判断（Condition）**

根据用户属性/行为决定执行路径：
- 用户标签：是否VIP、是否新用户
- 行为历史：是否购买过A品类
- 属性条件：地区、性别、消费金额

**3. 动作（Action）**

具体执行的营销动作：
- 发送邮件
- 发送短信
- 推送APP通知
- 添加/移除标签
- 发送微信消息
- 更新用户属性

**4. 延时（Delay）**

设置等待时间：
- 等待N小时/天
- 等待到特定时间
- 等待用户完成某行为

### 三、常见自动化场景

**1. 欢迎流程（Welcome Series）**

新用户注册后的自动化引导：
```
Day 0: 欢迎邮件 + 10%优惠码
Day 2: 品牌故事 + 热门产品推荐
Day 5: 新人专属优惠（限时）
Day 10: 引导完成首单（如果未购买）
```

**2. 弃购挽回（Abandoned Cart）**

```
弃购后1小时: 邮件提醒"您购物车的商品"
弃购后24小时: 邮件 + 小额优惠券
弃购后72小时: 最后提醒 + 库存紧张提示
```

**3. 复购提醒（Replenishment）**

根据产品消耗周期提醒：
```
购买后30天: 邮件"您的XX可能快用完了"
附上购买链接 + 小额优惠
```

**4. 流失挽回（Win-back）**

```
90天未购买: 邮件"我们想念您" + 大额优惠
120天未购买: 最后机会 + 更大优惠
180天未购买: 移出活跃名单
```

**5. 生日营销**

```
生日前7天: 预热邮件
生日当天: 专属优惠
生日后3天: 延长优惠提醒（如果未使用）
```

**6. 浏览挽回**

```
浏览某商品未购买: 24小时后推送相关商品
浏览某品类: 推送该品类热门商品
```

### 五、实施步骤

**第一步：梳理场景**
- 列出所有可能的自动化场景
- 评估优先级（ROI + 实施难度）
- 选择2-3个高价值场景优先实施

**第二步：设计流程**
- 绘制流程图
- 定义触发条件、判断逻辑、执行动作
- 设计内容模板

**第三步：技术对接**
- 埋点：确保行为数据采集
- API对接：邮件、短信、推送渠道
- 测试：全流程测试

**第四步：内容准备**
- 邮件/短信/推送模板
- 个性化变量设置
- A/B测试方案

**第五步：上线与优化**
- 小流量测试
- 监控数据（打开率、点击率、转化率）
- 持续优化

### 六、关键指标

**流程指标：**
- 触发人数
- 完成率
- 各节点转化率

**内容指标：**
- 打开率（邮件10%-30%为正常）
- 点击率（2%-5%为正常）
- 转化率
- 退订率（控制在0.5%以下）

**业务指标：**
- GMV贡献
- ROI
- 客单价提升

### 七、常见问题与解决

**问题1：触发过于频繁骚扰用户**
解决：设置频率控制，同一用户每周最多N条消息

**问题2：内容千篇一律，打开率低**
解决：个性化内容、测试不同主题行、优化发送时间

**问题3：流程复杂，维护困难**
解决：简化流程，模块化设计，定期清理无效流程

**问题4：与其他渠道冲突**
解决：统一营销日历，避免同一用户多渠道重复触达

## 关键案例

### 案例：Klaviyo赋能的电商营销自动化

某中型时尚电商使用Klaviyo实现营销自动化：

**实施的自动化流程：**

**1. 欢迎流程**
- 注册后立即发送欢迎邮件 + 15%折扣码
- 3天后发送新品推荐
- 7天后对未购买用户发送首单激励
- **成果**：欢迎流程贡献15%首单转化

**2. 弃购挽回**
- 弃购后2小时发送提醒邮件
- 24小时后发送5%优惠券
- 三段式挽回，挽回率12%

**3. 浏览挽回**
- 浏览商品页未加购：24小时后推送相似商品
- 浏览某品类：推送该品类热销榜

**4. 复购提醒**
- 购买后根据品类周期推送
- 服装：30天推送新品
- 消耗品：根据使用周期提醒

**5. 流失挽回**
- 60天未购买：10%优惠
- 90天未购买：20%优惠
- 120天未购买：30%优惠+限量商品
- **成果**：挽回流失用户贡献8%营收

**6. 生日营销**
- 收集用户生日
- 生日当月发送专属优惠
- **成果**：生日邮件打开率45%，是常规邮件2倍

**整体成果：**
- 营销自动化贡献35%营收
- ROI达到20:1
- 客服成本降低（自动化解答常见问题）
- 复购率提升25%

**可借鉴点：**
- 从高价值场景开始（欢迎、弃购）
- 持续测试优化
- 内容个性化
- 控制发送频率

## 实操建议

1. **场景优先**：先明确要解决的问题，再选工具
2. **小步快跑**：从2-3个场景开始，跑通再扩展
3. **内容为王**：自动化只是工具，内容质量决定效果
