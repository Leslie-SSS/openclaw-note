📋 可申报政策详解

1 立即可申报项目（2026年）

软件企业认定

政策依据：《关于软件和集成电路产业企业所得税优惠政策》

最高优先级
💰 核心优惠
  • • 增值税即征即退：软件产品销售税负超3%部分退还
  • • 企业所得税"两免三减半"：获利年度起，前两年免征，后三年减半
  • • 研发费用加计扣除：75%-100%加计扣除
✅ 申报条件
  • • 拥有核心关键技术
  • • 研发人员占比 ≥ 20%
  • • 研发费用占比 ≥ 6%
  • • 软件产品收入占比 ≥ 50%
📂 所需材料
1. 软件企业认定申请表
2. 企业营业执照复印件
3. 软件产品测试报告
4. 研发人员名单及证明
5. 研发费用专项审计报告
6. 软件产品销售合同/发票

创业带动就业补贴

政策名称：广州市创业带动就业补贴

高优先级（招人后）
💰 补贴标准
  • • 初创企业补贴：10,000元
  • • 招用人员补贴：每人2,000-3,000元
  • • 社保补贴：单位缴纳部分（最长3年）
✅ 申报条件
  • • 注册3年内的小微企业
  • • 招用员工并缴纳社保满6个月
💡 行动步骤：招聘计划 → 签订合同 → 缴纳社保 → 满6个月后申报

高新技术企业认定

可申报时间：2027年（需注册满1年）

核心目标
💰 核心优惠：企业所得税减按 15% 征收（标准25%）

附加收益：品牌提升、政府项目优先支持、融资加分

✅ 申报条件
  • • 注册满1年以上
  • • Ⅰ类知识产权1件 或 Ⅱ类知识产权6件
  • • 研发费用占比达标（营收<5000万需≥5%）
  • • 高新技术产品收入占比≥60%
  • • 科技人员占比≥10%

  • □ 保持研发人员占比≥10%
  • □ 规范财务管理制度
