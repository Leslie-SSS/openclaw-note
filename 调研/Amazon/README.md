# Amazon 卖家运营指南

## 一、平台概述

### Amazon 适合谁？

| 适合人群 | 不适合人群 |
|---------|-----------|
| 有供应链优势的卖家 | 纯新手卖家 |
| 能接受较低利润率 | 追求高利润率 |
| 愿意囤货做 FBA | 只做代发货 |
| 有一定资金实力 | 启动资金有限 |

### 市场对比

| 市场 | 卖家数量 | 竞争程度 | 增长潜力 |
|------|---------|---------|---------|
| **美国** | 最多 | 激烈 | 中 |
| **欧洲** | 较多 | 中等 | 高 |
| **日本** | 中等 | 中等 | 高 |
| **新兴市场** | 较少 | 较低 | 最高 |

## 三、选品策略

### 选品标准

| 指标 | 标准 | 说明 |
|------|------|------|
| 售价 | $20-50 | 利润空间足够 |
| 利润率 | >30% | 覆盖广告成本 |
| 月销量 | 300-3000 | 有市场需求 |
| 竞争度 | <100 reviews | 能进入市场 |
| 重量 | <2lb | 降低FBA成本 |
| 季节性 | 弱 | 全年销售 |

### 选品工具

| 工具 | 功能 | 费用 |
|------|------|------|
| **Helium 10** | 综合选品 | $99-399/月 |
| **Jungle Scout** | 市场分析 | $49-189/月 |
| **AMZScout** | 利润计算 | $45-600/年 |
| **Keepa** | 价格追踪 | €19/月 |

### 选品流程

```
Step 1：市场调研
├── 使用工具筛选类目
├── 分析 TOP 100 产品
├── 找出差异化机会
└── 确定目标产品

Step 2：竞争分析
├── 分析竞品 listing
├── 研究 reviews 找痛点
├── 确定差异化卖点
└── 制定定价策略

Step 3：供应链对接
├── 1688/阿里巴巴找货
├── 对比 3-5 家供应商
├── 获取样品测试
└── 谈判价格和MOQ

Step 4：小批量测试
├── 首批 50-100 件
├── 发 FBA 测试
├── 开广告引流
└── 分析数据决策
```

## 四、ACOS 优化策略

### ACOS 基础

```
ACOS = 广告成本 / 广告销售额 × 100%

健康指标：
- 新品期：30-50%（可接受高）
- 成长期：20-30%
- 成熟期：10-20%
```

### 优化步骤

**Step 1：关键词优化**
- 否定无效搜索词
- 添加精准匹配关键词
- 定期更新关键词列表

**Step 2：竞价优化**
- 高转化词提高竞价
- 低转化词降低竞价
- 使用动态竞价策略

**Step 3：广告结构**
- 自动广告：发现新关键词
- 手动广告：精准投放
- 品牌广告：防御+曝光

### ACOS 优化表

| ACOS | 问题 | 优化方向 |
|------|------|---------|
| >50% | 严重亏损 | 检查产品/关键词 |
| 30-50% | 新品正常 | 持续优化 |
| 20-30% | 健康范围 | 适度扩量 |
| 10-20% | 优秀 | 扩大预算 |
| <10% | 可能机会不足 | 检查曝光量 |

## 五、Listing 优化

### Listing 要素

| 要素 | 权重 | 优化要点 |
|------|------|---------|
| **标题** | 30% | 关键词前置，品牌+核心卖点 |
| **图片** | 25% | 主图白底，多角度展示 |
| **五点描述** | 20% | 痛点+解决方案 |
| **A+页面** | 15% | 品牌故事+产品细节 |
| **评论** | 10% | 持续积累好评 |

### 标题公式

```
[品牌名] + [核心关键词] + [材质/特性] + [适用场景] + [规格]

示例：
Sony WH-1000XM5 Wireless Noise Canceling Headphones, Black
```

### 图片规范

| 图片类型 | 要求 |
|---------|------|
| 主图 | 白底，1000x1000px+，占框85% |
| 场景图 | 展示使用场景 |
| 细节图 | 展示产品特性 |
| 尺寸图 | 展示尺寸规格 |
| 包装图 | 展示包含内容 |

## 七、风险提示

### 常见坑点

1. **账号关联** - 多账号被封
2. **侵权投诉** - 产品/图片侵权
3. **库存积压** - 卖不动占资金
4. **政策违规** - 被警告/封号
5. **评论造假** - 被检测处罚

### 避坑建议

- 每个账号独立IP/设备
- 确保产品无侵权风险
- 小批量测试再放大
- 遵守平台规则
- 通过正规渠道获取评论
