# eBay 运营指南

## 一、平台定位

### eBay 适合谁？

| 适合人群 | 不适合人群 |
|---------|-----------|
| 汽配卖家 | 快时尚卖家 |
| 收藏品卖家 | 低客单价卖家 |
| 二手/翻新卖家 | 纯新手卖家 |
| 有供应链优势的卖家 | 无货源卖家 |

### 平台特点

- **买家年龄偏大**：35-55岁为主
- **价格敏感度高**：比价文化盛行
- **重视评价体系**：好评率影响流量
- **拍卖模式独特**：部分品类适合拍卖

## 五、运营技巧

### Listing 优化

**标题公式**：
```
[品牌] [关键词1] [关键词2] [规格/型号] [适用车型/型号]
```

**示例**：
```
Bosch Ceramic Brake Pads Front Rear Kit for Honda Civic 2016-2021
```

**图片要求**：
- 主图白底，1200x1200像素以上
- 至少 5 张图片
- 包含尺寸图/安装图

### 定价策略

| 策略 | 适用场景 | 操作方法 |
|------|---------|---------|
| **一口价** | 标准商品 | 略低于竞品 5-10% |
| **拍卖** | 稀缺品/收藏品 | 起拍价设低价吸引竞价 |
| **Best Offer** | 高价商品 | 设置可接受最低价 |

### 物流建议

| 方案 | 时效 | 成本 | 推荐度 |
|------|------|------|--------|
| ePacket | 15-25天 | 低 | ⭐⭐⭐ |
| 专线物流 | 7-15天 | 中 | ⭐⭐⭐⭐ |
| 海外仓 | 3-7天 | 高 | ⭐⭐⭐⭐⭐ |

**建议**：高价值商品使用海外仓，提升买家体验

## 七、风险提示

### 常见坑点

1. **账号被限制** - 新账号突然增加销量
2. **差评影响** - 一个差评可能毁掉新品
3. **平台政策变化** - 费率上涨、规则调整
4. **竞争激烈** - 价格战压缩利润

### 避坑建议

- 新账号首月控制在额度 80% 以内
- 严格品控，减少退货率
- 关注 eBay 卖家中心政策更新
- 差异化选品，避免红海竞争
