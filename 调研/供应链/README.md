## 三、库存管理

### 库存策略

```
库存管理模型：

1. 安全库存
   公式：安全库存 = 日均销量 × 安全天数
   
   安全天数参考：
   ├── 热销品：14-21天
   ├── 常规品：7-14天
   ├── 长尾品：3-7天
   └── 季节性：根据周期

2. 补货点
   公式：补货点 = (日均销量 × 采购周期) + 安全库存
   
   示例：
   ├── 日均销量：50件
   ├── 采购周期：30天
   ├── 安全库存：1000件
   └── 补货点 = 50×30 + 1000 = 2500件

3. 补货量
   公式：补货量 = 补货点 - 当前库存 - 在途库存
   
   考虑因素：
   ├── 最低起订量
   ├── 经济订货量
   ├── 资金占用
   └── 仓储成本

4. 库存周转
   目标周转天数：
   ├── 快消品：30-45天
   ├── 一般商品：45-90天
   ├── 长尾商品：90-120天
   └── 季节性：季内清空
```

### 库存优化

```
库存优化方法：

1. ABC分类
   ├── A类（70-80%销售）：精细管理
   ├── B类（15-20%销售）：常规管理
   └── C类（5-10%销售）：简化管理

2. 滞销处理
   ├── 30天未售：关注
   ├── 60天未售：促销
   ├── 90天未售：清仓
   └── 120天+：捐赠/报废

3. 库存预测
   ├── 历史数据分析
   ├── 季节性因素
   ├── 促销活动影响
   ├── 市场趋势
   └── 安全边际

4. 库存盘点
   ├── 定期盘点（月/季）
   ├── 循环盘点（重点SKU）
   ├── 系统记录
   └── 差异分析
```

## 五、风险管理

### 供应链风险

```
供应链风险类型：

1. 供应风险
   ├── 供应商停产
   ├── 质量问题
   ├── 产能不足
   └── 应对：多供应商备份

2. 物流风险
   ├── 时效延误
   ├── 货物丢失/损坏
   ├── 清关问题
   └── 应对：保险+备选方案

3. 库存风险
   ├── 滞销积压
   ├── 缺货断供
   ├── 质量问题
   └── 应对：精准预测+安全库存

4. 政策风险
   ├── 贸易壁垒
   ├── 关税变化
   ├── 合规要求
   └── 应对：多市场布局

风险应对措施：
├── 建立备份供应商
├── 货物保险
├── 安全库存
├── 多渠道布局
├── 合规自查
└── 信息监控
```

### 质量控制

```
质量控制流程：

1. 产前
   ├── 样品确认
   ├── 规格锁定
   └── 质量标准制定

2. 生产中
   ├── 首件检验
   ├── 过程抽检
   └── 问题及时反馈

3. 出货前
   ├── 全检/抽检
   ├── 包装检查
   ├── 文件确认
   └── 不合格处理

4. 到货后
   ├── 入库检验
   ├── 问题记录
   └── 供应商反馈

质量指标：
├── 良品率：>98%
├── 退货率：<5%
├── 投诉率：<1%
└── 供应商质量评分：>90分
```
