## 三、爆款选品方法论

### 选品三原则

1. **视觉冲击力** - 视频展示效果好
2. **价格敏感度低** - $10-30 客单价最佳
3. **复购率高** - 消耗品/快消品

### 爆款特征

| 特征 | 权重 | 说明 |
|------|------|------|
| 视觉效果 | 30% | 视频展示吸引人 |
| 价格竞争力 | 25% | 比竞品便宜10-20% |
| 新奇特程度 | 20% | 有差异化卖点 |
| 实用性 | 15% | 解决实际问题 |
| 话题性 | 10% | 可引发讨论 |

### 选品工具

| 工具 | 功能 | 费用 |
|------|------|------|
| **TikTok Creative Center** | 热门内容分析 | 免费 |
| **EchoTik** | 爆款追踪 | $29-99/月 |
| **FastMoss** | 达人/商品分析 | $49-199/月 |
| **TikTok Ads Library** | 广告创意参考 | 免费 |

### 选品流程

```
Week 1：市场调研
├── 浏览 TikTok 热门标签
├── 分析 Top 100 爆款
├── 确定目标品类
└── 列出 20-30 个候选产品

Week 2：供应链对接
├── 1688/阿里巴巴找货
├── 获取样品
├── 对比价格/质量
└── 确定供应商

Week 3：小批量测试
├── 首批 50-100 件
├── 制作 3-5 条视频
├── 投放小额广告测试
└── 数据分析选爆款

Week 4：放量推广
├── 增加爆款库存
├── 扩大广告预算
├── 达人合作推广
└── 持续优化
```

## 四、流量获取策略

### 三大流量来源

| 来源 | 占比 | 成本 | 起效时间 |
|------|------|------|---------|
| **短视频** | 40-50% | 低 | 1-2周 |
| **直播** | 30-40% | 中 | 即时 |
| **广告** | 10-20% | 高 | 立即 |

### 短视频运营

**内容公式**：
```
爆款视频 = 痛点(3秒) + 解决方案(15秒) + 引导购买(5秒)
```

**发布技巧**：
- 每天 1-3 条视频
- 黄金时段：当地 18:00-22:00
- 标签：3-5个相关 + 1-2个热门
- 音乐：热门BGM

**数据指标**：
| 指标 | 健康值 | 说明 |
|------|--------|------|
| 播放完成率 | >50% | 内容吸引力 |
| 点赞率 | >5% | 用户喜爱度 |
| 评论率 | >0.5% | 互动程度 |
| 分享率 | >0.3% | 传播力 |

### 直播带货

| 类型 | 适合场景 | 时长 | 频率 |
|------|---------|------|------|
| 日常直播 | 固定粉丝 | 1-2小时 | 每天 |
| 大促直播 | 活动/新品 | 3-6小时 | 周末/节日 |
| 达人直播 | 品牌曝光 | 1-3小时 | 合作时 |

**直播技巧**：
- 固定直播时间培养习惯
- 互动抽奖增加停留
- 限时限量制造紧迫感
- 多机位展示产品细节

### 广告投放

| 广告类型 | 适合场景 | 平均 CPM | ROAS |
|---------|---------|---------|------|
| 信息流广告 | 品牌曝光 | $5-10 | 1.5-3 |

### 合作方式

| 方式 | 说明 | 费用模式 |
|------|------|---------|
| **纯佣金** | 按销量分成 | 10-20%佣金 |

1. 筛选达人：粉丝画像匹配、互动率高

3. 寄送样品：确保达人收到产品
4. 内容审核：确认视频符合要求
5. 发布推广：选择最佳时间发布
6. 效果追踪：用专属链接追踪销量

### 利润计算

```
单品利润 = 售价 - 采购成本 - 平台佣金 - 物流成本 - 广告成本

示例（$25 客单价）：
- 售价：$25
- 采购成本：$8（32%）
- 平台佣金：$1.5（6%）
- 物流成本：$4（16%）
- 广告成本：$5（20%）
- 利润：$6.5（26%）
```

## 八、风险提示

### 常见坑点

1. **内容违规** - 账号被限流/封禁
2. **产品质量** - 差评影响店铺评分
3. **物流延迟** - 影响用户体验
4. **政策变化** - 规则调整影响运营
5. **达人合作纠纷** - 预期不一致

### 避坑建议

- 学习平台内容规范
- 严格把控产品质量
- 选择靠谱物流服务商
- 关注官方政策更新
- 合作前明确条款
