## 二、Shopee 完整指南

### 平台概览

| 指标 | 数据 |

| 母公司 | Sea Group |

### 费用结构

```
Shopee费用构成：

1. 佣金（Commission Fee）
   ├── 普通卖家：5-6%
   ├── 平台卖家（跨境）：5-6%
   ├── Preferred卖家：4-5%
   └── Mall卖家：3-5%

2. 交易费（Transaction Fee）
   ├── 2%（含支付处理）
   └── 每笔订单

3. 物流费
   ├── SLS（Shopee Logistics Service）
   ├── 按重量/体积计算
   └── 按目的地不同

4. 其他费用
   ├── 活动参与费
   ├── 仓储费（如有）
   └── 广告费（按消耗）

综合费率：15-25%
```

### 入驻要求

```
Shopee跨境店入驻：

必备资料：
├── 企业营业执照
├── 法人身份证
├── 手机号/邮箱
├── 店铺Logo
└── 品牌授权（如有）

审核周期：
├── 资料审核：1-3天
├── 店铺开通：3-5天
└── 首批上架：1周

注意事项：
├── 一个营业执照可开多国站点
├── 首站建议：台湾/马来西亚
├── 跨境店与本土店有区别
└── 流量政策不同
```

**东南亚电商总额**：~$1,600亿+（2025年），预计2026年突破$2,000亿

### TikTok Shop vs Shopee 东南亚竞争

| 维度 | Shopee | TikTok Shop |
|------|--------|-------------|
| 市场份额 | ~50% | ~12%（快速增长） |
| 核心优势 | 全品类、COD、游戏化 | 内容电商、达人生态 |
| 用户画像 | 全年龄层 | 18-34岁为主 |
| 直播GMV占比 | 15-20% | 40-50% |
| 卖家策略建议 | 基础盘+转化 | 内容引流+品牌曝光 |

**最佳策略**：双平台运营——Shopee做转化复购，TikTok Shop做内容引流

### Shopee 最新政策变化（2025-2026）

1. **佣金调整**：跨境卖家佣金统一调整为5-6%（部分品类6-8%）
2. **SLS物流优化**：配送时效从5-15天缩短至3-10天（主要城市）
3. **Shopee Live激励**：直播卖家享受额外流量扶持和佣金减免
4. **Shopee Video**：短视频功能全面上线，类似TikTok的内容电商模式
5. **本地化仓**：在印尼、泰国推出本地仓服务，降低物流时效
