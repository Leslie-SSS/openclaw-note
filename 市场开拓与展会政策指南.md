## 二、广州市市场开拓专项资金

### 2.1 政策概述

广州市市场开拓专项资金是广州市商务局为支持企业开拓国际市场、参与境内外展会而设立的专项财政资金。该政策旨在帮助企业抢抓国际订单、稳定外贸增长、拓展市场渠道。

**政策依据：**
- 《2024年省级促进经济高质量发展专项资金开拓国际市场项目申报指南》
- 《2024年市级外贸稳增长资金开拓国际市场项目申报指南》
- 《广州市关于加快会展业发展的若干措施》（穗商务规字〔2024〕3号）

### 2.2 支持对象

**基本条件：**
1. 在广州市内注册经营的企业
2. 未被列入"失信联合惩戒黑名单"
3. 具有独立法人资格
4. 依法纳税、合规经营

**重点支持企业类型：**
- 外贸进出口企业
- 制造业企业
- 跨境电商企业
- 会展服务企业
- 品牌推广企业

### 2.3 支持内容及标准

#### 2.3.1 境外线下展会

**（一）省"粤贸全球"重点线下展会**

对参加经广东省商务厅比选确定的"粤贸全球"境外展会：

| 项目 | 标准 |
|-----|------|
| 展位数量 | 每个境外展会最多1个标准展位（9平方米） |
| 补贴比例 | 不超过展位费的80% |
| 最高金额 | 30,000元/展位 |
| 资金来源 | 省级资金 |
| 国家/地区划分 | 按附件5标准分档支持 |

**（二）一般性线下展会**

对参加重点展会以外的其他境外展会：

| 项目 | 标准 |
|-----|------|
| 展位数量 | 每个境外展会项目最多1个标准展位（9平方米） |
| 补贴比例 | 不超过展位费的50% |
| 最高金额 | 18,000元/展位 |
| 资金来源 | 省级资金 |

**（三）广州市重点境外展会**

对参加经广州市商务局公开发布的重点境外展会：

| 项目 | 标准 |
|-----|------|
| 展位数量 | 每个境外展会项目最多1个标准展位（9平方米） |
| 省市合计比例 | 不超过展位费的80% |
| 省市合计最高 | 50,000元 |
| 市级单展会最高 | 30,000元 |
| 备注 | 省市资金可叠加申报 |

**重要提示：**
- **原则上每家企业全年申报线下展会不得超过5个**
- **相同线下展位可同时申请省、市资金**

#### 2.3.2 线上展会

对经广东省商务厅比选确定的"粤贸全球"广东商品线上境外展览平台参展：

| 项目 | 标准 |
|-----|------|
| 补贴比例 | 不超过实际支出的80% |
| 最高金额 | 5,000元/展会 |
| 展会限制 | 每家企业全年不超过5个 |

#### 2.3.3 会展业专题项目

**（一）新展会培育奖励**

| 展会规模 | 奖励标准 |
|---------|---------|
| 新办展会 | 最高300万元 |
| 首届规模要求 | 需达到一定展览面积 |
| 举办地点 | 广州市范围内 |

**（二）企业"出海"办展支持**

| 项目 | 标准 |
|-----|------|
| 支持内容 | 企业自主在境外举办展会 |
| 最高金额 | 50万元/项目 |
| 申请条件 | 需提前备案 |

**（三）展会规模扩大奖励**

| 展览面积 | 补贴比例 | 备注 |
|---------|---------|------|
| 10万㎡（含）以上 | 新增面积场租80% | 展期不超过5天 |
| 5万-10万㎡ | 新增面积场租50% | 需面积增长 |
| 6,000㎡-5万㎡ | 新增面积场租50% | 面积增长20%以上 |

### 2.4 申报材料清单

#### 2.4.1 线下展会申报材料

**必备材料：**

1. **申请表格**
   - 《2024年省级促进经济高质量发展专项资金（促进外贸发展方向）开拓国际市场项目申请表》（附件1）
   - 申请市资金的需填写《2024年市级稳外贸资金开拓国际市场项目申请表》（附件2）

2. **企业资质材料**
   - 国家企业信用信息公示系统各栏目资料查询结果（https://gd.gsxt.gov.cn/index.html下载打印）
   - 企业营业执照副本复印件（加盖公章）

3. **展会证明材料**
   - 邀请函或展会主办方出具的展位确认函（须注明展位数量、面积）
   - 参展企业与境内（外）展会主办方或组展方签订的展位数量合同并注明展位编号
   - 展会主办方对组展企业的授权书或同等效力证明材料

4. **财务凭证**
   - 支付参展费的银行付款凭证或支出外汇认购展位的银行付汇凭证
   - 参展费发票原件及复印件

5. **参展人员材料**
   - 参展人员的护照首页或通行证（正、反面）
   - 境外展会举办地的出境和入境记录页
   - 我国出境和入境记录页
   - 签证页
   - 电子通关无出入境盖章的，需自行打印出入境记录凭证

6. **现场材料**
   - 展会主办方官方发布的参展企业名录及平面图
   - 展会现场拍摄的展位照片或视频（须包含参展企业名称和展位号信息）

7. **其他**
   - 涉外文材料需提供中文翻译件

#### 2.4.2 线上展会申报材料

1. 《2024年省级促进经济高质量发展专项资金（促进外贸发展方向）开拓国际市场项目申请表》（附件1）

2. 国家企业信用信息公示系统各栏目资料查询结果

3. 企业营业执照

4. 展位网页截图（须含有网址信息和参展商品信息）

5. 磋商截图（须含有买家和参展商信息，磋商日期）

6. 企业与"粤贸全球"广东线上展览平台承办单位签订的参展合同（若展会时间变更，另需提供承办单位盖章情况说明）

7. 支付参展费的银行付款凭证或支出外汇认购展位的银行付汇凭证，以及参展费发票

8. 涉外文材料需提供中文翻译件

### 2.5 支持时间范围

**2024年度政策支持时间：**
- 上半年项目：2023年1月1日至6月30日
- 下半年项目：2023年7月1日至12月31日

**注意事项：**
- 展会举办时间需在支持时间范围内
- 费用支付时间可适当延后
- 以展会实际举办日期为准

## 三、广东省展会扶持政策

### 3.1 省级政策体系概述

广东省展会扶持政策主要围绕"粤贸全球"和"粤贸全国"两大品牌展开，形成境外展会和境内展会双轮驱动的政策格局。

**政策框架：**
- 省级促进经济高质量发展专项资金（开拓国际市场项目）
- 中央外经贸发展专项资金（开拓重点市场事项）
- "粤贸全国"活动目录支持政策

### 3.2 "粤贸全球"境外展会补贴

#### 3.2.1 支持范围

参加列入《"粤贸全球"广东商品境外展览平台列表》的展会项目。

**2024年"粤贸全球"重点展会特点：**
- 经广东省商务厅比选确定
- 覆盖主要国际市场
- 兼顾传统市场和新兴市场
- 优先支持"一带一路"沿线国家展会

#### 3.2.2 补贴标准

| 展会类型 | 补贴比例 | 单展位最高 | 展位限制 | 年度限制 |
|---------|---------|-----------|---------|---------|
| "粤贸全球"重点展会 | 80% | 30,000元 | 2个标准展位 | 5场 |
| 其他境外展会 | 50% | 18,000元 | 1个标准展位 | 5场 |

**标准展位定义：**
- 每个标准展位按9平方米计算
- 超过9平方米按实际面积比例折算
- 特装展位按标准展位面积核定

#### 3.2.3 国家/地区补贴分档

根据目标市场的经济发展水平和开拓难度，补贴标准分档设置：

**一档地区（最高支持）：**
- 欧美发达国家和地区
- 展位费较高的大型国际展会

**二档地区（中等支持）：**
- 日韩、东南亚等周边市场
- 中等规模区域性展会

**三档地区（基础支持）：**
- "一带一路"沿线新兴市场
- 开发中国家重点展会

### 3.3 "粤贸全国"境内展会补贴

#### 3.3.1 政策概述

"粤贸全国"是广东省推动企业开拓国内市场的重要举措，通过支持企业参加境内重点展会，助力构建国内国际双循环新发展格局。

**支持范围：**
- 入选当年度"粤贸全国"活动目录的展会
- 由广东省商务厅比选确定
- 覆盖全国主要城市和重点行业

#### 3.3.2 补贴标准

**省级统筹标准：**
- 补贴比例：最高可达展位费实际支出的100%
- 具体标准以各地市商务局政策为准
- 优先支持目录内展会

**2024年"粤贸全国"目录特点：**
- 涵盖国内重点行业展会
- 兼顾综合性展会和专业性展会
- 优先支持与广东优势产业相关展会

#### 3.3.3 申报要求

1. 展会需在当年度目录内
2. 企业需按规定程序申报
3. 提供完整的参展证明材料
4. 配合做好绩效评价工作

### 3.4 重点展会目录

#### 3.4.1 境外重点展会目录

**目录发布机制：**
- 广东省商务厅每年定期发布
- 通过公开比选程序确定
- 企业可提前关注目录信息

**目录查询渠道：**
- 广东省商务厅官网
- 粤企政策通平台
- 各地市商务局通知

#### 3.4.2 境内重点展会目录

**典型目录展会类型：**

| 展会类型 | 代表展会 | 举办城市 |
|---------|---------|---------|
| 综合性展会 | 中国进出口商品交易会（广交会） | 广州 |
| 消费电子 | 中国国际消费电子博览会 | 青岛 |
| 服装纺织 | 中国国际服装服饰博览会 | 上海 |
| 家具家居 | 中国国际家具展览会 | 上海 |
| 食品饮料 | 全国糖酒商品交易会 | 成都/济南 |
| 建材五金 | 中国国际建筑装饰博览会 | 广州 |

### 3.5 申报流程

#### 3.5.1 项目入库

**入库申请时间：**
- 一般在每年上半年开放
- 需提前做好项目规划
- 关注商务部门通知

**入库材料：**
1. 《开拓国际市场项目入库申报表》
2. 参展企业营业执照
3. 参展计划说明
4. 与组展单位签订的合同

#### 3.5.2 资金拨付

**拨付申请材料：**
1. 《资金项目拨付申请表》（商务部业务系统统一平台下载）
2. 与组展单位签订的合同（注明展位数量和展位编号）
3. 支付参展费的银行付款凭证
4. 参展费发票
5. 参展人员护照及出入境记录
6. 展位现场照片

**拨付流程：**
1. 企业在线填报
2. 区级初审
3. 市级复核
4. 省级审核
5. 公示公告
6. 资金拨付

## 四、广州市境外展会补贴

### 4.1 市级外贸稳增长资金

#### 4.1.1 政策定位

广州市市级外贸稳增长资金是省级政策的重要补充，旨在与省级资金形成合力，为企业参展提供更全面的支持。

**政策特点：**
- 可与省级资金叠加申报
- 重点支持广州市重点境外展会目录内项目
- 简化申报流程，提高审核效率

#### 4.1.2 支持标准

**重点境外展会：**
| 项目 | 标准 |
|-----|------|
| 省市合计比例 | 不超过展位费的80% |
| 省市合计最高 | 50,000元 |
| 市级单展会最高 | 30,000元 |
| 展位限制 | 1个标准展位（9平方米） |

**目录外展会：**
- 按省级政策予以支持
- 不享受市级叠加补贴

#### 4.1.3 广州市重点境外展会目录

**目录构成：**
- 涵盖主要出口目标市场
- 优先支持优势产业展会
- 兼顾传统市场和新兴市场

**典型目录展会：**
- 美国拉斯维加斯国际消费电子展（CES）
- 德国法兰克福国际春季消费品展览会
- 中东（迪拜）国际秋季商品博览会
- 日本东京国际礼品展览会
- 俄罗斯莫斯科国际食品展览会

### 4.2 会展业高质量发展资金

#### 4.2.1 政策依据

《广州市关于加快会展业发展的若干措施》（穗商务规字〔2024〕3号）

#### 4.2.2 支持项目

**（一）新展会培育**

支持对象：
- 在广州市新创办的展会项目
- 达到一定规模要求
- 具有发展潜力

支持标准：
- 最高奖励300万元
- 根据展会规模和影响力分级支持

**（二）展会规模扩大**

支持对象：
- 已连续举办的展会项目
- 展览面积实现增长
- 参展商数量增加

支持标准：
| 展览面积 | 补贴标准 |
|---------|---------|
| 10万㎡以上 | 新增面积场租80% |
| 5-10万㎡ | 新增面积场租50% |
| 6,000㎡-5万㎡（增长20%以上） | 新增面积场租50% |

**（三）企业"出海"办展**

支持对象：
- 广州市企业自主在境外举办展会
- 组织广州企业参展
- 推广广州品牌和产品

支持标准：
- 最高支持50万元/项目

**（四）会展企业落户**

支持对象：
- 新落户广州的会展企业
- 达到一定经营规模
- 带动产业发展

支持标准：
- 按企业规模和贡献度给予奖励

### 4.3 申报指南

#### 4.3.1 申报时间

**2024年度项目：**
- 项目完成时间：2024年1月1日至12月31日
- 申报时间：2025年1-3月（以实际通知为准）

#### 4.3.2 申报渠道

**线上申报：**
- 广州市商务局官网
- 政务服务平台

**线下提交：**

## 七、申报流程指南

### 7.1 申报前准备

#### 7.1.1 资质审查

**自查清单：**
- [ ] 企业注册地在广州市内
- [ ] 营业执照有效且经营范围涵盖相关业务
- [ ] 未被列入"失信联合惩戒黑名单"
- [ ] 依法纳税、合规经营
- [ ] 无重大违法违规记录

#### 7.1.2 项目规划

**规划要点：**
1. 确定参展展会及时间
2. 核对展会是否在重点目录内
3. 评估可申请的政策类型
4. 制定申报计划时间表

#### 7.1.3 材料准备

**核心材料：**
- 企业基础证照
- 参展合同
- 付款凭证
- 发票原件
- 参展人员证件

### 7.2 申报流程详解

#### 7.2.1 在线申报

**平台入口：**
- 商务部业务系统统一平台：zxkt.mofcom.gov.cn
- 广东省政务服务网
- 广州市商务局官网

**操作步骤：**
1. 注册/登录账号
2. 选择对应项目类型
3. 填写申请表信息
4. 上传证明材料
5. 提交审核

#### 7.2.2 材料递交

**纸质材料要求：**
- 一式两份（部分项目一式一份）
- 加盖申报单位公章
- 加盖骑缝章
- 按顺序装订

**递交地点：**
- 所在区政务服务中心
- 指定受理窗口
- 邮寄（需保留邮寄凭证）

#### 7.2.3 区级初审

**初审内容：**
- 材料完整性审查
- 企业资质核验
- 项目真实性核查
- 出具初审意见

**初审时限：**
- 一般为5-10个工作日
- 材料不齐的一次性告知补正

#### 7.2.4 市级/省级评审

**评审环节：**
1. 形式审查
2. 第三方机构评审
3. 现场核查（必要时）
4. 专家评审（重大项目）
5. 集体审议

**评审时限：**
- 一般为30-60个工作日
- 复杂项目适当延长

#### 7.2.5 公示与拨付

**公示要求：**
- 公示期不少于5个工作日
- 接受社会监督
- 异议处理机制

**资金拨付：**
- 公示无异议后拨付
- 拨付至企业银行账户
- 保留相关凭证备查

### 7.3 注意事项

#### 7.3.1 时间节点

**关键时间点：**
- 展会举办前：确认展会是否在目录内
- 展会举办中：收集保存参展证明材料
- 展会举办后：及时整理申报材料
- 申报窗口期：关注申报通知，按时提交

#### 7.3.2 材料规范

**常见问题及解决方案：**

| 问题 | 解决方案 |
|-----|---------|
| 材料不全 | 按清单逐项核对，提前准备 |
| 材料过期 | 及时更新证照、年检材料 |
| 格式不符 | 按照模板要求整理 |
| 签章遗漏 | 检查所有需盖章的材料 |
| 翻译缺失 | 涉外材料提供中文翻译件 |

#### 7.3.3 常见误区

**误区一：重复申报**
- 同一项目不能重复申报同一层级资金
- 省、市级资金可叠加但总额有限制

**误区二：超出时限**
- 严格按照申报时间窗口
- 超期项目一般不予受理

**误区三：材料造假**
- 严格保证材料真实性
- 造假行为将被追究责任

### 7.4 申报技巧

#### 7.4.1 政策组合

**优化策略：**
- 优先选择目录内展会
- 合理规划年度参展计划
- 最大化利用政策红利

**案例示范：**
某企业参加美国CES展会：
1. 展会在"粤贸全球"目录内 → 申请省级80%补贴（最高3万）
2. 展会在广州市重点目录内 → 申请市级叠加补贴（最高3万）
3. 省市合计可获得最高5万元补贴

#### 7.4.2 时间管理

**建议时间表：**
| 时间节点 | 工作内容 |
|---------|---------|
| 展前3个月 | 确认展会目录，准备合同 |
| 展前1个月 | 完成报名缴费，保留凭证 |
| 展中 | 收集现场照片、证明材料 |
| 展后1个月 | 整理材料，准备申报 |
| 申报窗口期 | 提交申报材料 |

### 附录二：常见问题解答

**Q1：同一展会可以同时申请省、市资金吗？**
A：可以。相同线下展位可同时申请省、市资金，但省市合计不超过展位费的80%，合计最高不超过5万元。

**Q2：年度参展数量有限制吗？**
A：有。原则上每家企业全年申报线下展会不得超过5个，线上展会不得超过5个。

**Q3：展会目录在哪里查询？**
A：可通过广东省商务厅官网、广州市商务局官网、粤企政策通平台查询当年度重点展会目录。

**Q4：申报材料需要公证吗？**
A：一般不需要。但涉外文材料需提供中文翻译件，翻译件需加盖企业公章。

**Q5：资金拨付需要多长时间？**
A：一般情况下，从申报到拨付需要2-4个月，具体视评审进度而定。

## 结语

本指南系统梳理了广州市及广东省范围内的市场开拓与展会政策，涵盖省级、市级、区级三个层面的扶持措施，以及粤港澳大湾区的合作机制。企业在实际申报过程中，应重点关注以下几点：

### 关键提醒

1. **及时关注政策动态**
   - 政策每年可能调整，请关注最新发布的申报指南
   - 加入政策通知群组或关注官方公众号

2. **提前做好规划**
   - 年初制定年度参展计划
   - 优先选择目录内展会
   - 合理分配年度申报额度

3. **规范准备材料**
   - 严格按照清单准备材料
   - 注意材料的完整性和规范性
   - 妥善保存原始凭证

4. **善用政策组合**
   - 省、市资金可叠加申报
   - 关注各区配套政策
   - 最大化利用政策红利

5. **诚信合规申报**
   - 严格保证材料真实性
   - 不重复申报同级别资金
   - 接受监督检查

### 政策红利窗口期

当前正值国家和地方大力支持企业开拓市场的关键时期，各级财政资金投入力度较大。建议符合条件的优质企业抓住政策窗口期，积极参展，拓展市场，抢占先机。
