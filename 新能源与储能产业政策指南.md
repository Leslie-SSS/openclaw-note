## 二、广州市新能源产业发展政策

### 2.1 光伏发电政策

#### 2.1.1 《广州市分布式光伏发电项目管理办法》

**发文单位**：广州市发展和改革委员会  
**文号**：穗发改规字〔2021〕3号  
**有效期**：2021年6月1日至2026年5月31日

**核心内容**：

1. **适用范围**：在广州市行政区域内，利用工商业厂房、公共机构建筑、居民住宅等屋顶建设的分布式光伏发电项目。

2. **备案要求**：
   - 项目容量≤6MW，向所在区发改部门备案
   - 项目容量>6MW，向市发改委备案
   - 备案材料包括：项目申请表、产权证明、屋顶荷载证明、并网接入方案

3. **并网服务**：
   - 电网企业应在受理并网申请后20个工作日内出具接入系统方案
   - 项目应在备案后2年内开工建设
   - 并网验收合格后30个工作日内完成电表安装

4. **发电量补贴**：
   - 2021-2023年并网项目：0.15元/kWh，连续5年
   - 2024-2025年并网项目：0.1元/kWh，连续3年
   - 单个项目年度补贴上限200万元

5. **申报流程**：
   - 第一步：项目备案（区/市发改部门）
   - 第二步：并网申请（属地供电局）
   - 第三步：工程建设（具备资质单位）
   - 第四步：并网验收（电网企业）
   - 第五步：补贴申报（每年1月和7月）

6. **申报材料**：
   - 补贴申请表
   - 项目备案文件
   - 并网验收意见
   - 发电量证明（电网企业出具）
   - 企业营业执照、银行账户信息

#### 2.1.2 《广州市整区屋顶分布式光伏试点实施方案》

**发文单位**：广州市发展和改革委员会  
**文号**：穗发改能源〔2021〕28号  
**试点区域**：白云区、花都区、从化区

**核心内容**：

1. **试点目标**：
   - 到2025年，试点区屋顶光伏覆盖率不低于40%
   - 党政机关建筑屋顶覆盖率≥50%
   - 学校、医院等公共建筑≥40%
   - 工商业厂房≥30%
   - 农村居民屋顶≥20%

2. **一次性补助标准**：
   - 工商业项目：0.2元/W，最高100万元
   - 公共机构项目：0.3元/W，最高200万元
   - 农户项目：0.4元/W，最高2万元/户

3. **申报条件**：
   - 项目纳入整区试点方案
   - 组件效率≥20%
   - 采用"自发自用、余电上网"模式
   - 2021-2025年期间并网

4. **运维要求**：
   - 建立项目运维台账
   - 每季度上报发电数据
   - 配套安装监测系统

#### 2.1.3 《广州市绿色金融支持光伏发展实施方案》

**发文单位**：广州市地方金融监督管理局  
**文号**：穗金融〔2022〕15号  
**有效期**：2022年1月1日至2026年12月31日

**核心内容**：

1. **绿色信贷支持**：
   - 光伏项目贷款贴息：不超过贷款利息的50%
   - 单个项目年度贴息上限500万元
   - 贴息期限：最长3年

2. **绿色债券支持**：
   - 发行绿色债券贴息：发行费用的30%
   - 单笔债券贴息上限100万元

3. **融资担保支持**：
   - 担保费补贴：担保费用的50%
   - 单个项目担保补贴上限50万元

4. **申报条件**：
   - 在广州市注册的独立法人企业
   - 项目纳入广州市光伏项目库
   - 贷款用途为光伏项目建设

5. **申报材料**：
   - 贴息申请表
   - 贷款合同、借款凭证
   - 利息支付凭证
   - 项目备案文件
   - 项目竣工验收报告

#### 2.1.4 光伏项目申报流程图

```
项目可行性分析
      ↓

      ↓
```

### 2.2 风电产业政策

#### 2.2.1 《广州市海上风电产业发展规划（2021-2030年）》

**发文单位**：广州市发展和改革委员会  
**文号**：穗发改能源〔2021〕45号  
**规划期限**：2021-2030年

**核心内容**：

1. **发展目标**：
   - 到2025年：海上风电装机容量达到1GW
   - 到2030年：海上风电装机容量达到3GW
   - 形成千亿级海上风电产业集群

2. **重点任务**：
   - 建设南沙海上风电装备制造基地
   - 推进深水远海风电示范项目
   - 发展风电运维服务业
   - 培育风电核心技术装备

3. **支持政策**：
   - 海上风电项目投资补助：项目投资的3%
   - 单个项目补助上限：5000万元
   - 装备制造企业落户奖励：最高1亿元
   - 研发费用加计扣除：100%

4. **申报条件**：
   - 项目核准文件
   - 海域使用权证
   - 接入系统方案
   - 环境影响评价批复

#### 2.2.2 《广州市分散式风电发展指导意见》

**发文单位**：广州市发展和改革委员会  
**文号**：穗发改能源〔2022〕18号  
**有效期**：2022-2025年

**核心内容**：

1. **适用范围**：
   - 单机容量≤5MW
   - 总装机容量≤50MW
   - 接入电压等级≤35kV

2. **布局要求**：
   - 工业园区周边
   - 高速公路沿线
   - 防护林地带
   - 城市郊区空地

3. **项目核准**：
   - 总容量≤10MW：区发改部门核准
   - 总容量>10MW：市发改委核准

4. **用地要求**：
   - 不占用基本农田
   - 不涉及生态红线
   - 噪音距居民区≥300米

5. **电价政策**：
   - 执行燃煤发电基准电价
   - 参与电力市场交易
   - 享受绿色电力证书

### 2.3 新型储能政策

#### 2.3.1 《广州市新型储能产业发展行动方案（2023-2025年）》

**发文单位**：广州市发展和改革委员会、广州市工业和信息化局  
**文号**：穗发改能源〔2023〕22号  
**有效期**：2023-2025年

**核心内容**：

1. **发展目标**：
   - 到2025年：新型储能装机规模达到1GW
   - 培育产值超10亿元企业10家
   - 建设国家级储能创新平台2个
   - 打造千亿级储能产业集群

2. **重点技术路线**：
   - 锂离子电池储能：主导方向
   - 钠离子电池储能：重点培育
   - 液流电池储能：示范应用
   - 压缩空气储能：示范探索
   - 氢储能：前瞻布局

3. **支持政策**：

   **（1）投资补助**：
   - 独立储能电站：项目投资的10%，最高5000万元
   - 用户侧储能：项目投资的20%，最高1000万元
   - 配套储能：与发电项目同步建设，补助5%

   **（2）容量补贴**：
   - 独立储能：300元/kW·年，连续3年
   - 用户侧储能：150元/kW·年，连续3年

   **（3）电价支持**：
   - 峰谷价差套利：执行市场化峰谷电价
   - 辅助服务收益：参与调峰、调频市场
   - 容量租赁收益：向新能源项目出租容量

4. **示范项目**：

   **（1）电网侧储能示范**：
   - 南沙独立储能电站（200MW/400MWh）
   - 增城电网侧储能项目（100MW/200MWh）
   - 从化抽水蓄能配套储能（50MW/100MWh）

   **（2）用户侧储能示范**：
   - 开发区工业园区储能（50MW/100MWh）
   - 商业综合体储能项目（20MW/40MWh）
   - 数据中心配套储能（30MW/60MWh）

   **（3）新能源配套储能**：
   - 光伏+储能项目：配置比例≥10%
   - 风电+储能项目：配置比例≥15%

5. **申报条件**：
   - 在广州市注册的独立法人
   - 储能系统循环寿命≥6000次
   - 能量效率≥85%
   - 安全性符合国家标准

6. **申报材料**：
   - 项目备案/核准文件
   - 储能系统技术参数
   - 接入系统方案
   - 安全评估报告
   - 投资预算及资金证明

## 三、广东省新型储能产业政策

### 3.1 《广东省新型储能参与电力市场交易实施方案》

**发文单位**：广东省能源局、国家能源局南方监管办  
**文号**：粤能电力〔2023〕15号  
**实施日期**：2023年6月1日

**核心内容**：

1. **市场准入条件**：
   - 独立储能：功率≥5MW，持续时长≥1小时
   - 用户侧储能：功率≥1MW，持续时长≥1小时
   - 配套储能：与发电项目同步调度

2. **交易品种**：
   - 现货市场交易：电能量
   - 调峰辅助服务：削峰填谷
   - 调频辅助服务：频率调节
   - 备用容量市场：备用容量

3. **价格机制**：
   - 现货市场：按节点电价结算
   - 调峰补偿：按调用时段补偿
     - 深度调峰（>4小时）：0.5元/kWh
     - 一般调峰：0.3元/kWh
   - 调频补偿：
     - 调频里程收益：10元/MW
     - 调频容量收益：5元/MW·日

4. **容量补贴标准**：
   - 2023年并网：300元/kW·年
   - 2024年并网：200元/kW·年
   - 2025年并网：100元/kW·年
   - 补贴期限：3年

5. **申报流程**：
   - 第一步：市场注册（广东电力交易中心）
   - 第二步：资格审核
   - 第三步：计量装置安装
   - 第四步：参与市场交易
   - 第五步：补贴申请（每年）

### 3.2 《广东省加快新型储能融合发展若干措施》

**发文单位**：广东省发展和改革委员会  
**文号**：粤发改能源〔2023〕28号  
**有效期**：2023-2025年

**核心内容**：

1. **融合发展方向**：

   **（1）新能源+储能**：
   - 集中式光伏：配置10%×2小时储能
   - 集中式风电：配置15%×2小时储能
   - 海上风电：配置20%×2小时储能
   - 储能可与新能源项目联合或独立建设

   **（2）电网+储能**：
   - 变电站配套储能：10%×2小时
   - 负荷中心储能：缓解配电网压力
   - 应急保供储能：重要用户保障

   **（3）用户+储能**：
   - 工业用户：峰谷套利、需量管理
   - 商业用户：降低容量电费
   - 数据中心：备用电源+峰谷套利

2. **支持措施**：

   **（1）投资补助**：
   - 示范项目：项目投资的30%
   - 单个项目补助上限：1亿元
   - 储能装备制造项目：固定资产投资补助15%

   **（2）土地支持**：
   - 纳入土地利用总体规划
   - 优先保障用地指标
   - 鼓励利用存量建设用地

   **（3）金融支持**：
   - 绿色贷款：利率优惠20%
   - 融资租赁：支持储能资产证券化
   - 保险保障：储能安全责任险

3. **示范项目清单**：

| 项目名称 | 地点 | 规模 | 投资额 | 建设期限 |
|---------|------|------|--------|---------|
| 珠海金湾储能项目 | 珠海 | 300MW/600MWh | 12亿元 | 2023-2024 |
| 惠州大亚湾储能项目 | 惠州 | 200MW/400MWh | 8亿元 | 2023-2024 |
| 东莞虎门储能项目 | 东莞 | 150MW/300MWh | 6亿元 | 2024-2025 |
| 中山翠亨储能项目 | 中山 | 100MW/200MWh | 4亿元 | 2024-2025 |

### 3.3 《广东省先进制造业发展专项资金管理办法》

**发文单位**：广东省工业和信息化厅、广东省财政厅  
**文号**：粤工信规字〔2022〕4号  
**有效期**：2022-2025年

**储能产业支持**：

1. **支持方向**：
   - 储能电池材料研发
   - 储能系统集成
   - 电池管理系统（BMS）
   - 能量管理系统（EMS）
   - 储能变流器（PCS）

2. **奖励标准**：

   **（1）固定资产投资奖励**：
   - 投资1-5亿元：奖励5%，最高2500万元
   - 投资5-10亿元：奖励8%，最高8000万元
   - 投资10亿元以上：奖励10%，最高1亿元

   **（2）技术改造奖励**：
   - 设备投资：奖励10%
   - 单个项目最高5000万元

   **（3）首台套装备奖励**：
   - 首台套销售奖励：销售收入的30%
   - 单项装备最高1000万元

3. **申报条件**：
   - 在广东省注册的制造业企业
   - 项目符合产业政策
   - 固定资产投资≥5000万元
   - 项目已开工或具备开工条件

4. **申报时间**：
   - 每年3月：项目申报
   - 每年6月：项目评审
   - 每年9月：资金下达

- 组件效率≥19%（单晶）、≥18%（多晶）
- 逆变器效率≥98%
- 系统寿命≥25年
- 衰减率：首年≤2%，逐年≤0.55%

**储能项目**：
- 系统能量效率≥85%
- 循环寿命≥6000次（锂离子）
- 响应时间≤100ms
- 安全标准：GB/T 36276、GB/T 34131

**氢能项目**：
- 燃料电池电堆功率密度≥3kW/L
- 系统寿命≥10000小时
- 氢气纯度≥99.97%
- 加氢站安全距离符合GB 50516

1. **光伏政策**：广州市分布式光伏补贴0.1-0.15元/kWh，整区试点补助0.2-0.4元/W，绿色贷款贴息50%。

2. **储能政策**：独立储能容量补贴100-300元/kW·年，投资补助最高1亿元，参与电力市场获取调峰调频收益。

3. **氢能政策**：加氢站补助200-800万元/站，燃料电池汽车补贴20-100万元/辆，核心零部件销售额5-8%奖励。

4. **番禺区专项**：光伏补贴100-150元/kW，节能降碳200元/吨标煤，装备制造投资补助5%。

5. **粤港澳大湾区合作**：跨境电力交易、氢能交通示范、海上风电联合开发、绿证互认。

### 10.2 重点支持方向

| 领域 | 重点支持 | 政策力度 |
|-----|---------|---------|
| 光伏发电 | 工商业分布式、整区推进 | ★★★★ |
| 海上风电 | 深水远海、装备制造 | ★★★★ |
| 新型储能 | 独立储能、电网侧储能 | ★★★★★ |
| 氢能产业 | 燃料电池、加氢站 | ★★★★ |
| 新能源装备 | 本地化制造、技术升级 | ★★★★ |

### 10.3 申报建议

1. **优先申报储能项目**：政策力度大，市场机制成熟，收益稳定。

2. **光伏项目把握时机**：补贴逐年下降，尽早并网锁定较高补贴。

3. **氢能项目关注示范**：示范项目支持力度大，适合有技术优势的企业。

4. **区级政策叠加申请**：番禺、黄埔、南沙等区有专项政策，可叠加申请。

5. **关注政策更新**：政策每年更新，密切关注最新动态。
