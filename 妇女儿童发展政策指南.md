## 二、广州市妇女儿童发展规划

### 2.1 广州市妇女发展规划(2021-2030)

#### 2.1.1 总体目标

坚持男女平等基本国策，促进妇女全面发展，保障妇女合法权益，推动妇女走在时代前列。到2030年，广州妇女发展水平达到全国领先、国际先进，成为展示我国妇女事业发展成就的重要窗口。

#### 2.1.2 发展领域与主要目标

**（一）妇女与健康**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 人均预期寿命 | ≥83.5岁 | ≥84.5岁 |
| 孕产妇死亡率 | ≤10/10万 | ≤8/10万 |
| 妇女宫颈癌筛查覆盖率 | ≥85% | ≥90% |
| 妇女乳腺癌筛查覆盖率 | ≥80% | ≥85% |
| 孕产妇系统管理率 | ≥90% | ≥92% |

**核心政策要点：**

1. **免费"两癌"筛查**
   - 申报条件：35-64岁广州市户籍妇女或持有居住证的常住妇女
   - 服务内容：宫颈癌筛查（HPV检测+TCT）、乳腺癌筛查（乳腺超声+钼靶）
   - 申请流程：携带身份证/居住证到居住地社区卫生服务中心
   - 筛查频率：每3年免费筛查一次

2. **孕产妇健康管理**
   - 免费项目：孕前优生健康检查、孕期保健（至少5次产检）、产后访视
   - 补助标准：农村孕产妇住院分娩补助1000元
   - 高危孕产妇管理：建立专案管理，开通绿色通道

3. **妇女心理健康服务**
   - 服务网点：各区妇幼保健院设立妇女心理咨询门诊
   - 特色项目：更年期妇女健康管理、孕产期心理筛查

**（二）妇女与教育**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 女性高中阶段毛入学率 | ≥98% | ≥99% |
| 高等教育女性毛入学率 | ≥70% | ≥75% |
| 女性劳动年龄人口平均受教育年限 | ≥13年 | ≥14年 |

**核心政策要点：**

1. **女性职业教育支持**
   - 补贴标准：参加职业技能培训，按培训费80%补贴，最高5000元/人
   - 重点领域：家政服务、养老护理、婴幼儿照护、电子商务
   - 申报流程：通过"粤省事"小程序或各区人社局

2. **女性科技人才培养**
   - 项目名称：广州市女性科技人才专项
   - 支持对象：45周岁以下女性科研人员
   - 资助标准：一般项目10万元，重点项目30万元
   - 申报时间：每年3-4月

**（三）妇女与经济**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 城镇单位就业人员中女性比例 | ≥42% | ≥45% |
| 女性负责人企业比例 | ≥30% | ≥35% |
| 女性高级专业技术人员比例 | ≥45% | ≥50% |

**核心政策要点：**

1. **妇女创业担保贷款**
   - 贷款额度：个人最高50万元，小微企业最高300万元
   - 贴息标准：政府全额贴息（LPR-150BP以下部分）
   - 申报条件：
     - 法定劳动年龄内，具有创业意愿和能力的妇女
     - 广州市户籍或持有居住证满1年
     - 从事个体经营或创办小微企业
   - 申请流程：区妇联推荐→经办银行审核→贷款发放
   - 联系电话：广州市妇联妇女发展部 020-83126198

2. **女大学生就业帮扶**
   - 服务内容：就业指导、岗位推荐、就业见习
   - 补贴标准：就业见习补贴按最低工资80%发放
   - 服务平台：广州市女性就业服务中心

3. **农村妇女创业支持**
   - 专项计划："乡村振兴巾帼行动"
   - 支持领域：农业种植、农产品加工、乡村旅游、电商直播
   - 培训补贴：免费参加农村妇女技能培训

**（四）妇女参与决策和管理**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 市级党代会代表中女性比例 | ≥35% | ≥38% |
| 市人大代表中女性比例 | ≥30% | ≥32% |
| 市政府领导班子中女干部数量 | 逐步提高 | 逐步提高 |
| 村委会成员中女性比例 | ≥30% | ≥35% |

**核心政策要点：**

1. **女性干部培养选拔**
   - 后备干部库：各级组织部门建立女干部后备人才库
   - 培训计划：每年举办女干部培训班，不少于2期
   - 交流挂职：安排女干部到基层、企业、重点项目锻炼

2. **基层妇女参政**
   - 村（社区）妇联主席进"两委"：100%实现
   - 妇女议事会：村（社区）普遍建立妇女议事会
   - 支持措施：开展基层女干部培训，提高参政能力

**（五）妇女与社会保障**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 妇女基本医疗保险参保率 | ≥96% | ≥98% |
| 妇女基本养老保险参保率 | ≥95% | ≥97% |
| 女性失业保险参保率 | ≥90% | ≥93% |

**核心政策要点：**

1. **生育保险待遇**
   - 生育医疗费：顺产3000元，剖宫产5000元
   - 生育津贴：按用人单位上年度月平均工资÷30×产假天数
   - 产假标准：顺产98天+80天奖励假=178天

2. **灵活就业妇女参保**
   - 参保方式：可按灵活就业人员参加养老、医疗保险
   - 补贴政策：就业困难人员社保补贴，按实际缴费的2/3给予补贴
   - 申请渠道：街道（镇）劳动保障服务中心

**（六）妇女与环境**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 农村自来水普及率 | 100% | 100% |
| 农村卫生厕所普及率 | 100% | 100% |
| 城市污水处理率 | ≥97% | ≥98% |

**核心政策要点：**

1. **妇女活动阵地建设**
   - 市级：广州市妇女儿童发展中心（天河区）
   - 区级：11个区均设有妇女儿童活动中心
   - 社区级：村（社区）妇女之家覆盖率达100%

2. **妇女友好环境建设**
   - 母婴设施：公共场所母婴室配置率达90%以上
   - 无障碍设施：公共设施无障碍改造
   - 女性卫生间：女厕位比例不低于2:1

**（七）妇女与法律**

**核心政策要点：**

1. **妇女法律援助**
   - 服务内容：法律咨询、代写文书、诉讼代理
   - 申请条件：经济困难妇女（家庭人均收入低于最低工资标准）
   - 服务热线：12338妇女维权热线
   - 服务站点：各区法律援助中心、妇联维权站

2. **反家庭暴力**
   - 详见本指南第三章

### 2.2 广州市儿童发展规划(2021-2030)

#### 2.2.1 总体目标

坚持儿童优先原则，保障儿童生存、发展、受保护和参与权利，促进儿童全面发展。到2030年，广州儿童发展水平达到全国领先、国际先进，建成儿童友好型城市。

#### 2.2.2 发展领域与主要目标

**（一）儿童与健康**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 婴儿死亡率 | ≤3‰ | ≤2.5‰ |
| 5岁以下儿童死亡率 | ≤4‰ | ≤3.5‰ |
| 新生儿筛查率 | ≥98% | ≥99% |
| 0-6岁儿童眼保健覆盖率 | ≥95% | ≥98% |
| 儿童国家免疫规划疫苗接种率 | ≥95% | ≥98% |

**核心政策要点：**

1. **新生儿疾病免费筛查**
   - 筛查项目：先天性甲状腺功能减低症(CH)、苯丙酮尿症(PKU)、葡萄糖-6-磷酸脱氢酶缺乏症(G6PD)、先天性肾上腺皮质增生症(CAH)
   - 听力筛查：新生儿听力筛查全覆盖
   - 申请方式：助产机构直接提供服务，无需额外申请

2. **儿童预防接种**
   - 免费疫苗：国家免疫规划疫苗（乙肝、卡介苗、脊髓灰质炎、百白破、麻腮风、乙脑、流脑等）
   - 接种点：社区卫生服务中心预防接种门诊
   - 预约方式："粤苗"APP或"广州健康通"小程序

3. **0-6岁儿童健康管理**
   - 服务内容：新生儿访视、生长发育监测、营养指导、心理行为评估
   - 服务频次：0-1岁至少4次，1-3岁每年2次，3-6岁每年1次
   - 服务机构：社区卫生服务中心、乡镇卫生院

**（二）儿童与安全**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 儿童伤害死亡率 | 持续下降 | 持续下降 |
| 中小学生交通安全知识普及率 | 100% | 100% |
| 校园安全达标率 | 100% | 100% |

**核心政策要点：**

1. **儿童伤害预防**
   - 重点领域：溺水、交通、跌落、烧烫伤、中毒
   - 干预措施：安全知识教育、环境改造、产品监管
   - 宣传渠道：学校、社区、媒体

2. **校园安全**
   - 安全设施：视频监控全覆盖、一键报警系统
   - 人员配备：专职保安、护学岗
   - 交通安全：校车安全管理、校园周边交通整治

3. **儿童食品安全**
   - 监管重点：校园食堂、校园周边食品经营单位
   - 检查频次：每学期不少于2次全覆盖检查
   - 举报热线：12315消费者投诉热线

**（三）儿童与教育**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 学前三年毛入园率 | ≥100% | ≥100% |
| 义务教育巩固率 | ≥98% | ≥99% |
| 高中阶段毛入学率 | ≥99% | ≥99.5% |
| 普惠性幼儿园覆盖率 | ≥85% | ≥90% |

**核心政策要点：**

1. **学前教育**
   - 公办幼儿园：每区公办幼儿园在园幼儿占比达50%以上
   - 普惠性幼儿园：普惠性幼儿园覆盖率达85%以上
   - 保教费标准：省一级720元/月，市一级617元/月，区一级525元/月
   - 入园申请：每年5月通过"广州市幼儿园招生报名系统"报名

2. **义务教育**
   - 入学政策：免试就近入学，公民同招
   - 随迁子女：以居住证为主要依据的入学政策
   - 特殊教育：残疾儿童随班就读，特殊教育学校就读

3. **课后服务**
   - 服务时间：课后延时服务至下午6:30
   - 服务内容：作业辅导、兴趣培养、体育锻炼
   - 收费标准：实行政府指导价，每生每月不超过200元

**（四）儿童与福利**

| 指标 | 2025年目标 | 2030年目标 |
|-----|-----------|-----------|
| 机构集中养育孤儿基本生活保障标准 | 持续提高 | 持续提高 |
| 社会散居孤儿基本生活保障标准 | 持续提高 | 持续提高 |
| 残疾儿童康复救助覆盖率 | ≥95% | ≥98% |
| 3岁以下婴幼儿托位数 | 每千人口4.5个 | 每千人口5.5个 |

**核心政策要点：**

1. **孤儿基本生活保障**

### 3.1 反家庭暴力

#### 3.1.1 法律依据

- 《中华人民共和国反家庭暴力法》（2016年3月1日施行）
- 《广东省实施〈中华人民共和国反家庭暴力法〉办法》（2020年10月1日施行）

#### 3.1.2 核心制度

**（一）家庭暴力告诫制度**

| 项目 | 内容 |
|-----|-----|
| 适用对象 | 情节较轻，依法不予治安管理处罚的家暴行为 |
| 实施主体 | 公安派出所 |
| 告诫书内容 | 加害人信息、受害人信息、家暴事实陈述、禁止性行为、违反后果 |
| 法律效力 | 作为法院认定家暴的证据，可申请人身安全保护令 |

**申报流程：**
1. 受害人报警（110）
2. 公安机关调查取证
3. 符合条件的，制作《家庭暴力告诫书》
4. 送达加害人、受害人，并通知村（居）委会

**（二）人身安全保护令**

| 项目 | 内容 |
|-----|-----|
| 申请条件 | 遭受家庭暴力或面临家庭暴力现实危险 |
| 申请主体 | 受害人本人；因故无法申请的，可由近亲属、公安机关、妇联、村（居）委会代为申请 |
| 管辖法院 | 申请人或被申请人居住地、家暴发生地基层法院 |
| 保护措施 | 禁止施暴、禁止骚扰跟踪、责令迁出住所、及其他措施 |
| 有效期限 | 不超过6个月，可申请延长 |
| 违反后果 | 罚款、拘留；构成犯罪的，追究刑事责任 |

**申请流程：**
1. 准备材料：申请书、身份证明、家暴证据（报警记录、伤情照片、医疗诊断、证人证言等）
2. 向法院提交申请
3. 法院审查（情况紧急的24小时内作出裁定）
4. 裁定送达执行（公安机关、村（居）委会协助执行）

**（三）家暴庇护救助**

| 项目 | 内容 |
|-----|-----|
| 救助机构 | 民政部门设立的家暴庇护中心、救助管理站 |
| 救助内容 | 临时住所、饮食、医疗、心理疏导、法律援助 |
| 救助期限 | 一般不超过10天 |
| 入住条件 | 家暴受害人自愿申请 |
| 联系方式 | 12345政务服务便民热线、当地民政部门 |

**（四）家暴受害人医疗救助**

- 医疗机构：对家暴受害人及时救治，做好诊疗记录
- 诊疗记录：作为认定家暴的重要证据
- 费用减免：经济困难的家暴受害人可申请医疗救助

### 3.2 未成年人保护

#### 3.2.1 法律依据

- 《中华人民共和国未成年人保护法》（2021年6月1日修订施行）
- 《广东省未成年人保护条例》（2023年修订）
- 《广东省预防未成年人犯罪条例》（2023年修订）

#### 3.2.2 六大保护体系

**（一）家庭保护**

| 义务主体 | 核心义务 |
|---------|---------|
| 父母或其他监护人 | 抚养教育、安全保障、尊重人格、不得实施家暴 |
| 禁止行为 | 虐待、遗弃、非法送养、允许童工、强迫婚嫁 |

**监护干预机制：**
1. **监护评估**：民政部门对困境未成年人监护情况进行评估
2. **监护干预**：监护人存在问题的，责令改正；拒不改正的，依法撤销监护人资格
3. **临时监护**：紧急情况下，民政部门承担临时监护责任
4. **国家监护**：监护人丧失监护能力或被撤销资格的，由国家承担监护责任

**（二）学校保护**

| 项目 | 内容 |
|-----|-----|
| 校园安全 | 建立安全管理制度，配备安全设施和人员 |
| 校园欺凌防治 | 建立欺凌防控机制，发现欺凌及时处置 |
| 性侵害防范 | 建立预防性侵害制度，开展性教育 |
| 心理健康 | 配备心理健康教育教师，提供心理咨询服务 |

**欺凌处置流程：**
1. 发现欺凌→立即制止
2. 通知监护人
3. 对相关未成年学生及时给予心理辅导
4. 进行调查处理，情节严重的报告公安机关
5. 联络相关部门进行处置

**（三）社会保护**

| 主体 | 责任 |
|-----|-----|
| 村（居）委会 | 建立儿童之家，开展关爱服务，发现报告 |
| 公共场所 | 设有母婴室、儿童设施，安全提示 |
| 媒体 | 保护未成年人隐私，传播有益信息 |
| 经营场所 | 不得向未成年人销售烟酒，禁止未成年人进入不适宜场所 |

**强制报告制度详解：**

| 项目 | 内容 |
|-----|-----|
| 报告主体 | 学校、幼儿园、医疗机构、村（居）委会、社会工作服务机构、救助管理机构、福利机构及其工作人员 |
| 报告情形 | 发现未成年人遭受或疑似遭受不法侵害、面临不法侵害危险 |
| 报告时限 | 发现后立即报告 |
| 报告方式 | 向公安机关报案，同步向民政、教育等主管部门报告 |
| 法律责任 | 未履行报告义务造成严重后果的，依法给予处分 |

**（四）网络保护**

| 项目 | 内容 |
|-----|-----|
| 网络产品和服务 | 不得含有危害未成年人身心健康的内容 |
| 网络游戏 | 实施未成年人防沉迷系统，限制游戏时间 |
| 网络直播 | 不得为未满16周岁未成年人提供直播发布者账号注册服务 |
| 网络欺凌 | 制止网络欺凌，保护受害者 |

**未成年人网络保护措施：**
- 网络游戏时间限制：未成年人仅周五、六、日及法定节假日每日20时至21时可玩1小时
- 充值消费限制：8岁以下禁充值，8-16岁单次充值不超过50元、月累计不超过200元，16-18岁单次不超过100元、月累计不超过400元
- 举报渠道：12321网络不良与垃圾信息举报受理中心

**（五）政府保护**

| 责任部门 | 主要职责 |
|---------|---------|
| 民政部门 | 牵头协调未成年人保护工作，承担临时监护、国家监护 |
| 教育部门 | 保障受教育权，学校保护监督 |
| 公安机关 | 查处侵害未成年人案件，处置校园安全事件 |
| 检察机关 | 未成年人检察，督促监护落实 |
| 法院 | 未成年人案件审理，监护权撤销与指定 |
| 卫健部门 | 未成年人健康服务，托育机构管理 |

**未成年人保护工作协调机制：**
- 市级：广州市未成年人保护工作领导小组
- 区级：各区未成年人保护委员会
- 街道（镇）：未成年人保护工作站
- 社区（村）：儿童主任

**（六）司法保护**

| 项目 | 内容 |
|-----|-----|
| 专门机构 | 公检法设立未成年人案件专门机构或专人办理 |
| 特别程序 | 社会调查、合适成年人到场、犯罪记录封存 |
| 法律援助 | 未成年人刑事、民事、行政案件一律提供法律援助 |
| 心理干预 | 对涉案未成年人进行心理评估、心理疏导 |

**未成年人司法特殊制度：**

1. **合适成年人到场**
   - 适用情形：讯问、审判未成年人
   - 到场人员：法定代理人不能到场的，可邀请合适成年人（教师、社工、妇联干部等）到场

2. **犯罪记录封存**
   - 适用对象：犯罪时不满18周岁，被判处5年有期徒刑以下刑罚
   - 封存效果：犯罪记录不得向任何单位和个人提供（司法机关办案需要除外）
   - 法律后果：在入伍、就业时免除前科报告义务

3. **社会调查报告**
   - 调查内容：未成年人成长经历、犯罪原因、监护教育条件
   - 调查主体：司法社工、社区矫正机构
   - 报告用途：作为量刑参考、制定矫治方案依据
