## 二、广州市文旅产业发展政策

### 2.1 广州市文化和旅游产业发展专项资金管理办法

#### 2.1.1 政策概述

广州市文化和旅游产业发展专项资金（以下简称"专项资金"）是由广州市财政安排，专项用于支持广州市文化和旅游产业发展的资金。该政策旨在推动文旅融合、促进全域旅游发展、培育文旅新业态、提升文旅产业竞争力。

#### 2.1.2 支持方向

**（一）文旅融合项目**

| 项目类型 | 支持内容 | 最高金额 |
|----------|----------|----------|
| 文旅融合示范区 | 文化与旅游深度融合的示范区建设 | 500万元 |
| 文旅综合体 | 集文化展示、旅游观光、休闲娱乐于一体的综合体 | 300万元 |
| 文化创意旅游 | 以文化创意为核心吸引物的旅游项目 | 200万元 |
| 非遗+旅游 | 非遗项目与旅游结合的体验项目 | 100万元 |
| 文博旅游 | 博物馆、美术馆等文博设施旅游化改造 | 150万元 |

**（二）全域旅游发展**

| 项目类型 | 支持内容 | 最高金额 |
|----------|----------|----------|
| 全域旅游示范区 | 国家级、省级全域旅游示范区创建 | 200万元 |
| 旅游风景道 | 自驾车、自行车旅游风景道建设 | 150万元 |
| 旅游集散中心 | 旅游集散咨询服务中心建设运营 | 100万元 |
| 旅游厕所 | A级旅游厕所建设改造 | 30万元/座 |
| 旅游标识系统 | 旅游交通标识、导览系统建设 | 50万元 |

**（三）文旅新业态培育**

| 项目类型 | 支持内容 | 最高金额 |
|----------|----------|----------|
| 红色旅游 | 红色文化主题旅游项目 | 150万元 |
| 研学旅游 | 研学旅行基地建设运营 | 100万元 |
| 康养旅游 | 健康养生主题旅游项目 | 200万元 |
| 工业旅游 | 工业遗产改造、工厂观光项目 | 150万元 |
| 体育旅游 | 体育赛事与旅游结合项目 | 100万元 |
| 数字文旅 | VR/AR文旅体验、智慧景区 | 200万元 |

**（四）文旅品牌打造**

| 项目类型 | 支持内容 | 最高金额 |
|----------|----------|----------|
| 5A级景区创建 | 新评定为国家5A级旅游景区 | 500万元 |
| 4A级景区创建 | 新评定为国家4A级旅游景区 | 200万元 |
| 省级旅游度假区 | 新评定为省级旅游度假区 | 300万元 |
| 国家级旅游度假区 | 新评定为国家级旅游度假区 | 500万元 |
| 星级酒店升级 | 新评定为五星级酒店 | 100万元 |

#### 2.1.3 申报条件

**基本条件：**

1. **主体资格**：在广州市依法注册登记的企业、社会组织或事业单位
2. **经营年限**：成立满1年以上（特殊项目可适当放宽）
3. **财务状况**：财务管理制度健全，经营状况良好
4. **信用记录**：未被列入严重失信主体名单
5. **项目要求**：项目在广州实施，符合产业政策导向

**项目条件：**

1. 项目已立项或备案，手续齐全
2. 项目投资额达到规定标准（一般不低于100万元）
3. 项目建设周期合理，一般不超过2年
4. 项目具有示范性、带动性

#### 2.1.4 申报材料

**必备材料：**

1. 专项资金申请表
2. 企业营业执照、组织机构代码证复印件
3. 法定代表人身份证复印件
4. 上年度财务审计报告
5. 项目可行性研究报告或实施方案
6. 项目立项/备案文件
7. 项目投资预算及资金使用计划
8. 项目建设进度计划

**补充材料：**

1. 土地使用权证明（涉及用地项目）
2. 环评批复文件（涉及环评项目）
3. 相关资质证书
4. 合作协议（有合作方的项目）
5. 专利、软著等知识产权证明
6. 业绩证明材料

### 2.2 广州市全域旅游发展实施方案

#### 2.2.1 政策背景

2019年，广州市出台《广州市全域旅游发展实施方案》，提出打造"花城广州·全域旅游"品牌，构建"一核、两翼、三带、多节点"的旅游空间格局，推动旅游产业转型升级。

#### 2.2.2 发展目标

| 时间节点 | 目标内容 | 具体指标 |
|----------|----------|----------|
| **2025年** | 全域旅游示范区创建 | 5个以上省级全域旅游示范区 |
| **2025年** | 旅游总收入 | 超过5000亿元 |
| **2025年** | 接待游客量 | 超过3亿人次 |
| **2027年** | 旅游业增加值占GDP比重 | 超过7% |

#### 2.2.3 重点任务

**（一）旅游产品体系建设**

1. **都市旅游核心区**
   - 越秀、天河、海珠为核心
   - 打造成国际都市旅游目的地
   - 建设一批都市旅游示范街区

2. **滨海休闲旅游带**
   - 南沙、番禺滨海区域
   - 发展邮轮游艇、滨海度假
   - 建设粤港澳滨海旅游合作区

3. **生态文化旅游带**
   - 从化、增城北部生态区
   - 发展生态旅游、乡村旅游
   - 建设一批生态旅游示范区

4. **历史文化旅游带**
   - 荔湾、越秀历史文化街区
   - 发展文化旅游、非遗旅游
   - 打造历史文化街区精品线路

**（二）旅游基础设施提升**

| 设施类型 | 建设内容 | 投资标准 |
|----------|----------|----------|
| 旅游交通 | 旅游专线、集散中心 | 每个项目500万元以上 |
| 旅游厕所 | 新建、改建A级厕所 | 每座10-30万元 |
| 智慧旅游 | 导览系统、信息平台 | 每个项目100-300万元 |
| 旅游标识 | 全域旅游标识系统 | 每套50-100万元 |

**（三）旅游公共服务完善**

1. **旅游集散中心网络**
   - 建成1个市级旅游集散中心
   - 各区建设区级集散中心
   - 重点景区建设咨询点

2. **旅游咨询服务体系**
   - 机场、火车站、客运站设咨询点
   - 商业中心设旅游咨询台
   - 开通旅游服务热线

3. **旅游安全保障体系**
   - 建立旅游安全应急预案
   - 完善旅游保险制度
   - 建设旅游救援体系

#### 2.2.4 全域旅游示范区创建

**创建条件：**

| 条件类别 | 具体要求 |
|----------|----------|
| 组织领导 | 成立创建工作领导小组，建立联席会议制度 |
| 规划引领 | 编制全域旅游发展规划，纳入国土空间规划 |
| 产业发展 | 旅游业增加值占GDP比重达到5%以上 |
| 公共服务 | 建成完善的旅游公共服务体系 |
| 品牌打造 | 形成具有区域特色的旅游品牌 |
| 市场秩序 | 建立健全旅游市场监管机制 |

### 2.3 广州市文旅融合发展专项政策

#### 2.3.1 政策框架

广州市文旅融合政策围绕"文化+旅游"、"科技+旅游"、"体育+旅游"、"农业+旅游"等融合发展方向，支持新业态、新模式发展。

#### 2.3.2 文旅融合项目支持

**（一）文化场馆旅游化改造**

| 场馆类型 | 支持内容 | 最高补贴 |
|----------|----------|----------|
| 博物馆 | 展陈提升、互动体验设施 | 200万元 |
| 美术馆 | 数字化展示、公共教育区 | 150万元 |
| 图书馆 | 文创空间、阅读体验区 | 100万元 |
| 文化馆 | 非遗展示、互动体验区 | 100万元 |
| 剧院 | 文旅演出、沉浸式剧场 | 200万元 |

**（二）文化遗产旅游开发**

| 项目类型 | 支持内容 | 最高补贴 |

## 附录二：申报常见问题解答

### Q1：新注册企业可以申报文旅项目吗？

**A：** 部分项目可以。一般来说，企业需成立满1年以上才能申报专项资金项目，但部分创业类、人才类项目对新企业有专门支持。建议提前咨询主管部门了解具体要求。

### Q2：项目投资额如何认定？

**A：** 项目投资额以实际支出为准，需提供正规发票、合同、银行流水等凭证。土建、设备、软件等均可计入，但土地出让金一般不计入。

### Q3：获得补贴后资金如何使用？

**A：** 补贴资金须按申报用途专款专用，不得挪用。使用后需提交资金使用报告，接受审计检查。

### Q4：同一项目可以同时申报多个政策吗？

**A：** 一般原则是同一项目同一内容不重复享受财政资金支持，但可以叠加享受不同层级的奖励（如同时获得市级和区级奖励）。具体以各政策规定为准。

### Q5：申报不通过的原因主要有哪些？

**A：** 常见原因包括：
- 申报材料不齐全或不规范
- 企业或项目不符合申报条件
- 项目可行性不足
- 投资预算不合理
- 评审分数未达标

### Q6：项目验收不通过会怎样？

**A：** 如验收不通过，需按整改要求限期整改后重新验收。如仍不通过，可能被追回已拨付资金，并影响后续申报资格。
