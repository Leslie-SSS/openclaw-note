### 5.2 ROI计算

```python
# KOL营销ROI计算

def calculate_kol_roi(revenue, cost):
    """
    计算KOL营销ROI
    
    参数:
    revenue: 带来的收入
    cost: 总投入成本
    
    返回:
    ROI值
    """
    return (revenue - cost) / cost * 100

# 示例
revenue = 50000  # 带来的收入
cost = 10000     # 总投入
roi = calculate_kol_roi(revenue, cost)
print(f"ROI: {roi}%")  # ROI: 400%
```
