# 285. 代发货模式

## 一、代发货模式核心

### 模式定义

**代发货（Dropshipping）**：卖家不囤货，客户下单后由供应商直接发货

**三方关系**：
```
客户 → 下单 → 卖家 → 转单 → 供应商 → 发货 → 客户
```

**优势**：
- 零库存风险
- 低启动资金（$500-2000）
- 可快速测试新品
- 地理位置灵活

**劣势**：
- 毛利率低（15-30%）
- 质量控制难
- 发货时效不可控
- 竞争激烈

---

## 二、供应商选择标准

### 核心评估维度

**1. 产品质量（权重30%）**

| 评估项 | 方法 |
|-------|------|
| 样品测试 | 购买3-5个样品自测 |
| 评价分析 | 查看其他卖家评价 |
| 认证证书 | CE、FCC、RoHS等 |
| 一致性 | 批量与样品是否一致 |

---

**2. 发货能力（权重25%）**

| 评估项 | 标准 |
|-------|------|
| 处理时间 | ≤48小时 |
| 发货方式 | 支持专线/海外仓 |
| 物流时效 | 7-15天（美国） |
| 追踪信息 | 100%可追踪 |
| 包装质量 | 无破损率<1% |

---

**3. 价格竞争力（权重20%）**

**对比维度**：
- 与1688/Alibaba批发价对比
- 与竞品代发价对比
- 是否有阶梯价格（量大优惠）

**毛利测算**：
```
代发毛利率 = (售价 - 代发价 - 运费 - 平台费) ÷ 售价
```
目标毛利率：≥20%

---

**4. 服务能力（权重15%）**

| 评估项 | 标准 |
|-------|------|
| 响应速度 | <4小时回复 |
| 沟通语言 | 支持英语沟通 |
| 售后处理 | 接受退货/换货 |
| 定制能力 | 支持Logo/包装定制 |
| 数据对接 | 支持API/CSV自动下单 |

---

**5. 稳定性（权重10%）**

| 评估项 | 风险 |
|-------|------|
| 经营年限 | <1年风险高 |
| 公司规模 | 1-2人抗风险弱 |
| 库存深度 | 爆款缺货风险 |
| 替代供应商 | 必须有备选 |

---

## 三、供应商平台对比

### 主流平台

| 平台 | 优势 | 劣势 | 适合卖家 |
|-----|------|-----|---------|
| AliExpress | 品类全、价格低 | 时效慢（15-30天） | 新手、测试期 |
| CJ Dropshipping | 时效较快、支持定制 | 价格略高 | 进阶卖家 |
| Spocket | 欧美供应商、时效快 | 品类少、价格高 | 欧美市场 |
| Zendrop | 自动化程度高 | 月费$49起 | 规模化卖家 |
| 1688+代发 | 价格最低 | 需自己对接 | 有采购能力 |

---

### 供应商筛选流程

```
1. 平台搜索 → 2. 初筛（评分/销量/年限） → 3. 样品测试
→ 4. 小批量试单 → 5. 评估数据 → 6. 确定合作
```

**筛选标准**：
- AliExpress：评分≥4.7、订单数≥1000、经营≥2年
- Alibaba：星级≥3、响应率≥90%、金牌供应商

---

## 四、供应商管理

### 分级管理

| 等级 | 标准 | 管理方式 |
|-----|------|---------|
| 核心供应商 | 月订单>500、退货率<2% | 1对1对接、优先新品 |
| 主要供应商 | 月订单100-500 | 定期沟通、监控表现 |
| 备选供应商 | 随时可替代 | 保持联系、定期测试 |

---

### 绩效追踪

**KPI指标**：

| 指标 | 目标值 | 预警线 |
|-----|-------|-------|
| 发货时效 | ≤48小时 | >72小时 |
| 物流时效 | ≤15天 | >25天 |
| 退货率 | <2% | >5% |
| 缺货率 | <5% | >10% |
| 纠纷率 | <1% | >3% |

**月度评估**：
- 连续2月不达标→警告
- 连续3月不达标→启动备选供应商

---

## 五、风险管理

### 常见风险及应对

**1. 缺货风险**
- 预防：保持2-3家备选供应商
- 应对：及时通知客户，提供替代品或退款

**2. 质量风险**
- 预防：定期抽检样品
- 应对：建立退货SOP，向供应商索赔

**3. 时效风险**
- 预防：选择海外仓供应商
- 应对：提前沟通时效预期，提供追踪信息

**4. 价格波动**
- 预防：签订价格协议
- 应对：建立价格预警，及时调整售价

## 六、实操清单

**供应商筛选（第1-2周）**
- [ ] 确定产品品类
- [ ] 在2-3个平台搜索供应商
- [ ] 初筛10-20家候选
- [ ] 深入沟通5-10家
- [ ] 购买样品测试3-5家

**合作确认（第3周）**
- [ ] 小批量试单（10-20单）
- [ ] 评估发货时效和质量
- [ ] 确定核心+备选供应商
- [ ] 建立沟通渠道

**持续管理**
- [ ] 每周监控发货数据
- [ ] 每月评估供应商绩效
- [ ] 每季度复盘供应商结构
- [ ] 持续寻找更优供应商

## 七、关键案例

### 案例：Gymshark早期代发模式

**背景**：健身服饰品牌，2012年从代发起步

**早期策略**：

**1. 供应商选择**
- 平台：Alibaba+本地供应商
- 标准：质量>价格，选择能定制的供应商
- 测试：每款产品购买3个样品对比

**2. 差异化**
- 定制Logo和包装（代发也可以）
- 选择小众设计（减少比价）
- 内容营销（KOL合作）

**3. 转型时机**
- 月订单稳定>500单
- 核心产品明确
- 有足够现金流备货

**转型后**：
- 自建供应链
- 毛利率从20%提升到60%
- 质量和时效全面可控

**可复制要点**：
1. 代发是起步方式，不是终极模式
2. 质量和差异化比价格更重要
3. 始终准备转型自有供应链
4. 用代发验证市场，用自有放大利润

## 八、代发货避坑指南

**常见错误**：
- ❌ 只看价格不看质量
- ❌ 只有一家供应商
- ❌ 不测试样品直接上架
- ❌ 不监控发货表现
- ❌ 过度承诺时效

**成功要素**：
- ✅ 多供应商备份
- ✅ 严格样品测试
- ✅ 定期绩效评估
- ✅ 透明沟通时效
- ✅ 准备转型计划

## 九、供应商评估表模板

| 评估维度 | 权重 | 评分（1-5） | 加权得分 |
|---------|------|-----------|---------|
| 产品质量 | 30% | | |
| 发货能力 | 25% | | |
| 价格竞争力 | 20% | | |
| 服务能力 | 15% | | |
| 稳定性 | 10% | | |
| **总分** | 100% | | |

**决策标准**：
- ≥4分：优先合作
- 3-4分：备选合作
- <3分：不考虑

---

**核心原则**：代发货模式下，供应商就是你的"合伙人"。选对供应商，成功一半；选错供应商，步步是坑。
