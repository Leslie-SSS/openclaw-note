## 二、广州市人才引进政策

### 2.1 高层次人才认定

#### 2.1.1 政策依据

- 《广州市高层次人才认定和服务保障条例》（广州市人民代表大会常务委员会公告第13号）
- 《广州市高层次人才认定标准（2024年修订版）》
- 《广州市人民政府办公厅关于印发广州市引进人才入户管理办法的通知》（穗府办规〔2020〕10号）

#### 2.1.2 人才分类体系

广州市高层次人才分为三个层次，建立金字塔型的人才认定体系：

**（一）杰出专家**

认定标准（满足以下条件之一即可）：

1. 获得诺贝尔奖、菲尔兹奖、图灵奖、普利兹克奖等国际顶级奖项；
2. 中国科学院院士、中国工程院院士；
3. 国家最高科学技术奖获得者；
4. 国家"万人计划"杰出人才；
5. 相当于上述层次的顶尖人才。

**（二）优秀专家**

认定标准（满足以下条件之一即可）：

1. 国家"万人计划"领军人才；
2. 国家"千人计划"入选者（不含青年项目）；
3. 国家杰出青年科学基金获得者；
4. 国家自然科学奖、国家技术发明奖、国家科学技术进步奖一等奖前三完成人；
5. 国家重点学科、重点实验室首席科学家；
6. 教育部"长江学者奖励计划"特聘教授；
7. 相当于上述层次的人才。

**（三）青年后备人才**

认定标准（满足以下条件之一即可）：

1. 国家"万人计划"青年拔尖人才；
2. 国家"千人计划"青年项目入选者；
3. 国家优秀青年科学基金获得者；
4. 广东省"珠江人才计划"领军人才；
5. 广东省"特支计划"杰出人才；
6. 相当于上述层次的人才。

#### 2.1.3 认定条件

**基本条件：**

1. 遵守中华人民共和国宪法和法律法规；
2. 拥护中国共产党的领导，具有良好的道德品质和职业操守；
3. 在广州市工作、创业或与广州市用人单位签订3年以上劳动合同；
4. 在广州市连续缴纳社会保险或个人所得税满6个月；
5. 年龄要求：杰出专家无年龄限制；优秀专家一般不超过60周岁；青年后备人才一般不超过45周岁。

**特殊情况处理：**

1. 柔性引进的人才，每年在广州市工作时间不少于6个月的，可申请认定；
2. 非华裔外国人才，可放宽部分条件，按照《广州市外籍高层次人才认定标准》执行。

#### 2.1.4 奖励标准

**住房补贴：**

- 杰出专家：1000万元
- 优秀专家：200万元
- 青年后备人才：100万元

**人才津贴（按年度发放）：**

- 杰出专家：50万元/年
- 优秀专家：20万元/年
- 青年后备人才：10万元/年

**科研经费支持：**

高层次人才主持的科研项目，可优先推荐申报国家、省、市各级科技计划项目，符合条件的给予配套资助。

#### 2.1.5 服务保障

**医疗保障：**

- 建立高层次人才医疗保健档案
- 每年安排一次健康体检
- 提供就医绿色通道服务

**子女入学：**

- 高层次人才子女可优先安排入读公办幼儿园、义务教育阶段学校
- 高中阶段可根据意愿协调安排入学

**配偶就业：**

- 协助解决高层次人才配偶就业问题
- 提供就业推荐和培训服务

**出入境便利：**

- 外籍高层次人才可办理2-5年多次出入境签证
- 可申请在华永久居留

#### 2.1.6 申报材料

1. 《广州市高层次人才认定申请表》；
2. 身份证明（身份证或护照复印件）；
3. 学历学位证书复印件；
4. 职称证书或职业资格证书复印件；
5. 劳动合同或聘用合同复印件；
6. 社会保险缴纳证明或个人所得税缴纳证明；
7. 符合认定标准的证明材料（获奖证书、项目批文、论文专著等）；
8. 其他相关证明材料。

### 2.3 产业领军人才

#### 2.3.1 政策依据

- 《广州市产业领军人才奖励制度》
- 《广州市创新创业领军团队评审办法》
- 《广州市企业人才培育三年行动计划》

#### 2.3.2 产业领军人才认定条件

**基本条件：**

1. 在广州市战略性新兴产业领域工作；
2. 具有国际领先的技术水平和创新能力；
3. 能够引领相关产业发展方向；
4. 在广州市用人单位担任核心技术或管理岗位。

**重点支持产业：**

1. 新一代信息技术
2. 人工智能与数字经济
3. 生物医药与健康
4. 智能网联汽车与新能源汽车
5. 智能装备与机器人
6. 新材料与精细化工
7. 新能源与节能环保
8. 现代金融与商务服务
9. 现代商贸与会展
10. 文化创意与数字内容

#### 2.3.3 奖励标准

**个人奖励：**

- 产业领军人才：项目资助最高500万元 + 住房补贴300万元
- 产业高端人才：项目资助最高200万元 + 住房补贴100万元

**团队奖励：**

- 创新创业领军团队：项目资助最高3000万元
- 创新创业高端团队：项目资助最高1000万元

**配套支持：**

1. 优先推荐申报国家、省各类人才计划
2. 优先安排产业用地
3. 提供融资对接服务
4. 协助开拓市场

#### 2.3.4 申报流程

1. **项目征集**：每年第一季度发布申报指南
2. **材料申报**：通过广州市人才综合服务平台在线申报
3. **形式审查**：各区人力资源和社会保障局进行初审
4. **专家评审**：组织专家进行评审
5. **现场答辩**：重点项目进行现场答辩
6. **公示**：评审结果公示7个工作日
7. **审定发布**：由市人才工作领导小组审定后发布

### 6.1 政策背景

### 6.2 港澳人才个税优惠政策

#### 6.2.1 政策依据

- 《关于粤港澳大湾区个人所得税优惠政策的通知》（财税〔2019〕31号）
- 《广东省关于贯彻落实粤港澳大湾区个人所得税优惠政策的通知》

#### 6.2.2 政策内容

在大湾区工作的境外（含港澳台）高端人才和紧缺人才，其在珠三角九市缴纳的个人所得税已缴税额超过其按应纳税所得额的15%计算的税额部分，给予财政补贴。

#### 6.2.3 适用对象

**一、境外高端人才**

认定条件（满足以下条件之一）：
1. 国家、省、市重大人才工程入选者
2. 国家、省、市认定的海外高层次人才
3. 在国际知名高校、科研机构担任教授或相当职务
4. 在世界500强企业担任高级管理或技术职务
5. 获得国际公认的重大科技成果

**二、境外紧缺人才**

认定条件：
1. 符合珠三角九市紧缺人才目录
2. 在重点产业领域从事核心技术研发或管理工作
3. 具有本科及以上学历和3年以上相关工作经验

#### 6.2.4 补贴计算

**计算公式：**

补贴金额 = 实际缴纳个人所得税 - 应纳税所得额 × 15%

**示例：**

某境外高端人才年度应纳税所得额为100万元，适用最高税率45%，实际缴纳个人所得税约30万元。

个税补贴 = 30万元 - 100万元 × 15% = 15万元

#### 6.2.5 申报流程

1. 个人在年度终了后通过广东省政务服务网提交申请
2. 上传个人所得税完税证明、人才资格证明等材料
3. 相关部门进行审核
4. 审核通过后公示
5. 公示无异议后拨付补贴

## 七、申报流程指南

### 7.1 广州市高层次人才认定流程

#### 7.1.1 认定流程图

```
准备材料 → 网上申报 → 单位审核 → 区级初审 → 市级审核 → 公示 → 发证
```

#### 7.1.2 详细流程

**第一步：准备材料**

1. 收集整理申报材料
2. 确保材料真实、完整
3. 准备电子版材料（扫描件）

**第二步：网上申报**

1. 登录广州市人才综合服务平台（http://rc.gz.gov.cn）
2. 注册个人账号并完成实名认证
3. 选择"高层次人才认定"栏目
4. 在线填写申请表并上传材料
5. 提交申请

**第三步：单位审核**

1. 用人单位登录系统进行审核
2. 核实申请人信息和材料真实性
3. 审核通过后提交至区级部门

**第四步：区级初审**

1. 区人力资源和社会保障局进行形式审查
2. 核实申请人是否符合基本条件
3. 初审通过后提交至市级部门

**第五步：市级审核**

1. 市人力资源和社会保障局组织专家评审
2. 根据认定标准进行审核
3. 审核通过后进入公示环节

**第六步：公示**

1. 在广州市人才综合服务平台公示
2. 公示期为7个工作日
3. 公示无异议后进入发证环节

**第七步：发证**

1. 制作高层次人才证书
2. 通知申请人领取或邮寄
3. 完成认定

#### 7.1.3 办理时限

- 网上申报：随时
- 区级初审：10个工作日
- 市级审核：20个工作日
- 公示：7个工作日
- 发证：5个工作日

**总时限：**约45个工作日

### 7.2 广州市人才绿卡申领流程

#### 7.2.1 申领流程图

```
准备材料 → 网上申报 → 审核 → 制卡 → 发卡
```

#### 7.2.2 详细流程

**第一步：准备材料**

1. 身份证明
2. 学历学位证书
3. 职称或职业资格证书
4. 劳动合同
5. 社会保险缴纳证明
6. 收入证明（申请A卡、B卡需提供）

**第二步：网上申报**

1. 登录广州市人才绿卡管理系统
2. 在线填写申请表
3. 上传申报材料
4. 提交申请

**第三步：审核**

1. 系统自动校验部分信息
2. 人工审核申报材料
3. 审核通过后进入制卡环节

**第四步：制卡**

1. 制作人才绿卡
2. 录入人才信息

**第五步：发卡**

1. 通知申请人领取
2. 或邮寄至申请人

#### 7.2.3 办理时限

- 审核时限：15个工作日
- 制卡时限：10个工作日

**总时限：**约25个工作日

### 7.3 人才住房补贴申请流程

#### 7.3.1 租房补贴申请流程

**第一步：资格确认**

1. 确认已认定为高层次人才或持有人才绿卡
2. 确认在广州无自有产权住房
3. 确认在广州市租赁住房

**第二步：网上申请**

1. 登录广州市人才综合服务平台
2. 选择"住房补贴"栏目
3. 在线填写申请表
4. 上传租房合同、无房证明等材料

**第三步：审核**

1. 系统自动校验无房信息
2. 人工审核租房合同等材料
3. 审核通过后进入公示环节

**第四步：公示**

公示期为5个工作日

**第五步：发放**

公示无异议后，按季度发放至申请人银行账户

#### 7.3.2 购房补贴申请流程

1. 确认已认定为高层次人才
2. 确认在广州工作满1年
3. 确认购买首套住房

**第二步：网上申请**

1. 登录广州市人才综合服务平台
2. 选择"购房补贴"栏目
3. 在线填写申请表
4. 上传购房合同、不动产权证书等材料

**第三步：审核**

1. 核实购房信息真实性
2. 核实申请人资格条件
3. 审核通过后进入公示环节

**第四步：公示**

公示期为7个工作日

**第五步：发放**

公示无异议后，一次性发放至申请人银行账户
