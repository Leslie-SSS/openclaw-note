## 二、语言本地化

### 2.1 翻译策略

```yaml
翻译层级:

机器翻译:
  优点: 快速、低成本
  缺点: 质量不稳定
  适用: 初期测试

人工翻译:
  优点: 质量高
  缺点: 成本高、周期长
  适用: 正式内容

本地化翻译:
  优点: 符合当地表达
  缺点: 成本最高
  适用: 核心市场

翻译范围:
  网站:
    - 产品描述
    - 导航菜单
    - 政策条款
    - 帮助中心
  
  营销:
    - 广告文案
    - 社媒内容
    - 邮件模板
    - 活动素材
  
  客服:
    - FAQ
    - 自动回复
    - 话术模板
```

### 2.2 本地化表达

```yaml
文化适配:

美国:
  风格: 直接、自信
  表达: "Best seller"、"Top rated"
  避免: 过于谦虚

英国:
  风格: 正式、含蓄
  表达: "Popular choice"
  避免: 过度推销

日本:
  风格: 礼貌、细节
  表达: "人気商品"、"おすすめ"
  避免: 直接比较

德国:
  风格: 严谨、客观
  表达: 数据说话
  避免: 夸大宣传

法国:
  风格: 优雅、品味
  表达: "Notre best-seller"
  避免: 粗俗表达
```

```yaml
规格调整:

  欧洲: 厘米、公斤
  英国: 混合使用

  美国: 110V
  欧洲: 220V
  英国: 230V
  日本: 100V

  美国: Type A/B
  欧洲: Type C/F

  - 原产地标识

通用原则:

```
