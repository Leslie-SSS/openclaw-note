# 支付体验优化

## 一、理论框架

### 1.1 支付体验的核心价值

支付是电商转化的最终环节，支付体验直接影响用户决策和品牌信任：

**支付体验数据洞察：**
- 13%的弃单因支付方式不足
- 简化支付可提升转化率35%
- 提供多种支付方式可降低弃单率20%
- 移动支付用户增长每年超过30%

**数字钱包：** Apple Pay、Google Pay、支付宝、微信支付、PayPal

```javascript
// Luhn算法验证
function validateCard(number) {
  let sum = 0;
  let isEven = false;
  
  for (let i = number.length - 1; i >= 0; i--) {
    let digit = parseInt(number[i]);
    if (isEven) {
      digit *= 2;
      if (digit > 9) digit -= 9;
    }
    sum += digit;
    isEven = !isEven;
  }
  
  return sum % 10 === 0;
}

// 卡片类型识别
function detectCardType(number) {
  const patterns = {
    visa: /^4/,
    mastercard: /^5[1-5]/,
    amex: /^3[47]/
  };
  
  for (const [type, pattern] of patterns.entries()) {
    if (pattern.test(number)) return type;
  }
  return null;
}
```

```javascript
// 支付状态轮询
function pollPaymentStatus(orderId) {
  const interval = setInterval(async () => {
    const status = await fetch(`/api/payment/status/${orderId}`);
    const { paid } = await status.json();
    
    if (paid) {
      clearInterval(interval);
      redirectToSuccess(orderId);
    }
  }, 2000);
}
```

```javascript
async function process3DS(paymentData) {
  // 1. 创建支付意图
  const intent = await createPaymentIntent(paymentData);
  
  // 2. 检查是否需要3DS
  if (intent.requiresAction) {
```

**友好的错误处理：**

```javascript
class PaymentErrorHandler {
  handle(error) {
    const errorMap = {
      'insufficient_funds': {
        title: '余额不足',
        message: '您的账户余额不足以完成支付',
        actions: ['更换支付方式', '联系银行']
      },
      'card_declined': {
        title: '卡片被拒绝',
        message: '您的银行卡无法完成此交易',
        actions: ['检查卡片信息', '联系发卡行']
      },
      'expired_card': {
        title: '卡片已过期',
        message: '您使用的卡片已超过有效期',
        actions: ['更新卡片信息', '更换支付方式']
      },
      'network_error': {
        title: '网络错误',
        message: '支付请求失败，请检查网络连接',
        actions: ['重试支付']
      }
    };
    
    const errorInfo = errorMap[error.code] || {
      title: '支付失败',
      message: '抱歉，支付过程中出现问题',
      actions: ['重试', '联系客服']
    };
    
    this.showErrorModal(errorInfo);
  }
  
  showErrorModal(info) {
    const modal = document.createElement('div');
    modal.className = 'payment-error-modal';
    modal.innerHTML = `
      <div class="modal-content">
        <h3>${info.title}</h3>
        <p>${info.message}</p>
        <div class="actions">
          ${info.actions.map(a => `<button>${a}</button>`).join('')}
        </div>
      </div>
    `;
    document.body.appendChild(modal);
  }
}
```

**自动重试机制：**

```javascript
class PaymentRetry {
  constructor(maxRetries = 3) {
    this.maxRetries = maxRetries;
    this.retryCount = 0;
  }
  
  async processWithRetry(paymentFn) {
    while (this.retryCount < this.maxRetries) {
      try {
        return await paymentFn();
      } catch (error) {
        this.retryCount++;
        
        if (this.isRetryable(error) && this.retryCount < this.maxRetries) {
          await this.delay(1000 * this.retryCount);
          continue;
        }
        
        throw error;
      }
    }
  }
  
  isRetryable(error) {
    const retryableErrors = ['network_error', 'timeout', 'service_unavailable'];
    return retryableErrors.includes(error.code);
  }
  
  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```
