# 软件企业认定指南

> 本指南涵盖软件企业认定的申请流程、所需材料、税收优惠、常见问题及时间节点，帮助企业高效完成认定并享受税收优惠。

---

## 一、软件企业认定概述

### 1.1 什么是软件企业认定

软件企业认定是指企业按照国家相关规定，经过相关部门审核后，获得"软件企业"资格的认定。通过认定后，企业可享受企业所得税优惠等政策。

### 1.2 认定依据

- 《关于软件产业和集成电路产业企业所得税优惠政策有关问题的通知》（财税〔2012〕27号）
- 《关于集成电路设计和软件产业企业所得税政策的公告》（财税〔2019〕68号）
- 《关于促进集成电路产业和软件产业高质量发展企业所得税政策的公告》（财税〔2020〕45号）
- 《软件企业评估标准》（T/SIA 001-2017）

### 1.3 认定条件（核心要求）

| 条件类别 | 具体要求 |
|---------|---------|
| **企业性质** | 在中国境内（不包括港、澳、台地区）依法注册的居民企业 |
| **主营业务** | 以软件开发生产、系统集成、应用服务和其他相应技术服务为主营业务 |
| **人员比例** | 具有本科以上学历的月平均职工人数占企业月平均职工总人数的比例不低于40%，研究开发人员占比不低于20% |
| **研发费用** | 软件产品开发销售（营业）收入占企业收入总额的比例不低于50%（嵌入式软件产品和信息系统集成产品不低于40%），其中软件产品自主开发销售（营业）收入占企业收入总额的比例不低于40%（嵌入式软件产品和信息系统集成产品不低于30%） |
| **知识产权** | 拥有核心关键技术，并以此为基础开展经营活动，且汇算清缴年度研究开发费用总额占企业销售（营业）收入总额的比例不低于6% |
| **质量保证** | 具有保证设计产品质量的手段和能力，并建立符合软件工程要求的质量管理体系并提供有效运行的过程文档记录 |
| **经营环境** | 具有与软件开发相适应的生产经营场所、软硬件设施等开发环境 |
| **诚信记录** | 汇算清缴年度未发生重大安全、重大质量事故或严重环境违法行为 |

## 二、软件企业认定详细申请流程

### 2.1 流程概览

```
准备材料 → 网上申报 → 提交纸质材料 → 初审 → 专家评审 → 公示 → 备案/发证
```

### 2.2 详细步骤

#### 第一步：自我评估

1. 核对企业是否符合软件企业认定条件
2. 梳理企业财务数据、人员构成、研发费用等
3. 评估知识产权和技术情况

#### 第二步：材料准备

1. 收集企业基本信息材料
2. 整理财务报表和审计报告
3. 准备研发费用明细和人员名单
4. 获取软件著作权等知识产权证书
5. 建立质量管理体系文件

#### 第三步：网上申报

1. 登录当地软件行业协会或科技部门指定平台
2. 填写《软件企业认定申请表》
3. 上传相关证明材料
4. 提交申请

#### 第四步：提交纸质材料

1. 按要求装订纸质申请材料
2. 送达指定受理窗口
3. 获取受理回执

#### 第五步：审核流程

| 环节 | 时间 | 内容 |
|-----|------|-----|
| 形式审查 | 5-10个工作日 | 材料完整性、合规性检查 |
| 初审 | 10-15个工作日 | 地方部门初步审核 |
| 专家评审 | 15-20个工作日 | 专家组进行技术、财务评审 |
| 公示 | 5-7个工作日 | 通过企业名单公示 |
| 发证 | 5个工作日 | 颁发软件企业证书 |

#### 第六步：后续备案

1. 取得软件企业证书后，需向税务部门备案
2. 在企业所得税汇算清缴时享受优惠
3. 每年进行年度报告和重新认定

### 2.3 认定机构

- **国家层面**：工业和信息化部
- **地方层面**：各省市软件行业协会、经信委/工信厅
- **具体操作**：多数地区由软件行业协会负责具体评估工作

## 四、税收优惠计算示例

### 4.1 主要税收优惠政策

#### 4.1.1 "两免三减半"政策

**政策内容：**
- 2011年1月1日后依法成立且符合条件的软件企业，经认定后，在2018年12月31日前自获利年度起计算优惠期
- **第一年至第二年**：免征企业所得税
- **第三年至第五年**：按照25%的法定税率减半征收（即12.5%）

**注意：** 2019年1月1日后新成立的软件企业，需关注最新政策动态。

#### 4.1.2 国家规划布局内重点软件企业

符合条件的国家规划布局内重点软件企业，如当年未享受免税优惠，可减按10%的税率征收企业所得税。

#### 4.1.3 研发费用加计扣除

软件企业的研发费用可享受75%（2023年起提高到100%）的加计扣除优惠。

### 4.2 计算示例一：新设软件企业"两免三减半"

#### 企业背景

- **企业名称**：XX软件科技有限公司
- **成立时间**：2022年3月
- **认定时间**：2023年取得软件企业认定
- **获利年度**：2023年（首次盈利）

#### 年度利润及应纳税所得额

| 年度 | 应纳税所得额（万元） | 优惠情况 |
|-----|---------------------|---------|
| 2023（第1年） | 200 | 免税 |
| 2024（第2年） | 350 | 免税 |
| 2025（第3年） | 500 | 减半征收（12.5%） |
| 2026（第4年） | 650 | 减半征收（12.5%） |
| 2027（第5年） | 800 | 减半征收（12.5%） |
| 2028及以后 | 按正常税率25% |  |

#### 税款计算

| 年度 | 应纳税所得额 | 税率 | 应纳税额 | 节税金额 |
|-----|-------------|------|---------|---------|
| 2023 | 200万 | 0% | **0元** | 50万元 |
| 2024 | 350万 | 0% | **0元** | 87.5万元 |
| 2025 | 500万 | 12.5% | **62.5万元** | 62.5万元 |
| 2026 | 650万 | 12.5% | **81.25万元** | 81.25万元 |
| 2027 | 800万 | 12.5% | **100万元** | 100万元 |
| **五年合计** | - | - | **243.75万元** | **381.25万元** |

#### 收益分析

- 五年应纳税所得额总额：2,500万元
- 五年实际缴纳税额：243.75万元
- 五年节税总额：381.25万元
- 平均有效税率：9.75%

### 4.3 计算示例二：研发费用加计扣除

#### 企业背景

- **企业名称**：YY信息技术有限公司
- **年度**：2024年
- **研发费用实际发生额**：500万元
- **应纳税所得额（扣除前）**：1,500万元

#### 计算过程

**（1）研发费用加计扣除金额**
- 加计扣除比例：100%（2023年起提高）
- 加计扣除金额 = 500万 × 100% = **500万元**

**（2）调整后应纳税所得额**
- 应纳税所得额 = 1,500万 - 500万 = **1,000万元**

**（3）企业所得税计算**

| 情况 | 应纳税所得额 | 税率 | 应纳税额 |
|-----|-------------|------|---------|
| 不享受加计扣除 | 1,500万 | 25% | 375万 |
| 享受加计扣除 | 1,000万 | 25% | 250万 |
| **节税额** | - | - | **125万** |

### 4.4 综合计算示例

#### 企业背景

- **企业名称**：ZZ科技有限公司
- **认定状态**：软件企业（处于"两免三减半"减半期）
- **年度**：2025年
- **应纳税所得额（扣除前）**：2,000万元
- **研发费用**：400万元

#### 计算步骤

**步骤一：研发费用加计扣除**
- 加计扣除金额 = 400万 × 100% = 400万元

**步骤二：调整后应纳税所得额**
- 应纳税所得额 = 2,000万 - 400万 = 1,600万元

**步骤三：适用税率计算**
- 软件企业减半期税率 = 25% × 50% = 12.5%

**步骤四：计算应纳税额**

| 项目 | 金额 |
|-----|------|
| 原应纳税所得额 | 2,000万元 |
| 研发费用加计扣除 | -400万元 |
| 调整后应纳税所得额 | 1,600万元 |
| 适用税率 | 12.5% |
| **应纳税额** | **200万元** |

**节税效果对比**

| 方案 | 应纳税所得额 | 税率 | 税额 |
|-----|-------------|------|------|
| 无优惠 | 2,000万 | 25% | 500万 |
| 仅软件企业优惠 | 2,000万 | 12.5% | 250万 |
| 软件企业 + 加计扣除 | 1,600万 | 12.5% | 200万 |
| **综合节税** | - | - | **300万** |

## 五、常见问题和注意事项

### 5.1 常见问题

#### Q1：软件企业认定是否有有效期？

**A：** 软件企业认定通常**每年需要进行年度评估**。企业应在每年汇算清缴时向税务机关备案，并接受相关部门的监督检查。部分省市要求每年提交年度报告。

#### Q2：获利年度如何确定？

**A：** 获利年度是指企业开始生产经营后，第一个有应纳税所得额的年度。如果企业年度亏损，则该年度不计入优惠期。

**示例：**
- 2022年成立，2022年亏损
- 2023年盈利，则2023年为获利年度（第1年）
- 2024年为第2年，2025年为第3年，以此类推

#### Q3：集团内多家公司可以分别申请吗？

**A：** 可以。每家独立法人企业可以单独申请软件企业认定，但需各自满足认定条件。

#### Q4：软件企业认定和高新企业认定有什么区别？

| 对比项目 | 软件企业认定 | 高新技术企业认定 |
|---------|-------------|-----------------|
| 主管单位 | 工信部/软件行业协会 | 科技部/科技厅 |
| 核心条件 | 软件开发为主营业务 | 高新技术领域 |
| 税收优惠 | 两免三减半、重点企业10% | 15%税率 |
| 有效期 | 需年度评估 | 3年有效 |
| 研发费用比例 | ≥6% | 3%-5%（按收入规模） |

#### Q5：可以同时享受软件企业优惠和高新技术企业优惠吗？

**A：** 企业可以根据实际情况选择最有利的优惠政策，但**同一笔所得不能重复享受优惠**。建议进行比较后选择：

- 处于免税期：选择软件企业"两免三减半"
- 减半期或优惠期结束后：可选择高新技术企业15%税率

#### Q6：嵌入式软件企业如何认定？

**A：** 嵌入式软件企业需满足以下条件：
- 嵌入式软件产品开发销售（营业）收入占企业收入总额的比例不低于40%
- 嵌入式软件产品自主开发销售（营业）收入占企业收入总额的比例不低于30%
- 其他条件与一般软件企业相同

#### Q7：分公司可以单独申请认定吗？

**A：** 不可以。软件企业认定主体必须是**独立法人企业**。分公司可随总公司一并享受优惠。

#### Q8：外资企业可以申请软件企业认定吗？

**A：** 可以。在中国境内依法注册的外商投资企业，符合条件均可申请软件企业认定并享受相应优惠。

#### Q9：研发费用归集有什么注意事项？

**A：**
1. **范围界定**：严格按照《财政部 国家税务总局 科技部关于完善研究开发费用税前加计扣除政策的通知》（财税〔2015〕119号）规定范围
2. **费用类型**：人员人工、直接投入、折旧费用、无形资产摊销、新产品设计费、其他相关费用
3. **辅助账设置**：需建立研发费用辅助账，保留原始凭证
4. **分摊原则**：共同费用需合理分摊

#### Q10：未通过认定怎么办？

**A：**
1. 根据反馈意见，分析未通过原因
2. 补充完善相关材料
3. 可在下一批次重新申请
4. 建议提前进行预评估，确保条件符合

### 5.2 注意事项

#### 5.2.1 材料准备注意事项

| 事项 | 说明 |
|-----|------|
| **一致性** | 所有材料中数据应保持一致，如收入、人员、费用等 |
| **真实性** | 所有材料必须真实有效，不得伪造或虚假 |
| **完整性** | 按照清单逐项准备，不遗漏任何材料 |
| **规范性** | 按要求格式填写，签字盖章齐全 |
| **时效性** | 财务数据、审计报告等应为最新年度 |

#### 5.2.2 财务核算注意事项

1. **单独核算**：软件产品收入应单独核算
2. **研发费用**：建立研发费用辅助账，按项目归集
3. **成本分摊**：共同成本合理分摊，保留分摊依据
4. **收入确认**：按会计准则正确确认收入

#### 5.2.3 人员管理注意事项

1. **社保缴纳**：研发人员应正常缴纳社保
2. **人员稳定性**：避免频繁变动影响比例计算
3. **学历证明**：提前准备学历证书备查
4. **岗位说明**：研发岗位应与技术相关

#### 5.2.4 知识产权注意事项

1. **提前布局**：提前申请软件著作权，避免时间不够
2. **权属清晰**：确保知识产权权属明确
3. **登记备案**：及时进行软件著作权登记
4. **持续积累**：每年新增软件著作权

#### 5.2.5 税务备案注意事项

1. **时间要求**：在企业所得税汇算清缴时一并备案
2. **资料留存**：备查资料应妥善保管10年
3. **优惠台账**：建立税收优惠台账，跟踪优惠期
4. **政策变化**：关注税收政策变化，及时调整

#### 5.2.6 风险提示

| 风险点 | 防范措施 |
|-------|---------|
| **条件变化** | 定期自查，确保持续符合条件 |
| **数据不一致** | 财务、税务、统计等数据保持一致 |
| **资料缺失** | 建立档案管理制度，妥善保存 |
| **政策变化** | 关注政策动态，及时咨询专业机构 |
| **税务稽查** | 规范核算，备查资料齐全 |

## 六、申请时间节点

### 6.1 年度申请时间安排

#### 常规申请时间

| 时间节点 | 事项 |
|---------|------|
| **1-3月** | 准备上年度财务审计报告、研发费用专项报告 |

## 八、总结

软件企业认定是软件行业企业享受税收优惠政策的重要途径。企业在申请过程中应注意：

1. **提前规划**：至少提前3-6个月准备相关材料
2. **确保合规**：严格按照条件要求，确保数据真实准确
3. **完善材料**：材料齐全、规范、一致
4. **持续维护**：定期自查，保持认定条件
5. **专业支持**：必要时寻求专业机构帮助

通过软件企业认定，企业可显著降低税负，将更多资源投入研发创新，提升核心竞争力。
