## 前言

### 编制目的

在全球应对气候变化和我国推进碳达峰碳中和战略目标的大背景下，绿色制造与低碳转型已成为企业发展的必然选择。本指南旨在为企业、产业园区和相关部门提供系统、全面的政策解读和实务操作指导，帮助各类主体把握政策机遇，实现绿色高质量发展。

### 适用范围

本指南主要适用于：
- 广州市及粤港澳大湾区各类制造企业
- 工业园区和产业集聚区
- 节能环保服务企业
- 金融机构和投资主体
- 政府主管部门和行业协会

### 政策依据

- 《中华人民共和国国民经济和社会发展第十四个五年规划和2035年远景目标纲要》
- 《2030年前碳达峰行动方案》
- 《工业领域碳达峰实施方案》
- 《广东省碳达峰实施方案》
- 《广州市绿色工厂梯度培育及管理暂行办法》（穗工信规字〔2024〕4号）
- 《番禺区落实<广州市打造美丽中国城市样板工作方案>的实施方案》

## 第一章 政策背景与发展趋势

### 1.1 国家战略背景

#### 1.1.1 双碳目标的确立

2020年9月，国家主席习近平在第七十五届联合国大会一般性辩论上宣布，中国将提高国家自主贡献力度，采取更加有力的政策和措施，二氧化碳排放力争于2030年前达到峰值，努力争取2060年前实现碳中和。

这一重大战略决策对我国经济社会发展产生深远影响，工业领域作为碳排放的主要来源，面临着深刻的转型压力和机遇。

#### 1.1.2 绿色制造体系建设

工业和信息化部积极推进绿色制造体系建设，通过创建绿色工厂、绿色园区、绿色供应链管理企业，开发绿色设计产品，构建高效、清洁、低碳、循环的绿色制造体系。

**绿色制造体系四大核心要素：**

| 类别 | 定义 | 主要要求 |
|------|------|----------|
| 绿色工厂 | 实现用地集约化、原料无害化、生产洁净化、废物资源化、能源低碳化的工厂 | 能源利用率高、污染排放低、环境管理体系完善 |
| 绿色园区 | 以绿色制造理念为指导，实现园区内企业绿色发展的产业园区 | 产业集聚、资源循环利用、基础设施共享 |
| 绿色供应链 | 将绿色理念贯穿于产品全生命周期的供应链管理 | 供应商绿色评估、绿色采购、逆向物流 |
| 绿色设计产品 | 符合绿色设计理念和评价要求的产品 | 全生命周期环境影响小、资源效率高 |

### 1.2 广东省政策导向

#### 1.2.1 省级战略部署

广东省作为经济大省和制造业大省，在绿色低碳转型方面承担着重要的示范引领作用。省人民政府出台《广东省碳达峰实施方案》，明确工业领域碳达峰的路线图和时间表。

**广东省碳达峰主要目标：**

| 时间节点 | 能源结构目标 | 碳排放目标 | 产业结构目标 |
|----------|--------------|------------|--------------|
| 2025年 | 非化石能源消费比重达到32%左右 | 单位GDP能耗下降14%以上 | 规上工业企业数字化改造全覆盖 |
| 2030年 | 非化石能源消费比重达到35%左右 | 碳排放达峰 | 高新技术制造业占比显著提升 |
| 2035年 | 清洁能源成为主体能源 | 碳排放稳中有降 | 绿色制造体系全面建立 |

#### 1.2.2 工业绿色发展重点任务

**一、推动能源结构优化**
- 大力发展海上风电、光伏发电
- 加快发展新型储能产业
- 推进天然气发电项目建设
- 积极发展氢能产业

**二、促进产业结构调整**
- 严格限制"两高"项目盲目发展
- 推动传统产业绿色化改造
- 培育绿色低碳新兴产业
- 发展循环经济

**三、强化技术创新驱动**
- 突破低碳零碳负碳技术
- 推广节能降碳先进技术
- 建设绿色技术创新平台
- 培育绿色技术创新主体

### 1.3 广州市发展定位

#### 1.3.1 城市发展目标

广州市作为国家中心城市和粤港澳大湾区核心引擎，在绿色低碳发展方面承担着重要使命。市政府提出打造"美丽中国城市样板"的战略目标，以绿色制造体系建设为抓手，推动制造业高质量发展。

**广州市绿色制造发展目标（2025-2030）：**

| 指标类别 | 2025年目标 | 2030年目标 | 备注 |
|----------|------------|------------|------|
| 绿色工厂数量 | 300家以上 | 500家以上 | 国家级+省级+市级 |
| 绿色园区数量 | 10个以上 | 20个以上 | 省级以上 |
| 绿色供应链企业 | 50家以上 | 100家以上 | 省级以上 |
| 单位工业增加值能耗 | 下降15% | 下降30% | 较2020年 |
| 工业固体废物综合利用率 | 90%以上 | 95%以上 | 综合利用率 |

#### 1.3.2 政策支持体系

广州市建立了较为完善的绿色制造政策支持体系：

**财政支持政策：**
- 绿色工厂认定奖励：国家级100万元、省级50万元、市级30万元
- 绿色产品开发奖励：每个产品10-20万元
- 节能技术改造补贴：按节能量给予补贴
- 清洁生产审核补贴：通过验收企业给予补贴

**金融支持政策：**
- 绿色信贷优惠利率
- 绿色债券发行支持
- 绿色保险产品创新
- 绿色产业基金引导

## 第二章 广州市绿色制造体系建设政策

### 2.1 绿色工厂梯度培育制度

#### 2.1.1 政策依据

《广州市绿色工厂梯度培育及管理暂行办法》（穗工信规字〔2024〕4号）于2024年12月27日正式发布实施，这是广州市首个系统性的绿色工厂梯度培育政策文件。

**政策要点：**

| 政策要素 | 内容说明 |
|----------|----------|
| 制定目的 | 完善绿色制造和服务体系，发挥绿色工厂在制造业绿色低碳转型中的基础性和导向性作用 |
| 培育机制 | 建立国家、省、市三级联动的梯度培育机制 |
| 申报主体 | 在广州市行政区域内登记注册、具有独立法人资格的工业企业 |
| 申报时间 | 每年组织一次，具体时间以通知为准 |
| 有效期 | 绿色工厂称号有效期为三年 |

#### 2.1.2 梯度培育体系

**三级培育机制：**

```
┌─────────────────────────────────────────┐
│           国家级绿色工厂                  │
│     （申报省级基础上择优推荐）            │
├─────────────────────────────────────────┤
│           省级绿色工厂                    │
│     （申报市级基础上择优推荐）            │
├─────────────────────────────────────────┤
│           市级绿色工厂                    │
│     （基础层级，自主申报）                │
└─────────────────────────────────────────┘
```

**各级绿色工厂认定标准对比：**

| 评价指标 | 市级绿色工厂 | 省级绿色工厂 | 国家级绿色工厂 |
|----------|--------------|--------------|----------------|
| 基本要求 | 合法合规经营、三年内无重大事故 | 市级绿色工厂称号 | 省级绿色工厂称号 |
| 能源利用 | 能源计量器具配备率≥95% | 能源利用效率达到省内先进水平 | 达到国际或国内领先水平 |
| 资源利用 | 工业固体废物综合利用率≥80% | ≥85% | ≥90% |
| 污染防治 | 污染物排放达标 | 排放强度低于行业平均水平 | 达到清洁生产先进水平 |
| 管理体系 | 建立环境管理体系 | 通过ISO14001认证 | 多体系融合认证 |
| 技术创新 | 采用节能环保技术 | 拥有自主知识产权 | 主持或参与标准制定 |

#### 2.1.3 申报条件详解

**基本条件：**

1. **主体资格**
   - 在广州市行政区域内登记注册
   - 具有独立法人资格
   - 从事实际生产的工业企业
   - 生产经营状况良好

2. **合规要求**
   - 符合国家产业政策
   - 依法取得相关证照
   - 环保手续齐全
   - 安全生产条件符合要求

3. **业绩要求**
   - 上年度主营业务收入不低于1000万元
   - 建成投产满2年
   - 近三年内无重大安全、环境、质量等事故
   - 未被列入失信被执行人名单

**评价指标体系：**

| 一级指标 | 二级指标 | 权重 | 评分标准 |
|----------|----------|------|----------|
| 基础设施 | 建筑、照明、设备 | 15% | 按绿色建筑标准、节能照明比例、高效设备比例评分 |
| 管理体系 | 能源、环境、质量、职业健康 | 20% | 按体系建立和认证情况评分 |
| 能源与资源投入 | 能源、水资源、原材料 | 15% | 按清洁能源占比、水重复利用率、绿色采购比例评分 |

| 环境排放 | 大气、水体、固废、噪声 | 25% | 按排放强度和达标情况评分 |
| 绩效 | 用地、能耗、水耗、污染物 | 15% | 按单位产品指标与行业标杆对比评分 |

### 2.2 绿色园区建设政策

#### 2.2.1 绿色园区定义与标准

绿色园区是指以绿色制造理念为指导，通过优化园区空间布局、调整产业结构、完善基础设施，实现园区内企业绿色发展、资源能源梯级利用、污染物集中治理的产业园区。

**绿色园区评价指标：**

| 指标类别 | 评价指标 | 基本要求 | 优秀值 |
|----------|----------|----------|--------|
| 能源利用 | 可再生能源利用率 | ≥10% | ≥20% |
| 资源利用 | 工业用水重复利用率 | ≥75% | ≥90% |
| 污染控制 | 工业固体废物综合利用率 | ≥85% | ≥95% |
| 绿色产业 | 绿色产业增加值占比 | ≥30% | ≥50% |
| 基础设施 | 污水集中处理率 | 100% | 100% |
| 管理水平 | 环境监测体系 | 已建立 | 智慧化管理 |

#### 2.2.2 申报条件

**园区基本条件：**

1. 具有明确的园区边界和统一的管理机构
2. 符合国家产业政策和园区发展规划
3. 园区已建成并正常运行3年以上
4. 园区内工业用地占比不低于60%
5. 园区内企业符合环保要求，无重大环境违法记录

**园区绿色化水平要求：**

| 评估领域 | 评估内容 | 权重 |
|----------|----------|------|
| 园区规划 | 绿色发展规划、产业定位 | 15% |
| 产业结构 | 绿色产业占比、产业链完善度 | 20% |
| 能源系统 | 清洁能源利用、能源梯级利用 | 20% |
| 资源循环 | 循环产业链、废物资源化 | 20% |
| 污染防治 | 污染集中治理、风险防控 | 15% |
| 信息服务 | 智慧园区建设、信息公开 | 10% |

#### 2.2.3 政策支持措施

**财政支持：**

| 支持类型 | 支持标准 | 申报条件 |
|----------|----------|----------|
| 园区规划编制补贴 | 最高100万元 | 完成绿色发展规划编制并通过评审 |
| 基础设施建设补贴 | 按投资额10%最高500万元 | 污水处理、固废处置、能源设施等 |
| 公共服务平台建设 | 最高200万元 | 绿色制造公共服务平台 |
| 国家级绿色园区奖励 | 一次性奖励300万元 | 获得国家级认定 |
| 省级绿色园区奖励 | 一次性奖励150万元 | 获得省级认定 |

**用地支持：**
- 优先安排新增建设用地指标
- 支持园区开展低效用地再开发
- 鼓励建设多层标准厂房

### 2.3 绿色供应链管理企业认定

#### 2.3.1 认定标准

绿色供应链管理企业是指将绿色理念贯穿于产品设计、原材料采购、生产、运输、储存、销售、使用和报废处理全过程的企业。

**绿色供应链管理企业评价指标体系：**

| 一级指标 | 二级指标 | 评价要点 |
|----------|----------|----------|
| 绿色采购 | 供应商绿色评估 | 建立绿色供应商评估标准和程序 |
| | 绿色采购比例 | 绿色原材料、零部件采购占比 |
| 生产制造 | 生产过程绿色化 | 节能、减排、清洁生产水平 |
| | 绿色设计 | 产品全生命周期环境影响考量 |
| 物流配送 | 绿色物流 | 运输方式优化、包装减量 |
| 产品使用 | 产品能效 | 产品使用过程能耗水平 |
| 回收利用 | 逆向物流 | 废旧产品回收处理体系 |
| | 资源化利用 | 再制造、再利用比例 |

#### 2.3.2 申报条件

**企业基本条件：**

1. 具有独立法人资格
2. 在行业内具有较强的代表性和影响力
3. 建立完善的供应链管理体系
4. 供应商管理规范，具有稳定的供应商队伍
5. 近三年内无重大安全、环境、质量事故

**供应链绿色化要求：**

| 评估维度 | 具体要求 | 评分权重 |
|----------|----------|----------|
| 供应商管理 | 绿色供应商比例≥30% | 25% |
| 绿色采购 | 绿色原材料采购比例≥50% | 20% |
| 生产控制 | 符合绿色工厂标准 | 25% |
| 物流优化 | 绿色物流措施有效 | 15% |
| 回收利用 | 产品回收利用率≥60% | 15% |

#### 2.3.3 申报流程与材料

**申报流程：**

```
┌────────────────────────────────────────────────────────────┐

│  2. 材料准备 ─→ 编制申报材料                                │
│  3. 区级推荐 ─→ 区工信部门初审推荐                          │
│  4. 市级审核 ─→ 市工信局组织评审                            │
│  5. 省级认定 ─→ 省工信厅认定发布                            │
│  6. 国家推荐 ─→ 择优推荐申报国家级                          │
```
