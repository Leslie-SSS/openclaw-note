## 一、政策概述

### 1.1 广州科技创新发展背景

广州市作为国家中心城市和粤港澳大湾区核心引擎，高度重视科技创新发展。2024-2025年，广州市继续实施创新驱动发展战略，出台了一系列支持科技创新和产业发展的政策措施。

**广州科技创新主要指标（2024年）：**
- 全社会研发经费支出占GDP比重：3.2%以上
- 高新技术企业数量：超过13,000家
- 科技型中小企业入库数量：超过25,000家
- 技术合同成交额：超过2,500亿元

### 1.2 番禺区产业定位

番禺区是广州市重要的科技创新基地和数字经济集聚区，重点发展：
- **软件与信息技术服务业**
- **人工智能与大数据产业**
- **数字文化创意产业**
- **智能网联汽车产业**
- **高端装备制造**

**番禺区科技创新优势：**
- 广州大学城所在地，人才资源丰富
- 广州国际创新城核心区
- 万博商务区数字经济集聚区
- 国家数字家庭应用示范产业基地

## 二、广州市科技创新政策体系

### 2.1 科技计划体系

广州市科技计划主要包括以下类别：

| 计划类别 | 支持方向 | 支持额度 | 申报时间 |
|---------|---------|---------|---------|
| 基础研究计划 | 基础科学研究、应用基础研究 | 20-100万元 | 每年3-4月 |
| 重点研发计划 | 关键核心技术攻关 | 100-1000万元 | 每年4-6月 |
| 科技成果转化计划 | 成果转化、产业化 | 50-500万元 | 每年5-7月 |
| 创新平台计划 | 实验室、工程技术中心 | 100-2000万元 | 每年4-5月 |
| 科技企业培育计划 | 高企认定、研发补贴 | 10-100万元 | 全年滚动 |

### 2.2 重点科技政策清单

#### 2.2.1 广州市科技创新条例（2024年修订）

**核心内容：**
- 市本级财政科技投入占财政支出比重不低于6%
- 建立科技资源开放共享机制
- 完善科技金融结合机制
- 强化知识产权保护

**企业支持措施：**
- 研发费用后补助：按研发费用的一定比例给予补助
- 科技创新券：用于购买科技服务
- 贷款贴息：科技型中小企业贷款可获贴息支持

#### 2.2.2 广州市促进科技成果转化条例

**核心内容：**
- 科技成果转化收益分配
- 技术转移机构扶持
- 成果转化激励政策

**支持方式：**
| 支持类型 | 具体内容 | 支持标准 |
|---------|---------|---------|
| 技术合同认定登记 | 对经认定的技术合同给予奖励 | 按技术交易额的1-3%奖励 |
| 成果转化项目 | 支持重大成果转化 | 最高500万元 |
| 技术转移机构 | 支持技术转移服务机构发展 | 最高100万元 |

#### 2.2.3 广州市科技金融结合政策

**科技信贷风险补偿：**
- 设立科技型中小企业贷款风险补偿资金池
- 对合作银行向科技型中小企业发放贷款产生的本金损失给予补偿
- 补偿比例：最高可达贷款本金的50%

**科技保险补贴：**
- 对高新技术企业购买科技保险给予保费补贴
- 补贴比例：保费的50%
- 年度补贴上限：20万元

**股权投资支持：**
- 广州市科技成果转化引导基金
- 广州市科技创新企业创业投资引导基金
- 区级产业投资基金配套

### 2.3 广州市软件产业发展政策

#### 2.3.1 广州市软件产业发展行动计划（2024-2025年）

**发展目标：**
- 软件业务收入达到8,000亿元
- 培育营业收入超10亿元的软件企业100家以上
- 建设国家级软件名园3个以上

**重点支持领域：**

| 领域 | 重点方向 | 支持政策 |
|-----|---------|---------|
| 基础软件 | 操作系统、数据库、中间件 | 研发补贴、应用推广 |
| 工业软件 | CAD/CAE/CAM、MES、ERP | 产业化支持、首版次补贴 |
| 嵌入式软件 | 智能制造、汽车电子 | 产业链协同支持 |
| 新兴软件 | 云计算、大数据、人工智能 | 专项项目支持 |
| 信息安全 | 网络安全产品与服务 | 资质认证补贴 |

**主要支持措施：**

1. **企业落户奖励**
   - 世界500强企业设立软件研发中心：最高1,000万元
   - 中国软件百强企业设立总部：最高500万元
   - 重点软件企业新落户：最高300万元

2. **租金补贴**
   - 入驻市级以上软件园区的企业
   - 补贴标准：前三年分别给予50%、30%、20%的租金补贴
   - 年度补贴上限：100万元

3. **资质认证奖励**

| 资质类型 | 奖励标准 |
|---------|---------|
| CMMI 5级认证 | 50万元 |
| CMMI 3级认证 | 20万元 |
| ISO27001认证 | 10万元 |
| ITSS认证 | 10万元 |
| 系统集成资质 | 10-50万元 |

4. **首版次软件产品补贴**
   - 对首版次高端软件产品给予补贴
   - 补贴标准：产品销售额的30%
   - 单个产品最高300万元

5. **软件出口奖励**
   - 年度软件出口额超过100万美元的企业
   - 奖励标准：出口额的3%
   - 年度奖励上限：200万元

## 四、高新技术企业认定指南

### 4.1 高新技术企业认定概述

#### 4.1.1 高新技术企业定义

高新技术企业是指在《国家重点支持的高新技术领域》内，持续进行研究开发与技术成果转化，形成企业核心自主知识产权，并以此为基础开展经营活动，在中国境内（不包括港、澳、台地区）注册的居民企业。

#### 4.1.2 认定意义

| 方面 | 具体内容 |
|-----|---------|
| 税收优惠 | 企业所得税减按15%征收 |
| 资金奖励 | 市、区两级财政奖励，广州地区合计可达50-80万元 |
| 品牌提升 | 提升企业形象和市场竞争力 |
| 政策优先 | 优先获得政府项目、人才落户、用地等支持 |
| 融资便利 | 更容易获得银行贷款和风险投资 |

### 4.2 认定条件

#### 4.2.1 基本条件（必须同时满足）

**条件一：注册年限**
- 企业申请认定时须注册成立一年以上

**条件二：知识产权**
- 企业通过自主研发、受让、受赠、并购等方式，获得对其主要产品（服务）在技术上发挥核心支持作用的知识产权的所有权

**知识产权分类与评分：**

| 知识产权类型 | 分值 | 备注 |
|------------|-----|-----|
| 发明专利（含国防专利、植物新品种、国家级农作物品种、国家新药、国家一级中药保护品种、集成电路布图设计专有权） | I类 | 可重复使用 |
| 实用新型专利 | II类 | 仅限使用一次 |
| 外观设计专利 | II类 | 仅限使用一次 |
| 软件著作权 | II类 | 仅限使用一次 |

**条件三：核心技术领域**
- 对企业主要产品（服务）发挥核心支持作用的技术属于《国家重点支持的高新技术领域》规定的范围

**国家重点支持的高新技术领域（软件相关）：**

```
一、电子信息技术
├── 1. 软件
│   ├── 1.1 系统软件
│   ├── 1.2 支撑软件
│   ├── 1.3 中间件软件
│   ├── 1.4 嵌入式软件
│   ├── 1.5 计算机辅助工程管理/产品开发工艺
│   ├── 1.6 中文及多语种处理软件
│   ├── 1.7 图形和图像软件
│   ├── 1.8 金融信息化软件
│   ├── 1.9 地理信息系统
│   ├── 1.10 电子商务软件
│   ├── 1.11 电子政务软件
│   └── 1.12 企业管理软件
├── 2. 微电子技术
├── 3. 计算机及网络技术
├── 4. 通信技术
└── ...
```

**条件四：人员占比**
- 企业从事研发和相关技术创新活动的科技人员占企业当年职工总数的比例不低于10%

**科技人员认定标准：**
- 直接从事研发和相关技术创新活动的人员
- 专门从事上述活动管理和提供直接技术服务的人员
- 累计工作时间在183天以上

**条件五：研发费用占比**

| 企业类型 | 近一年销售收入 | 研发费用占比要求 |
|---------|--------------|----------------|
| 小型企业 | 小于5,000万元 | 不低于5% |
| 中型企业 | 5,000万元-2亿元 | 不低于4% |
| 大型企业 | 大于2亿元 | 不低于3% |

**条件六：高新技术产品收入**
- 近一年高新技术产品（服务）收入占企业同期总收入的比例不低于60%

**条件七：创新能力评价**
- 企业创新能力评价应达到相应要求（总分70分以上）

#### 4.2.2 创新能力评价指标

**指标体系与评分标准：**

| 一级指标 | 二级指标 | 分值 | 评分要点 |
|---------|---------|-----|---------|

## 五、研发费用加计扣除操作指南

### 5.1 政策概述

#### 5.1.1 政策定义

研发费用加计扣除是指企业为开发新技术、新产品、新工艺发生的研究开发费用，可以在计算应纳税所得额时，在实际发生额的基础上，再加计一定比例，作为扣除额。

#### 5.1.2 最新政策标准（2024年）

| 企业类型 | 加计扣除比例 | 生效时间 |
|---------|------------|---------|
| 所有企业 | 100% | 2023年1月1日起 |
| 以前（一般企业） | 75% | 2023年前 |
| 以前（制造业企业） | 100% | 2021-2022年 |
| 以前（科技型中小企业） | 100% | 2022年 |

**政策解读：**
自2023年起，所有行业企业研发费用加计扣除比例统一提高到100%，这意味着企业每投入100万元研发费用，可在税前扣除200万元。

#### 5.1.3 政策效益分析

**案例：某软件企业研发费用加计扣除效益**

假设企业投入研发费用100万元，企业所得税税率25%

| 项目 | 金额 |
|-----|-----|
| 实际研发费用 | 100万元 |
| 加计扣除额 | 100万元 |
| 可扣除总额 | 200万元 |
| 减少应纳税所得额 | 100万元 |
| 减少企业所得税 | 25万元 |
| 实际税后成本 | 75万元 |
| 节税比例 | 25% |

### 5.2 适用范围

#### 5.2.1 适用企业

- 会计核算健全、实行查账征收的居民企业
- 不限制企业类型、行业、规模
- 包括软件企业、科技型中小企业、高新技术企业等

#### 5.2.2 适用活动

**允许加计扣除的研发活动：**

| 活动类型 | 具体内容 |
|---------|---------|
| 基础研究 | 为获取新知识而进行的独创性研究 |
| 应用研究 | 为获取新知识而进行的创造性研究 |
| 试验发展 | 利用研究成果进行的新产品、新工艺开发 |

**软件企业典型研发活动：**

```
软件企业研发活动示例
├── 新产品开发
│   ├── 新软件产品研发
│   ├── 新模块/功能开发
│   └── 新算法研究
├── 技术改进
│   ├── 性能优化
│   ├── 架构升级
│   └── 技术重构
├── 工艺创新
│   ├── 开发流程优化
│   ├── 测试方法改进
│   └── 运维技术创新
└── 应用研究
    ├── 人工智能应用
    ├── 大数据技术应用
    └── 云计算技术应用
```

**不属于研发活动的情形：**
1. 企业产品（服务）的常规性升级
2. 对某项科研成果的直接应用
3. 企业商品流通中的常规性服务
4. 调查、统计、收集信息活动
5. 软件的复制和简单程序编制
6. 行政管理、法律服务、会计核算等事务性活动

### 5.3 研发费用归集范围

#### 5.3.1 研发费用分类

**一、人员人工费用**

| 费用项目 | 具体内容 | 归集要点 |
|---------|---------|---------|
| 工资薪金 | 研发人员的基本工资、绩效工资 | 按实际工时分摊 |
| 五险一金 | 研发人员的社会保险、住房公积金 | 与工资同步归集 |
| 外聘研发人员费用 | 劳务费、顾问费 | 需有合同、发票 |

**人员人工费用归集公式：**
```
某项目人员人工费用 = 该项目研发人员工时 × (工资总额 + 五险一金) ÷ 总工时
```

**二、直接投入费用**

| 费用项目 | 具体内容 | 归集要点 |
|---------|---------|---------|
| 材料费 | 研发过程中消耗的材料、燃料 | 需有领用记录 |
| 动力费 | 研发过程中的水、电、气费用 | 按实际使用分摊 |
| 检验费 | 测试、检验、化验等费用 | 需有检测报告 |
| 设备费 | 中间试验和产品试制的设备费用 | 非固定资产 |

**三、折旧费用与长期待摊费用**

| 费用项目 | 具体内容 | 归集要点 |
|---------|---------|---------|

| 实验室折旧 | 研发用房屋建筑物折旧 | 按面积分摊 |
| 长期待摊费用 | 研发设施改建、装修等 | 按受益期摊销 |

**四、无形资产摊销费用**

| 费用项目 | 具体内容 | 归集要点 |
|---------|---------|---------|
| 软件摊销 | 研发使用的软件摊销 | 如IDE、测试工具等 |
| 专利权摊销 | 用于研发的专利权摊销 | 需证明与研发相关 |
| 非专利技术摊销 | 专有技术摊销 | 需有技术资料 |

**五、新产品设计费**

| 费用项目 | 具体内容 | 归集要点 |
|---------|---------|---------|
| 设计费 | 新产品设计、工艺设计 | 需有设计文件 |
| 模具费 | 新产品试制模具 | 需有实物或图片 |
| 原型费 | 原型机制作费用 | 需有原型证明 |

**六、其他相关费用**

此项目有限额限制，不得超过可加计扣除研发费用总额的10%。

| 费用项目 | 具体内容 | 备注 |
|---------|---------|-----|
| 技术图书资料费 | 专业书籍、数据库使用费 | 计入其他费用 |
| 资料翻译费 | 外文资料翻译 | 计入其他费用 |

| 差旅费 | 研发相关差旅 | 计入其他费用 |
| 会议费 | 技术研讨会费用 | 计入其他费用 |

**其他相关费用限额计算：**
```
其他相关费用限额 = （人员人工费用 + 直接投入费用 + 折旧费用 + 
                    无形资产摊销 + 新产品设计费）÷ (1-10%) × 10%
```

## 六、科技型中小企业评价指南

### 6.1 科技型中小企业评价概述

#### 6.1.1 定义

科技型中小企业是指依托一定数量的科技人员从事科学技术研究开发活动，取得自主知识产权并将其转化为高新技术产品或服务，从而实现可持续发展的中小企业。

#### 6.1.2 评价意义

| 方面 | 具体内容 |
|-----|---------|
| 税收优惠 | 研发费用加计扣除比例提高到100% |
| 政策扶持 | 优先获得科技项目支持 |
| 资金奖励 | 部分地区给予入库奖励（番禺区1万元） |
| 融资支持 | 优先获得科技贷款、股权投资 |
| 品牌形象 | 提升企业形象和信用等级 |

### 6.2 评价条件

#### 6.2.1 基本条件（必须同时满足）

1. **注册地要求**
   - 在中国境内（不包括港、澳、台地区）注册的居民企业

2. **企业规模**
   - 职工总数不超过500人
   - 年销售收入不超过2亿元
   - 资产总额不超过2亿元

3. **合规要求**
   - 企业在填报上一年及当年内未发生重大安全、重大质量事故和严重环境违法、科研严重失信行为
   - 企业未列入经营异常名录和严重违法失信企业名单

4. **评价指标要求**
   - 企业根据科技型中小企业评价指标进行综合评价，所得分值不低于60分，且科技人员指标得分不得为0分

#### 6.2.2 评价指标体系

**科技型中小企业评价指标（满分100分）：**

| 指标类别 | 具体指标 | 分值 | 评分标准 |
|---------|---------|-----|---------|
| 科技人员指标（20分） | 科技人员占职工总数比例 | 20 | 30%（含）以上：20分<br>25%（含）-30%：16分<br>20%（含）-25%：12分<br>15%（含）-20%：8分<br>10%（含）-15%：4分<br>10%以下：0分 |
| 研发投入指标（50分） | 研发费用占销售收入比例 | 50 | 6%（含）以上：50分<br>5%（含）-6%：40分<br>4%（含）-5%：30分<br>3%（含）-4%：20分<br>2%（含）-3%：10分<br>2%以下：0分 |
| 科技成果指标（30分） | 拥有有效知识产权数量 | 30 | I类知识产权≥1项：30分<br>II类知识产权≥3项：24分<br>II类知识产权≥2项：18分<br>II类知识产权≥1项：12分<br>无知识产权：0分 |

**科技人员定义：**
- 直接从事研发和相关技术创新活动的人员
- 专门从事上述活动管理和提供直接技术服务的人员

**知识产权分类：**
- I类：发明专利（含国防专利、植物新品种、国家级农作物品种、国家新药、国家一级中药保护品种、集成电路布图设计专有权）
- II类：实用新型专利、外观设计专利、软件著作权

## 九、常见问题与注意事项

### 9.1 高新技术企业认定常见问题

#### 9.1.1 知识产权相关问题

| 问题 | 解答 |
|-----|-----|
| 软件著作权可以作为I类知识产权吗？ | 不可以，软件著作权属于II类知识产权，且只能使用一次 |
| 知识产权必须授权吗？ | 是的，必须是已获得授权的知识产权，受理通知书不认可 |
| 知识产权可以转让获得吗？ | 可以，通过受让、受赠、并购等方式获得的知识产权均可使用 |
| II类知识产权可以重复使用吗？ | 不可以，II类知识产权在申报高企时只能使用一次 |
| 需要多少项知识产权？ | 建议：I类知识产权≥1项，或II类知识产权≥6项 |

#### 9.1.2 研发费用相关问题

| 问题 | 解答 |
|-----|-----|
| 研发费用占比如何计算？ | 研发费用占比=研发费用总额÷销售收入总额 |
| 研发费用必须专项审计吗？ | 是的，需由具有资质的会计师事务所出具专项审计报告 |
| 研发人员工资如何归集？ | 按实际从事研发工作的工时分摊 |
| 委托研发费用如何处理？ | 按实际发生额的80%计入研发费用总额 |
| 研发费用辅助账格式有要求吗？ | 没有统一格式，但需按项目、费用类别归集 |

#### 9.1.3 科技成果转化相关问题

| 问题 | 解答 |
|-----|-----|
| 什么是科技成果转化？ | 将科技成果转化为产品、服务、工艺等 |
| 如何证明科技成果转化？ | 技术合同、销售合同、检测报告、用户报告等 |
| 需要多少项成果转化？ | 近三年年均≥5项（三年共≥15项）可得高分 |
| 同一成果可以多次转化吗？ | 同一成果转化为多个产品可分别计算 |

### 9.2 研发费用加计扣除常见问题

#### 9.2.1 政策适用问题

| 问题 | 解答 |
|-----|-----|
| 哪些企业可以享受？ | 会计核算健全、查账征收的居民企业均可享受 |
| 失败的研发活动可以加计扣除吗？ | 可以，失败的研发活动发生的费用也可加计扣除 |
| 研发与生产如何区分？ | 以研发项目为载体，研发活动具有明确目标和创新性 |
| 软件开发都是研发活动吗？ | 不是，常规性软件开发、简单程序编制不属于研发活动 |

#### 9.2.2 费用归集问题

| 问题 | 解答 |
|-----|-----|
| 研发人员差旅费计入哪个科目？ | 计入"其他相关费用"，受10%限额限制 |
| 软件企业云服务器费用如何归集？ | 可计入"直接投入费用"或"无形资产摊销" |
| 其他相关费用超限怎么办？ | 超出部分不得加计扣除，只能按实际发生额扣除 |
| 共用费用如何分摊？ | 按合理方法分摊，如工时比例、使用面积比例等 |

### 9.3 科技型中小企业评价常见问题

| 问题 | 解答 |
|-----|-----|
| 科技型中小企业评价有什么好处？ | 研发费用加计扣除100%、政策扶持、融资支持 |
| 评价需要现场审核吗？ | 一般不需要，主要进行形式审查 |
| 评价有效期多长？ | 当年有效，每年需重新评价 |
| 科技人员如何认定？ | 直接从事研发或研发相关技术工作的人员 |
| 高新技术企业需要评价吗？ | 高新技术企业可直接确认为科技型中小企业 |

### 9.4 申报注意事项

#### 9.4.1 时间节点把握

| 注意事项 | 具体要求 |
|---------|---------|
| 知识产权申请 | 提前1-2年申请，确保申报时已授权 |

#### 9.4.2 材料质量要求

| 要求 | 具体内容 |
|-----|---------|

| 风险点 | 防范措施 |

## 附录一：软件企业认定标准

### 1. 软件企业认定条件

| 条件 | 具体要求 |
|-----|---------|
| 企业资质 | 在中国境内依法注册的居民企业 |
| 主营业务 | 以软件开发生产、系统集成、应用服务为主 |
| 技术人员 | 从事软件产品开发和技术服务的技术人员占企业职工总数的比例不低于50% |
| 研发投入 | 软件产品开发销售（营业）收入占企业收入总额的比例不低于50%（嵌入式软件不低于40%） |
| 质量管理 | 具有保证软件产品质量的技术手段和设施 |

### 2. 软件企业优惠政策

| 优惠类型 | 优惠内容 |
|---------|---------|
| 企业所得税 | 国家规划布局内的重点软件企业，减按10%的税率征收企业所得税 |
| 增值税 | 增值税一般纳税人销售其自行开发生产的软件产品，按13%税率征收增值税后，对其增值税实际税负超过3%的部分实行即征即退 |
| 企业所得税（免税） | 我国境内新办的集成电路设计企业和符合条件的软件企业，经认定后，在2018年12月31日前自获利年度起计算优惠期，第一年至第二年免征企业所得税，第三年至第五年按照25%的法定税率减半征收企业所得税 |

## 结语

本指南系统梳理了广州市及番禺区软件企业可享受的主要政策措施，包括：

1. **广州市科技创新政策体系**：涵盖科技计划、成果转化、科技金融等多个方面
2. **番禺区软件企业专项扶持**：包括认定奖励、租金补贴、人才支持等
3. **高新技术企业认定**：详细说明认定条件、流程和材料要求
4. **研发费用加计扣除**：提供完整的操作指南和风险防范建议
5. **科技型中小企业评价**：说明评价指标和申报流程

**建议企业：**

1. **提前规划**：知识产权申请、研发费用归集需提前1-2年准备
2. **系统布局**：各项政策相互关联，应统筹规划
3. **专业咨询**：必要时寻求专业机构帮助
4. **持续关注**：政策会调整更新，需及时关注最新信息
5. **材料积累**：平时做好材料积累，为申报打好基础

希望本指南能够帮助广州番禺区的软件企业更好地了解和利用各项政策，助力企业快速发展！
