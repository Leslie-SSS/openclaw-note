## 一、概述与政策背景

### 1.1 工业互联网与数字化转型体系架构

工业互联网是新一代信息通信技术与工业经济深度融合的关键基础设施，是实现产业数字化转型的重要支撑。在广东省和广州市的政策体系中，工业互联网与数字化转型主要包括以下几个核心领域：

| 领域类型 | 核心内容 | 政策重点 | 主管单位 |
|---------|---------|---------|---------|
| 工业互联网 | 平台建设、标识解析、网络基础设施 | 创新应用、示范推广 | 省/市工信部门 |
| 数字化转型 | 制造业数字化改造、智能化升级 | 中小企业上云用数、产业集群 | 省/市工信部门 |
| 智能制造 | 智能工厂、数字化车间、灯塔工厂 | 示范引领、标杆打造 | 省/市工信部门 |
| 工业软件 | CAD/CAE/CAM、MES、ERP、PLM | 自主可控、国产替代 | 省/市科技/工信部门 |
| 两化融合 | 信息化与工业化深度融合 | 贯标认证、贯标奖励 | 省/市工信部门 |

### 1.2 政策演进历程

**2015-2018年：战略布局期**
- 《中国制造2025》发布，智能制造上升为国家战略
- 工业互联网产业联盟成立
- 广东省出台智能制造发展规划

**2019-2022年：加速推进期**
- 《工业互联网创新发展战略》实施
- 工业互联网标识解析体系建设
- 粤港澳大湾区工业互联网一体化发展加速
- 广州市工业互联网平台培育计划启动

**2023-2025年：高质量发展期**
- 《广东省制造业数字化转型实施方案》发布
- 工业互联网创新发展工程深入推进
- 智能制造示范工厂、灯塔工厂建设加速
- 工业软件自主可控战略提速

### 1.3 政策目标与意义

**对企业的价值**：
1. **降本增效**：通过数字化手段优化生产流程，降低运营成本15-30%
2. **提质升级**：提升产品质量稳定性，缩短交付周期
3. **资金支持**：享受财政补助、税收优惠等多重政策红利
4. **竞争优势**：在供应链协同、客户服务中获得差异化优势

**对政府的目标**：
1. 推动制造业高质量发展，构建现代化产业体系
2. 培育数字经济新动能，促进产业转型升级
3. 打造工业互联网产业集群，形成区域竞争优势
4. 实现碳达峰碳中和目标，推动绿色制造

**重点支持的应用场景**：

| 场景类别 | 具体场景 | 典型效益 |
|---------|---------|---------|
| 智能生产 | 生产计划优化、工艺参数优化、质量追溯 | 生产效率提升20-30% |
| 智能运维 | 设备预测性维护、远程诊断、能耗管理 | 设备故障率降低40% |
| 供应链协同 | 供应商管理、物流优化、库存管理 | 库存周转率提升25% |
| 产品服务 | 产品远程监控、故障预警、增值服务 | 服务收入占比提升15% |
| 协同研发 | 云端协同设计、仿真分析、知识管理 | 研发周期缩短30% |

#### 2.3.2 示范项目申报条件

| 条件类型 | 具体要求 |
|---------|---------|
| 申报主体 | 在广州市注册的制造业企业或工业互联网服务商 |
| 项目投资 | 总投资≥500万元 |
| 建设周期 | 1-2年 |
| 应用效果 | 具有可量化、可复制的应用效果 |
| 知识产权 | 拥有自主知识产权或核心技术的优先 |

### 2.4 政策支持措施

#### 2.4.1 财政资金支持

| 支持类型 | 支持标准 | 申请条件 |
|---------|---------|---------|
| 双跨平台培育 | 最高5000万元 | 获评国家级双跨平台 |
| 行业级平台建设 | 最高500万元 | 通过市级认定 |
| 企业级平台建设 | 最高100万元 | 通过市级认定 |
| 应用示范项目 | 项目投资的20%，最高300万元 | 列入市级示范 |
| 上云用数项目 | 服务费用的30%，最高50万元 | 中小企业上云 |

#### 2.4.2 税收优惠政策

1. **研发费用加计扣除**：
   - 工业互联网相关研发费用100%加计扣除
   - 委托外部研发费用的80%可加计扣除

2. **固定资产加速折旧**：
   - 工业软件可缩短折旧年限至2年
   - 工业设备可缩短折旧年限至3年

3. **软件产品增值税优惠**：
   - 工业软件销售增值税实际税负超过3%部分即征即退

#### 2.4.3 金融支持政策

| 政策类型 | 具体内容 |
|---------|---------|
| 科技信贷 | 工业互联网企业优先纳入科技信贷风险补偿 |
| 融资担保 | 对工业互联网项目给予担保费补贴 |
| 产业基金 | 广州市工业互联网产业基金股权投资支持 |

## 八、常见问题解答

### 8.1 资格条件类问题

**Q1：外商投资企业能否申报工业互联网项目？**
A：可以。只要在广州市/广东省注册，具有独立法人资格，符合申报条件即可。部分涉及国家安全的领域可能有特殊限制。

**Q2：企业成立不足三年能否申报？**
A：部分项目可以。如中小企业上云用数项目、工业软件产品认定等对成立年限要求较低。但智能制造示范工厂等项目通常要求3年以上经营经验。

**Q3：分公司能否申报？**
A：一般不能。申报主体应为具有独立法人资格的企业。分公司可通过总公司统一申报。

**Q4：高新技术企业是否优先？**
A：是的。高新技术企业、专精特新企业、科技型中小企业在评审中通常有加分。

### 8.2 资金支持类问题

**Q5：同一项目能否同时申请多个资金支持？**
A：原则上不能重复享受。同一项目已获得省级以上财政资金支持的，市级资金不再重复支持。但不同环节、不同内容的项目可以分别申请。

**Q6：项目资金如何拨付？**
A：通常分期拨付。首次拨付30-50%，中期评估后拨付30-40%，验收合格后拨付剩余部分。

**Q7：项目失败是否需要退还资金？**
A：视情况而定。如因企业原因导致项目失败或验收不合格，需退还已拨付资金。因不可抗力导致的，可申请延期或调整。

**Q8：资金使用有哪些限制？**
A：财政资金必须专款专用，主要用于设备购置、软件开发、技术服务等项目建设支出，不得用于人员工资、日常办公、偿还债务等。

### 8.3 技术方案类问题

**Q9：如何选择工业互联网平台？**
A：应考虑以下因素：
- 平台功能是否满足业务需求
- 行业经验与成功案例
- 技术成熟度与可扩展性
- 服务能力与响应速度
- 成本效益比

**Q10：中小企业数字化转型从哪里入手？**
A：建议按以下优先级：
1. 先上云：将IT基础设施迁移到云端
2. 再联网：实现核心设备联网采集数据
3. 后应用：部署MES/ERP等核心应用
4. 持优化：数据分析、智能决策

**Q11：如何评估数字化转型的投入产出？**
A：主要从以下维度评估：
- 直接效益：生产效率提升、成本降低、质量提高
- 间接效益：管理效率、决策质量、客户满意度
- 长期价值：数据资产、能力提升、竞争优势

### 8.4 申报流程类问题

**Q12：申报周期一般多长？**
A：从申报到认定通常需要3-6个月。其中：
- 材料准备：1-2个月
- 形式审查：2-4周
- 专家评审：1-2个月
- 公示认定：2-3周

**Q13：评审主要看哪些方面？**
A：评审主要考察：
- 企业资质与基础条件（20%）
- 技术方案先进性（30%）
- 实施能力与保障（20%）
- 预期效益与可复制性（20%）
- 答辩表现（10%）

**Q14：未通过评审能否再次申报？**
A：可以。下次申报周期可重新申请，建议根据评审意见完善方案后再申报。

**Q15：如何获取最新的申报信息？**
A：主要渠道：
- 市/区工信局官网
- 市/区科技局官网
- 产业园区、行业协会通知
- 政府服务热线12345

### 8.5 政策享受类问题

**Q16：认定后有哪些持续义务？**
A：主要包括：
- 定期报送运行数据
- 配合绩效评估
- 参与经验分享推广
- 接受监督检查

**Q17：认定有效期多长？**
A：通常3-5年。到期需重新认定或复核。

**Q18：企业信息变更如何处理？**
A：企业名称、地址、股权等发生变更的，应及时向认定部门报备，办理变更手续。
