## 前言与政策背景

### 一、时代背景

当前，全球科技竞争日趋激烈，人才资源已成为国家和区域发展的第一资源。党的二十大报告明确提出"加快建设世界重要人才中心和创新高地"，强调要"聚天下英才而用之"。在此背景下，各类高端人才平台的建设成为地方政府人才工作的重要抓手。

广州市作为国家中心城市、粤港澳大湾区核心引擎城市，在人才引进和培养方面始终走在前列。近年来，广州市持续完善"1+1+N"人才政策体系，构建起涵盖院士、博士后、高技能人才等各个层次的立体化人才支持网络。

### 二、政策体系概述

广东省和广州市的高端人才平台建设政策主要呈现以下特点：

**（一）政策层次分明**

从国家级的博士后科研流动站、工作站，到省级的院士工作站，再到市级的技能大师工作室、区级的人才培训补贴，形成了多层次、全覆盖的政策体系。

**（二）支持力度持续加大**

近年来，各级财政对人才平台的支持力度显著提升。以博士后为例，广州市博士后日常经费从最初的每人每年8万元提升至目前的每人每年20万元，在国际顶尖人才方面更是给予每人每年100万元的支持。

**（三）政策协同性增强**

各部门政策相互衔接，形成了完整的政策链条。例如，企业设立博士后工作站后，不仅可获得建站资助，还可享受博士后人员的生活补贴、科研经费支持，以及出站留穗的安家费补贴。

### 三、适用对象

本指南适用于以下对象：

| 类型 | 适用对象 |
|------|----------|
| 博士后工作站 | 符合条件的企业、科研院所、高校、产业园区 |
| 院士工作站 | 具备较强研发能力的企业、高校、科研机构 |
| 技能大师工作室 | 高技能人才领办的技术技能创新平台 |
| 培训补贴 | 番禺区企业员工、新入职员工、在职提升人员 |
| 高端人才引进 | 港澳籍人才、海外高层次人才、行业领军人才 |

## 第一章 博士后工作站政策

### 1.1 国家博士后制度概述

#### 一、博士后制度发展历程

中国博士后制度于1985年正式建立，由著名物理学家李政道先生倡议，经邓小平同志亲自批准实施。经过近40年的发展，博士后制度已成为我国培养高层次创新型人才的重要制度安排。

博士后制度的核心是"流动站+工作站"双轨运行模式。博士后科研流动站主要设立在高等院校和科研院所，博士后科研工作站则主要设立在企业、高新技术园区等单位。

#### 二、博士后培养目标

博士后研究人员是在获得博士学位后，在博士后流动站或工作站从事科研工作的青年人才。博士后阶段的主要任务包括：

1. **深化专业研究**：在博士阶段研究基础上，进一步深化专业领域的理论研究和实践探索
2. **拓展学术视野**：通过跨学科、跨单位的研究经历，拓展学术视野和研究思路
3. **提升创新能力**：参与重大项目研究，提升独立开展科研工作的能力
4. **积累学术成果**：发表高水平学术论文，申请专利，形成知识产权

#### 三、国家层面政策支持

国家层面主要通过以下政策支持博士后事业发展：

| 政策内容 | 具体措施 |
|----------|----------|
| 日常经费 | 每人每年8万元，由中央财政和地方财政共同承担 |
| 科研资助 | 中国博士后科学基金面上资助、特别资助 |
| 生活保障 | 协助解决户口、子女入学等问题 |
| 出站安置 | 鼓励博士后出站后留在设站单位工作 |

### 1.2 广东省博士后政策体系

#### 一、广东省博士后政策发展

广东省高度重视博士后工作，先后出台《广东省博士后工作管理规定》《广东省加快新时代博士和博士后人才创新发展若干意见》等一系列政策文件，构建了较为完善的博士后政策体系。

#### 二、主要政策内容

**（一）生活补贴**

广东省对在站博士后给予生活补贴支持：

- 省财政给予每人每年15万元的生活补贴
- 补贴期限最长不超过2年
- 资金直接发放至博士后个人账户

**（二）科研资助**

广东省设立博士后科研资助项目：

| 资助类型 | 资助额度 | 申报时间 |
|----------|----------|----------|
| 面上资助 | 5-8万元 | 每年两次 |
| 特别资助 | 15-20万元 | 每年一次 |
| 国际交流 | 10-15万元 | 按需申报 |

**（三）出站留粤支持**

对出站后留粤工作的博士后：

- 给予不少于30万元的安家费补贴
- 协助解决配偶就业、子女入学等问题
- 优先推荐申报省级以上人才项目

### 1.3 广州市博士后工作站专项政策

#### 一、广州市博士后政策框架

广州市博士后政策以《广州市博士后管理服务工作实施办法》为核心，形成了较为完善的政策框架。政策特点包括：

1. **覆盖范围广**：涵盖博士后工作站、流动站、创新实践基地等各类平台
2. **支持力度大**：在国家和省级支持基础上叠加市级财政支持
3. **服务体系完善**：设立博士后管理服务机构，提供"一站式"服务

#### 二、主要支持政策

**（一）建站资助**

对新设立的博士后工作站、流动站给予建站资助：

| 平台类型 | 资助标准 |
|----------|----------|
| 国家级博士后工作站 | 100万元 |
| 博士后流动站 | 50万元 |
| 博士后创新实践基地 | 30万元 |

**（二）在站博士后支持**

对在站博士后给予多方面支持：

1. **日常生活补贴**
   - 市财政给予每人每年20万元的生活补贴
   - 鼓励设站单位给予配套支持
   - 支持期限不超过2年

2. **科研经费支持**
   - 优秀博士后科研项目资助10-30万元
   - 支持申报各类科研项目
   - 鼓励参与重大科研项目

3. **国际交流支持**
   - 支持参加国际学术会议
   - 鼓励开展国际科研合作
   - 资助赴境外访学研究

**（三）出站留穗支持**

对出站后留在广州工作的博士后：

1. **安家费补贴**：给予20-30万元的安家费补贴
2. **住房保障**：纳入人才住房保障范围
3. **子女入学**：协调解决子女入学问题
4. **配偶就业**：协助解决配偶就业问题

#### 三、广州市各区博士后政策

广州市各区在市级政策基础上，叠加区级支持政策：

| 区域 | 区级支持政策 |
|------|-------------|
| 天河区 | 一次性安家费10万元 |
| 黄埔区 | 最高给予50万元购房补贴 |
| 南沙区 | 额外生活补贴10万元/年 |
| 番禺区 | 给予人才公寓优先配租 |
| 白云区 | 优先纳入重点人才服务 |

### 1.4 博士后工作站申报指南

#### 一、申报条件

**（一）基本条件**

申请设立博士后工作站，应当具备以下基本条件：

1. 具有独立法人资格，经营或运行状况良好
2. 具有一定规模，并具有专门的研究与开发机构
3. 拥有高水平的研究队伍，具有创新理论和创新技术的博士后科研项目
4. 领导重视，能够为博士后研究人员提供较好的科研条件和必要的生活条件

**（二）具体条件**

根据不同类型的申请单位，具体条件有所差异：

| 单位类型 | 具体条件 |
|----------|----------|
| 企业 | 具有省级以上企业技术中心或工程技术研究中心，或属于国家高新技术企业，研发投入占销售收入比例不低于3% |
| 高新技术园区 | 园区内高新技术企业不少于10家，设有专门服务机构 |
| 科研院所 | 具有博士学位授权点或国家级重点学科 |
| 事业单位 | 具有较强科研实力和较高学术水平 |

#### 二、申报程序

**（一）准备阶段**

1. 评估自身条件是否符合申报要求
2. 准备申报材料
3. 与拟合作的流动站进行对接

**（二）申报阶段**

```
第一步：在线填报
├── 登录中国博士后网
├── 填写申报信息
└── 上传证明材料

第二步：逐级审核
├── 单位所在区人社部门审核
├── 广州市人社局复核
└── 广东省人社厅审核

第三步：专家评审
├── 全国博士后管委会组织专家评审
├── 专家组实地考察（部分）
└── 评审委员会审议

第四步：批准设立
├── 公示拟设立单位名单
├── 正式发文批准
└── 授予博士后工作站牌匾
```

**（三）申报材料清单**

1. 申请报告
2. 申请表
3. 单位法人证明文件
4. 研发机构设立证明
5. 科研人员名册及学历证明
6. 近三年科研项目清单
7. 近三年科研成果清单
8. 科研设备清单
9. 博士后管理办法
10. 经费保障承诺函

#### 三、评审标准

博士后工作站评审主要从以下几个方面进行评估：

| 评估维度 | 权重 | 评估要点 |
|----------|------|----------|
| 科研实力 | 30% | 科研人员数量、结构；科研成果数量、质量 |
| 研发投入 | 20% | 研发经费投入强度；科研设备配置情况 |
| 项目质量 | 25% | 拟开展博士后研究项目的创新性、可行性 |
| 平台条件 | 15% | 实验室条件、办公环境、生活设施 |
| 管理水平 | 10% | 管理制度完善程度、服务保障能力 |

### 1.5 资助标准与资金管理

#### 一、资助标准汇总

| 资助类型 | 国家标准 | 广东省标准 | 广州市标准 | 合计 |
|----------|----------|-----------|-----------|------|
| 建站资助 | - | - | 100万元 | 100万元 |
| 日常经费 | 8万元/年 | - | - | 8万元/年 |
| 生活补贴 | - | 15万元/年 | 20万元/年 | 35万元/年 |

| 出站安家费 | - | 30万元 | 20-30万元 | 50-60万元 |

#### 二、资金管理办法

**（一）经费使用范围**

博士后日常经费主要用于以下方面：

| 支出类别 | 支出范围 | 占比建议 |
|----------|----------|----------|
| 人员费用 | 博士后工资、社保、公积金等 | 60-70% |
| 科研费用 | 实验材料、测试化验、差旅费等 | 20-30% |
| 管理费用 | 办公费用、培训费用等 | 5-10% |

**（二）经费监管要求**

1. 设立专门账户，独立核算
2. 定期接受审计检查
3. 年度报告经费使用情况
4. 专款专用，不得挪用

#### 三、绩效评估

博士后工作站绩效评估主要指标：

| 评估指标 | 评估标准 |
|----------|----------|
| 招收人数 | 年均招收博士后不少于2人 |
| 出站留用率 | 出站后留用率不低于50% |
| 科研产出 | 人均发表SCI/EI论文不少于2篇 |
| 专利申请 | 人均申请发明专利不少于1件 |
| 科研项目 | 主持省部级以上项目不少于1项 |
| 人才培养 | 获得国家级人才项目支持人数 |

## 第二章 院士工作站建设

### 2.1 院士工作站制度框架

#### 一、院士工作站概念

院士工作站是指以中国科学院院士、中国工程院院士及其团队为核心，以企事业单位为依托，以技术创新和人才培养为目的的科技创新平台。院士工作站是推动产学研合作、促进科技成果转化的重要载体。

#### 二、制度起源与发展

院士工作站制度起源于地方探索实践。2008年，中国工程院正式推动院士工作站建设，此后全国各省市相继出台相关支持政策。广东省自2009年开始推进院士工作站建设，目前已建立院士工作站300余家。

#### 三、功能定位

院士工作站主要承担以下功能：

1. **科技创新**：开展前沿技术研究和关键技术攻关
2. **成果转化**：推动科研成果产业化应用
3. **人才培养**：培养高层次科技人才和创新团队
4. **决策咨询**：为区域发展和行业规划提供智力支持

### 2.2 广东省院士工作站管理办法

#### 一、管理办法要点

《广东省院士工作站管理办法》对院士工作站的申报、建设、运行、评估等各个环节进行了规范。主要内容包括：

**（一）建站条件**

1. 具有独立法人资格
2. 具备良好的科研基础和技术创新条件
3. 与院士已建立稳定的合作关系
4. 有明确的研究方向和可行的研发计划
5. 能够为院士团队提供必要的工作条件和生活保障

**（二）合作院士要求**

1. 中国科学院院士或中国工程院院士
2. 每位院士同期在建工作站原则上不超过3家
3. 院士本人每年应有一定时间到站指导工作

**（三）运行期限**

院士工作站运行期限一般为3年，期满后可申请续建。

#### 二、支持政策

广东省对院士工作站给予以下支持：

| 支持类型 | 支持内容 |
|----------|----------|
| 建站经费 | 省级财政给予100万元支持 |
| 科研经费 | 优先推荐申报省级科技项目 |
| 人才项目 | 优先推荐院士团队申报人才项目 |
| 税收优惠 | 享受研发费用加计扣除等优惠政策 |

#### 三、组织管理

院士工作站的组织管理架构：

### 2.3 申报条件与程序

#### 一、申报条件详解

**（一）依托单位条件**

| 条件类别 | 具体要求 |
|----------|----------|
| 法人资格 | 具有独立法人资格的企业、高校、科研院所、医疗机构等 |

## 第三章 技能大师工作室

### 3.1 技能大师工作室建设背景

#### 一、政策背景

高技能人才是推动技术创新和实现科技成果转化的重要力量。为加强高技能人才队伍建设，人力资源和社会保障部、财政部等部门自2011年起在全国推动技能大师工作室建设。

广州市积极响应国家号召，先后出台《广州市高技能人才振兴计划实施方案》《广州市技能大师工作室管理办法》等政策文件，大力推动技能大师工作室建设。

#### 二、建设意义

技能大师工作室建设具有以下重要意义：

1. **传承绝技绝活**：通过师带徒的方式，传承老一辈技术能手的精湛技艺
2. **推动技术创新**：汇聚高技能人才，开展技术攻关和工艺改进
3. **培养技能人才**：搭建技能人才培养平台，培养青年技能人才
4. **促进产业发展**：服务产业转型升级，提升产业核心竞争力

#### 三、建设原则

技能大师工作室建设遵循以下原则：

| 原则 | 具体内涵 |
|------|----------|
| 需求导向 | 紧密对接产业发展需求，服务企业技术进步 |
| 突出特色 | 发挥行业和技能大师的特色优势，形成品牌效应 |
| 注重实效 | 以解决实际问题为导向，推动技术成果转化 |
| 动态管理 | 建立绩效评估机制，优胜劣汰 |

### 3.5 技能传承与人才培养

#### 一、师带徒模式

技能大师工作室的核心功能是师带徒技艺传承。主要模式包括：

**（一）一对一传授**

技能大师直接指导学徒，手把手传授技艺。这种方式适合需要精细操作技能的传统工艺和现代制造技术。

**（二）一对多培养**

技能大师同时指导多名学徒，通过集中授课、示范操作、分组练习等方式，培养一批技能人才。

**（三）团队协作培养**

组建技术攻关团队，在解决实际技术问题的过程中，培养学徒的技术能力和创新思维。

#### 二、培养内容

技能大师工作室培养内容主要包括：

| 培养类别 | 培养内容 | 培养方式 |
|----------|----------|----------|
| 基本功训练 | 基本操作技能、安全操作规程 | 示范讲解、反复练习 |
| 专项技能 | 行业核心技能、典型工艺 | 项目实训、案例分析 |
| 绝技绝活 | 技能大师独门绝技 | 手把手传授、观摩学习 |
| 创新能力 | 技术改进、工艺创新 | 技术攻关、项目研发 |
| 职业素养 | 职业道德、工匠精神 | 案例教学、榜样示范 |

#### 三、培养成果认定

技能大师工作室培养的人才，可通过以下方式获得技能水平认定：

1. 参加职业技能等级认定，取得职业资格证书
2. 参加职业技能竞赛，获得竞赛成绩和荣誉称号
3. 通过企业内部技能考核，获得岗位技能等级
4. 参加专项职业能力考核，取得专项能力证书

## 附录三 常见问题解答

### 一、博士后工作站相关问题

**Q1：企业申报博士后工作站需要哪些基本条件？**

A：基本条件包括：具有独立法人资格，经营状况良好；具有专门的研发机构；拥有高水平研究队伍；能够提供较好的科研和生活条件。具体条件可参考《博士后管理工作规定》。

**Q2：博士后在站期间可以流动吗？**

A：博士后在站期间原则上不得中途退出。如确需变更设站单位，须经原设站单位和新设站单位同意，并报全国博士后管委会备案。

**Q3：博士后出站后必须留穗工作才能获得安家费吗？**

A：广州市博士后安家费政策要求出站后留在广州工作，且与用人单位签订3年以上劳动合同。具体条件以最新政策为准。

### 二、院士工作站相关问题

**Q4：同一院士可以建立多个工作站吗？**

A：根据广东省院士工作站管理办法，每位院士同期在建工作站原则上不超过3家。

**Q5：院士工作站的建设周期是多久？**

A：院士工作站运行期限一般为3年，期满后可申请续建。

**Q6：院士本人必须亲自到站工作吗？**

A：院士本人每年应有一定时间到站指导工作，一般要求每年不少于30天。同时可以通过远程指导等方式开展工作。

### 三、技能大师工作室相关问题

**Q7：技能大师工作室的领办人需要具备什么资格？**

A：领办人应具备以下条件之一：获得中华技能大奖、全国技术能手称号；具有技师以上职业资格；具有绝技绝活；在传统工艺美术领域具有较高技艺水平。

**Q8：技能大师工作室经费可以用于哪些方面？**

A：经费主要用于：设备购置（40-50%）、技能培训（20-30%）、技术研发（15-25%）、交流推广（5-10%）。

### 四、培训补贴相关问题

**Q9：同一培训项目可以多次申请补贴吗？**

A：同一培训项目、同一等级证书只能申请一次补贴。提升等级后可再次申请。

**Q10：培训补贴什么时候可以申请？**

A：培训结束后6个月内申报。逾期不予受理。

### 五、人才引进相关问题

**Q11：港澳青年来穗创业有哪些支持政策？**

A：主要包括：创业启动资金（最高50万元）、办公场地免租或租金补贴、创业担保贷款（最高300万元）等。

**Q12：外籍人才如何申请永久居留？**

A：可通过投资类、就业类、特殊贡献类、家庭团聚类等途径申请。具体条件可咨询公安出入境管理部门。

## 结语

高端人才平台建设是推动科技创新、实现高质量发展的重要抓手。广州市、广东省及粤港澳大湾区各级政府高度重视人才工作，出台了一系列含金量高、针对性强的人才政策，构建起多层次、全覆盖的人才政策体系。

本指南系统梳理了博士后工作站、院士工作站、技能大师工作室、人才培训补贴、高端人才引进等方面的政策内容，希望能够为各类人才和用人单位提供实用的政策参考。

在实际操作过程中，建议：

1. **及时关注政策更新**：政策会根据实际情况进行调整，请以最新发布的政策文件为准

4. **合理规划申报时间**：注意各类项目的申报时间节点，提前做好准备

希望本指南能够帮助各类人才更好地了解和利用人才政策，在广州这片创新创业的热土上实现更大发展！
